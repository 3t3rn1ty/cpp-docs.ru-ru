# Ошибка компилятора C2145
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

синтаксическая ошибка: отсутствует "токен" перед идентификатором  
  
 Компилятор ожидал `token` и вместо этого обнаружил идентификатор.  
  
 Эта ошибка может быть вызвана отсутствием точки с запятой после последнего объявления в блоке.
