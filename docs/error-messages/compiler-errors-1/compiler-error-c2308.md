# Ошибка компилятора C2308
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

объединение неподходящих строк  
  
 Для объединения определены строки как расширенных, так и нерасширенных знаков.  Объединение строки расширенных знаков и строки нерасширенных знаков невозможно.
