# Ошибка компилятора C2188
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"число": слишком большое значение для расширенного символа  
  
 Это число превышает предельный размер для типа расширенных символов. Выберите больший тип.
