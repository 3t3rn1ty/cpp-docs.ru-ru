# Compiler Error C2222
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

непредвиденный тип type: ожидался базовый класс или член  
  
 Список инициализаторов может инициализировать только базовые классы или члены типа.  Чтобы устранить эту ошибку, убедитесь, что в списке инициализаторов выполняется инициализация только базовых классов или членов типа.
