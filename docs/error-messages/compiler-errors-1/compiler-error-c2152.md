# <a name="compiler-error-c2152"></a>Ошибка компилятора C2152
«Идентификатор»: указатели на функции с другими атрибутами  
  
 Указатель на функцию с определенным соглашением вызова (`__cdecl`, `__stdcall`, или `__fastcall`) присваивается указателю на функцию с другим соглашением вызова.
