# Неустранимая ошибка C1506
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

неустранимая ошибка области блока  
  
 Размеры блока слишком велики для его компиляции.  
  
### Чтобы устранить ошибку, следует проверить следующие возможные причины ее возникновения.  
  
1.  Непарные скобки;  
  
2.  Нетипично большая функция или класс.
