# Ошибка компилятора C2138
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

определение перечисления без членов недопустимо  
  
 Перечисление должно содержать по крайней мере один член, если выбран параметр \/Za \(отключить расширения Microsoft\).
