# Ошибка компилятора C2151
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

несколько атрибутов языка  
  
 В функции используется несколько ключевых слов \(`__cdecl`, `__stdcall` или `__fastcall`\), задающих соглашение о вызове.
