# <a name="compiler-error-c2101"></a>Ошибка компилятора C2101
"&" в константе  
  
 Оператор address-of ( `&` ) должен иметь операнд l-value.  
  
 В следующем примере возникает ошибка C2101:  
  
```  
// C2101.cpp  
int main() {  
   char test;  
   test = &'a';   // C2101  
   test = 'a';   // OK  
}  
```
