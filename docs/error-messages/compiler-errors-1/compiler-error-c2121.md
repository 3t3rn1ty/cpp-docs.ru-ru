# Ошибка компилятора C2121
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"\#": недопустимый знак: возможный результат подстановки макроопределения  
  
 Недопустимый знак \# мог быть вставлен неправильным макросом, который использует оператор вставки лексем \(\#\#\) вместо строкового оператора \(\#\).
