# <a name="compiler-error-c2338"></a>Ошибка компилятора C2338

> *Сообщение об ошибке*

Эта ошибка может быть вызвана `static_assert` ошибка во время компиляции. Сообщение предоставляется свойством `static_assert` параметров.

Это сообщение об ошибке можно также сформировать при внешних поставщиков для компилятора. В большинстве случаев эти ошибки выводятся поставщиком атрибута библиотеки DLL, например ATLPROV. Перечислены некоторые распространенные виды это сообщение.

> "*атрибут*" поставщик атрибутов Atl: ошибка ATL*номер* *сообщения*

> Неправильное использование атрибута "*атрибут*"

> "*использования*": неправильный формат для атрибута «использование»

Эти ошибки часто неустранимой и может следовать ошибка Неустранимая компилятора.

Для исправления этой проблемы исправьте использование атрибута. Например, в некоторых случаях атрибут параметры необходимо объявлять прежде, чем они могут использоваться. Если номер ошибки ATL, обратитесь к документации для этой ошибки, подробные сведения.
