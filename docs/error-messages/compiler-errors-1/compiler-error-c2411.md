# Ошибка компилятора C2411
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"идентификатор": недопустимый член структуры или объединения в "контекст"  
  
### Чтобы устранить ошибку, следует проверить следующие возможные причины ее возникновения.  
  
1.  Идентификатор не является членом отображаемой структуры или объединения в этом контексте.  
  
2.  Идентификатор не является членом структуры или объединения, указанных с помощью оператора выбора члена \(.\).
