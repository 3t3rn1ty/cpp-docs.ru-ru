# Ошибка компилятора C2446
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"оператор": не удается выполнить преобразования из "типа1" в "тип 2"  
  
 Компилятору не удается выполнить преобразование `type1` в `type2`.  Преобразование может не иметь смысла, поскольку оно нарушает семантику С\/С\+\+.
