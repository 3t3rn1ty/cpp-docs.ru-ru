# <a name="compiler-error-c2446"></a>Ошибка компилятора C2446
«оператор»: преобразование из «тип1» в «тип2»  
  
 Компилятор не может преобразовать `type1` для `type2`. Преобразование не имеет смысла, поскольку оно нарушает семантику C/C++.
