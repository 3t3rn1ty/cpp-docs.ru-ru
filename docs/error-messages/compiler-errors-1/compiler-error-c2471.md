# <a name="compiler-error-c2471"></a>Ошибка компилятора C2471
не удалось обновить базу данных программы «файл»  
  
 Компилятору не удается записать в файл базы данных.  
  
### <a name="to-fix-the-problem"></a>Устранение проблемы  
  
-   Убедитесь, что диск содержит недостаточно свободного места.  
  
-   Убедитесь, что файл не только для чтения.  
  
-   Убедитесь в отсутствии без нарушения общего доступа.
