# Ошибка компилятора C2471
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

невозможно обновить базу данных программы "файл"  
  
 Компилятору не удалось произвести запись в файл базы данных.  
  
### Устранение этой проблемы:  
  
-   Убедитесь, что диск имеет достаточно большой объем доступного пространства.  
  
-   Убедитесь, что файл не только для чтения.  
  
-   Убедитесь, что никакие файловом нарушений.
