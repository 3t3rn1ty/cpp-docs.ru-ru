# <a name="fatal-error-c1023"></a>Неустранимая ошибка C1023
"файл": непредвиденная ошибка в pch; попытайтесь построить pch заново  
  
 Ошибка C1023 может быть связана с различными проблемами, для устранения которых нужно выполнить повторную сборку файла предкомпилированного заголовка.  
  
 Ошибка C1023 может возникать при использовании файла предкомпилированного заголовка на компьютере, отличном от того, на котором он был создан.
