# Неустранимая ошибка C1051
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

формат файла базы данных программы "PDB\-файл" устарел, удалите файл и повторите компиляцию  
  
 В процессе компиляции не удается обновить файл базы данных программы, который имеет устаревший номер версии.  Удалите файл и выполните повторную компиляцию программы с использованием параметра **\/Zi** или **\/ZI**.  Дополнительные сведения см. в разделе [\/Z7, \/Zi, \/ZI \(формат отладочной информации\)](../Topic/-Z7,%20-Zi,%20-ZI%20\(Debug%20Information%20Format\).md).
