# Ошибка компилятора C2307
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

прагма\-директива "прагма" должна находиться вне функции в режиме добавочной компиляции  
  
 Прагма\-директива `data_seg` должна стоять между функциями при использовании добавочной компиляции.
