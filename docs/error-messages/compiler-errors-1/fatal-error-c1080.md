# Неустранимая ошибка C1080
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

предельное значение компилятора: параметр командной строки превысил допустимое число знаков  
  
 Длина аргумента, переданного компилятору, превышает 256 знаков.
