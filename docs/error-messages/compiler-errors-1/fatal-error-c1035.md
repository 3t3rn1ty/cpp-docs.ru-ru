# Неустранимая ошибка C1035
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

слишком сложное выражение; упростите его  
  
 Компилятору не удалось создать код для сложного выражения. Разбейте его на более простые выражения и перекомпилируйте.
