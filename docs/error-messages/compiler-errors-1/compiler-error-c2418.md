# Ошибка компилятора C2418
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

невозможно удалить файл браузера: имя файла  
  
 Компилятору не удалось удалить файл браузера.  
  
### Чтобы устранить ошибку, следует проверить следующие возможные причины ее возникновения.  
  
1.  Файл доступен только для чтения.  
  
2.  Файл используется другим процессом.
