# Неустранимая ошибка C1350
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ошибка при загрузке dll "библиотека dll": dll не найдена  
  
 Не найдена библиотека DLL, обеспечивающая поддержку запрашиваемой операции.  
  
 Это указывает на проблему установки. Следует переустановить библиотеку DLL с установочного компакт\-диска.
