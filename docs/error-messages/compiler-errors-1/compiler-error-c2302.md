# Ошибка компилятора C2302
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

выражение слева от ".~идентификатор" должно иметь тип класса, структуры или объединения  
  
 Выражение слева от оператора точки \(.\) не является классом, структурой или объединением.
