# <a name="compiler-error-c2175"></a>Ошибка компилятора C2175
«язык»: недопустимый язык  
  
 Указанный язык является недопустимым. В разделе [языка и страны или региона строк](../../c-runtime-library/locale-names-languages-and-country-region-strings.md) в *Справочник по библиотеке времени выполнения* для поддерживаемых языков.
