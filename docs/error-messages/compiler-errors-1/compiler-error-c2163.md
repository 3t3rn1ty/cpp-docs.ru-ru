# Ошибка компилятора C2163
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"функция": недоступна в качестве встроенной функции  
  
 В директиве pragma `intrinsic` или `function` функция перечислена как недоступная для встроенной формы.  Например, определенные встроенные функции недоступны при компиляции программы с помощью программных средств \/clr.
