# Неустранимая ошибка C1904
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

недопустимое взаимодействие поставщиков: "файл"  
  
 Эта ошибка указывает на сбой поставщика атрибутов.  
  
 Соберите сведения об обстоятельствах возникновения ошибки, попытайтесь изолировать проблему и создать воспроизводимый тестовый случай, а затем обратитесь в [Обращайтесь к нам](../Topic/Talk%20to%20Us.md).
