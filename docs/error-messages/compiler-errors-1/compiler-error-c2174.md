# <a name="compiler-error-c2174"></a>Ошибка компилятора C2174
«функция»: фактический параметр имеет тип «void»: параметр номер1, список параметров номер2  
  
 Параметр `number1` переданный в список параметров `number2` — `void` параметра. Параметры не могут иметь тип `void`. Взамен рекомендуется использовать `void*`.
