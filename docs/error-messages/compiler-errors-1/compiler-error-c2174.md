# Ошибка компилятора C2174
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"function" : фактический параметр имеет тип "void" : parameter number1, parameter list number2  
  
 Параметр `number1`, переданный в список параметров `number2` является параметром `void`.  Параметрам нельзя иметь тип `void`.  Взамен рекомендуется использовать `void*`.
