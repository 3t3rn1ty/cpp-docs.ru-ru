# <a name="compiler-error-c2223"></a>Ошибка компилятора C2223

слева от «->"идентификатор» должно указывать на структуру или объединение

Операнд слева от `->` не является указателем на класс, структуру или объединение.

Эта ошибка может быть вызвана, что левый операнд является неопределенной переменной (следовательно ввести `int`).
