# <a name="fatal-error-c1903"></a>Неустранимая ошибка C1903

не удается восстановить после предыдущих ошибок; остановка компиляции

Компилятор обнаружил слишком много ошибок, чтобы продолжить. Устраните некоторые ошибки и повторите компиляцию.
