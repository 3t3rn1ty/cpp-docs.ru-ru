# Неустранимая ошибка C1196
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"идентификатор": идентификатор, найденный в библиотеке типов "библиотека", не является допустимым идентификатором C\+\+  
  
 Один из идентификаторов в библиотеке типов не является допустимым идентификатором C\+\+. Библиотеку типов нельзя использовать с директивой [\#import](../Topic/%23import%20Directive%20\(C++\).md).
