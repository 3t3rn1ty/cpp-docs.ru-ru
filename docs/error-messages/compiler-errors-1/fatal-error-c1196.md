# <a name="fatal-error-c1196"></a>Неустранимая ошибка C1196
"идентификатор": идентификатор, найденный в библиотеке типов "библиотека", не является допустимым идентификатором C++  
  
 Один из идентификаторов в библиотеке типов не является допустимым идентификатором C++. Библиотека типов недоступна для использования с [#import](../../preprocessor/hash-import-directive-cpp.md).
