# Неустранимая ошибка C1192
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Сбой директивы \#using в файле "file"  
  
 В директиву [\#using](../../preprocessor/hash-using-directive-cpp.md) можно передать только файл в формате MSIL.  Параметр компилятора [\/clr](../../build/reference/clr-common-language-runtime-compilation.md) позволяет создать выходной файл MSIL.  Файлы MSIL также создаются с помощью других языков Visual Studiо.
