# Неустранимая ошибка C1008
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

не указан входной файл  
  
 Не указан исходный файл C или C\+\+ для компиляции. Проверьте спецификации файла в переменной среды и командной строке **CL**.
