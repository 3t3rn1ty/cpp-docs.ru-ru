# Ошибка компилятора C2224
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

слева от ".идентификатора" должна стоять структура или объединение  
  
 Операнд слева от точки \("."\) не является классом, структурой или объединением.  
  
 Причиной этой ошибки может быть левый операнд, являющийся не определенной переменной \(следовательно, имеющий тип\) `int`\).
