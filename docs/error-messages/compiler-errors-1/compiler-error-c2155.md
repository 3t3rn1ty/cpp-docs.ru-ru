# <a name="compiler-error-c2155"></a>Ошибка компилятора C2155
"?": недопустимый левый операнд, требуется арифметический тип или указатель  
  
 Выражение в левой части `?` нельзя сравнивать с нуля. Необходимо использовать выражение арифметический тип или указатель, можно сравнить с нуля.
