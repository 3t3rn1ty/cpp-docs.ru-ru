# Неустранимая ошибка C1093
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Не удалось выполнить вызов "имени функции" АPI. Сбой "расположения вызова": "текст из среды выполнения"  
  
 Вызов функции Windows завершился с ошибкой  ***Текст из среды выполнения*** предоставляется или не предоставляется средой выполнения СОМ.  
  
 Дополнительные сведения о сообщениях о системных ошибках см. в системном файле winerror.h, а также в разделе [Форматирование сообщения](http://msdn.microsoft.com/library/windows/desktop/ms679351).
