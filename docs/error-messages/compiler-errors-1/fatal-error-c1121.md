# Неустранимая ошибка C1121
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

не удалось выполнить вызов CryptoAPI  
  
 Компилятор выполнил вызов CryptoAPI, во время которого произошел сбой.  Переустановите Visual Studio и при необходимости переустановите операционную систему.
