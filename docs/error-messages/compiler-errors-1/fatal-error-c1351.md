# <a name="fatal-error-c1351"></a>Неустранимая ошибка C1351
Произошла ошибка при загрузке dll «библиотека dll»: несовместимая версия  
  
 Найден неправильная версия библиотеки DLL.  
  
 Это указывает на проблему с установкой, и необходимо переустановить библиотеки DLL с установочного компакт-диска.
