# <a name="fatal-error-c1309"></a>Неустранимая ошибка C1309
Несовпадение версий C2. Библиотеки DLL и PGODB\<версии настроек. БИБЛИОТЕКИ DLL  
  
 Используется для создания и использования набора инструментов [профильные оптимизации](../../build/reference/profile-guided-optimizations.md) содержит несовпадающие компоненты.  Если не удается устранить эту ошибку вручную, переустановите Visual C++.
