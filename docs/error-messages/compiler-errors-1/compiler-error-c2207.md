# Ошибка компилятора C2207
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

member: член шаблона класса не может получать тип функции  
  
 `member` шаблона класса был ранее проанализирован как нестатический элемент данных.  Его невозможно переопределить как функцию\-член.
