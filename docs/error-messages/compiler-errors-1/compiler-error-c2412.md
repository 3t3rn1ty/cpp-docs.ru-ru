# Ошибка компилятора C2412
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"метка": метка переопределена без учета регистра букв  
  
 В текущей функции метка определена несколько раз.  Измените написание метки и ссылок на нее.
