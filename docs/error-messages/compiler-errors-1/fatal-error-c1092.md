# Неустранимая ошибка C1092
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Режим "Изменить и продолжить" не поддерживает изменение типов данных, требуется построение.  
  
 Со времени последнего успешного построения тип данных был изменен или добавлен.  
  
-   Режим "Изменить и продолжить" не поддерживает изменения существующих типов данных, включая определения классов, структур или объединений.  Необходимо остановить отладку и выполнить построение приложения.  
  
-   Режим "Изменить и продолжить" не поддерживает добавление новых типов данных, если файл базы данных программы, такой как vc*x*0.pdb \(где *x* – основная используемая версия Visual C\+\+\), доступен только для чтения.  Чтобы добавить типы данных, компилятор должен открыть PDB\-файл в режиме записи.  
  
### Устранение этой ошибки без прекращения текущей сессии отладки  
  
1.  Следует изменить тип данных до состояния, предшествовавшего ошибке.  
  
2.  В меню **Отладка** выберите команду **Применить изменения кода**.  
  
### Для устранения этой ошибки без изменения вашего исходного кода  
  
1.  В меню **Отладка** выберите команду **Остановить отладку**.  
  
2.  Из меню **Построение** выберите подпункт **Построение**.
