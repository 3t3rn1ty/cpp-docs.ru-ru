# <a name="fatal-error-c1033"></a>Неустранимая ошибка C1033
не удается открыть PDB-файл базы данных программы  
  
 Эта ошибка может быть вызвано ошибка диска.  
  
 В Visual C++ .NET 2002 языковой стандарт пользователя необходимо присвоить правильно, если имя файла (или путь к каталогу, имя файла) содержит символы в многобайтовой Кодировке. Установки языкового стандарта системы недостаточно; для обработки символов в многобайтовой кодировке должен быть настроен языковой стандарт пользователя.  
  
 Дополнительные сведения см. в разделе [http://support.microsoft.com/default.aspx?scid=kb;en-us;246007](http://support.microsoft.com/default.aspx?scid=kb;en-us;246007).
