# Ошибка компилятора C2028
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

член структуры \(объединения\) должен находиться в ее \(его\) пределах  
  
 Члены структуры или объединения должны объявляться внутри структуры или объединения.
