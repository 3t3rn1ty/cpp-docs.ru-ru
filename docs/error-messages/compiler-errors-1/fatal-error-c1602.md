# Неустранимая ошибка C1602
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

неподдерживаемая встроенная функция  
  
 Эта ошибка указывает на несоответствие в EXE\-файлах компилятора, которое могло возникнуть в результате незавершенной установки.  Например, мог быть установлен пакет обновления, но пакет процессора при этом установлен не был.  Установите все необходимые продукты.
