# <a name="fatal-error-c1108"></a>Неустранимая ошибка C1108
не удается найти DLL: "библиотека dll"  
  
 По указанному пути не удалось найти библиотеку DLL (***библиотека dll***).  
  
 Чтобы устранить эту ошибку, переустановите Visual C++ или скопируйте соответствующий DLL-файл с установочного компакт-диска на компьютер.
