# Неустранимая ошибка C1508
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ограничение компилятора: "функция": размер аргументов превышает 65 535 байтов  
  
 Формальные параметры функции превышают ограничение 65 535 байтов.
