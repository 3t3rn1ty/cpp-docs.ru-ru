# Ошибка компилятора C2487
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"идентификатор" : член класса, входящего в интерфейс DLL, не может объявляться в интерфейсе DLL  
  
 В интерфейсе DLL можно объявить целый класс или определенные элементы класса, не входящего в интерфейс DLL.  При этом нельзя объявить в интерфейсе DLL класс, а затем объявить в интерфейсе DLL член этого класса.
