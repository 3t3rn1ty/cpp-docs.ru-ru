# Неустранимая ошибка C1098
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Несовпадение версий в механизме "Изменить и продолжить"  
  
 Используемая версия отладчика не совпадает с версией компилятора, использовавшегося для создания исполняемого файла. Если повторная компиляция не устранит эту ошибку, может потребоваться переустановить Visual C\+\+, чтобы обеспечить соответствие версий отладчика и компилятора.
