# Ошибка компилятора C2114
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"оператор": указатель слева; требуется значение целого типа справа  
  
 Левый операнд `operator` был указателем, поэтому правый операнд должен быть целым числом.
