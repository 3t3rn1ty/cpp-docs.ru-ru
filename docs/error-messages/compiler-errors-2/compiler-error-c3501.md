# Ошибка компилятора C3501
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

для идентификатора ProgID "идпрограммы" не зарегистрирована библиотека типов  
  
 С идентификатором класса указанного идентификатора программы не связана библиотека типов.  Соответственно, использовать этот идентификатор программы в операторе `#import` нельзя.
