# Ошибка компилятора C2557
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"идентификатор": закрытые и защищенные члены нельзя инициализировать без конструктора  
  
 Только члены и друзья могут назначать значение закрытому или защищенному члену. Закрытые члены должны быть инициализированы в конструкторе класса.
