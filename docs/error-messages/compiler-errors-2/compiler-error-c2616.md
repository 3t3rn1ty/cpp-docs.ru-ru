# Ошибка компилятора C2616
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"преобразование": не может неявно преобразовать значение "тип1", не являющееся lvalue, в "тип2", не являющееся константным  
  
 Не удается инициализировать ссылку из значения, не являющегося lvalue.  
  
 Эта ошибка отображается в режиме совместимости ANSI \(параметр [\/Za](../../build/reference/za-ze-disable-language-extensions.md)\). В режиме расширений Microsoft \(параметр **\/Ze**\) отображается предупреждение.
