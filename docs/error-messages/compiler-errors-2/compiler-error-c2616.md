# <a name="compiler-error-c2616"></a>Ошибка компилятора C2616
«Преобразование»: не удается неявно преобразовать не с левосторонними значениями «тип1» в «тип2», не являющееся константным  
  
 Не удается инициализировать ссылку из не являющегося lvalue.  
  
 Это ошибка в режиме совместимости с ANSI ([/Za](../../build/reference/za-ze-disable-language-extensions.md)) и предупреждение при использовании расширений Майкрософт (**/Ze**).
