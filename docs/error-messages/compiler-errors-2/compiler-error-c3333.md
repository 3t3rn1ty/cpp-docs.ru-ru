# <a name="compiler-error-c3333"></a>Ошибка компилятора C3333

"библиотека типов": невозможно импортировать (#import) поврежденную библиотеку типов

Библиотека типов, указанная в операторе `#import` , является нечитаемой для компилятора. Вы можете захотеть повторно создать библиотеку типов, если это возможно, или запросить новую библиотеку типов у вашего поставщика. Можно использовать средство OLE Viewer, поставляемое с Visual C++, для просмотра файла библиотеки типов и определения, что с ним случилось.
