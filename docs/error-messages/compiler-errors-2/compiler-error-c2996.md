# Ошибка компилятора C2996
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"функция": рекурсивное определение шаблона функции  
  
 Определение функции пытается создать экземпляр корневого класса шаблона. Рекурсивное создание шаблонов не допускается.
