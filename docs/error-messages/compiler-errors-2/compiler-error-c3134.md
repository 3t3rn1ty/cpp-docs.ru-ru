# <a name="compiler-error-c3134"></a>Ошибка компилятора C3134
«значение»: значение аргумента атрибута «атрибут» не имеет допустимый тип «тип»  
  
 Синтаксическая ошибка была обнаружена при назначении значения для атрибута.
