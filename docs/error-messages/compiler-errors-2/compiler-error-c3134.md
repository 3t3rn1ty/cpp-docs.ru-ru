# Ошибка компилятора C3134
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"значение": значение аргумента атрибута "имяАтрибута" не соответствует допустимому типу "тип"  
  
 При назначении значения атрибуту была обнаружена ошибка синтаксиса.
