# Ошибка компилятора C2858
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

параметр командной строки "\/Yc \(\/Fdfilename\)'" не соответствует предварительно скомпилированному заголовку, который использует "\/Fdfilename'"  
  
 База данных программы, указанная с помощью параметра "Использовать предварительно скомпилированный заголовок" \([\/Yu](../../build/reference/yu-use-precompiled-header-file.md)\), не является единственной указанной параметром "Создать предварительно скомпилированный заголовок" \([\/Yc](../../build/reference/yc-create-precompiled-header-file.md)\).
