# Ошибка компилятора C2829
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"operator оператор" не может иметь список параметров переменных  
  
 Только два оператора могут иметь списки параметров переменных:  
  
-   Вызов функции `()`  
  
-   `new`
