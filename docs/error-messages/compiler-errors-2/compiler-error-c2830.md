# <a name="compiler-error-c2830"></a>Ошибка компилятора C2830

только параметры размещения для «operator new» может иметь значения по умолчанию

Стандартные формальные параметры для [оператор new](../../standard-library/new-operators.md#op_new) не может иметь значения по умолчанию. Только параметры размещения пользовательских можно указать значения по умолчанию.
