# <a name="compiler-error-c2830"></a>Ошибка компилятора C2830
только параметры размещения для "operator new" могут иметь значения по умолчанию  
  
Стандартные формальные параметры для [оператор new](../../standard-library/new-operators.md#operator_new) не могут иметь значения по умолчанию. Только параметры определяемых пользователем размещения можно указать значения по умолчанию.
