# Ошибка компилятора C3068
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"функция": в функции с атрибутом naked не могут содержаться объекты, требующие развертывания при возникновении исключения C\+\+  
  
 Компилятору не удалось выполнить развертывание стека в функции [naked](../Topic/naked%20\(C++\).md), в которой возникло исключение вследствие создания в функции временного объекта, и была задана обработка исключений С\+\+ \([\/EHsc](../../build/reference/eh-exception-handling-model.md)\).  
  
 Чтобы устранить эту ошибку, необходимо выполнить следующие действия:  
  
-   Не компилировать с параметром \/EHsc.  
  
-   Не помечать функцию как `naked`.  
  
-   Не создавать временные объекты в функции.  
  
 Если функция создает в стеке временный объект и вызывает исключение, и при этом включена обработка исключений С\+\+, компилятор при вызове исключения выполнит очистку стека.  
  
 При вызове исключения для функции выполняется созданный компилятором код \(пролог и эпилог\), который не содержится в функции с атрибутом naked.  
  
## Пример  
 Следующий пример приводит к возникновению ошибки C3068:  
  
```  
// C3068.cpp  
// compile with: /EHsc  
// processor: x86  
class A {  
public:  
   A(){}  
   ~A(){}  
};  
  
void b(A){}  
  
__declspec(naked) void c() {  
   b(A());   // C3068   
};  
```
