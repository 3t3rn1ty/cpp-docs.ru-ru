# Ошибка компилятора C2948
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

явное создание экземпляра; спецификатор класса хранения "спецификатор" не разрешен при специализации  
  
 Не допускается использование спецификаторов класса хранения \(например `extern`\) в специализации класса шаблона, экземпляр которой был ранее явно создан.
