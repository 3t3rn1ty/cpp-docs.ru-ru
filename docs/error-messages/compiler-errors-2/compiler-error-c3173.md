# <a name="compiler-error-c3173"></a>Ошибка компилятора C3173
Несоответствие версий при слиянии idl  
  
 Эта ошибка возникает, если объектный файл содержит внедренные idl, который был создан с помощью предыдущей версии компилятора. Компилятор кодирует номер версии, что обеспечивает тот же компилятор, используемый для создания содержимого idl, внедренных в OBJ-файлы также же компилятор, используемый для слияния встроенных idl.  
  
 Обновление установки Visual C++, таким образом, все средства из последней версии.
