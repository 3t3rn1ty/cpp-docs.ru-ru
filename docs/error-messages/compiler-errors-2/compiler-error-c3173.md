# Ошибка компилятора C3173
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

несоответствие версий при слиянии idl  
  
 Эта ошибка возникает, если объектный файл содержит встроенный idl, созданный предыдущей версией компилятора.  Компилятор кодирует номер версии, чтобы тот же компилятор, который использовался для создания содержимого idl, встраиваемого в OBJ\-файлы, также использовался для слияния встроенных idl.  
  
 Обновите Visual C\+\+, чтобы использовать средства последней версии.
