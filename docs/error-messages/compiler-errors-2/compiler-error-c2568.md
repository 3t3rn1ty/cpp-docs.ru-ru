# Ошибка компилятора C2568
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"identifier1" : не удается разрешить перегрузку функции  
  
 Компилятору невозможно определить какую перегруженную функцию вызвать.  Фактические параметры, переданные в функцию, должны быть приведены для совмещения формальных параметров перегруженной функции, но ни одно совмещение однозначно не лучше, чем все остальные.
