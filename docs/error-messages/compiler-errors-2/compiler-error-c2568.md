# <a name="compiler-error-c2568"></a>Ошибка компилятора C2568
«идентификатор1»: не удалось разрешить перегрузку функции  
  
 Компилятор не может определить, какую из перегруженных вызываемой функции. Фактические параметры, переданные функции должен быть приведен к соответствует формальные параметры для одной перегруженной функции, но ни одно совмещение однозначно лучше, чем все остальные.
