# <a name="compiler-error-c3661"></a>Ошибка компилятора C3661
в списке явных переопределений не найден любой из переопределяемых методов  
  
 Явное переопределение указано одно или несколько имен типа.  Однако не было обнаружено функции с необходимой сигнатурой в типы, соответствует подписи переопределяющей функции.  При попытке переопределения на основе имени типа, должен существовать один или несколько виртуальных функций в указанных типов, которые соответствуют подписи переопределяющей функции.  
  
 Дополнительные сведения см. в разделе [явное переопределение](../../windows/explicit-overrides-cpp-component-extensions.md).
