# <a name="compiler-error-c3661"></a>Ошибка компилятора C3661
в списке явных переопределений не найден ни один из переопределяемых методов  
  
 Явное переопределение указано одно или несколько имен типов.  Однако не было обнаружено функции с необходимой сигнатурой в типы, которое соответствует подписи переопределяющей функции.  При попытке переопределения на основе имени типа, в указанном типе, которые соответствуют подписи переопределяющей функции должен быть один или несколько виртуальных функций.  
  
 Дополнительные сведения см. в разделе [явное переопределение](../../windows/explicit-overrides-cpp-component-extensions.md).
