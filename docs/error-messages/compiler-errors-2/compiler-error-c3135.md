# Ошибка компилятора C3135
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'property' : свойство не может быть типа const или volatile  
  
 Ключевые слова [const](../../cpp/const-cpp.md) и [volatile](../../cpp/volatile-cpp.md) нельзя использовать со свойствами.
