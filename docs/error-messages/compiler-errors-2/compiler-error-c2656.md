# Ошибка компилятора C2656
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"function" : функция в качестве битового поля не разрешена  
  
 Функция объявлена как член битового поля.  
  
 Это ошибка может быть вызвана синтаксической ошибкой в списке инициализаторов конструктора.
