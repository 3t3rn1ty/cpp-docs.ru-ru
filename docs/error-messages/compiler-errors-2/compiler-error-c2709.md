# Ошибка компилятора C2709
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"идентификатор": длина формальных параметров в байтах отличается от предыдущего объявления  
  
 Сигнатура в вызове указанной функции отличается от прототипа.
