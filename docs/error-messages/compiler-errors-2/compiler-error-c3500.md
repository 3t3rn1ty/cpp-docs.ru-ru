# Ошибка компилятора C3500
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

недопустимый ProgID "идентификатор программы"  
  
 В операторе `#import` указан недопустимый идентификатор программы.  Проверьте правильность указываемого идентификатора программы в реестре Windows.
