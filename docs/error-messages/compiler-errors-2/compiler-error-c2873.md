# <a name="compiler-error-c2873"></a>Ошибка компилятора C2873
«символ»: символ не может использоваться в объявлении использования  
  
 Объект `using` отсутствует директива [имен](../../cpp/namespaces-cpp.md) ключевое слово. Это указывает компилятору неправильно интерпретировать код как [с помощью объявления](../../cpp/using-declaration.md) вместо [с помощью директивы](../../cpp/namespaces-cpp.md#using_directives).
