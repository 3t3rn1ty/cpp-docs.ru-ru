# Ошибка компилятора C2654
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"идентификатор" : попытка обращения к члену вне пределов функции\-члена  
  
 Обращение к члену используется в объявлении.  Обращение к данным членов допускаются только в функциях\-членах.  
  
 Эта ошибка может возникать при попытке инициализации переменных в объявлении.  Для этого следует использовать конструктор.
