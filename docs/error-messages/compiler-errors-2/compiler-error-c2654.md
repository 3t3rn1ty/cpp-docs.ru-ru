# <a name="compiler-error-c2654"></a>Ошибка компилятора C2654
«Идентификатор»: попытка обращения к члену вне функции-члена  
  
 Доступ к члену в объявлении. Элемент данных может осуществляться только в функциях-членах.  
  
 Эта ошибка может возникать при попытке инициализации переменных в объявлении. Используйте конструктор для этой цели.
