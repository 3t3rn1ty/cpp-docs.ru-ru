# Ошибка компилятора C3180
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"имя типа": имя превысило ограничение "ограничение" по метаданным  
  
 Компилятор выполнил усечение имени для управляемого типа в метаданных.  Вследствие усечения тип не может использоваться вместе с директивой `#using` \(или эквивалентом в другом языке\).  
  
 Ограничение имени типа включает все характеристики пространства имен.
