# <a name="compiler-error-c3180"></a>Ошибка компилятора C3180
«имя_типа»: длина имени превышает ограничение метаданных символов «ограничение»  
  
 Компилятор усечение имени для управляемого типа в метаданных. Усечения тип не может использоваться вместе `#using` директивы (или его эквивалент в другом языке).  
  
 Ограничение имени типа включает все характеристики пространства имен.
