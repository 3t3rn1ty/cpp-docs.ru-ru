# Ошибка компилятора C3167
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Не удалось инициализировать .NET Framework: следует проверить, установлена ли платформа  
  
 .NET Framework не установлен на данном компьютере; необходимо установить .NET Framework.
