# <a name="compiler-error-c2667"></a>Ошибка компилятора C2667
«функция»: нет перегрузок нет наилучшего преобразования  
  
 Вызов перегруженной функции является неоднозначным и не удается разрешить.  
  
 Преобразование, требуемое для соответствия фактических параметров в вызове функции в один из перегруженных функций должно быть строго лучше, чем преобразования, необходимые для всех перегруженных функций.  
  
 В статье базы знаний Q240869 подробнее на частичное Упорядочение шаблонов функций.
