# Ошибка компилятора C3846
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"символ" : не удается импортировать символ из "сборки2": символ уже был импортирован из другой сборки "сборка1"  
  
 Символ не удалось импортировать из указанной сборки, поскольку он уже был ранее импортирован из указанной сборки.  
  
 Следующий пример приводит к возникновению ошибки C3846:  
  
```  
// C3846a.cpp  
// compile with: /LD /clr  
public ref struct G  
{  
};  
```  
  
 Затем скомпилируйте следующий код:  
  
```  
// C3846b.cpp  
// compile with: /clr  
#using "c3846a.dll"  
#using "c3846a.obj"   // C3846  
  
int main()  
{  
}  
```  
  
 Следующий пример приводит к возникновению ошибки C3846:  
  
```  
// C3846c.cpp  
// compile with: /LD /clr:oldSyntax  
#using <mscorlib.dll>  
using namespace System;  
  
public __gc struct G  
{  
};  
```  
  
 Затем скомпилируйте следующий код:  
  
```  
// C3846d.cpp  
// compile with: /clr:oldSyntax  
#using <mscorlib.dll>  
using namespace System;  
#using "c3846c.dll"  
#using "c3846c.obj"   // C3846  
  
int main()  
{  
}  
```
