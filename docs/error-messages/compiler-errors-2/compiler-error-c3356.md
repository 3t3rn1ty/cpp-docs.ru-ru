# <a name="compiler-error-c3356"></a>Ошибка компилятора C3356

"атрибут": вызов неоднозначно используемого атрибута с полным именем не допускается

Атрибут, используемый в нескольких процессах, например компилятором и поставщиком ATL, был задан неправильно.
