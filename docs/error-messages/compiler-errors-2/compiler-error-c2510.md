# <a name="compiler-error-c2510"></a>Ошибка компилятора C2510
"идентификатор": слева от "::" должен быть класс, структура или объединение  
  
 Слева от оператора разрешения области действия (`::`) должно быть указано имя класса, структуры или объединения.
