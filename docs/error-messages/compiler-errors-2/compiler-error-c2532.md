# <a name="compiler-error-c2532"></a>Ошибка компилятора C2532
«Идентификатор»: недопустимый модификатор для ссылки  
  
 Ссылка была изменена. Ссылки нельзя изменять для указания на другой объект. Вместо этого используйте указатель.
