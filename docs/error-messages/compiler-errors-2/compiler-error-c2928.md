# <a name="compiler-error-c2928"></a>Ошибка компилятора C2928
явное создание экземпляра; "идентификатор" не является функцией или статическими данными-членом класса-шаблона "класс"  
  
 Нельзя явно создать экземпляр члена `class` , который не является функцией или переменной `static` .
