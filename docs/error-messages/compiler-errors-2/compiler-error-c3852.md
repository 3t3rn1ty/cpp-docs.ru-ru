# <a name="compiler-error-c3852"></a>Ошибка компилятора C3852
«member» с типом «тип»: агрегатной инициализации не удалось инициализировать этот элемент  
  
 Попытка назначить элемент данных, который не может получить инициализацию по умолчанию в агрегатной инициализации инициализации по умолчанию как часть агрегатной инициализации.  
  
 Следующей ошибке C3852:  
  
```  
// C3852.cpp  
struct S  
{  
   short s;  
};  
  
struct S1  
{  
   int i;  
   const S s;  
};  
  
struct S2  
{  
   int i;  
   char & rc;  
};  
  
int main()  
{  
   S1 s1 = { 1 };   // C3852 const member   
   // try the following line instead  
   // S1 s1 = { 1, 2 };  
  
   S2 s2 = { 2 };   // C3852 reference member  
   // try the following line instead  
   // char c = 'a';  
   S2 s2 = { 2, c };  
}  
```
