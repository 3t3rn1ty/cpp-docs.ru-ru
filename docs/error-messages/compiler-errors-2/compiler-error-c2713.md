# Ошибка компилятора C2713
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

В каждой функции допускается применение только одной формы обработки исключений  
  
 Нельзя использовать структурную обработку исключений \(`__try`\/`__except`\) и обработку исключений C\+\+ \(`try`\/`catch`\) в одной и той же функции.
