# Ошибка компилятора C2855
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

параметр командной строки "параметр" несовместим с предкомпилированным заголовком  
  
 Данный параметр командной строки отличается от параметра, использованного для создания предварительно скомпилированного заголовка.  Генерирует еще раз предварительно скомпилированный заголовок при помощи параметра командной строки.
