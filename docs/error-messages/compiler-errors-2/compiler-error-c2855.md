# <a name="compiler-error-c2855"></a>Ошибка компилятора C2855

параметр командной строки «параметр» несовместим с предкомпилированным заголовком

Указанный параметр командной строки отличается от параметра, используемый для создания предкомпилированного заголовка. Повторное создание предкомпилированного заголовка с помощью заданного параметра командной строки.
