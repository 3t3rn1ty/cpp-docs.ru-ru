# <a name="compiler-error-c2545"></a>Ошибка компилятора C2545
«оператор»: не удается найти перегруженный оператор  
  
 Оператор не может использоваться с предоставленными операндами. Вы должны предоставить перегруженного оператора необходимые операнды.  
  
 Эта ошибка может возникнуть, если операнды имеют неправильный тип.  
  
 Чтобы устранить эту ошибку, определить оператор преобразования или конструктор, принимающий один параметр.
