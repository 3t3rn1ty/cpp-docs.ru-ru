# Ошибка компилятора C2545
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"оператор": не удается обнаружить перегруженный оператор  
  
 Невозможно использовать оператор с предоставленными операндами.  Следует предоставить для перегруженного оператора необходимые операнды.  
  
 Данная ошибка может возникнуть в результате использования неверного типа операндов.  
  
 Устранить данную ошибку можно посредством определения оператора преобразования или конструктора, который принимает один параметр.
