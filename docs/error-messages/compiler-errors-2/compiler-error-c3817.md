# <a name="compiler-error-c3817"></a>Ошибка компилятора C3817

«объявление»: свойство может применяться только к функции

`property` Ключевое слово может быть только примененные к определению функции.

C3817 доступен только с помощью параметра компилятора устаревшие **/CLR: oldSyntax**.
