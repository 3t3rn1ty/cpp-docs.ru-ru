# Ошибка компилятора C3276
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"ключевое слово" : для перехода из блока \_\_FINALLY FINALLY не определено поведение при обработке завершения  
  
 Эта ошибка аналогична предупреждению [C4532](../../error-messages/compiler-warnings/compiler-warning-level-1-c4532.md).  Однако при использовании \/clr это условие не может быть отключено с помощью pragma [warning](../../preprocessor/warning.md).
