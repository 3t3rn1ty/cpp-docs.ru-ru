# <a name="compiler-error-c3276"></a>Ошибка компилятора C3276
«ключевое_слово»: переход из блока __finally/finally не определено поведение при обработке завершения  
  
 Эта ошибка является таким же, как [C4532](../../error-messages/compiler-warnings/compiler-warning-level-1-c4532.md) предупреждение. Тем не менее, при использовании/CLR это условие не может быть отключено с [предупреждение](../../preprocessor/warning.md) pragma.
