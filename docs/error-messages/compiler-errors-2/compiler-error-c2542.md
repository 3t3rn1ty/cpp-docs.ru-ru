# Ошибка компилятора C2542
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"идентификатор": класс не имеет конструктора для инициализации  
  
 Отсутствует конструктор со списком параметров, соответствующим инициализации.  
  
 Причиной этой ошибки могут служить неправильные параметры при инициализации.
