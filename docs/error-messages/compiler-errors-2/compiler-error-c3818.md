# <a name="compiler-error-c3818"></a>Ошибка компилятора C3818
объявление свойства массива «свойство1» не должно перегружать свойство индекса «property2»  
  
 Перегрузка для свойства невозможна, когда он индексатор, а второе — свойство массива. 
  
 Ошибка C3818 доступен только с помощью параметра компилятора устаревшие **/CLR: oldSyntax**.
