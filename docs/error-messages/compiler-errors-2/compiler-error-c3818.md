# <a name="compiler-error-c3818"></a>Ошибка компилятора C3818
объявление свойства массива «свойство1» не будет перегружать свойство индекса «свойство2»  
  
 Перегруженная версия не поддерживается для свойства при один является индексатором, а другой — свойства массива. 
  
 C3818 доступен только с помощью параметра компилятора устаревшие **/CLR: oldSyntax**.
