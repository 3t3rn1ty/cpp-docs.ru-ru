# <a name="compiler-error-c2795"></a>Ошибка компилятора C2795
«Super::функция» не является функцией-членом  
  
 Это сообщение об ошибке появляется при попытке использовать [super](../../cpp/super.md) для доступа к членам, не являющимся функцию-член.
