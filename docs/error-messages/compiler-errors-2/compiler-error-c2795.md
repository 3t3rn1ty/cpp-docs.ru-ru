# Ошибка компилятора C2795
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"super::функция" не является функцией\-членом  
  
 Это сообщение об ошибке отображается при использовании ключевого слова [super](../../cpp/super.md) для обращения к членам, не являющимся функциями\-членами.
