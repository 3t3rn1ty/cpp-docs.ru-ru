# Ошибка компилятора C2651
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"тип данных" : слева от "оператора" должен стоять класс, структура или объединение  
  
 Для использования параметра шаблона в контексте класса, следует создать специализацию шаблона с классом вместо целочисленного типа.
