# Ошибка компилятора C2544
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

требуется "\)" для оператора "\(\)"  
  
 В операторе вызова функции отсутствует левая круглая скобка.  
  
 Эта ошибка может возникать при использовании расширения макроса.
