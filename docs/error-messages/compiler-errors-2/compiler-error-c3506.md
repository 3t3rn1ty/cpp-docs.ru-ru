# Ошибка компилятора C3506
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

нет библиотеки типов, зарегистрированной для LIBID "идентификатор"  
  
 Библиотека типов была зарегистрирована неверно.  Используйте файл regsvr32.exe для регистрации библиотеки типов.
