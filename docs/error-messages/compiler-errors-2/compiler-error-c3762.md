# Ошибка компилятора C3762
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

не удается обработать атрибут "атрибут"  
  
 Атрибут, определенный пользователем, который наследует `System.Security.Permissions.SecurityAttribute`, используется для определения атрибута безопасности.  Такой атрибут нельзя использовать в той же сборке, где он определен.
