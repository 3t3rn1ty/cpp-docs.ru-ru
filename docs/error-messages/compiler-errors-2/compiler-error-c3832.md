# <a name="compiler-error-c3832"></a>Ошибка компилятора C3832
«Библиотека типов»: библиотека типов выглядит, как если бы он был построен для 32-разрядных указателей; Измените квалификатор «ptrsize»  
  
 Явная информация, указанная с `ptrsize` атрибут [#import](../../preprocessor/hash-import-directive-cpp.md) директива не согласуется с компилятор обнаружил в библиотеке типов.
