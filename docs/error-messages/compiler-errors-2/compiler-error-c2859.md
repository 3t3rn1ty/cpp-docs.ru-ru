# Ошибка компилятора C2859
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"имяФайла" не представляет файл того "типа", который использовался при создании предкомпилированного заголовка; создайте предкомпилированный заголовок заново  
  
 Файлы предкомпилированных заголовков и базы данных проекта должны создаваться вместе, чтобы обеспечить согласованность информации.  Выполните повторное построение проекта, чтобы заново создать предкомпилированный заголовок.  
  
 Сведения о предкомпилированных заголовках см. в разделе [\/Y \(Предварительно скомпилированные заголовки\)](../../build/reference/y-precompiled-headers.md).
