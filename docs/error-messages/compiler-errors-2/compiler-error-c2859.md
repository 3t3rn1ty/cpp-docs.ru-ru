# <a name="compiler-error-c2859"></a>Ошибка компилятора C2859
Имя файла не тип файла, который использовался при создании предкомпилированного заголовка, предкомпилированного заголовка.  
  
 Файлы предкомпилированного заголовка и базы данных проекта должны создаваться вместе, чтобы обеспечить согласованность информации. Перестройте проект для повторного создания предкомпилированного заголовка.  
  
 Сведения о предкомпилированных заголовках см. в разделе [/Y (предварительно скомпилированные заголовки)](../../build/reference/y-precompiled-headers.md).
