# <a name="compiler-error-c3130"></a>Ошибка компилятора C3130
Внутренняя ошибка компилятора: не удалось записать блок подставляемого кода в PDB-ФАЙЛ  
  
 Эта ошибка возникает, если компилятору не удается записать блок подставляемого кода в PDB-файл. Наиболее распространенной причиной ошибки является нехватка дискового пространства.
