# Ошибка компилятора C3130
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

внутренняя ошибка компилятора: не удалось записать в файл PDB подставляемый блок кода  
  
 Эта ошибка возникает, если компилятору не удается записать подставляемый блок кода в файл PDB.  Наиболее частая причина возникновения этой ошибки — нехватка дискового пространства.
