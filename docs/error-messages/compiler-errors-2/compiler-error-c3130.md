# <a name="compiler-error-c3130"></a>Ошибка компилятора C3130

Внутренняя ошибка компилятора: не удалось записать блок подставляемого кода в PDB-ФАЙЛ

Эта ошибка возникает, если компилятору не удалось записать блок подставляемого кода в PDB-файл. Наиболее распространенной причиной возникновения этой ошибки является нехватка дискового пространства.
