# Ошибка компилятора C2721
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"спецификатор": спецификатор класса хранения нельзя использовать между оператором и типом  
  
 Пользовательские преобразования типов применяются ко всем классам хранения, в связи с чем нельзя указать класс хранения в преобразовании типов.
