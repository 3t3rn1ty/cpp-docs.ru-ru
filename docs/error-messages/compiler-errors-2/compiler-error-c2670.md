# Ошибка компилятора C2670
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"идентификатор": функция\-шаблон не может преобразовать номер параметра из типа "тип"  
  
 Параметр нельзя преобразовать в требуемый тип.  
  
 Ошибку можно устранить путем создания явного преобразования.
