# Ошибка компилятора C3752
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"attribute class": не удается классифицировать атрибут; "keyword" не должно использоваться в этом контексте  
  
 Заданный пользователем атрибут был применен неправильно.
