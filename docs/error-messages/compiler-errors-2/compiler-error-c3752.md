# <a name="compiler-error-c3752"></a>Ошибка компилятора C3752
«класс атрибута»: не удается классифицировать атрибут; в этом контексте не следует использовать «ключевое слово»  
  
 Определяемый пользователем атрибут был применен неправильно.
