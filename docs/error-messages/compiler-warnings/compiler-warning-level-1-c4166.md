# Предупреждение компилятора (уровень 1) C4166
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**недопустимое соглашение о вызове для конструктора или деструктора**  
  
 Конструкторы и деструкторы не могут иметь соглашения о вызовах, отличные от используемых по умолчанию для платформы \(за исключением случаев, когда явно задано **\_\_clrcall**\).
