# <a name="compiler-warning-level-1-c4119"></a>Предупреждение компилятора (уровень 1) C4119
Указаны разные базы "база1" и "база2"  
  
 Два относительных указателя несовместимы, поскольку они имеют разные базы. Компилятор не может выполнить их преобразование.
