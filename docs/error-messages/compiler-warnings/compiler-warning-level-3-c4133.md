# Предупреждение компилятора (уровень 3) C4133
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

тип: несовместимые типы \- из "типа 1" в "тип 2"  
  
 Это предупреждение может быть вызвано попыткой вычислить разность двух указателей различных типов.  
  
 Чтобы избежать этого предупреждения, обеспечьте соответствующее приведение типов.
