# Предупреждение компилятора (уровень 1) C4340
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"значение" : значение из положительного стало отрицательным  
  
 Значение `enum` превышает самое большое положительное значение `enum`, перенесенное в отрицательное.
