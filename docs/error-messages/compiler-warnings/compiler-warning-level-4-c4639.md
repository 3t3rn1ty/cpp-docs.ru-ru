# Предупреждение компилятора (уровень 4) C4639
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ошибка MSXML, комментарии XML\-документа не будет обработан. причина  
  
 Данное предупреждение может возникнуть по ряду причин.  
  
 Чтобы устранить это предупреждение:  
  
-   Выполните компиляцию заново.  
  
-   Переустановите MSXML с помощью повторной установки среды CLR.  
  
-   Измените или удалите комментарий документации, вызывающий предупреждение, и выполните компиляцию заново.  
  
 Когда возникает предупреждение компилятора C4639, вся последующая обработка комментариев XML\-документа отключается и файл XDC не создается.
