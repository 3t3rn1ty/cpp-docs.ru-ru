# <a name="compiler-warning-level-4-c4639"></a>Предупреждение компилятора (уровень 4) C4639
Ошибка MSXML, не будут обрабатываться комментарии XML-документа. reason  
  
 Это предупреждение может возникать по любой причине.  
  
 Чтобы устранить это предупреждение:  
  
-   Выполните повторную компиляцию.  
  
-   Переустановите MSXML, переустановив общеязыковая среда выполнения.  
  
-   Измените или удалите комментарий документации, вызывающий предупреждение и перекомпилировать.  
  
 При выдаче C4639 отключены все дальнейшей обработке комментария XML и XDC-файл не создается.
