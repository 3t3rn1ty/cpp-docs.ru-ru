# <a name="compiler-warning-level-1-c4821"></a>Предупреждение компилятора (уровень 1) C4821
Не удается определить тип кодировки Юникод; сохраните файл с сигнатурой (BOM)  
  
 Компилятору не удалось определить тип кодировки файла. Чтобы устранить это предупреждение, сохраните файл с отметкой порядка байтов. В разделе [управление файлами с кодировкой](http://msdn.microsoft.com/en-us/ab3a74da-5c8f-4680-8882-f1def4909e82) подробнее.
