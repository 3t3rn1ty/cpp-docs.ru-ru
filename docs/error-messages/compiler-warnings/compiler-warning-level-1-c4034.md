# Предупреждение компилятора (уровень 1) C4034
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

sizeof возвращает 0  
  
 Оператор `sizeof` применяется к операнду нулевого размера \(пустой структуре, объединению, классу, перечисляемому типу или типу `void`\).
