# <a name="compiler-warning-level-4-c4931"></a>Предупреждение компилятора (уровень 4) C4931

> предполагается, библиотека типов была собрана для *номер*-разрядных указателей

Явные сведения не было задано с помощью **ptrsize** атрибут [#import](../../preprocessor/hash-import-directive-cpp.md) директив; компилятор предполагает, что размер указателей в библиотеке типов составляет *номер*.

Это предупреждение отключено по умолчанию. Подробнее: [Выключенные по умолчанию предупреждения компилятора](../../preprocessor/compiler-warnings-that-are-off-by-default.md) .
