# <a name="compiler-warning-level-4-c4931"></a>Предупреждение компилятора (уровень 4) C4931
предполагается, что при построении библиотека типов была рассчитана на "число"-разрядные указатели  
  
 Явная информация не предоставлен с **ptrsize** атрибут [#import](../../preprocessor/hash-import-directive-cpp.md) директив; компилятор предполагает, что размер указателей в библиотеке типов составляет ***номер***.  
  
 Это предупреждение отключено по умолчанию. Подробнее: [Выключенные по умолчанию предупреждения компилятора](../../preprocessor/compiler-warnings-that-are-off-by-default.md) .
