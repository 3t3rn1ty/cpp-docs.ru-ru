# Предупреждение компилятора (уровень 1) C4089
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"функция": различные типы в фактическом параметре "номер" и формальном параметре "номер"  
  
 Различные типы соответствующих фактических и формальных параметров.  Фактический параметр передается без изменения.  Фактический параметр автоматически приводится к типу, указанному в определении функции.
