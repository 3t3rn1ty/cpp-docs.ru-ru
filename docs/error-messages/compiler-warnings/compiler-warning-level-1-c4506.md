# <a name="compiler-warning-level-1-c4506"></a>Предупреждение компилятора (уровень 1) C4506
отсутствует определение для встроенной функции «функция»  
  
 Данная функция объявлена до помечено для встраиванию, но не был определен.  
  
 Компилятор не встроенную функцию.  
  
 Убедитесь, что внешние функции, подлежащие встраиванию, объявляются с `extern` ключевое слово.
