# Предупреждение компилятора (уровень 2) C4275
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

classkey "identifier", не являющийся DLL\-интерфейсом, использован в качестве базового для DLL\-интерфейса classkey "identifier"  
  
 Экспортированный класс является производным от неэкспортированного класса.  
  
 Для минимизации возможности повреждения данных во время экспортирования класса с параметром [\_\_declspec\(dllexport\)](../../cpp/dllexport-dllimport.md) убедитесь, что соблюдаются следующие условия.  
  
-   Все статистические данные доступны посредством функций, экспортированных из библиотеки DLL.  
  
-   Ни один из встроенных методов класса не может изменить статистические данные.  
  
-   Ни один из встроенных методов класса не использует CRT\-функции, а другие функции библиотек используют статистические данные.  
  
-   Ни одна из встроенных функций класса не использует CRT\-функции или другие функции библиотек, где, например, можно получить доступ к статистическим данным.  
  
-   Ни один из методов класса \(независимо от встраивания\) не может использовать типы, в которых присутствует отличие статистических данных экземпляров в EXE и DLL.  
  
 Экспортирование классов можно предотвратить путем определения библиотеки DLL, определяющей класс с виртуальными функциями, и функций, которые можно вызвать для создания и удаления объектов типа.  Затем можно вызвать виртуальные функции типа.  
  
 Дополнительные сведения о экспортировать шаблоны см. в разделе [http:\/\/support.microsoft.com\/default.aspx?scid\=KB;EN\-US;168958](http://support.microsoft.com/default.aspx?scid=KB;EN-US;168958).  
  
 Ошибка C4251 может быть пропущена в Visual C\+\+ при наследовании от типа из библиотеки Standard C\+\+ с компилированием отладочного выпуска \(с параметром **\/MTd**\), где сообщение об ошибке компилятора ссылается на \_Container\_base.  
  
```  
// C4275.cpp  
// compile with: /EHsc /MTd /W2 /c  
#include <vector>  
using namespace std;  
class Node;  
class __declspec(dllimport) VecWrapper : vector<Node *> {};   // C4275  
```
