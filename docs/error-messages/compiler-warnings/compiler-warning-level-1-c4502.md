# Предупреждение компилятора (уровень 1) C4502
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"linkage specification" требует использования зарезервированного слова "extern" и должен предшествовать любым другим спецификаторам  
  
 Компоновка указана без ключевого слова `extern`.  Компоновка не относится к типам "non\-extern".  
  
 Компилятор принял ключевое слово `extern`.
