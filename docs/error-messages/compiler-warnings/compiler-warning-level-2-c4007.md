# Предупреждение компилятора (уровень 2) C4007
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"функция": должна являться "атрибутом"  
  
 Атрибут, требуемый для функции, не указан явно.  Например, функция **main** должна иметь атрибут `__cdecl`.  Компилятор назначает атрибут принудительно.
