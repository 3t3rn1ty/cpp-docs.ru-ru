# <a name="compiler-warning-level-1-c4411"></a>Предупреждение компилятора (уровень 1) C4411
«Идентификатор»: символ соответствует регистру смещения  
  
 Идентификатор является локальным символом, который соответствует регистру смещения и таким образом, могут быть использованы в операнде с другим символом.
