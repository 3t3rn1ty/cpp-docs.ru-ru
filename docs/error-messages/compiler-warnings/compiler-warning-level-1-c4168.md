# Предупреждение компилятора (уровень 1) C4168
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ограничение компилятора: не хватает типов для отладчика; удалите базу данных программы "база\_данных" и повторите сборку  
  
 Файл базы данных программы необходимо перестроить, чтобы вместить все типы в программе.
