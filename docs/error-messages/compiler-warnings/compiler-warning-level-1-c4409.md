# Предупреждение компилятора (уровень 1) C4409
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

недопустимый размер инструкции  
  
 У инструкции нет формы с указанным размером.  Использован минимальный допустимый размер.
