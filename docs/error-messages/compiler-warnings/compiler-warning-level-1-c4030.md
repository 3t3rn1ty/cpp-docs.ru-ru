# Предупреждение компилятора (уровень&#160;1) C4030
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**первый список формальных параметров длиннее второго**  
  
 Функция была повторно объявлена с другими формальными параметрами. Компилятор использует формальные параметры в первом объявлении.
