# <a name="compiler-warning-level-1-c4015"></a>Предупреждение (уровень 1) C4015 компилятора
«Идентификатор»: тип битового поля должен быть целым  
  
 Битовое поле не объявлено как целочисленный тип. Компилятор предполагает базовый тип битового поля без знака. Битовые поля должны объявляться как типы целое число без знака.
