# Предупреждение компилятора (уровень 1) C4015
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"идентификатор": тип битового поля должен быть целым  
  
 Битовое поле объявлено с использованием типа, отличного от целого.  В качестве базового типа битового поля предполагается тип без знака.  Битовые поля необходимо объявлять с использованием целых типов без знака.
