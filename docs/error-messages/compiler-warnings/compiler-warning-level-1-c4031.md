# <a name="compiler-warning-level-1-c4031"></a>Компилятор C4031 предупреждение (уровень 1)
второй список формальных параметров длиннее первого  
  
 Функция будет повторно объявлена с другими формальными параметрами. Компилятор использует формальные параметры в первом объявлении.
