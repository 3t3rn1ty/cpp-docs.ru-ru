# Предупреждение компилятора (уровень 3) C4543
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Подставляемый текст запрещен атрибутом "no\_injected\_text"  
  
 В исходном окне использован атрибут [no\_injected\_text](../../windows/no-injected-text.md). Это означает, что компилятор будет блокировать подстановку кода с помощью атрибутов.  
  
 Это предупреждение служит для напоминания о том, что подстановка кода с помощью атрибутов невозможна.
