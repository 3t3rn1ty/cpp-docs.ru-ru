# <a name="compiler-warning-level-1-c4375"></a>Предупреждение компилятора (уровень 1) C4375

не являющийся открытым метод «метод2» переопределяет «метод1»

Тип, реализующий другой тип определен метод переопределение, но переопределение не открытый. Таким образом метод не переопределяет метод базового типа.
