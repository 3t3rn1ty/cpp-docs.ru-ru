# Предупреждение компилятора (уровень 1) C4375
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

метод "метод\_2", который не является общим, не переопределяет "метод\_1"  
  
 Тип, реализующий другой тип, был использован для определения метода переопределения, однако переопределение не является открытым.  Поэтому метод не переопределяет метод базового типа.
