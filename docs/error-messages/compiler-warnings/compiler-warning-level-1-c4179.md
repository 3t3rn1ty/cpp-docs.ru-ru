# <a name="compiler-warning-level-1-c4179"></a>Предупреждение компилятора (уровень 1) C4179
"/ / *": интерпретировать как «/» и "/\*": несогласованность со стандартными "/ /" комментарии  
  
 **//\*** является неправильным разделителем комментариев. Вместо этого следует использовать **//** или **/\*** .
