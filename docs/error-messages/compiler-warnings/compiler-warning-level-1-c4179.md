# <a name="compiler-warning-level-1-c4179"></a>Предупреждение компилятора (уровень 1) C4179
"или / *": анализируется как «/» и "или\*": несогласованность со стандартными "или или" комментарии  
  
 **//\***Это является корректным разделителем комментариев. Use **//** or **/\*** instead.
