# Предупреждение компилятора (уровень 1) C4264
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"виртуальная\_функция" : не существует доступного переопределения для виртуальной функции\-члена из базового класса "класс"; функция скрыта  
  
 Предупреждение C4264 всегда возникает после предупреждения [C4263](../../error-messages/compiler-warnings/compiler-warning-level-4-c4263.md).  
  
 Данное предупреждение по умолчанию отключено.  Дополнительные сведения см. в разделе [Отключенные по умолчанию предупреждения компилятора](../Topic/Compiler%20Warnings%20That%20Are%20Off%20by%20Default.md).
