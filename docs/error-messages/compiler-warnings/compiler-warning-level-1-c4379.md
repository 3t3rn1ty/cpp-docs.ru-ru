# <a name="compiler-warning-level-1-c4379"></a>Предупреждение компилятора (уровень 1) C4379
Версии среды CLR не поддерживается на этом компьютере. Использование этой версии может привести к непредвиденным результатам.  
  
 У вас есть предыдущей версии CLR на компьютере, но не текущей версии. Чтобы устранить C4379, установите версию среды common language runtime, которая поставлялась с вашим компилятором.
