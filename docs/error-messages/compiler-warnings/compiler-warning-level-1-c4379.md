# Предупреждение компилятора (уровень 1) C4379
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Версия среды CLR не поддерживается на этом компьютере.Использование этой версии может привести к непрогнозируемым результатам.  
  
 Установите предыдущую версию среды CLR на вашем компьютере, но не текущую версию.  Чтобы устранить С4379, установите версию среды CLR, которая будет выполняться на вашем компьютере.
