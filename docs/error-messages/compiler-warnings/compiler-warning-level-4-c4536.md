# <a name="compiler-warning-level-4-c4536"></a>Предупреждение компилятора (уровень 4) C4536
«имя_типа»: имя типа превышает ограничение метаданных «limit» знаков  
  
 Имя типа усекается в метаданных, если он был управляемого типа. В разделе [C3180](../../error-messages/compiler-errors-2/compiler-error-c3180.md) для получения дополнительной информации.  
  
 Это предупреждение отключено по умолчанию. Подробнее: [Выключенные по умолчанию предупреждения компилятора](../../preprocessor/compiler-warnings-that-are-off-by-default.md) .
