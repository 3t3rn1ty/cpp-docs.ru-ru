# <a name="compiler-warning-level-1-c4120"></a>Предупреждение компилятора (уровень 1) C4120
несоответствие указателей - с базой и без базы  
  
 Компилятор не может выполнить преобразование между двумя указателями, так как один является относительным, а другой — нет.
