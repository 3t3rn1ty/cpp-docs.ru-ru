# Предупреждение компилятора (уровень 1) C4656
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

символ: тип данных изменился со времени последнего построения или имеются разные объявления  
  
 Был добавлен или изменен тип данных, делающий его новым для вашего кода с момента последнего построения.  Режим "Изменить и продолжить" не поддерживает изменения существующих типов данных.  
  
 За этим предупреждением всегда будет следовать [Неисправимая ошибка C1092](../../error-messages/compiler-errors-1/fatal-error-c1092.md).  Дополнительные сведения см. в разделе [Поддерживаемые изменения кода](../Topic/Supported%20Code%20Changes%20\(C++\).md).  
  
### Удаление данного предупреждения без завершения текущего сеанса отладки  
  
1.  Следует изменить тип данных до состояния, предшествовавшего ошибке.  
  
2.  В меню **Отладка** выберите команду **Применить изменения кода**.  
  
### Для устранения этой ошибки без изменения вашего исходного кода  
  
1.  В меню **Отладка** выберите команду **Остановить отладку**.  
  
2.  Из меню **Построение** выберите подпункт **Построение**.
