# Предупреждение компилятора (уровень 3) C4511
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"класс": не удается создать конструктор копии  
  
 Компилятору не удалось создать для класса конструктор копии по умолчанию. Возможно, базовый класс имеет закрытый конструктор копии.
