# <a name="compiler-warning-level-1-c4381"></a>Предупреждение компилятора (уровень 1) C4381
«функция1»: метод интерфейса не будет реализован закрытым методом «функция2»  
  
 Класс должен реализовывать всю функцию в интерфейсе. Класс может удовлетворять этому условию, если один из его базовых классов реализует функции. Тем не менее функции должны быть реализованы как открытые функции.
