# <a name="compiler-warning-level-1-c4381"></a>Предупреждение компилятора (уровень 1) C4381
«функция1»: метод интерфейса не будет реализован методом закрытые «функция2»  
  
 Класс должен реализовывать все функции в интерфейсе. Класс может удовлетворять этому условию, если один из его базовых классов реализует функции. Однако функция должна быть реализована как общая.
