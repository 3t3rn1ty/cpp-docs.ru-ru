# Предупреждение компилятора (уровень 1) C4731
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"указатель": регистр указателя фрейма "регистр" изменен встроенным ассемблерным кодом  
  
 Регистр указателя фрейма был изменен.  Необходимо сохранить и восстановить регистр во встроенном блоке сборки или переменной фрейма \(локальная переменная или параметр, зависящий от измененного регистра\). В противном случае код может функционировать неправильно.  
  
 Следующий пример приводит к возникновению ошибки C4731:  
  
```  
// C4731.cpp  
// compile with: /W1 /LD  
// processor: x86  
// C4731 expected  
void bad(int p) {  
   __asm  
   {  
      mov ebp, 1  
   }  
  
   if (p == 1)  
   {  
      // ...  
   }  
}  
```  
  
 EBP представляет собой указатель фрейма \(FPO не допускается\), в который вносятся изменения.  Если далее упоминается `p`, то он упоминается в связи с `EBP`.  Однако кодом была выполнена перезапись `EBP`, поэтому программа не будет работать корректно, и даже может произойти сбой.
