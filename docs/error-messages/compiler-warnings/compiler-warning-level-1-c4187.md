# <a name="compiler-warning-level-1-c4187"></a>Предупреждение компилятора (уровень 1) C4187
\#атрибуты import «атрибут1» и «атрибут2» несовместимы. Оба игнорируется  
  
 Объект [#import](../../preprocessor/hash-import-directive-cpp.md) инструкция, указанная в `no_implementation` и `implementation_only` атрибуты. Оба атрибута игнорируются.
