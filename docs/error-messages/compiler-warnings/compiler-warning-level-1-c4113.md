# <a name="compiler-warning-level-1-c4113"></a>Предупреждение (уровень 1) C4113 компилятора
«идентификатор1» отличается в списках параметров от «идентификатор2»  
  
 Указатель функции присваивается другому указателю функции, но списки формальных параметров функции не совпадают. Присвоение компилируется без изменений.
