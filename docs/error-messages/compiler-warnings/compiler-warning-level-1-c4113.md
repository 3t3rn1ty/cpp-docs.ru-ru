# Предупреждение компилятора (уровень 1) C4113
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

идентификатор 'identifier1' отличается списком параметров от идентификатора 'identifier2'  
  
 Указатель функции присваивается другому указателю функции, но списки формальных параметров функций не совпадают.  Присвоение компилируется без изменений.
