# Предупреждение компилятора (уровень 1) C4116
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

определение безымянного типа в скобках  
  
 В выражении со скобками объявлена безымянная структура, объединение или перечисление.  Определение типа не имеет смысла.  
  
 В вызове функции C определение относится к глобальной области видимости.  В вызове функции C\+\+ определение относится к той же области видимости, что и вызываемая функция.
