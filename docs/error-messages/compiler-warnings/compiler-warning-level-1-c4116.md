# <a name="compiler-warning-level-1-c4116"></a>Предупреждение (уровень 1) C4116 компилятора
Определение именованного типа в круглых скобках  
  
 Структуры, объединения или перечисляемого типа без имени, определен в скобках. Определение типа не имеет смысла.  
  
 В вызове функции C определение имеет глобальную область. В вызове функции C++ определение имеет совпадает с областью вызываемой функции.
