# Предупреждение компилятора (уровень 1) C4344
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

изменение поведения: использование явно заданных аргументов шаблона приводит к вызову "функции"  
  
 Вызов функции, выполняемый с использованием явно заданных аргументов шаблона, вызывает функцию, отличную от той, что была бы вызвана без указания явно заданных аргументов.
