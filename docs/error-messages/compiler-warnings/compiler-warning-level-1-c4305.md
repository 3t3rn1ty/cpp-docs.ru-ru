# Предупреждение компилятора (уровень 1) C4305
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"идентификатор": усечение из "типа 1" к "типу 2"  
  
 Идентификатор преобразуется в меньший тип, что приводит к потере данных.  
  
 Если использовался мастер настройки точки подключения ATL, см. статью информационной базы данных:  
  
-   Q250847 : Код для события с аргументом VARIANT, созданный мастером настройки точки подключения ATL, выводит предупреждение C4305.
