# <a name="compiler-warning-level-1-c4305"></a>Предупреждение компилятора (уровень 1) C4305
«Идентификатор»: усечение из «тип1» в «тип2»  
  
 Идентификатор преобразуется в меньший тип, приведет к потере данных.  
  
 Если используется мастер точки подключения ATL, см. статью базы знаний:  
  
-   Q250847: Мастером настройки точки подключения ATL созданный код для события с аргументом VARIANT дает предупреждение C4305
