# Предупреждение компилятора (уровень 1) C4098
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"функция": функция типа void, возвращающая значение  
  
 В функции, объявленной с типом возвращаемого значения [void](../../cpp/void-cpp.md), используется возвращающий значение оператор `return`.  При компиляции предполагается, что функция возвращает значение типа `int`.
