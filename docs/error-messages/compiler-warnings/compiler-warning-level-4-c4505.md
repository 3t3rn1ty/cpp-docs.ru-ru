# Предупреждение компилятора (уровень 4) C4505
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"функция" : удалена локальная неиспользуемая функция  
  
 Указанная функция является локальной и не используется в теле модуля. Таким образом, функция является неиспользуемым кодом.  
  
 Компилятор не создал код для этой неиспользуемой функции.
