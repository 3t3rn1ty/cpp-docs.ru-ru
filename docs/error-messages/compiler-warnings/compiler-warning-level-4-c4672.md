# Предупреждение компилятора (уровень&#160;4) C4672
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"идентификатор1": неоднозначность. Первое появление в качестве "идентификатор2"  
  
 Указанный объект, создаваемый в качестве исключения в блоке **try**, является неоднозначным. Однозначная трактовка объекта при его создании в качестве исключения невозможна.
