# <a name="resource-compiler-fatal-error-rc1203"></a>Неустранимая ошибка компилятора ресурсов RC1203
недопустимый шестнадцатеричный идентификатор языка по умолчанию указано.  
  
 Определение языка по умолчанию (/ l) параметр последовало недопустимый шестнадцатеричный идентификатор языка. В разделе [языка и страны или региона строк](../../c-runtime-library/locale-names-languages-and-country-region-strings.md) в *Справочник по библиотеке времени выполнения* список допустимых кодов языков.
