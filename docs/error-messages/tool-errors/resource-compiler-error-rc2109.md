# Ошибка компилятора ресурсов RC2109
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ожидается численная константа диалога  
  
 Оператор **DIALOG** требует целые значения для полей *x, y, ширина* и *высота*.  Убедитесь, что эти значения включены после ключевого слова **DIALOG** и являются отрицательными.
