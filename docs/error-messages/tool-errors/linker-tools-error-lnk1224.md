# <a name="linker-tools-error-lnk1224"></a>Ошибка средств компоновщика LNK1224
Недопустимый базовый адрес изображения  
  
 Указан недопустимый базовый адрес для изображения. Базовые адреса должны быть выровнены 64 КБ (последние четыре шестнадцатеричные цифры должен быть равен нулю) и базовый образ должны умещаться в 32-разрядное значение со знаком или без знака.
