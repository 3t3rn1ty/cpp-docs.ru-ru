# Ошибка средств компоновщика LNK1224
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Недопустимый базовый адрес изображения  
  
 Для изображения задан недопустимый базовый адрес.  Размер базовых адресов должен выравниваться по значению 64КБ \(последние четыре шестнадцатеричные цифры — нули\), а база образа должна умещаться в 32\-битовом значении со знаком или без знака.
