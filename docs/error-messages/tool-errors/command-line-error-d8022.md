# <a name="command-line-error-d8022"></a>Ошибка командной строки D8022
не удается открыть «messagefile»  
  
 Данный файл был не в текущем каталоге или в каталоге, указанном в переменной среды PATH. Файл сообщений содержит краткое описание синтаксиса командной строки компилятора и параметры.  
  
 Переместите этот файл в текущем каталоге или каталог в текущем пути. Если не удается найти этот файл, запустите программу установки, скопируйте его с установочных дисков.
