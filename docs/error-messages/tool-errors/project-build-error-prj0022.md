# Ошибка построения проекта PRJ0022
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Неизвестный инструмент, свойство "свойство" содержит неверное имя файла "файл".  
  
 Имя файла ***файл***, заданное в свойстве ***свойство***, недопустимо.  
  
 Возможно, был использован неизвестный или недействительный макрос.  
  
 Сведения о доступе к свойствам проекта см. в разделе [Настройка свойств проекта Visual C\+\+](../../ide/working-with-project-properties.md).
