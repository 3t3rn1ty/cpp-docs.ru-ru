# Неустранимая ошибка NMAKE U1083
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

пустой после подстановки конечный макрос "конечный макрос"  
  
 Данный целевой объект является вызовом макроса, который не определен или имеет значение NULL.  NMAKE не может обработать объект null.
