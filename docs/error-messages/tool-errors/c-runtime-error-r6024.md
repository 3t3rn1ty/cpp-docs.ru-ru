# Ошибка во время выполнения C R6024
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

для таблицы \_onexit\/atexit недостаточно памяти  
  
 Для `_onexit` или функции `atexit` нет доступной памяти.  Эта ошибка вызвана недостатком памяти.
