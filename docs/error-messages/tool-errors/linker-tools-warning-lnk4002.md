# Предупреждение средств компоновщика LNK4002
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

в объекте определен символ  
  
 Символ, отображаемый в декорированной форме, был указан в `object` в недекорированной форме, однако уникальное соответствие с декорированным символом найти не удалось.  Этому предупреждению всегда предшествует предупреждение [LNK4022](../../error-messages/tool-errors/linker-tools-warning-lnk4022.md), а за предупреждением следует неустранимая ошибка [LNK1152](../../error-messages/tool-errors/linker-tools-error-lnk1152.md).
