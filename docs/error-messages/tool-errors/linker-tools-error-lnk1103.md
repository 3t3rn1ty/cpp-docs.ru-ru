# Ошибка средств компоновщика LNK1103
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

отладочная информация повреждена; перекомпилируйте модуль  
  
 Эта ошибка может возникать из\-за того, что компиляция была прервана до создания допустимого объектного файла.  
  
 Перекомпилируйте данный объектный файл.  Если перекомпиляция не помогает, соберите сведения об обстоятельствах появления ошибки, после чего попытайтесь выделить возникшую проблему и создать воспроизводимый тестовый случай.  Затем обратитесь к файлу справки и технической поддержки или разделам технической поддержки в доступных пользовательских руководствах.
