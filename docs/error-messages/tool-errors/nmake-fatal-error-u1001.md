# Неустранимая ошибка NMAKE U1001
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

синтаксическая ошибка: недопустимый символ "символ" в макросе  
  
 Данный символ отображается в макросе, но не является буквенным символов, числом или нижним подчеркиванием.  
  
 Данная ошибка может возникать вследствие пропущенного двоеточия в макроопределении:  
  
```  
syntax error : illegal character '=' in macro  
```
