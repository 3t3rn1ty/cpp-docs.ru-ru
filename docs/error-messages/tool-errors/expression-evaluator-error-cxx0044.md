# Ошибка вычислителя выражений CXX0044
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

для использования указателя \_based \(void\):\> оператор  
  
 Указатель, основанный на `void`, не может использоваться непосредственно.  Форма будет полный указатель с помощью оператора **:\>**.  
  
 Эта ошибка идентична ошибке CAN0044.
