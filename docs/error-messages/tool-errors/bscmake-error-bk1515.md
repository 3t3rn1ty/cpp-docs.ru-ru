# <a name="bscmake-error-bk1515"></a>Ошибка BSCMAKE BK1515
BSC-файл: несовместимая версия: добавочное обновление невозможно  
  
 BSC-файл был создан с помощью другой версии BSCMAKE. Повторное создание BSC-файл.
