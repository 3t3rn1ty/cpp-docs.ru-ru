# <a name="project-build-error-prj0027"></a>Ошибка построения проекта PRJ0027
Юникод сообщение журнала «содержимое» содержит содержимое, которое не может быть преобразована к кодовой странице ANSI пользователя.  
  
 Обычно только появится это предупреждение вместе с сообщениями об ошибках создания пакетного файла или файла ответов.  
  
 Способ устранения этой ошибки является обновление содержимое журнала построения с использованием стандарта ANSI или установите кодовую страницу на компьютере и задать его в качестве системы по умолчанию.
