# <a name="c-runtime-error-r6019"></a>Ошибка во время выполнения R6019 C
не удается открыть устройство консоли  
  
> [!NOTE]
>  Если это сообщение об ошибке возникает во время выполнения приложения, приложения была завершена, так как она была предпринята попытка доступа к консоли, но он не имеет достаточных разрешений. Существует несколько возможных причин этой ошибки, но это происходит потому, что программа должна запускаться с правами администратора, или имеется ошибка в программе.  
>   
>  Для устранения этой ошибки попробуйте выполнить следующие действия:  
>   
>  -   Запустите программу от имени администратора.  
> -   Используйте **программы и компоненты** или **программы и компоненты** страницы в **панели управления** восстановите или переустановите программу.  
> -   Проверьте **центра обновления Windows** в **панели управления** для обновлений программного обеспечения.  
> -   Проверьте наличие обновленной версии приложения. Обратитесь к поставщику приложения, если ошибка повторится.  
  
 **Сведения для программистов**  
  
 Эта ошибка возникает, поскольку приложения вызывается функция консоли, но операционная система не предоставляет доступ к консоли. За исключением того, в режиме отладки, в консоли функции обычно не разрешены в приложениях для магазина Microsoft. Если для приложения требуются права администратора для выполнения, убедитесь, что установлен для запуска от имени администратора по умолчанию.
