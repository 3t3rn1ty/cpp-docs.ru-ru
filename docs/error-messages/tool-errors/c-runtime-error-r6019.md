# Ошибка во время выполнения C R6019
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

не удается открыть консольное устройство  
  
 В программе вызывается консольная функция, объявленная в файле CONIO.h, однако в операционной системе запрещен доступ к консоли.
