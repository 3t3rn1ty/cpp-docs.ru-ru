# <a name="linker-tools-error-lnk1166"></a>Ошибка средств компоновщика LNK1166
не удается настроить код по смещению = смещение va = значение  
  
 СВЯЗИ не удалось заполнить код, как требуется.  
  
 Некоторые инструкции не разрешено выходить за пределы страницы на некоторых процессорах. LINK предпринята попытка добавить пробелы для исправления этой ситуации. В этом случае ссылка может не работать для исправления этой ошибки.
