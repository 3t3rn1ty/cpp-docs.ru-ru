# Ошибка средств компоновщика LNK1166
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

не удается скорректировать код со смещением offset\="смещение", va\="значение"  
  
 LINK не удалось заполнить код, как требуется.  
  
 Определенным инструкциям не разрешено переходить границу страницы на некоторых процессорах.  LINK предпринята попытка добавить пробелы для исправления ситуации.  В этом случае LINK не удалось устранить проблему.
