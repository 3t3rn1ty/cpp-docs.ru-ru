# <a name="linker-tools-error-lnk1166"></a>Ошибка средств компоновщика LNK1166

не удается настроить код смещением = смещение, va = значение

ССЫЛКУ не удалось заполнить код, как требуется.

Некоторые инструкции не могут выходить за границы страницы на некоторых процессорах. ССЫЛКА пытается добавить пробелы для исправления этой ситуации. В этом случае ссылки не удалось обойти проблему.
