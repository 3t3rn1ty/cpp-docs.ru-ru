# Предупреждение средств компоновщика LNK4065
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"функция" нельзя упорядочить; не учитывается  
  
 Данная функция не была скомпилирована как упакованная.  Перекомпилируйте функцию с параметром [\/Gy](../../build/reference/gy-enable-function-level-linking.md).
