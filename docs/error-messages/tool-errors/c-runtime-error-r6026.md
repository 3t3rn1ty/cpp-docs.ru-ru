# Ошибка во время выполнения C R6026
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

недостаточно памяти для инициализации stdio  
  
 Эта ошибка возникает в том случае, если свободной памяти недостаточно для инициализации стандартных функций ввода\-вывода.
