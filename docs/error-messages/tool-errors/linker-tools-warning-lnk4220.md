# <a name="linker-tools-warning-lnk4220"></a>Предупреждение средств компоновщика LNK4220
Недопустимый «параметр компоновщика» значение «значение»; принято значение по умолчанию  
  
 Указанное значение вне диапазона [/TLBID](../../build/reference/tlbid-specify-resource-id-for-typelib.md) параметр. Значение по умолчанию для **/TLBID** -1.
