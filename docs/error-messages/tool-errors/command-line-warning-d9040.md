# Предупреждение командной строки D9040
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

пропуск параметра "\/analyze"; предупреждения анализа кода недоступны в этой версии компилятора  
  
 Параметр командной строки **\/analyze** не доступен во всех версиях Visual Studio.  Чтобы исправить это предупреждение, либо перейдите на нужно выпуска Visual Studio, либо удалите параметр командной строки.
