# Ошибка вычислителя выражений CXX0039
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

неоднозначный символ  
  
 Вычислитель выражений C не может определить, какой экземпляр символа необходимо использовать в выражении.  В дереве наследования содержится несколько экземпляров символа.  
  
 Явно укажите экземпляр, который необходимо использовать в выражении, с помощью оператора разрешения области действия \(`::`\).  
  
 Эта ошибка идентична ошибке CAN0039.
