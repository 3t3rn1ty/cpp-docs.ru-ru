# <a name="resource-compiler-error-rc2107"></a>Ошибка компилятора ресурсов RC2107
**Ожидается числовое командное значение**  
  
 Компилятор ресурсов ожидает числовое *idvalue* в **УСКОРИТЕЛИ** инструкции. Убедитесь, что вы использовали `#define` константу для указания значения и константа была написана верно.
