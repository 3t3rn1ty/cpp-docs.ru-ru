# Неустранимая ошибка NMAKE U1076
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

слишком длинное имя  
  
 Строка превышает одно из следующих предельных значений:  
  
-   1024 символа в имени макроса.  
  
-   256 символов для обозначения пути к целевому объекту  
  
-   2048 символов для обозначения команды.
