# Ошибка компилятора ресурсов RC2164
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

непредвиденное значение в RCDATA  
  
 Значения *raw\-data* в операторе **RCDATA** должны быть целыми числами или строками и разделяться запятыми.  Убедитесь, что не вы не забыли поставить запятую или заключить строку в кавычки.
