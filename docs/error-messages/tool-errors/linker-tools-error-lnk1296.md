# Ошибка средств компоновщика LNK1296
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

не удается загрузить "имя файла"  
  
 Указанная библиотека DLL недоступна.  
  
 Поиск библиотеки DLL осуществляется в текущем каталоге, в каталогах системы и Windows, а также в каталогах, указанных в переменной среды PATH.
