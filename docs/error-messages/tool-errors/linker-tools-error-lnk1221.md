# Ошибка средств компоновщика LNK1221
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

подсистема не может быть логически выведена, а должна быть определена  
  
 Компоновщик обладает недостаточными сведениями для выведения целевой подсистемы.  
  
 Чтобы устранить эту ошибку, используйте параметр [\/SUBSYSTEM](../../build/reference/subsystem-specify-subsystem.md).
