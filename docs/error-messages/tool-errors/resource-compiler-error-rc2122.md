# <a name="resource-compiler-error-rc2122"></a>Ошибка компилятора ресурсов RC2122

Неизвестный подтип меню

*Определение элемента* поле **меню** оператор может содержать только **MENUITEM** и **всплывающее ОКНО** инструкций.
