# <a name="resource-compiler-error-rc2167"></a>Ошибка компилятора ресурсов RC2167
Неизвестное поле VERSIONINFO; Ожидается BEGIN или запятая  
  
 Нераспознанное поле было найдено в **ФИКСИРОВАННЫХ** частью **VERSIONINFO** объявление структуры. Объект **VERSIONINFO** поля должны быть разделены запятыми тех.
