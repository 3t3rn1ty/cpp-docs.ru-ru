# Неустранимая ошибка компилятора ресурсов RW1022
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**ошибка ввода\-вывода при записи в файл**  
  
 Компилятору ресурсов не удалось выполнить запись в файл.  
  
### Чтобы устранить ошибку, следует проверить следующие возможные причины ее возникновения.  
  
1.  Нехватка места на диске.  Минимально необходимый объем свободного места должен быть равен двойному размеру создаваемого исполняемого файла.  
  
2.  Том доступен только для чтения.  
  
3.  Поврежденный сектор.  
  
4.  Ошибка общего доступа.
