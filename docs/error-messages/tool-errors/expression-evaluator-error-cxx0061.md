# Ошибка вычислителя выражений CXX0061
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

превышена глубина вложения функций  
  
 Выражение имеет глубину вложения функции, превышающую предельное значение.  Уменьшите глубину вложения в выражении.  
  
 Эта ошибка идентична ошибке CAN0061.
