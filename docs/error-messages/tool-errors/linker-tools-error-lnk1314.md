# Ошибка средств компоновщика LNK1314
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

поврежденная или недействительная таблица символов COFF \(неопределенный статический символ или символ метки\)  
  
 Сгенерированное компилятором содержимое для секции указанного объекта повреждено.  
  
 Эта ошибка может возникать вследствие повреждения компилятора.
