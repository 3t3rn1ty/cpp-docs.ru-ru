# Неустранимая ошибка NMAKE U1007
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

не разрешено использование двойных кавычек в имени  
  
 В указанном целевом имени или имени файла содержатся двойные кавычки \(**"**\).  
  
 Имя файла может быть заключено в двойные кавычки, но не может содержать их.
