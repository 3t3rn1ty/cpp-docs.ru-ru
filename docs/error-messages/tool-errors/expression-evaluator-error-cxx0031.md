# Ошибка вычислителя выражений CXX0031
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

выражение не подлежит расширению  
  
 В процессе вычисления выражения произошла внутренняя ошибка.  
  
 Можно попробовать написать эквивалентное выражение, которое будет вычислено корректно.  
  
 Эта ошибка идентична ошибке CAN0031.
