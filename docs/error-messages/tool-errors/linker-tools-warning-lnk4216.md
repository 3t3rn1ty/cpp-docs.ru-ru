# Предупреждение средств компоновщика LNK4216
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Экспортирована точка входа  
  
 Точка ввода была экспортирована из библиотеки DLL.  Точка ввода библиотеки DLL не должна экспортироваться.
