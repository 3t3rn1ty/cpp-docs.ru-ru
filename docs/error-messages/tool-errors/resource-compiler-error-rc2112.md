# Ошибка компилятора ресурсов RC2112
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

в диалоговом окне ожидается ключевое слово BEGIN  
  
 Сразу за ключевым словом **DIALOG** должно следовать ключевое слово **BEGIN**.
