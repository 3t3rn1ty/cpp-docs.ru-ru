# Ошибка компилятора ресурсов RC2114
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

требуется имя класса элемента управления  
  
 Поле `class` оператора CONTROL в операторе **DIALOG** должно быть одного из следующих типов: BUTTON, COMBOBOX, EDIT, LISTBOX, SCROLLBAR, STATIC или пользовательский.  Убедитесь, что класс указан правильно.
