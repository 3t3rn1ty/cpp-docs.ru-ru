# <a name="resource-compiler-fatal-error-rc1205"></a>Неустранимая ошибка компилятора ресурсов RC1205
Недопустимая кодовая страница  
  
 Указание кодовой страницы (/ c) следует параметр недопустимая кодовая страница. В разделе [кодовые страницы](../../c-runtime-library/code-pages.md) в *Справочник по библиотеке времени выполнения* для получения дополнительной информации.
