# <a name="resource-compiler-error-rc2152"></a>Ошибка компилятора ресурсов RC2152
Недопустимый управляющий символ  
  
 Недопустимый управляющий символ в операторе **ACCELERATORS** . Управляющий знак должен состоять из одной буквы (только) после знак вставки (**^**).
