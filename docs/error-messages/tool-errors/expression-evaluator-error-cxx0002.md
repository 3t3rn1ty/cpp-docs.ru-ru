# Ошибка вычислителя выражений CXX0002
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ошибка доступа к пользовательской памяти  
  
 Выражение пытается ссылаться на память, не выделенную для программы, отладка которой производится в данный момент.  
  
 Эта ошибка идентична ошибке CAN0002.
