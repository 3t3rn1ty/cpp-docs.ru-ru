# Неустранимая ошибка компилятора ресурсов RC1121
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ошибка ввода\/вывода при чтении файла  
  
 Компилятору ресурсов не удалось выполнить чтение файла.  Убедитесь, что диск, содержащий файл, доступен, и что этот файл является допустимым.
