# Ошибка вычислителя выражений CXX0041
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

элемент класса должен быть статическим членом или функцией\-членом  
  
 Нестатический член класса \(либо структуры или объединения\) был использован без указания используемого экземпляра класса.  
  
 Без указания экземпляра класса можно использовать только статические члены данных или функции\-члены.  
  
 Эта ошибка идентична ошибке CAN0041.
