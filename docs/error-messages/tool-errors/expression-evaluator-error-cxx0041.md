# <a name="expression-evaluator-error-cxx0041"></a>Ошибка вычислителя выражений CXX0041
элемент класса должен быть статическим членом или функцией-членом  
  
 Нестатический член класса (или структуры или объединения) был использован без указания используемого экземпляра класса для использования.  
  
 Без указания экземпляра можно использовать только статические данные-члены или функции-члены.  
  
 Эта ошибка идентична ошибке CAN0041.
