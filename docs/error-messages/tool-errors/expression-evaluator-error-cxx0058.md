# <a name="expression-evaluator-error-cxx0058"></a>Ошибка вычислителя выражений CXX0058
Перегруженный оператор не найден  
  
 Тип класса был указан как левый операнд в выражении, но перегруженный оператор не был определен для класса.  
  
 Эта ошибка идентична ошибке CAN0058.
