# <a name="nmake-warning-u4007"></a>Предупреждение программы NMAKE U4007
Имя файла «имя файла» слишком длинное; Сокращение до формата 8.3  
  
 Базовое имя данного файла содержит более восьми символов и расширение имеет более трех символов. NMAKE ПРОИЗВОДИТ усечение имени базе восьми символов и расширение трех символов.  
  
 Если файловая система поддерживает длинные имена файлов, заключите имя в двойные кавычки (**»**).
