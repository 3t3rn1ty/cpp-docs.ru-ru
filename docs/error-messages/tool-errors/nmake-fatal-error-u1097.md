# <a name="nmake-fatal-error-u1097"></a>Неустранимая ошибка NMAKE U1097
при использовании обозначений компонентов имени файла требуется указать зависимость  
  
 Текущая зависимость не имеет явного или скрытого зависимого. Синтаксис части имени файла, который использует процентов (`%`) описатель, представляет компоненты первого зависимого текущей цели.
