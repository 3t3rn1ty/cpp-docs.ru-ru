# <a name="nmake-fatal-error-u1097"></a>Неустранимая ошибка NMAKE U1097

обозначение компонентов требуется указать зависимость

Текущей зависимости не имеет явного или скрытого зависимого. Синтаксис обозначение компонентов имени файла, который использует процентов (`%`) описатель, представляет компоненты первого зависимого текущего целевого объекта.
