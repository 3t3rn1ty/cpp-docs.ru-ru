# Ошибка вычислителя выражений CXX0053
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

обнаружен символ, не являющийся функцией, во время привязки точек останова  
  
 Символ, использованный в качестве точки останова, не является функцией.  Эта ошибка может быть вызвана, если в качестве точки останова был указан член данных.  
  
 Эта ошибка идентична ошибке CAN0053.
