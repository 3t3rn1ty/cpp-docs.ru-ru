# Ошибка компилятора ресурсов RC2017
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

недопустимая управляющая последовательность  
  
 Escape\-последовательность используется там, где использоваться не должна.  
  
 Escape\-последовательность —обратная косая черта \( **\\** \), за которой следует число или буквенный символ —может использоваться только в символьном или строковом константном выражении.
