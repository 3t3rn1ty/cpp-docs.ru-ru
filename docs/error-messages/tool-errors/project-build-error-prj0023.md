# Ошибка построения проекта PRJ0023
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Средство 'tool', неизвестное свойство содержит недопустимое имя файла 'file'.  
  
 Имя файла, ***file***, указанное для средства, ***tool***, является недопустимым.  
  
 Возможно, был использован неизвестный или недействительный макрос.  
  
 Сведения о доступе к свойствам проекта см. в разделе [Настройка свойств проекта Visual C\+\+](../../ide/working-with-project-properties.md).
