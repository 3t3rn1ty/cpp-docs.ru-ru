# Предупреждение командной строки D9024
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Нераспознанный тип исходного файла "имя\_файла", использование объектного файла  
  
 Расширение указанного файла не распознано.  Файл признан объектным и передан компоновщику.  
  
 Поддерживаются только файлы со следующими расширениями:  
  
-   C \(исходный файл на языке C\)  
  
-   CXX \(исходный файл на языке C\+\+\)  
  
-   CPP \(исходный файл на языке C\+\+\)  
  
-   OBJ \(объектный файл\)  
  
-   LIB \(файл библиотеки\)  
  
-   DEF \(файл определения модуля\)  
  
-   EXP \(файлы экспорта компоновщика, созданные LINK \/LIB\)
