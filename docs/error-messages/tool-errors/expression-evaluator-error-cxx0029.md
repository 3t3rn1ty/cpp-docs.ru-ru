# <a name="expression-evaluator-error-cxx0029"></a>Ошибка вычислителя выражений CXX0029
не является указателем на структуру  
  
 Оператор выбора члена (**->**) был применен к выражению, которое не является указателем на структуру.  
  
 Проверьте, все выражение представляет скобки правильно, или тип привести выражение адреса к типу указателя, подходящей структурой.  
  
 Эта ошибка идентична ошибке CAN0029.
