# <a name="expression-evaluator-error-cxx0029"></a>Ошибка вычислителя выражений CXX0029

не является указателем на структуру

Оператор выбора члена (**->**) была применена к выражение, которое не является указателем на структуру.

Проверьте, что все выражение скобках неправильно, или адрес выражения к типу указателя, подходящей структурой приведения типов.

Эта ошибка идентична ошибке CAN0029.
