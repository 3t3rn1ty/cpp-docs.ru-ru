# Ошибка вычислителя выражений CXX0029
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

не является указателем на структуру  
  
 Оператор член\- выделения \(**\-\>**\) был применен к выражению, которое не является указателем на структуру.  
  
 Следует проверить, правильно ли выражение заключено целиком в круглые скобки, или выполнить приведение типа для выражения адреса к соответствующему типу указателя на структуру.  
  
 Эта ошибка идентична ошибке CAN0029.
