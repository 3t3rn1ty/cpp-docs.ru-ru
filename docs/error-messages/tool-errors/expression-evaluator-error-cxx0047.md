# <a name="expression-evaluator-error-cxx0047"></a>Ошибка вычислителя выражений CXX0047

список аргументов не соответствует функции

Выражение, которое вызвало функцию со списком фактического параметра, не соответствует список формальных параметров любой функции с тем же именем, определенным в программе.

Перегруженные функции могут вызываться только в том случае, если нет точного соответствия параметров или результат, который не требует создания объекта.

Эта ошибка идентична ошибке CAN0047.
