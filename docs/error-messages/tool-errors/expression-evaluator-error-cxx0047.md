# <a name="expression-evaluator-error-cxx0047"></a>Ошибка вычислителя выражений CXX0047
список аргументов не соответствует функции  
  
 Выражение вызвало функцию с фактическим списком параметров, не совпадает с тем же именем, определенных в программе список формальных параметров любой функции.  
  
 Перегруженные функции могут вызываться только в том случае, если нет точного соответствия параметров или сопоставление с не требуется создание объекта.  
  
 Эта ошибка идентична ошибке CAN0047.
