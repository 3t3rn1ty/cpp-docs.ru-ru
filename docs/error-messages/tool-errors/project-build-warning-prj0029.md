# <a name="project-build-warning-prj0029"></a>Предупреждение при построении проекта PRJ0029
Свойство «Выходные данные» этапа настраиваемого построения уровня проекта не задано. Настраиваемый этап сборки будет пропущен.  
  
 Этап настраиваемого построения не была выполнена, так как выходные данные не был указан.  
  
 Чтобы устранить эту ошибку, выполните одно из следующих:  
  
-   Исключите настраиваемый этап сборки из сборки.  
  
-   Добавление выходных данных.  
  
-   Удалите содержимое команды этапа настраиваемого построения.
