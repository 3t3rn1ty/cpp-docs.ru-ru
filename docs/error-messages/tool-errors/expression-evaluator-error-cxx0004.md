# Ошибка вычислителя выражений CXX0004
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

синтаксическая ошибка  
  
 Неправильный синтаксис выражения.  
  
 Введите выражение повторно с правильным синтаксисом.  
  
 Эта ошибка идентична ошибке CAN0004.
