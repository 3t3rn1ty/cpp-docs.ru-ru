# Неустранимая ошибка NMAKE U1052
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

файл "имя файла" не найден  
  
 NMAKE не может найти файл, указанный с одним из следующих:  
  
-   Параметр **\/F**.  
  
-   Директива препроцессора **\!INCLUDE**.  
  
-   Подпись \(**@**\) спецификатора для файла ответа.  
  
 Проверьте, что файл существует, и имя файла написано правильно.
