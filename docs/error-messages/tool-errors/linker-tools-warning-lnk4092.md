# <a name="linker-tools-warning-lnk4092"></a>Предупреждение средств компоновщика LNK4092

Общий доступный для записи раздел «раздел» содержит перемещения; образ может неправильно выполняться

Каждый раз, когда у вас есть общий раздел, чтобы предупреждать вас о потенциальных проблемах, компоновщик выдает это предупреждение.

Один способ обмена данными между несколькими процессами — отметить раздел как «общий». Однако создание раздела как общие может вызвать проблемы. Например у вас есть библиотеку DLL, которая содержит объявления в общем разделе данных:

```
int var = 1;
int *pvar = &var;
```

Не удается разрешить компоновщик `pvar` так, как его значение зависит от того, где библиотека DLL загружается в память, поэтому он помещает запись о перемещении в библиотеку DLL. При загрузке библиотеки DLL в память, адрес `var` можно разрешить и `pvar` назначен. Если другой процесс загружает той же библиотеки DLL, но не удается загрузить ее в то же адрес, перемещение для адреса `var` будет обновляться, второй процесс и первый процесс адресное пространство будет указывать на неправильный адрес.
