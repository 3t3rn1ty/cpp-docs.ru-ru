# Ошибка вычислителя выражений CXX0032
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

деление на 0  
  
 Делитель выражения равен нулю, что недопустимо.  В этом случае делителем может быть литерал, соответствующий нулю, или выражение, при вычислении которого получается нулевое значение.  
  
 Эта ошибка идентична ошибке CAN0032.
