# Предупреждение средств компоновщика LNK4229
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

обнаруженная недопустимая директива \/directive будет пропущена  
  
 Директива, переданная с помощью директивы pragma[comment](../../preprocessor/comment-c-cpp.md), имеет недопустимый формат.  Компоновщик пропустит директиву `/directive`.
