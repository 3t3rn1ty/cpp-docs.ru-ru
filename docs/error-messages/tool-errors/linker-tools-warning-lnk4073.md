# <a name="linker-tools-warning-lnk4073"></a>Предупреждение средств компоновщика LNK4073
не удается создать сопоставление для ILK-файла; компоновка выполняется не инкрементно  
  
 Не было большого недостаточно непрерывного пространства в общей памяти ССЫЛКУ, чтобы создать файл инкрементного состояния (). СВЯЗИ выполняется недобавочная сборка.
