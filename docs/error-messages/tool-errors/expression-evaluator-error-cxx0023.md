# Ошибка вычислителя выражений CXX0023
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

неверное основание системы счисления  
  
 Вычислитель выражений языка C не распознает указанное основание системы счисления.  Допускается только десятичное, шестнадцатеричное или восьмеричное основание системы счисления.  
  
 Эта ошибка идентична ошибке CAN0023.
