# Неустранимая ошибка компилятора ресурсов RW1023
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ошибка ввода\-вывода при записи в файл, диск заполнен  
  
 Минимально необходимый объем свободного места должен быть равен двойному размеру создаваемого исполняемого файла.
