# <a name="resource-compiler-fatal-error-rw1023"></a>Неустранимая ошибка компилятора ресурсов RW1023
Ошибка ввода-вывода при записи в файл, диск заполнен  
  
 Свободного места должно быть по крайней мере дважды размер исполняемого файла, который вы создаете.
