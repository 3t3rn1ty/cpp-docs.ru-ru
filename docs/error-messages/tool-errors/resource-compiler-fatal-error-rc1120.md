# Неустранимая ошибка компилятора ресурсов RC1120
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

нехватка памяти, необходимо количество байтов  
  
 Компилятору ресурсов недостаточно места для хранения элементов, хранимых в его куче.  Как правило, это результат использования чрезмерного количества символов.  
  
### Возможные способы устранения данной ошибки  
  
1.  Следует увеличить размер файла подкачки Windows.  Дополнительные сведения об увеличении размера файла подкачки см. в разделе "виртуальная память" в справке Windows.  
  
2.  Необходимо удалить ненужные включаемые файлы, в частности ненужные директивы `#define` и прототипы функций.  
  
3.  Можно также разбить текущий файл на два или более файлов и компилировать их по отдельности.  
  
4.  Можно удалить другие программы или драйвера, работающие в системе, которые потребляют значительные объемы памяти.
