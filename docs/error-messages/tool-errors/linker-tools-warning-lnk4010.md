# <a name="linker-tools-warning-lnk4010"></a>Предупреждение средств компоновщика LNK4010
Недопустимый номер версии подсистемы чисел; принята версия подсистемы по умолчанию  
  
 Можно указать версию подсистемы изображения ([/SUBSYSTEM](../../build/reference/subsystem-specify-subsystem.md)). Каждая подсистема имеет минимальное требование к версии. Если указанная версия меньше минимального, это предупреждение может возникнуть и компоновщик будет использовать подсистему по умолчанию.
