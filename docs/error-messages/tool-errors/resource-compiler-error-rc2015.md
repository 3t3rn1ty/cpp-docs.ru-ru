# Ошибка компилятора ресурсов RC2015
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

слишком много знаков в константе  
  
 Знаковая константа содержит более двух знаков.  
  
 Знаковые константы могут содержать только один знак \(стандартные знаковые константы\) или не более двух знаков \(длинные знаковые константы\).  
  
 Обратите внимание, что escape\-последовательность \(например "\\t" для табуляции\) преобразуется в один знак.
