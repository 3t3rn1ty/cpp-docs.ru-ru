# <a name="resource-compiler-error-rc2015"></a>Ошибка компилятора ресурсов RC2015

слишком много знаков в константе

Символьная константа содержит более двух знаков.

Символьные константы для одного символа (стандартные знаковые константы) или двух символов (long знаковые константы) ограничены.

Обратите внимание на то, что escape-последовательности (например, \t для вкладки) преобразуются в один символ.
