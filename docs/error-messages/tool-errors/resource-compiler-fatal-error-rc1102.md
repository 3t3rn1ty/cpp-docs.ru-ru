# <a name="resource-compiler-fatal-error-rc1102"></a>Неустранимая ошибка компилятора ресурсов RC1102
Внутренняя ошибка: слишком много аргументов для RCPP  
  
 Передано слишком много аргументов препроцессору компилятора ресурсов. Сократите количество символов, определенных с использованием определения символов (/ d), определив их в источнике. Эта ошибка также может быть вызвана указанием слишком многих указаны пути поиска файлов, используя параметр путь поиска включаемых файлов (/ i).
