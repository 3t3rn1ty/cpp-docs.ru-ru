# Предупреждение CVTRES CVT4001
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**тип машины не указан; предполагается**   
 ***тип***  
  
 Программа CVTRES не нашла спецификацию машины.  Предполагается, что используется указанный тип машины.  Если используемый по умолчанию тип неверен, перезапустите CVTRES с параметром \/MACHINE.
