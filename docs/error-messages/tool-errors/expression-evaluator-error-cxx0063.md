# <a name="expression-evaluator-error-cxx0063"></a>Ошибка вычислителя выражений CXX0063
неподдерживаемый перегруженный оператор ->  
  
 В выражении использован перегруженный оператор членам класса доступа (**->**).  
  
 Эта ошибка идентична CAN0063.
