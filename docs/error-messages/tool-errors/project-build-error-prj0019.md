# <a name="project-build-error-prj0019"></a>Ошибка построения проекта PRJ0019
Инструмент вернул код ошибки из  
  
 Уровень ошибки не является нулевым для этапа настраиваемого построения или события построения.  
  
 Вы также увидите PRJ0019 когда инструмент возвращает код ошибки, но не сообщение об ошибке. Это может произойти, например, при перенаправлении вывода MIDL в NUL.  
  
 В разделе [действия по устранению неполадок пользовательского построения и события построения](../../ide/troubleshooting-build-customizations.md) для получения дополнительной информации.  
  
 Эта ошибка также может возникнуть при при работе в качестве члена группы «Пользователи» и требуется доступ с правами администратора. Дополнительные сведения см. в разделе [работы в качестве члена группы «Пользователи»](../../security/running-as-a-member-of-the-users-group.md).
