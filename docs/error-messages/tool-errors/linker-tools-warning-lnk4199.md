# Предупреждение средств компоновщика LNK4199
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

параметр \/DELAYLOAD:dllname игнорируется; не найдены операции импорта из dllname  
  
 Компоновщик не обрабатывает параметр `dllname`, поскольку экспортируемые из соответствующей библиотеки функции не используются.
