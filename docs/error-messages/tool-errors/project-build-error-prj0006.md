# Ошибка построения проекта PRJ0006
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Не удалось открыть временный файл "файл".Убедитесь, что файл существует, и что каталог не защищен от записи.  
  
 В процессе построения Visual C\+\+ не удалось создать временный файл.  Возможные причины:  
  
-   Отсутствует временный каталог.  
  
-   Временный каталог защищен от записи.  
  
-   Диск переполнен.  
  
-   Каталог $\(IntDir\) защищен от записи или содержит защищенные от записи временные файлы.  
  
 Эта ошибка также может возникать вслед за ошибкой PRJ0007: Не удалось создать выходной каталог "каталог".  Ошибка PRJ0007 означает, что не удалось создать каталог $\(IntDir\); это также означает, что создание временных файлов будет невозможно.  
  
 Временные файлы создаются при указании:  
  
-   Файла ответов;  
  
-   Пользовательского шага построения;  
  
-   События построения.
