# Ошибка вычислителя выражений CXX0043
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

указатель "this" использован за пределами функции\-члена  
  
 Указатель **this** может быть использован только для функций\-членов, не являющихся статическими.  
  
 Эта ошибка идентична ошибке CAN0043.
