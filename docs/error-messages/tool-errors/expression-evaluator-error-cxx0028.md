# <a name="expression-evaluator-error-cxx0028"></a>Ошибка вычислителя выражений CXX0028
элемент структуры или объединения  
  
 Выражения формы «Struct.Member» или «pStruct -> член» был указан, но \<член > не элемент структуры.  
  
 Выражение не может быть скобках правильно.  
  
 Эта ошибка идентична CAN0028.
