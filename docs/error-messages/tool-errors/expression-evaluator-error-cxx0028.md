# <a name="expression-evaluator-error-cxx0028"></a>Ошибка вычислителя выражений CXX0028
не элемент структуры или объединения  
  
 Выражение в форме «Struct.Member» или «pStruct-> член» был указан, но \<элемента настроек не является элементом структуры.  
  
 Выражение не может быть скобки правильно.  
  
 Эта ошибка идентична CAN0028.
