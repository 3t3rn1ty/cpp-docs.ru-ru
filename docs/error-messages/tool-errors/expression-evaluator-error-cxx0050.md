# <a name="expression-evaluator-error-cxx0050"></a>Ошибка вычислителя выражений CXX0050
слева от оператора:: должен быть класс, структура или объединение  
  
 Значок слева от оператора разрешения области действия (`::`) не был класса, структуры или объединения.  
  
 Эта ошибка идентична CAN0050.
