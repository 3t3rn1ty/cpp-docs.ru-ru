# Ошибка вычислителя выражений CXX0050
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

левая сторона :: должна быть классом\/структурой\/объединением  
  
 Символ справа от оператора разрешения области действия \(`::`\) не является классом, структурой или объединением.  
  
 Эта ошибка идентична CAN0050.
