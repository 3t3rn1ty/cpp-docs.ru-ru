# <a name="expression-evaluator-error-cxx0021"></a>Ошибка вычислителя выражений CXX0021
использование структуры или объединения как скаляра  
  
 Структура или объединение используется в выражении, но не указан элемент.  
  
 При обработке структуры или объединения, имя переменной может отображаться самостоятельно, без квалификатора поля. Если структура или объединение используется в выражении, он должны быть дополнены конкретного необходимого элемента.  
  
 Укажите элемент, значение которого будет использоваться в выражении.  
  
 Эта ошибка идентична ошибке CAN0021.
