# <a name="project-build-error-prj0026"></a>Ошибка построения проекта PRJ0026
Файл ответа «файл», содержащий знаки Юникода, не может быть преобразована в кодовую страницу ANSI пользователя.  
  
 ***Содержимое файла в кодировке Юникод***  
  
 Система проектов найти содержимое в кодировке Юникод в файл ответов, не будет правильно преобразован в текущую кодовую страницу ANSI пользователя.  
  
 Способ устранения этой ошибки является обновления содержимого файла ответов в формате ANSI или установите кодовую страницу на компьютере и укажите ее как системный объект по умолчанию.
