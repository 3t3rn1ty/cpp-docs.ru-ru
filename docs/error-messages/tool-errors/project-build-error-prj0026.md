# Ошибка построения проекта PRJ0026
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

В файле ответа "файл" присутствует содержимое в кодировке Юникод, которое не удается преобразовать в кодовую страницу ANSI пользователя.  
  
 ***Содержимое файла в кодировке Юникод***  
  
 Система проектов обнаружила в файле ответа содержимое в кодировке Юникод, которое не удается правильно преобразовать в текущую кодовую страницу ANSI пользователя.  
  
 Способ устранения этой ошибки состоит в обновлении содержимого файла ответа или в установке кодовой страницы на компьютере в качестве системной страницы по умолчанию.
