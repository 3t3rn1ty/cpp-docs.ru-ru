# Ошибка построения проекта PRJ0044
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Свойство "Дополнительные зависимости" правила пользовательского построения "правило", заданное для файла "файл", является недопустимым.Свойство содержит "строку", которая является "значением".  
  
 Свойство **Дополнительные зависимости** являлось пустой строкой, либо строкой, содержащей недопустимые символы \(любые символы, которые не могут содержаться в имени файла или каталога\).  Для правил пользовательского построения необходимы выходные данные процесса построения.  
  
 Для получения дополнительной информации см. [Задание пользовательских средств построения](../../ide/specifying-custom-build-tools.md).
