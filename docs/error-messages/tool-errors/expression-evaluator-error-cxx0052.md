# <a name="expression-evaluator-error-cxx0052"></a>Ошибка вычислителя выражений CXX0052

функция-член не существует

Функция-член был указан как точка останова, но не найден. Задание точки останова в функции, который был развернут таким образом можно вызывать эту ошибку.

Перекомпилировать файл с встраивания отключен (/ Ob0) чтобы установить точку останова в этой функции.

Выражение, которое вызывается функция, которая не была определена.

Эта ошибка идентична ошибке CAN0052.
