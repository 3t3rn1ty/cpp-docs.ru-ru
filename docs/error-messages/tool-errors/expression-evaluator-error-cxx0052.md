# <a name="expression-evaluator-error-cxx0052"></a>Ошибка вычислителя выражений CXX0052
отсутствует функция-член  
  
 Функция-член, заданного в качестве точки останова, но не удалось найти. Задание точки останова во встроенной функции может вызвать эту ошибку.  
  
 Перекомпилировать файл с встраивания отключен (/ Ob0) чтобы установить точку останова в этой функции.  
  
 Выражение, вызывается функция, которая не была определена.  
  
 Эта ошибка идентична ошибке CAN0052.
