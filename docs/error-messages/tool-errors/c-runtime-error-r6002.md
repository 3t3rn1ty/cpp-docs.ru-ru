# <a name="c-runtime-error-r6002"></a>Ошибка во время выполнения C R6002
Поддержка плавающей запятой не загружена  
  
 Не был связан с плавающей запятой необходимые библиотеки.  
  
> [!NOTE]
>  Если это сообщение об ошибке возникает во время выполнения приложения, приложения была завершена из-за внутренней проблемы. Существует несколько возможных причин этой ошибки, но часто связанную с дефект в коде приложения или при попытке запустить приложение, которое не было создано для конкретного процессора.  
>   
>  Для устранения этой ошибки попробуйте выполнить следующие действия:  
>   
>  -   Используйте **программы и компоненты** или **программы и компоненты** страницы в **панели управления** восстановите или переустановите программу.  
> -   Проверьте **центра обновления Windows** в **панели управления** для обновлений программного обеспечения.  
> -   Проверьте наличие обновленной версии приложения. Обратитесь к поставщику приложения, если ошибка повторится.  
  
 **Сведения для программистов**  
  
 Эта ошибка может возникать в приложении, если не был связан с плавающей запятой библиотеки. Проверка для одного из этих причин:  
  
-   Строка формата для `printf_s` или `scanf_s` функция содержится спецификацию формата с плавающей запятой и программа не содержит все значения или переменные. Чтобы устранить эту проблему, используйте аргумент с плавающей запятой в соответствии со спецификацией формата с плавающей запятой или в другом месте выполните плавающей запятой в программе. В этом случае выполняется загрузка поддержки чисел с плавающей запятой.  
  
-   Компилятор позволяет свести к минимуму размер программы путем загрузки с плавающей запятой поддержки только при необходимости. Компилятор не обнаруживает спецификации формата с плавающей запятой или операций с плавающей запятой в строках формата, поэтому она не загружает необходимые подпрограммы. Чтобы устранить эту проблему, используйте спецификацию формата с плавающей запятой и указать аргумент с плавающей запятой или выполнить плавающей запятой в другом месте в программе. В этом случае выполняется загрузка поддержки чисел с плавающей запятой.  
  
-   В программе на разных языках библиотека C задана до библиотеки FORTRAN при компоновке программы. Повторное связывание и укажите библиотеку C последней.
