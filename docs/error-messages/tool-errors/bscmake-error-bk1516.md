# Ошибка BSCMAKE BK1516
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"BSC\-файл" поврежден; добавочное обновление невозможно  
  
 Возможно, BSC\-файл поврежден из\-за сбоя системы в процессе построения.  Удалите BSC\-файл, после чего выполните повторное построение всех SBR\-файлов и затем BSC\-файла.
