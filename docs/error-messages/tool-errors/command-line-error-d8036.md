# Ошибка командной строки D8036
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"\/option" не используется с несколькими исходными файлами  
  
 Эти параметры компилятора не используются с несколькими исходными файлами:  
  
-   Задайте имя списку файлов сборки \(\/Fa\).  
  
-   Измените имя объектного файла \(\/Fo\).  
  
-   Создайте сведения обозревателя исходного кода без локальных переменных \(\/Fr\).  
  
-   Создайте сведения обозревателя исходного кода, включающие локальные переменные \(\/FR\).
