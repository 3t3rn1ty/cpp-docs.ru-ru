# Ошибка вычислителя выражений CXX0057
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

нет кода в строке с номером  
  
 Не был сгенерирован код для указанного номера строки.  Ее нельзя использовать как точку останова.  
  
 Эта ошибка идентична ошибке CAN0057.
