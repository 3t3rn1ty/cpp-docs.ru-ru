# Ошибка средств компоновщика LNK1136
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

недопустимый или поврежденный файл  
  
 Входной файл имеет поврежденный заголовок или нулевой или ненормально малый размер.  
  
 Ошибка LNK1136 может также возникнуть, если файл неправильно указан как дополнительная зависимость.
