# Ошибка вычислителя выражений CXX0007
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

пропущена правая круглая скобка  
  
 Круглые скобки в выражении не сбалансированы.  Введите выражение заново со сбалансированными скобками.  
  
 Эта ошибка идентична ошибке CAN0007.
