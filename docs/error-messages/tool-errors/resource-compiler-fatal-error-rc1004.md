# <a name="resource-compiler-fatal-error-rc1004"></a>Неустранимая ошибка компилятора ресурсов RC1004
Непредвиденное обнаружение конца файла  
  
 Эта ошибка может быть вызвана с отсутствием символов перевода строки и возврата каретки в последней строке текстового файла.
