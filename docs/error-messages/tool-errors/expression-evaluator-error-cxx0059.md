# Ошибка вычислителя выражений CXX0059
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

левый операнд является классом, а не именем функции  
  
 Левый операнд вызова функции является именем класса, его нельзя использовать для вызова функции.  Эту ошибку может вызвать пропуск имени функции\-члена в выражении.  
  
 Эта ошибка идентична ошибке CAN0059.
