# <a name="expression-evaluator-error-cxx0066"></a>Ошибка вычислителя выражений CXX0066
отсутствует статический член  
  
 Не удалось найти статический член класса, или не был определен. Эта ошибка может возникнуть от статического класса, но не объявленный член определен, или только определенные или используемые в модулях, которые не содержат отладочной информации.  
  
 Эта ошибка идентична CAN0066.
