# Ошибка вычислителя выражений CXX0066
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

статический элемент отсутствует  
  
 Не удалось обнаружить статический элемент класса либо не был задан.  Ошибка происходит в результате того, что объявленный статический элемент класса не определен, либо только определен и на него имеются ссылки в модулях, которые не содержат отладочной информации.  
  
 Эта ошибка идентична CAN0066.
