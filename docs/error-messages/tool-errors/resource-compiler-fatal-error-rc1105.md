# Неустранимая ошибка компилятора ресурсов RC1105
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

недопустимый ключ, параметр: слишком много ключей \/d  
  
 Определено слишком много символов с помощью параметра \/d.  Определите некоторые символы в исходном коде и выполните повторную компиляцию.
