# <a name="resource-compiler-fatal-error-rc1105"></a>Неустранимая ошибка компилятора ресурсов RC1105
Недопустимый ключ, параметр: слишком много ключей /d  
  
 Слишком много символов, определенных с использованием символов определения (/ d) параметра. Определите некоторые символы в исходном коде и повторной компиляции.
