# Ошибка вычислителя выражений CXX0072
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ошибка: отсутствует информация о типе  
  
 Файл PCH не используется при компоновке или код ссылается на тип в модуле, скомпилированном без помощи параметра \/Zi.  
  
 Эта ошибка идентична ошибке CAN0067.
