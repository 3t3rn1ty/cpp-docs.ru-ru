# <a name="expression-evaluator-error-cxx0060"></a>Ошибка вычислителя выражений CXX0060
регистр недоступен  
  
 Выражение указало регистр, который не может использоваться. Эта ошибка может вызвана при попытке обращения к регистру, не существует на компьютере.  
  
 Эта ошибка идентична ошибке CAN0060.
