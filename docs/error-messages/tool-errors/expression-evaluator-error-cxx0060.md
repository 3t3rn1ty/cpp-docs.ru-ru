# Ошибка вычислителя выражений CXX0060
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

регистр недоступен  
  
 Выражение указало регистр, который нельзя использовать.  Эту ошибку может вызвать попытка обращения к регистру, который отсутствует на компьютере.  
  
 Эта ошибка идентична ошибке CAN0060.
