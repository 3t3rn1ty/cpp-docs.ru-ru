# <a name="nmake-fatal-error-u1077"></a>Неустранимая ошибка NMAKE U1077

«Программа»: «значение» код возврата

Указанной команды или программы вызывается ошибка и возвращается указанный код выхода (NMAKE).

Чтобы отключить эту ошибку и продолжить сеанс программы NMAKE, используйте параметр/i, **. ИГНОРИРОВАТЬ** директива точка или тире (**-**) модификатор команды. Чтобы продолжить сеанс программы NMAKE для несвязанных частей дерева зависимостей, используйте параметр /K.
