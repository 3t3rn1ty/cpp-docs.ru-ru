# Ошибка компилятора ресурсов RC2175
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

файл ресурсов "имя\_файла" не соответствует формату 3.00  
  
 Для указанного ресурса используется более ранний формат, чем формат версии 3.00.  Файл ресурса необходимо преобразовать или создать заново в формате версии 3.00 или более поздней версии.
