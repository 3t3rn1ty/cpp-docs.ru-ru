# <a name="c-runtime-error-r6031"></a>Ошибка во время выполнения R6031 C
Попытка инициализации CRT более одного раза. Это указывает на ошибку в приложении.  
  
> [!NOTE]
>  Если это сообщение об ошибке возникает во время выполнения приложения, приложения была завершена из-за внутренней проблемы. Это может быть вызвана ошибкой в приложении, или ошибки в надстройку или расширения, который использует приложение.  
>   
>  Для устранения этой ошибки попробуйте выполнить следующие действия:  
>   
>  -   Используйте **программы и компоненты** или **программы и компоненты** страницы в **панели управления** восстановите или переустановите программу.  
> -   Используйте **программы и компоненты** или **программы и компоненты** страницы в **панели управления** для удаления, восстановите или переустановите надстройку или расширение программ, используемых в приложении.  
> -   Проверьте **центра обновления Windows** в **панели управления** для обновлений программного обеспечения.  
> -   Проверьте наличие обновленной версии приложения. Обратитесь к поставщику приложения, если ошибка повторится.  
  
 **Сведения для программистов**  
  
 Это диагностическое указывает, что во время блокировки загрузчика выполнялись инструкции MSIL. Дополнительные сведения см. в разделе [Инициализация смешанных сборок](../../dotnet/initialization-of-mixed-assemblies.md).
