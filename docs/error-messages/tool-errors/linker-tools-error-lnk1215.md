# <a name="linker-tools-error-lnk1215"></a>Ошибка средств компоновщика LNK1215
Сбой операции метаданных (HRESULT): ошибка  
  
 Компоновщик получил ошибку от среды выполнения .NET при попытке выполнения обновления метаданных через среду выполнения .NET.  
  
 `HRESULT`представляет собой значение HRESULT из метода среды выполнения .NET. `error`—. Текст, предоставляемый NET.  
  
 Возможно, компоновщик не соответствует и среду выполнения .NET. Переустановите Visual C++.
