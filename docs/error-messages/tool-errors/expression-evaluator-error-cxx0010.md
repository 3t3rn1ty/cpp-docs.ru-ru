# Ошибка вычислителя выражений CXX0010
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

отсутствует левая квадратная скобка  
  
 Выражение содержит несогласованные квадратные скобки.  Проверьте правильность записи квадратных скобок в выражении.  
  
 Эта ошибка идентична ошибке CAN0010.
