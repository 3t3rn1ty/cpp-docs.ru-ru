# Ошибка компилятора ресурсов RC2135
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

файл не найден:имя файла  
  
 Файл, указанный в командной строке компилятора ресурсов, не найден.  Проверьте был ли файл перемещен в другой каталог и правильно ли были написаны имя файла и путь.  
  
 В файлы производится поиск на предмет использования **INCLUDE** переменной среды или параметра Visual C\+\+ **INCLUDE**.
