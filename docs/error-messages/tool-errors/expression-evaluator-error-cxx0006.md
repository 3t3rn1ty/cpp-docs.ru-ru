# Ошибка вычислителя выражений CXX0006
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

пропущена левая круглая скобка  
  
 Круглые скобки в выражении не сбалансированы.  Введите выражение заново со сбалансированными скобками.  
  
 Эта ошибка идентична CAN0006.
