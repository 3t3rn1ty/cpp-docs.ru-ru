# <a name="command-line-warning-d9035"></a>Предупреждение командной строки D9035
параметр «параметр» является устаревшим и будет удален в будущем выпуске  
  
 Указан параметр компилятора, который будет удален в будущей версии компилятора Visual C++. Это предупреждение сопровождается предупреждение [D9036](../../error-messages/tool-errors/command-line-warning-d9036.md) Если предложенная замена *параметр*.  
  
 Указанный параметр по-прежнему работает, но следует обновить конфигурацию построения. В результате проект чаще всего продолжить создание при обновлении компилятор.
