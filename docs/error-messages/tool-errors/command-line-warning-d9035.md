# Предупреждение командной строки D9035
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Использовать параметр "параметр" не рекомендуется, и в следующем выпуске он будет удален  
  
 Был задан параметр компилятора, который будет удален в следующей версии компилятора Visual C\+\+.  За этим предупреждением следует предупреждение [D9036](../../error-messages/tool-errors/command-line-warning-d9036.md), если рекомендуется заменить *option*.  
  
 Заданный параметр функционирует, но необходимо немедленно обновить конфигурацию построения.  Это повысит вероятность того, что проект продолжит построение после обновления компилятора.
