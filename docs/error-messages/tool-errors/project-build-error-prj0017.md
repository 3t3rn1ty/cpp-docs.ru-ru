# Ошибка построения проекта PRJ0017
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Текущий рабочий каталог недопустим.  
  
 Длина пути к текущему рабочему каталогу превышает значение \_MAXPATH.  Чтобы устранить эту ошибку, не создавайте проект на таком глубоком уровне вложения.
