# <a name="project-build-error-prj0017"></a>Ошибка построения проекта PRJ0017

Текущий рабочий каталог недопустим.

Путь к текущему рабочему каталогу длиннее, чем _MAXPATH. Чтобы устранить эту ошибку, не создавайте проект на таком глубоком уровне.
