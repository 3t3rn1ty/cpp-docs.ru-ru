# Ошибка компилятора ресурсов RC2116
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ожидается номер для идентификатора  
  
 Ожидается число для поля `id` оператора управляющего элемента в операторе **DIALOG**.  Убедитесь, что у вас есть число или оператор `#define` для идентификатора элемента управления.
