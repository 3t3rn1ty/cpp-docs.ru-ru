# Ошибка вычислителя выражений CXX0005
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

оператор не поддерживается  
  
 Неподдерживаемый оператор C будет указан в выражении.  Запишите эквивалентное выражение при помощи операторов, поддерживаемых языком С.  
  
 Эта ошибка идентична ошибке CAN0005.
