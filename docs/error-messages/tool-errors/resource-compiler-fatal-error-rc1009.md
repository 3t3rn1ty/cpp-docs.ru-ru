# Неустранимая ошибка компилятора ресурсов RC1009
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ограничение компилятора: слишком глубокое вложение макросов "макрос"  
  
 В файле превышен предел вложения макросов для компилятора ресурсов.  Проверьте указанный исходный файл для уменьшения глубины вложения макросов.
