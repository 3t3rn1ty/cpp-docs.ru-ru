# <a name="profile-guided-optimization-warning-pg1039"></a>Предупреждение профильной оптимизации PG1039
Не удалось дождаться момента, удобного для уборки.  
  
 Приложение не разрешает [pgosweep](../../build/reference/pgosweep.md) для запуска.  Если это возможно выполняются pgosweep, когда приложение находится в состоянии простоя.
