# Предупреждение профильной оптимизации PG1039
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Не удалось дождаться момента, удобного для уборки.  
  
 Запуск [pgosweep](../../build/reference/pgosweep.md) не разрешен приложением.  По возможности запуск pgosweep должен производиться, когда приложение находится в состоянии бездействия.
