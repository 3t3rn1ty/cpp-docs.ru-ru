# <a name="project-build-error-prj0005"></a>Ошибка построения проекта PRJ0005
Не удалось создать временный файл в каталог «directory».  
  
 Сбой вызова, чтобы создать временный файл. Возможные причины ошибки:  
  
-   Вышел за пределы имена временных файлов.  
  
-   Временная папка доступна только для чтения.  
  
-   Нет, каталог temp или переменная среды TMP.  
  
-   На компьютере недостаточно свободного места на диске.
