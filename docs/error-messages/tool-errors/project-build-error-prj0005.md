# Ошибка построения проекта PRJ0005
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Не удается создать временный файл в каталоге "каталог".  
  
 Вызов создания временного файла завершился с ошибкой.  Возможные причины ошибки:  
  
-   Закончились доступные имена временных файлов;  
  
-   Каталог временных файлов доступен только для чтения;  
  
-   Каталога временных файлов и переменной среды TMP не существует;  
  
-   Закончилось свободное место на диске компьютера.
