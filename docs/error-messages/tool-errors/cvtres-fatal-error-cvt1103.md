# Неустранимая ошибка CVTRES CVT1103
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

не удается прочитать имя файла  
  
 При попытке CVTRES прочитать данный файл произошла неустранимая ошибка.  
  
 Данная ошибка может быть вызвана повреждением файла.
