# Ошибка вычислителя выражений CXX0025
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

оператору требуется структура или объединение  
  
 Оператор, принимающий выражение типа структуры `struct` или **объединения**, был применен к выражению, которое не является структурой `struct` или **объединением**.  
  
 Компоненты переменных класса, структуры или объединения должны иметь полное имя.  Компоненты нельзя ввести без полной спецификации.  
  
 Эта ошибка идентична CAN0025.
