# <a name="expression-evaluator-error-cxx0025"></a>Ошибка вычислителя выражений CXX0025
оператор должен структуры или объединения  
  
 Оператор, принимающий выражение `struct` или **объединение** тип был применен к выражению, не `struct` или **объединение**.  
  
 Компоненты класса, структуры или объединения переменные должны иметь полное имя. Компоненты нельзя ввести без полной спецификации.  
  
 Эта ошибка идентична CAN0025.
