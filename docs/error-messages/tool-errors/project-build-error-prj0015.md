# <a name="project-build-error-prj0015"></a>Ошибка построения проекта PRJ0015

В системе отсутствует устройство NULL. Не удалось запустить построение.

Эта ошибка может быть вызвана нехваткой системных ресурсов. Закройте некоторые приложения или перезагрузке.

Он также может возникать, если у вас нет необходимых прав для устройство NULL на компьютере.
