# <a name="project-build-error-prj0015"></a>Ошибка построения проекта PRJ0015
В системе отсутствует устройство NULL. Не удается запустить построение.  
  
 Эта ошибка может быть вызвано нехваткой системных ресурсов. Закройте некоторые приложения или перезагрузки.  
  
 Он также может возникать, если на компьютере нет необходимых прав доступа для устройства NULL.
