# Ошибка построения проекта PRJ0015
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

В системе отсутствует устройство NULL.Не удается запустить построение.  
  
 Эта ошибка может быть вызвана нехваткой системных ресурсов.  Завершите несколько приложений или перезагрузите систему.  
  
 Также, причина может состоять в недостаточности привилегий на использование устройства NULL.
