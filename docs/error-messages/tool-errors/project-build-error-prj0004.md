# Ошибка построения проекта PRJ0004
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Не удалось создать командную строку для инструмента "tool".  
  
 Таким образом, было указано одно или несколько свойств недопустимости синтаксиса вызова средства***tool***:  
  
-   Можно задать неправильный или неизвестный макрос.  
  
-   На жестком диске компьютера может быть мало свободного места.  
  
 Можно просмотреть параметры инструмента путем просмотра [страницы со свойствами командной строки](../../ide/command-line-property-pages.md).
