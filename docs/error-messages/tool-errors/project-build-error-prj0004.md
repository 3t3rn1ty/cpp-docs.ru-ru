# <a name="project-build-error-prj0004"></a>Ошибка построения проекта PRJ0004
Не удалось создать командную строку для инструмента» tool ".  
  
 Одно или несколько свойств были указаны в таким образом, чтобы сделать синтаксис вызова для ***средство*** недопустимые:  
  
-   Указан неверный или неизвестный макрос.  
  
-   На компьютере не хватает свободного места на диске.  
  
 Вы можете просмотреть параметры инструмента путем просмотра [страницы свойств командной строки](../../ide/command-line-property-pages.md).
