# Ошибка BSCMAKE BK1505
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

невозможно прочитать файл "имя\_файла" \[: причина\]  
  
 Инструменту BSCMAKE не удалось произвести чтение из файла.  
  
### Чтобы устранить ошибку, следует проверить следующие возможные причины ее возникновения.  
  
1.  Повреждение файла.  
  
2.  Усечение файла из\-за нехватки места на диске для компилятора или прерывания работы компилятора в процессе создания SBR\-файла.
