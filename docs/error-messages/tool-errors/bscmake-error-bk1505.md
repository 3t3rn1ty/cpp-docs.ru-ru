# <a name="bscmake-error-bk1505"></a>Ошибка BSCMAKE BK1505
не удается прочитать файл «имя_файла» [: причина]  
  
 BSCMAKE не удается прочитать файл.  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>Чтобы устранить ошибку, проверьте указанные ниже возможные причины ее возникновения.  
  
1.  Повреждение файла.  
  
2.  Файл усечения в компилятор, нехватка места на диске или прерывания во время создания SBR-файл.
