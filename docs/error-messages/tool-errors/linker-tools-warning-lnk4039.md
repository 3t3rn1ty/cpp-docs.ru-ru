# Предупреждение средств компоновщика LNK4039
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

раздел 'name', указанный с параметром \/SECTION, не существует  
  
 [DUMPBIN](../../build/reference/dumpbin-reference.md) или [EDITBIN](../Topic/EDITBIN%20Reference.md) не смогли найти раздел с именем `name` во входном файле.  
  
 Для просмотра раздела в файле следует запустить DUMPBIN с параметром [\/HEADERS](../../build/reference/headers.md).
