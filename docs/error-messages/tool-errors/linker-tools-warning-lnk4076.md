# Предупреждение средств компоновщика LNK4076
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

недопустимый файл добавочного статуса "имяфайла"; добавочная компоновка не используется  
  
 Программе LINK не удалось осуществить запись в файл добавочного статуса \(ILK\).  Либо файл `filename` поврежден, либо он не является базой данных добавочной компоновки.  Удалите файл и проведите повторную компоновку.
