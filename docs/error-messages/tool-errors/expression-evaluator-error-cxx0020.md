# <a name="expression-evaluator-error-cxx0020"></a>Ошибка вычислителя выражений CXX0020
Неправильные типы операндов для этой операции  
  
 Оператор применяется к выражению с недопустимым типом для этого оператора.  
  
 Например не должно получить адрес регистра или индекс массива с помощью выражения с плавающей запятой.  
  
 Эта ошибка идентична ошибке CAN0020.
