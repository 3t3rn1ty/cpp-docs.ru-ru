# <a name="expression-evaluator-error-cxx0020"></a>Ошибка вычислителя выражений CXX0020

типы операндов для этой операции

Оператор применяется к выражению с недопустимый тип данного оператора.

Например не допускается, чтобы получить адрес регистра или индекс массив, содержащий выражение с плавающей запятой.

Эта ошибка идентична ошибке CAN0020.
