# <a name="expression-evaluator-error-cxx0020"></a>Ошибка вычислителя выражений CXX0020
неправильные типы операндов для этой операции  
  
 Оператор применяется к выражению с недопустимым типом для данного оператора.  
  
 Например недопустимо использовать адрес регистра или индекс массива выражение с плавающей запятой.  
  
 Эта ошибка идентична ошибке CAN0020.
