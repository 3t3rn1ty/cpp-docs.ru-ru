# <a name="linker-tools-error-lnk1282"></a>Ошибка средств компоновщика LNK1282
не удается применить параметр/REBASE файл; файл подписан  
  
 Предпринята попытка изменить базовый адрес подписанной сборки с помощью параметра/REBASE [editbin](../../build/reference/editbin-reference.md). Для этого сначала изменить базовый адрес и затем подписать сборку.
