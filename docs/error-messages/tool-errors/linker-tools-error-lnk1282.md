# Ошибка средств компоновщика LNK1282
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

не удается применить параметр \/REBASE к файлу; файл подписан  
  
 Предпринята попытка изменить базовый адрес подписанной сборки с помощью параметра \/REBASE программы [editbin](../Topic/EDITBIN%20Reference.md).  Для этого необходимо сначала изменить базовый адрес, а затем подписать сборку.
