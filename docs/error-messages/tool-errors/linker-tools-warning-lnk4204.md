# <a name="linker-tools-warning-lnk4204"></a>Предупреждение средств компоновщика LNK4204

«имя_файла» не содержит отладочной информации для ссылающегося модуля; компоновка объекта выполняется как при отсутствии отладочных данных

PDB-файл с ошибочной подписью. Компоновщик продолжит компоновку объекта без отладочной информации. Можно перекомпилировать файл объекта с помощью [/ZI](../../build/reference/z7-zi-zi-debug-information-format.md) параметр.

LNK4204 может произойти, если некоторые объекты в библиотеке обратиться к файлу, который больше не существует. Это может произойти при перестроении решения, например; объектный файл может быть удален и не может быть перестроен из-за ошибки компиляции. В этом случае либо компиляцию с параметром **/Z7**, или **/Fd**, предназначенных для обновления объектов для ссылки на один файл в библиотеке (то есть не по умолчанию имя файла PDB-файл).  Дополнительные сведения см. в разделе [/Fd (имя файла базы данных программы)](../../build/reference/fd-program-database-file-name.md).  Убедитесь, что PDB-файл сохраняется с библиотекой, каждый раз при его обновлении в системе управления версиями.
