# Предупреждение средств компоновщика LNK4204
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

для "filename" отсутствует отладочная информация для указанного модуля; компоновка объекта как при отсутствии отладочной информации  
  
 PDB\-файл содержит ошибочную подпись.  Компоновщик продолжит устанавливать связь с объектом при отсутствии отладочной информации.  Возможно, потребуется перекомпилировать файл объекта, используя параметр [\/Zi](../Topic/-Z7,%20-Zi,%20-ZI%20\(Debug%20Information%20Format\).md).  
  
 Ошибка LNK4204 может произойти, когда некоторые объекты в библиотеке ссылаются на несуществующий файл.  Это может произойти, если при перестроении решения файл объекта был удален и не перестроен из\-за ошибки компиляции.  В этом случае, либо компилируйте с помощью **\/Z7**, либо с помощью **\/Fd** для обновления объектов, чтобы они ссылались на один файл в библиотеке \(не с именем PDB\-файла по умолчанию\).  Для получения дополнительной информации см. [\/Fd \(имя файла базы данных программы\)](../../build/reference/fd-program-database-file-name.md).  Каждый раз убеждайтесь в том, что PDB\-файл, сохраненный в библиотеке, обновляется в системе управления версиями.
