# Неустранимая ошибка NMAKE U1071
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

цикл в дереве зависимостей для целевого объекта "имяобъекта"  
  
 В дереве зависимостей для указанного целевого объекта существует циклическая зависимость.  Указанный целевой объект зависит от одного из своих зависимых объектов.  Циклическая зависимость недопустима.
