# Математическая ошибка M6107
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

не поддерживается эмулятором  
  
 Предпринята попытка выполнения инструкции сопроцессора, которая является недопустимой или не поддерживается эмулятором.  
  
 Программа завершается с кодом завершения 135.
