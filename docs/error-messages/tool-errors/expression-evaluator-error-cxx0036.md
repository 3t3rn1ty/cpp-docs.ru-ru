# <a name="expression-evaluator-error-cxx0036"></a>Ошибка вычислителя выражений CXX0036
Неправильный контекст {...} specification  
  
 Это сообщение может возникать по любой из нескольких ошибок при использовании оператора контекста (**{}**).  
  
-   Синтаксис оператора контекста (**{}**) был задан неправильно.  
  
     Используется синтаксис оператора контекста:  
  
     {*функция*,*модуль*,*dll*}*выражение*  
  
     Это указывает контекст для *выражение*. Оператор контекста имеет тот же приоритет и использование как приведение типа.  
  
     В конце запятые можно опустить. Если какие-либо *функция*, *модуля*, или *dll* символом запятой, содержащий полное имя необходимо заключить в круглые скобки.  
  
-   Имя функции было задано неправильно или не существует в указанном модуле или библиотеки динамической компоновки.  
  
     Поскольку C — язык с учетом регистра, *функция* следует задавать в том же регистре, что он определен в источнике.  
  
-   Не найден модуля или библиотеки DLL.  
  
     Проверьте имя полного пути, указанного модуля или библиотеки DLL.  
  
 Эта ошибка идентична CAN0036.
