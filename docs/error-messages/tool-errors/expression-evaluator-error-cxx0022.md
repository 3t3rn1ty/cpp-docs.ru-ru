# <a name="expression-evaluator-error-cxx0022"></a>Ошибка вычислителя выражений CXX0022
в вызове функции перед _main  
  
 Вычислитель выражений C не удалось вычислить функцию, прежде чем отладчик перейдет в функции **_main**. Программа не инициализирован должным образом до **_main** был вызван.  
  
 Эта ошибка идентична ошибке CAN0022.
