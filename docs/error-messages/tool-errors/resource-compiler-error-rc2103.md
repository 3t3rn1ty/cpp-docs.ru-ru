# Ошибка компилятора ресурсов RC2103
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

непредвиденный конец файла в строковом литерале  
  
 Конец файла обнаружен перед концом строки.  В строке, вероятно, отсутствуют заключительные двойные кавычки \(**"**\).
