# <a name="linker-tools-error-lnk1240"></a>Ошибка средств компоновщика LNK1240

не удалось откомпилировать IDL-содержимое

Компоновщик порожденных MIDL для компиляции внедренные IDL, но возникла проблема. См. ошибки компилятора MIDL.
