# Ошибка средств компоновщика LNK1240
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

не удалось откомпилировать IDL\-содержимое  
  
 Ошибка при компиляции встроенного содержимого IDL с помощью компилятора MIDL, вызванного компоновщиком.  Проверьте ошибки компилятора MIDL.
