# Неустранимая ошибка компилятора ресурсов RC1017
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

недопустимое константное выражение целого типа  
  
 Выражение в директиве `#if` не существует, либо не является константным.
