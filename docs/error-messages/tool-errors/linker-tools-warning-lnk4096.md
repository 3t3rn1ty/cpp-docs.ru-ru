# <a name="linker-tools-warning-lnk4096"></a>Предупреждение средств компоновщика LNK4096
/ BASE значение «number» является недопустимым для Windows 95 и Windows 98; образ нельзя запустить  
  
 Указанный базовый адрес является недопустимым. Исполняемые файлы Windows 95 и Windows 98 должен иметь больше 0x400000 базовый адрес. Дополнительные сведения о базовых адресов в разделе [/BASE](../../build/reference/base-base-address.md) компоновщика.
