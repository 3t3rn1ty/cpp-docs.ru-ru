# Ошибка во время выполнения C R6027
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

недостаточно памяти для инициализации lowio  
  
 Эта ошибка возникает, если для инициализации поддержки низкоуровневого ввода\-вывода не хватает памяти.
