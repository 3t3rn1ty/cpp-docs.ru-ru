# Ошибка построения проекта PRJ0014
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Объект\-задание, используемый для контроля созданных процессов, не был выполнен.  Построение нельзя продолжить.  
  
 Ошибка среды разработки.  
  
 Чтобы устранить эту ошибку, закройте и повторно загрузите проект.  При необходимости завершите работу среды разработки Visual Studio и запустите ее повторно.
