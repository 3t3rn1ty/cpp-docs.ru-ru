# <a name="project-build-error-prj0007"></a>Ошибка построения проекта PRJ0007
Не удалось создать выходной каталог «каталог».  
  
 Visual C++ не удалось создать выходной каталог. Возможные причины.  
  
-   Каталог, в котором был быть создан в выходной каталог доступен только для чтения.  
  
-   Недостаточно места на диске.  
  
-   Указан недопустимый путь к каталогу.  
  
-   Пользователю разрешений недостаточно для создания каталога.
