# Ошибка построения проекта PRJ0007
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Не удалось создать выходной каталог "каталог".  
  
 Попытка Visual C\+\+ создать каталог завершилась неудачей.  Ниже перечислены возможные причины.  
  
-   Директория, в которой должен был быть создан каталог, доступна только для чтения.  
  
-   Недостаточно дискового пространства.  
  
-   Указан недопустимый путь к каталогу.  
  
-   Права пользователя недостаточны для создания каталога.
