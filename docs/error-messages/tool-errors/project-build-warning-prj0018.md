# <a name="project-build-warning-prj0018"></a>Предупреждение при построении проекта PRJ0018

Не найдены следующие переменные среды:

Переменная среды не определен. Эта ошибка список переменных среды, которые не были определены.

См. в разделе [макросы для команд сборки Custom](../../ide/common-macros-for-build-commands-and-properties.md) сведения о макросах в процессе построения.
