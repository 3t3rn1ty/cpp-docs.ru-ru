# <a name="project-build-warning-prj0018"></a>Предупреждение при построении проекта PRJ0018
Не найдены следующие переменные среды:  
  
 Переменная среды не определен. Ошибки перечислены переменные среды, которые не были определены.  
  
 В разделе [макросы для команд построения Custom](../../ide/common-macros-for-build-commands-and-properties.md) сведения о макросах в процессе построения.
