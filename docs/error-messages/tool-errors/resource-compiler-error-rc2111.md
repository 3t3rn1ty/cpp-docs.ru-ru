# <a name="resource-compiler-error-rc2111"></a>Ошибка компилятора ресурсов RC2111
Недопустимый тип элемента управления  
  
 Каждой инструкции УПРАВЛЕНИЯ **диалоговое ОКНО** инструкция должна быть одно из следующих: 3STATE AUTO3, автоматическая проверка, AUTORADIO, BEDIT, ФЛАЖОК, поле со СПИСКОМ, элемент УПРАВЛЕНИЯ, CTEXT, DEFPUSHBUTTON, EDITTEXT, GROUPBOX, HEDIT, значок, IEDIT, LISTBOX, LTEXT, PUSHBOX, КНОПКИ, RADIOBUTTON, RTEXT, ПОЛОСЫ ПРОКРУТКИ, USERBUTTON.  
  
 Убедитесь, что эти операторы УПРАВЛЕНИЯ написаны правильно.
