# Неустранимая ошибка NMAKE U1035
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ошибка синтаксиса: ожидается разделитель ":" или "\="  
  
 Ожидается либо двоеточие \(**:**\), либо знак равенства \(**\=**\).  
  
### Чтобы устранить ошибку, следует проверить следующие возможные причины ее возникновения.  
  
1.  После целевого буквенного символа не поставлено двоеточие.  
  
2.  После  одиночного целевого буквенного символа поставлено двоеточие без пробела \(например "a:"\).  NMAKE интерпретирует это как обозначение диска.  
  
3.  После правила определения не поставлено двоеточие.  
  
4.  После определения макроса не поставлен знак равенства.  
  
5.  Символ поставлен после обратной косой черты \(**\\**\), которая использовалась для продолжения команды в новой строке.  
  
6.  Отобразилась строка, которая не соответствует правилам синтаксиcа NMAKE.  
  
7.  Makefile был отформатирован текстовым процессором.
