# Математическая ошибка M6102
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

denormal  
  
 Операция генерировала очень маленькое число с плавающей запятой, которое недопустимо по причине незначительности.  Ненормальные исключения с плавающей запятой обычно маскируются, что приводит к их перехвату и обработке.  
  
 Программа завершается с кодом завершения 130.
