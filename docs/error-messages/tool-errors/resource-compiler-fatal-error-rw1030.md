# Неустранимая ошибка компилятора ресурсов RW1030
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ошибка вывода  
  
 Причиной этой ошибки может быть то, что компилятору ресурсов не удалось создать конечный файл.  Убедитесь в том, что на диске достаточно свободного места и имеются права записи в данном объеме.
