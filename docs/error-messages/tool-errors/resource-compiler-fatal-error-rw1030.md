# <a name="resource-compiler-fatal-error-rw1030"></a>Неустранимая ошибка компилятора ресурсов RW1030
Ошибка вывода  
  
 Эта ошибка может возникнуть, если компилятору ресурсов не удалось создать конечный файл. Убедитесь, что имеется достаточно места и что у вас есть права записи в томе.
