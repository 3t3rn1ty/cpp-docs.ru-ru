# Ошибка вычислителя выражений CXX0034
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**типы несовместимы с оператором**  
  
 Указанные типы операндов недопустимы для данной операции.  
  
 Например, указатель нельзя умножить на какое\-либо значение.  
  
 Может потребоваться выполнить приведение типов операндов к типу, совместимому с оператором.
