# <a name="expression-evaluator-error-cxx0034"></a>Ошибка вычислителя выражений CXX0034
**типы, совместимые с оператором**  
  
 Указанные типы операндов не допустимы для данной операции.  
  
 Например указатель не будет умножено на любое значение.  
  
 Может потребоваться ввод операнды к типу, совместимому с помощью оператора приведения.
