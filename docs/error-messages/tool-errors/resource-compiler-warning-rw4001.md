# Предупреждение компилятора ресурсов RW4001
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Параметры обработки EXE \(\/L \/M \/P \/T \/K \/E \/31 или \/30\)  
  
 Заданы параметры обработки EXE, однако отсутствует соответствующий исполняемый файл.  Не используйте эти параметры для работы с RES\-файлами.
