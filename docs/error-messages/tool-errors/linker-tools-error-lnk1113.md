# Ошибка средств компоновщика LNK1113
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

недопустимый тип компьютера "тип"  
  
 Тип компьютера, указанный в заголовке объектного файла, недопустим.  
  
 Эта ошибка может возникнуть, если файл поврежден.  Выполните повторное построение объекта.
