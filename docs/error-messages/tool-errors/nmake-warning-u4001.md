# <a name="nmake-warning-u4001"></a>Предупреждение программы NMAKE U4001
командный файл может быть вызвана только из командной строки  
  
 Командный файл, который может быть вызван с символа (**@**) описатель, не может содержать спецификацию на другой командный файл. Вложения не допускается. Спецификации был пропущен.
