# Предупреждение программы NMAKE U4001
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

командный файл может вызываться только из командной строки  
  
 Командный файл, вызванный знаком "**@**", не может содержать указание на другой командный файл.  Такое вложение недопустимо.  Вложенный командный файл игнорируется.
