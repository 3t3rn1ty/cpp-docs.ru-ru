# Ошибка средств компоновщика LNK1120
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

число неразрешенных внешних элементов  
  
 Ошибка LNK1120 представляет собой подсчет \(`number`\) неразрешенных внешних элементов для данной компоновки.  Условия, которые вызвали появление неразрешенных внешних элементов, описаны в ошибке [LNK2001](../Topic/Linker%20Tools%20Error%20LNK2001.md), которая предшествует данному сообщению об ошибке \(одно сообщение для каждого неразрешенного элемента\).
