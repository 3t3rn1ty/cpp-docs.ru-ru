# Ошибка средств компоновщика LNK1248
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

размер образа \(размер\) превышает максимально допустимый размер \(80000000\)  
  
 Компоновщик определил, что размер выходного файла будет превышать максимально возможный размер для программного образа.  Возможно, необходимо сделать программу с использованием нескольких библиотек DLL.  Дополнительные сведения о программе РЕ\) \(PE\), см. в статье «всматривается внутри PE\-ФАЙЛА: Учебник формата переносимый исполняемый Win32» в [Библиотека MSDN](http://go.microsoft.com/fwlink/?linkid=556).
