# <a name="nmake-fatal-error-u1095"></a>Неустранимая ошибка NMAKE U1095

После подстановки Командная строка «командная_строка» слишком длинное

После расширения макроса в данной командной строки превышает ограничение на длину командной строки для операционной системы.

В командной строке в MS-DOS разрешает до 128 символов.

Если команда выполняется для программы, может принимать входные данные командной строки из файла, изменить команду и укажите входное из файла на диске или встроенного файла. Например ссылки и LIB принимать входные данные из файла ответов.
