# Ошибка построения проекта PRJ0021
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Инструмент 'tool', свойство 'property' содержат недопустимое имя файла..  
  
 Имя файла, указанное в свойстве ***property*** для инструмента ***tool***, было некорректным.  
  
 Возможно, был использован неизвестный или недействительный макрос.  
  
 Сведения о доступе к свойствам проекта см. в разделе [Настройка свойств проекта Visual C\+\+](../../ide/working-with-project-properties.md).
