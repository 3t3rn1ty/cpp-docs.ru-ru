# Предупреждение при построении проекта PRJ0042
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**Не установлено свойство "Выходные данные" настраиваемого этапа построения для файла "**  ***файл***  **". Настраиваемый этап построения будет пропущен.**  
  
 Настраиваемый этап построения был пропущен, поскольку не были указаны выходные данные.  
  
 Чтобы устранить данную ошибку, выполните одно из следующих действий:  
  
-   Исключите настраиваемый этап из построения.  
  
-   Добавьте выходные данные.  
  
-   Удалите содержимое команды настраиваемого этапа построения.
