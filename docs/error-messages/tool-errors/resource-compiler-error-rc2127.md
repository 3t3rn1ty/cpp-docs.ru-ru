# Ошибка компилятора ресурсов RC2127
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

предполагается использование слов в формате WORD различных версий, разделенных запятыми  
  
 Номера версий в ресурсе версий должны быть в формате **WORD** и разделены запятыми.
