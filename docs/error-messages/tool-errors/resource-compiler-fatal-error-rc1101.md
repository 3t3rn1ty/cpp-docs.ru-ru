# Неустранимая ошибка компилятора ресурсов RC1101
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

не указано имя двоичного файла ресурса  
  
 В параметре для переименования выходного файла \(\/fo\) не указано имя файла.  
  
 Для параметра \/fo следует использовать следующий синтаксис:  
  
 **RC \/fo**   
 ***outfilename infilename.* rc**
