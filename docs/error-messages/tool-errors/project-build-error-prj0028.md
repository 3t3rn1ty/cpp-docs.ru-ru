# <a name="project-build-error-prj0028"></a>Ошибка построения проекта PRJ0028
Временный файл «файл», содержащий знаки Юникода, не может быть преобразована в кодовую страницу ANSI пользователя.  
  
 Указанное значение [/MIDL (указание параметров командной строки MIDL)](../../build/reference/midl-specify-midl-command-line-options.md) параметра компоновщика, который не удалось разрешить с помощью кодовой страницы системы.  
  
 Кодовая страница, используемая при указании команды MIDL (кодовая страница ввода) должен быть таким же, как в системную кодовую страницу.
