# Ошибка вычислителя выражений CXX0038
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

несовпадение числа и \(или\) типа аргументов функции  
  
 Вызов указанной функции не соответствует прототипу этой функции.  
  
 Перепишите вызов, указав правильное количество аргументов.  При необходимости приведите тип каждого аргумента в соответствие с прототипом.  
  
 Эта ошибка идентична ошибке CAN0038.
