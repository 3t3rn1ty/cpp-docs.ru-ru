# <a name="expression-evaluator-error-cxx0038"></a>Ошибка вычислителя выражений CXX0038
Несоответствие числа и тип аргумента функции  
  
 Вызов указанной функции не соответствует прототип функции.  
  
 Перепишите вызов, указав правильное число аргументов. Приведите тип каждого аргумента в соответствие с прототипом, при необходимости.  
  
 Эта ошибка идентична ошибке CAN0038.
