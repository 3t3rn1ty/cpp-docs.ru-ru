# <a name="resource-compiler-error-rc2169"></a>Ошибка компилятора ресурсов RC2169
Имя файла ресурсов не соответствует формату 2.03  
  
 Указанного ресурса используется более ранний формат версии 2.03. Файл ресурса необходимо преобразовать или создать заново в формате версии 3.00 или более поздней версии.
