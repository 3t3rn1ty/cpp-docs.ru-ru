# Ошибка вычислителя выражений CXX0009
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

отсутствует одинарная кавычка \('\) после символьной константы  
  
 Одиночная кавычка, ожидаемая в конце символьной константы, была утеряна.  Перепишите выражение, заключив символьную константу в одинарные кавычки.  
  
 Эта ошибка идентична ошибке CAN0009.
