# Ошибка средств компоновщика LNK1141
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

сбой при построении файла экспорта  
  
 Программе [LINK](../../build/reference/linker-command-line-syntax.md) не удается построить файл экспорта с расширением EXP.  
  
 Обычно эта ошибка связана с проблемами в системе, в том числе с нехваткой памяти и синтаксическими ошибками в операторах параметров или определений модуля.  
  
 Перед этой ошибкой возникает другая ошибка, содержащая дополнительные сведения.
