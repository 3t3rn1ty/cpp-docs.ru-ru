# Ошибка компилятора ресурсов RC2113
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

в диалоговом окне ожидается ключевое слово END  
  
 В конце оператора **DIALOG** должно стоять ключевое слово **END**.  Убедитесь, что в предыдущем операторе не осталось незакрытых скобок.
