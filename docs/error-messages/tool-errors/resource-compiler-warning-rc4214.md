# Предупреждение компилятора ресурсов RC4214
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Недопустимая кодовая страница: пропуск  
  
 В RC\-файле содержится аргумент кодовой страницы. При этом указана недопустимая кодовая страница.  Дополнительные сведения см. в описании функции [IsValidCodePage](http://msdn.microsoft.com/library/windows/desktop/dd318674).
