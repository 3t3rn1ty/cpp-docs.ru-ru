# <a name="linker-tools-error-lnk1169"></a>Ошибка средств компоновщика LNK1169
обнаружен многократно определенный символ - один или более  
  
 Построение завершилось неудачей из-за несколько определений одного или нескольких символов. Перед этой ошибкой возникает ошибка [LNK2005](../../error-messages/tool-errors/linker-tools-error-lnk2005.md).  
  
 [/FORCE](../../build/reference/force-force-file-output.md) или **/FORCE: Multiple** позволяют переопределить ошибку.
