# Неустранимая ошибка компилятора ресурсов RC1109
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ошибка при создании "имя ресурса"  
  
 Не удается создать указанный RES\-файл.  Убедитесь, что диск, на котором он создается, не защищен от записи.  Воспользуйтесь параметром \/V, чтобы проверить, создается ли файл.
