# <a name="project-build-error-prj0020"></a>Ошибка построения проекта PRJ0020
Инструмент «инструмент», свойство «свойство» содержит неверное имя файла «файл».  
  
 Имя файла ***файл***, указанного в свойстве ***свойство***, средства, ***средство***, является недопустимым.  
  
 Используется неизвестный или недопустимый макрос.  
  
 В разделе [задание свойств проекта Visual C++](../../ide/working-with-project-properties.md) сведения о том, как получить доступ к свойствам проекта.
