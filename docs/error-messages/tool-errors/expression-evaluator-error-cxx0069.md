# <a name="expression-evaluator-error-cxx0069"></a>Ошибка вычислителя выражений CXX0069
переменной требуется фрейм стека  
  
 Средство оценки выражений не удается вычислить переменной, так как она отсутствует в кадре стека. Это может быть вызвано переменные, объявленные как часть встроенной функции.
