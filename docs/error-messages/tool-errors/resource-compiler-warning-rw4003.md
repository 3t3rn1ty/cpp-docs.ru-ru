# Предупреждение компилятора ресурсов RW4003
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

SHIFT или CONTROL были использованы без VIRTKEY  
  
 В ресурсе таблицы сочетания клавиш для использования клавиш SHIFT или CONTROL требуется VIRTKEY.  Поскольку SHIFT и CONTROL обозначены как флаговые биты в сочетании клавиш VIRTKEY, они не могут существовать независимо от VIRTKEY.
