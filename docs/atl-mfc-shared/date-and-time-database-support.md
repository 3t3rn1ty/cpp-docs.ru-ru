# <a name="date-and-time-database-support"></a>Дата и время: поддержка базы данных

Начиная с версии 4.0, MFC database программирования использует [COleDateTime](../atl-mfc-shared/reference/coledatetime-class.md) и [COleDateTimeSpan](../atl-mfc-shared/reference/coledatetimespan-class.md) классы, представляющие данные даты и времени. Эти классы, также используется в службе автоматизации, являются производными от класса [COleVariant](../mfc/reference/colevariant-class.md). Оно содержит улучшенную поддержку управления данные даты и времени, чем у [CTime](../atl-mfc-shared/reference/ctime-class.md) и [CTimeSpan](../atl-mfc-shared/reference/ctimespan-class.md).
