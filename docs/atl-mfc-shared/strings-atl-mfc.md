# Strings (ATL/MFC)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Практически всю работу программы с данными строки.  Visual C\+\+ предоставляет несколько способов управления этими данные строк.  
  
## В этом подразделе  
 [Using CStringT](../atl-mfc-shared/using-cstringt.md)  
 Описывает программирование с помощью класса CStringT шаблона.  
  
 [Using CString](../atl-mfc-shared/using-cstring.md)  
 Описывает программирование с использованием CString, реализацию по умолчанию для CStringT.  
  
## Связанные подразделы  
 [Общие сведения о CStringT](../atl-mfc-shared/reference/cstringt-class.md)  
 Содержит справочные сведения об общем классе `CStringT`.  
  
 [Основные понятия MFC](../mfc/mfc-concepts.md)  
 Предоставляет и задача\-, основанные на концептуальные разделы помогут программирование с использованием библиотеки MFC.
