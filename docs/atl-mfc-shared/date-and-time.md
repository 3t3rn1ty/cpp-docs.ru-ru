# <a name="date-and-time"></a>Дата и время
MFC поддерживает несколько различных способов работы с датами и временем. Сюда входит следующее.  
  
-   Классы общего назначения времени. [CTime](../atl-mfc-shared/reference/ctime-class.md) и [CTimeSpan](../atl-mfc-shared/reference/ctimespan-class.md) классы инкапсулируют большая часть функций, связанных с библиотекой времени стандарт ANSI, который объявляется в времени. З.  
  
-   Поддержка системных часов. С MFC версии 3.0 добавлена поддержка `CTime` для Win32 `SYSTEMTIME` и `FILETIME` типов данных.  
  
-   Поддержка автоматизации [тип данных DATE](../atl-mfc-shared/date-type.md). **Дата** поддерживает даты, времени и даты и времени значения. [COleDateTime](../atl-mfc-shared/reference/coledatetime-class.md) и [COleDateTimeSpan](../atl-mfc-shared/reference/coledatetimespan-class.md) классы инкапсулируют эту функциональность. Они работают с [COleVariant](../mfc/reference/colevariant-class.md) класса с помощью поддержки автоматизации.
