# <a name="control-containment-classes"></a>Классы элементов управления вложения
Следующие классы обеспечивают вложения для размещения элементов управления.  
  
-   [CAxWindow](../atl/reference/caxwindow-class.md) предоставляет методы для работы с окном, на котором размещается элемент управления ActiveX.  
  
-   [CAxWindow2T](../atl/reference/caxwindow2t-class.md) содержит методы для работы окно, которое размещает элемент ActiveX, а также поддержка размещения Лицензированные элементы управления ActiveX.  
  
-   [IAxWinAmbientDispatch](../atl/reference/iaxwinambientdispatch-interface.md) вызывать методы для этого интерфейса для задания свойства окружения для размещенного элемента управления.  
  
-   [IAxWinHostWindow](../atl/reference/iaxwinhostwindow-interface.md) вызывать методы для этого интерфейса на создание или присоединить элемент управления к объекту узла или для получения интерфейса из размещенного элемента управления.
