# Thread Pooling Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Группировка потоков поддержки следующих классов:  
  
-   [CComAutoThreadModule](../atl/reference/ccomautothreadmodule-class.md) реализующий модуль EXE, с поддержкой квартир поток\- вместе скложенных кратен.  
  
-   [CComApartment](../atl/reference/ccomapartment-class.md) управляет подразделение в объединенный в поток исполняемый модуль.  
  
-   Поток управления [CComSimpleThreadAllocator](../atl/reference/ccomsimplethreadallocator-class.md) выделение для расширения EXE\-ФАЙЛА.
