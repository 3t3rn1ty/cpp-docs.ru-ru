# Using Task Manager
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Один из самых простых способов отладки службы с помощью диспетчера задач Windows NT 4.0 или Windows 2000.  Пока служба запущена, запустите диспетчер задач и перейдите на вкладку **Процессы**.  Щелкните правой кнопкой мыши имя исполняемого файла и нажмите кнопку **Отладка**.  Запустится Visual C\+\+ вложенный к этому выполняемый процесс.  Теперь, нажмите кнопку **Прервать** в меню **Отладка** разрешить можно установить точки останова в коде.  Щелкните **Запуск** для работы с выбранным точки останова.
