# <a name="interface-pointers-classes"></a>Указатели классов интерфейсов
Следующие классы управление указатель заданного интерфейса.  
  
-   [CComPtr](../atl/reference/ccomptr-class.md) выполняет подсчет автоматических ссылок.  
  
-   [CComQIPtr](../atl/reference/ccomqiptr-class.md) аналогичен `CComPtr`, а также выполняет автоматический запрос интерфейсов.  
  
-   [CInterfaceArray](../atl/reference/cinterfacearray-class.md) указателей на интерфейс предоставляет методы, которые полезны при создании массива COM.  
  
-   [CInterfaceList](../atl/reference/cinterfacelist-class.md) указателей на интерфейс предоставляет методы, которые полезны при построении список COM.  
  
-   [CComGITPtr](../atl/reference/ccomgitptr-class.md) предоставляет методы для работы с указателей интерфейса и глобальной таблицы интерфейсов (GIT).
