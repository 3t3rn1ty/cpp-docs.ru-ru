# Interface Pointers Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Следующие классы управляют заданный указатель интерфейса:  
  
-   [CComPtr](../atl/reference/ccomptr-class.md) выполняет автоматического подсчета ссылок.  
  
-   [CComQIPtr](../atl/reference/ccomqiptr-class.md) аналогичное `CComPtr`, но также выполняет автоматическое выполнение запросов интерфейсов.  
  
-   [CInterfaceArray](../atl/reference/cinterfacearray-class.md) предоставляет методы полезны при создании массива указателей интерфейса модели COM.  
  
-   [CInterfaceList](../Topic/CInterfaceList%20Class.md) предоставляет методы полезных для построения списка указателей интерфейса модели COM.  
  
-   [CComGITPtr](../Topic/CComGITPtr%20Class.md) предоставляет методы для ведения дела с указателями интерфейса и глобальный таблицей интерфейса \(GIT\).
