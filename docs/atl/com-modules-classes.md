# COM Modules Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Следующие классы предоставляют поддержку для модуля модели COM.  
  
-   [CAtlBaseModule](../Topic/CAtlBaseModule%20Class.md) этот класс создается в каждом проекте библиотеки ATL.  
  
-   [CAtlComModule](../atl/reference/catlcommodule-class.md) этот класс реализует модуль сервера модели COM.  
  
-   [CAtlModule](../atl/reference/catlmodule-class.md) этот класс предоставляет методы, используемые несколькими классами модуля библиотеки ATL.  
  
-   [CAtlModuleT](../atl/reference/catlmodulet-class.md) этот класс реализует модуль библиотеки ATL.  
  
-   [CAtlExeModuleT](../atl/reference/catlexemodulet-class.md) этот класс представляет модуль для приложения.  
  
-   [CAtlServiceModuleT](../Topic/CAtlServiceModuleT%20Class.md) этот класс реализует службу.  
  
-   [CAtlWinModule](../atl/reference/catlwinmodule-class.md) этот класс обеспечивает поддержку компонентов над окнами библиотеки ATL.  
  
-   [CComModule](../atl/reference/ccommodule-class.md) этот класс реализует модуль DLL или EXE.  Устарел в библиотеке ATL 7,0.  
  
-   [CComAutoThreadModule](../atl/reference/ccomautothreadmodule-class.md) этот класс реализует модуль EXE, с поддержкой квартир поток\- вместе скложенных кратен.  Устарел в библиотеке ATL 7,0.
