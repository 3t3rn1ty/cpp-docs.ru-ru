# Site Information Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Следующие классы позволяют объект для взаимодействия с сайтом:  
  
-   Возвращает и задает [IObjectWithSiteImpl](../atl/reference/iobjectwithsiteimpl-class.md) указатель к сайту объекта.  Общий для объектов, которые не являются элементов управления.  
  
-   Возвращает и задает [IOleObjectImpl](../atl/reference/ioleobjectimpl-class.md) указатель к сайту объекта.  Общий для элементов управления.
