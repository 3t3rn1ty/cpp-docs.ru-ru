# Persistence Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Следующая сохраняемость реализации классов объектов.  
  
-   [IPersistPropertyBagImpl](../atl/reference/ipersistpropertybagimpl-class.md) позволяет клиенту для загрузки и сохранения свойств объекта в контейнер свойств.  
  
-   [IPersistStreamInitImpl](../atl/reference/ipersiststreaminitimpl-class.md) позволяет клиенту загружать и сохранять данные объекта постоянные в поток.  
  
-   [IPersistStorageImpl](../atl/reference/ipersiststorageimpl-class.md) позволяет клиенту загружать и сохранять данные объекта постоянные в хранилище.
