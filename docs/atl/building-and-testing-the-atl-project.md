# Building and Testing the ATL Project
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Как упоминалось в [Вставка составной элемент управления](../atl/inserting-a-composite-control.md), один из исходных компонентов проекта по умолчанию страницы HTML, которая размещение новый составной элемент управления.  После завершения изменения составного элемента управления, щелкните **Построить решение** или **Перестроить решение** из меню **Построение**.  После успешного построения проекта, загружают html\-страницу, расположенную в корневом каталоге проекта составного элемента управления в Internet Explorer или другой браузер и проверяющие функциональность элемента управления.  
  
 Можно также тестирование составного элемента управления с помощью средства тестового контейнера или любое другое приложение, которое может размещение элементов управления ActiveX.  Сведения о доступе к тестовому контейнеру см. в разделе [Тестирование свойств и событий при помощи тестового контейнера](../mfc/testing-properties-and-events-with-test-container.md).
