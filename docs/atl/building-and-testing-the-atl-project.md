# <a name="building-and-testing-the-atl-project"></a>Построение и тестирование проекта ATL

Как упоминалось в [Вставка составного элемента управления](../atl/inserting-a-composite-control.md), основные компоненты проекта, он на HTML-страницу по умолчанию, на котором размещается новый составной элемент управления. После завершения изменения составного элемента управления, нажмите кнопку **построить решение** или **Перестроить решение** из **построения** меню. После успешного построения проекта, загрузки страницы HTML, расположенный в корневом каталоге проекта составного элемента управления, в Internet Explorer или другой браузер и проверить функциональные возможности элемента управления.

Также можно проверить в составной элемент управления, используя средство контейнере для тестирования или любого другого приложения, который может содержать элемент управления ActiveX. Сведения о том, как получить доступ к Контейнеру для тестирования, см. в разделе [Тестирование свойств и событий в Контейнере для тестирования](../mfc/testing-properties-and-events-with-test-container.md) .
