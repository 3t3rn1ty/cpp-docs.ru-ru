# <a name="active-template-library-atl-tutorial"></a>Учебник по активной библиотеке шаблонных классов (ATL)
ATL предназначен для упрощения процесса создания эффективных, гибкий и упрощенные элементы управления. Этот учебник описывает создания элементов управления ActiveX с демонстрацией многих основы ATL и COM.  
  
 Выполнив этот учебник, вы узнаете, как добавить элемент управления в проект ATL, который рисует окружность и закрашенного многоугольника. Затем добавьте свойство, указывающее число сторон многоугольника будет и создайте код прорисовки для обновления элемента управления при изменении значения свойства. Элемент управления будет отображаться на веб-странице, с помощью некоторых VBScript, чтобы сделать его в ответ на события.  
  
 Учебник разделен на семь шагов. Каждый шаг необходимо выполнить в порядке, как остальные шаги зависят от ранее выполненные задачи. Прежде чем начать, необходимо убедиться, что вы имеете права, необходимые для регистрации компонента ActiveX на конкретном компьютере. Эта проблема обычно только при запуске Visual Studio .NET через подключение служб терминалов.
