# Tear-Off Interfaces Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

-   Средства **IUnknown**[CComTearOffObject](../atl/reference/ccomtearoffobject-class.md), запрещены интерфейса.  
  
-   Средства **IUnknown**[CComCachedTearOffObject](../atl/reference/ccomcachedtearoffobject-class.md) для "кэшированного" запрещены интерфейса.
