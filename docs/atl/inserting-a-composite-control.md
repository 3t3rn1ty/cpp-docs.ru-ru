# Inserting a Composite Control
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Диалоговое окно **Добавление класса** позволяет вставить объект библиотеки ATL в проект.  Чтобы открыть это диалоговое окно, щелкните правой кнопкой мыши имя проекта в обозревателе решений, указать **Добавить**, а затем щелкните **Добавление класса**.  
  
 В диалоговом окне **Добавление класса** выберите **Элемент управления ATL**.  Запустится [Мастер элементов управления библиотеки ATL](../atl/reference/atl-control-wizard.md).  Чтобы создать составной элемент управления, выберите вкладку **Параметры** и установите флажок **Составной элемент управления**.  
  
 По умолчанию страница HTML будет создана для просмотра элемента управления.
