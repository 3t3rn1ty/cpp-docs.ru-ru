# <a name="inserting-a-composite-control"></a>Вставка составного элемента управления
**Добавить класс** служит диалоговое окно для вставки объекта ATL в проект. Доступ к данным диалоговым окном, щелкнув правой кнопкой мыши имя проекта в обозревателе решений, указывающие на **добавить**и выбрав **Добавление класса**.  
  
 В **добавить класс** диалогового окна выберите **управления ATL**. Это приведет к запуску [мастер элементов управления ATL](../atl/reference/atl-control-wizard.md). Для создания составного элемента управления, выберите **параметры** и нажмите кнопку **составного элемента управления** флажок.  
  
 HTML-страницей по умолчанию создается для просмотра элемента управления.
