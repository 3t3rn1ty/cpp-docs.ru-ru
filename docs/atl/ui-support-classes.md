# <a name="ui-support-classes"></a>Классы поддержки пользовательского интерфейса
Следующие классы обеспечивают общий пользовательский Интерфейс.  
  
-   [IDocHostUIHandlerDispatch](../atl/reference/idochostuihandlerdispatch-interface.md) интерфейс синтаксического анализа Microsoft HTML и механизм визуализации.  
  
-   [IOleObjectImpl](../atl/reference/ioleobjectimpl-class.md) предоставляет основные методы, через которые контейнер связывается с элементом управления. Управляет активацией и деактивацией элементов управления на месте.  
  
-   [IOleInPlaceObjectWindowlessImpl](../atl/reference/ioleinplaceobjectwindowlessimpl-class.md) управляет повторной активации элементов управления на месте. Позволяет элементу управления без окон для получения сообщений, а также участвовать в операции перетаскивания и вставки.  
  
-   [IOleInPlaceActiveObjectImpl](../atl/reference/ioleinplaceactiveobjectimpl-class.md) помогает взаимодействие элемента управления на месте и его контейнера.  
  
-   [IViewObjectExImpl](../atl/reference/iviewobjecteximpl-class.md) включает элемент управления для отображения окна напрямую и известить контейнера изменения в его отображении. Обеспечивает поддержку мерцания Рисование непрямоугольные и прозрачные элементы управления и проверки нажатия.
