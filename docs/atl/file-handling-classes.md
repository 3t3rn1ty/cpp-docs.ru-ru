# File Handling Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Эти классы предоставляют методы для обработки файлов, временные файлы, и сопоставленные в памяти файлы.  
  
-   [CAtlFile](../atl/reference/catlfile-class.md) этот класс содержит программу\-оболочку вокруг обработки файл\- тонкая Windows API.  
  
-   [CAtlFileMapping](../Topic/CAtlFileMapping%20Class.md) этот класс представляет размещенный в памяти файл, добавляя оператор приведения к методам [CAtlFileMappingBase](../atl/reference/catlfilemappingbase-class.md).  
  
-   [CAtlFileMappingBase](../atl/reference/catlfilemappingbase-class.md) этот класс представляет размещенный в памяти файл.  
  
-   [CAtlTemporaryFile](../atl/reference/catltemporaryfile-class.md) этот класс содержит методы для создания и использования временного файла.
