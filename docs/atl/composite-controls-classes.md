# Composite Controls Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Следующий класс обеспечивает поддержку создание составных элементов управления  
  
-   Элементы управления ActiveX [CComCompositeControl](../atl/reference/ccomcompositecontrol-class.md), производные от `CComCompositeControl` хозяйничаются стандартным диалоговым окном.  Эти типы элементов управления называются составными элементами управления, так как они смогут размещения других элементов управления \(элементов управления Windows собственного и управления ActiveX\).
