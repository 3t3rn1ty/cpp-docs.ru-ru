# <a name="string-and-text-classes"></a>Строки и Text
Эти классы обеспечивают поддержку для строк и преобразования строк текста.  
  
-   [CA2AEX](../atl/reference/ca2aex-class.md) этот класс используется макросы преобразования строк `CA2TEX` и `CT2AEX`и определение типа **CA2A**.  
  
-   [CA2CAEX](../atl/reference/ca2caex-class.md) этот класс используется макросы преобразования строк `CA2CTEX` и `CT2CAEX`и определение типа **CA2CA**.  
  
-   [CA2WEX](../atl/reference/ca2wex-class.md) этот класс используется макросы преобразования строк `CA2TEX`, `CA2CTEX`, `CT2WEX`, и `CT2CWEX`и определение типа **CA2W**.  
  
-   [CW2AEX](../atl/reference/cw2aex-class.md) этот класс используется макросы преобразования строк `CT2AEX`, `CW2TEX`, `CW2CTEX`, и `CT2CAEX`и определение типа **CW2A**.  
  
-   [CW2CWEX](../atl/reference/cw2cwex-class.md) этот класс используется макросы преобразования строк `CW2CTEX` и `CT2CWEX`и определение типа **CW2CW**.  
  
-   [CW2WEX](../atl/reference/cw2wex-class.md) этот класс используется макросы преобразования строк `CW2TEX` и `CT2WEX`и определение типа `CW2W`.  
  
-   [CComBSTR](../atl/reference/ccombstr-class.md) этот класс является оболочкой для `BSTR`s.  
  
-   [_U_STRINGorID](../atl/reference/u-stringorid-class.md) класс адаптера этот аргумент позволяет либо имена ресурсов (`LPCTSTR`s) или идентификаторы ресурсов (**UINT**s) должны быть переданы функции без необходимости вызывающий объект, преобразуемый в строку, используя идентификатор **MAKEINTRESOURCE** макрос.
