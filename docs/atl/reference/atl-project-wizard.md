# <a name="atl-project-wizard"></a>Мастер проектов ATL
Active Template Library (ATL) — это набор основанных на шаблонах классов C++, упрощающих написание компактных и быстрых COM-объектов. Мастер проектов ATL создает проект, содержащий структуры для размещения объектов COM..  
  
## <a name="overview"></a>Обзор  
 На этой странице мастера описываются текущего [параметры приложения для проекта ATL](../../atl/reference/application-settings-atl-project-wizard.md) вы создаете. По умолчанию проект имеет следующие параметры:  
  
-   Указывает, что сервер имеет библиотеки DLL и, следовательно, внутрипроцессным сервером библиотеки динамической компоновки.  
  
-   Атрибут указывает, что в проекте используются атрибуты.  
  
 Чтобы изменить эти значения по умолчанию, нажмите кнопку **параметры приложения** в левом столбце мастера и внести изменения на этой странице ATL Project Wizard.  
  
 Сведения о параметрах проекта по умолчанию, включая выбор кодировки и параметры компоновки по умолчанию, в разделе [конфигурации проекта ATL по умолчанию](../../atl/reference/default-atl-project-configurations.md).  
  
 После создания проекта ATL, можно добавить объекты и элементы управления в проект Visual C++ с помощью [мастеров кода](../../ide/adding-functionality-with-code-wizards-cpp.md). Можно производить следующие типы расширений для базового проекта ATL с помощью мастеров кода:  
  
-   [Добавление объектов и элементов управления в проект ATL](../../atl/reference/adding-objects-and-controls-to-an-atl-project.md)  
  
-   [Добавление нового интерфейса в проект ATL](../../atl/reference/adding-a-new-interface-in-an-atl-project.md)  
  
-   [Добавить компонент COM + 1.0 в проект ATL](../../atl/reference/adding-an-atl-com-plus-1-0-component.md)  
  
 Кроме того примите во внимание эти задачи при создании и улучшить проект ATL:  
  
-   [Сделать noncreatable объекта ATL](../../atl/reference/making-an-atl-object-noncreatable.md)  
  
-   [Оптимизации компилятора для проекта ATL](../../atl/reference/specifying-compiler-optimization-for-an-atl-project.md)  
  
 Можно задать свойства проекта (например, [ли статическую компоновку с CRT](../../atl/programming-with-atl-and-c-run-time-code.md)) в [свойства проекта](../../ide/general-property-page-project.md) страницы и можно задать [конфигурации построения](/visualstudio/ide/understanding-build-configurations) для проекта ATL. В разделе [с помощью проектов для построения приложений](http://msdn.microsoft.com/en-us/3339fa90-bac2-4b95-8361-662a2e0e7dfe) подробнее.
