# Connection Point Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Эти макросы указывают сопоставления и записи точки подключения.  
  
|||  
|-|-|  
|[BEGIN\_CONNECTION\_POINT\_MAP](../Topic/BEGIN_CONNECTION_POINT_MAP.md)|Отмечает начало записей сопоставления точки подключения.|  
|[CONNECTION\_POINT\_ENTRY](../Topic/CONNECTION_POINT_ENTRY.md)|Вставляет точки подключения в сопоставление.|  
|[END\_CONNECTION\_POINT\_MAP](../Topic/END_CONNECTION_POINT_MAP.md)|Помечает конец записей сопоставления точки подключения.|
