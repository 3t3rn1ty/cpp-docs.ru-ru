# <a name="options-atl-property-page-wizard"></a>Параметры, мастер страниц свойств ATL
Эта страница мастера используется для определения потоковой модели и агрегирование уровня страницы свойств, которые вы создаете.  
  
 **Потоковая модель**  
 Указывает потоковую модель, используемую на странице свойств.  
  
 См. в разделе [указание потоковой модели проекта](../../atl/specifying-the-threading-model-for-a-project-atl.md) Дополнительные сведения.  
  
|Параметр|Описание:|  
|------------|-----------------|  
|**Single**|Страница свойств выполняется только в основном потоке COM.|  
|**Подразделения**|На странице свойств могут создаваться в любом однопотоковом подразделении. По умолчанию.|  
  
 **Статистическая обработка**  
 Добавляет поддержку статистической обработки для страницы свойств, которые вы создаете. См. в разделе [статистической обработки](../../atl/aggregation.md) Дополнительные сведения.  
  
|Параметр|Описание:|  
|------------|-----------------|  
|**Да**|Создание страницы свойств, который может быть статистически вычислена.|  
|**No**|Создание страницы свойств, которое невозможно выполнить статистическую обработку.|  
|**Только**|Создание страницы свойств, которые могут быть созданы только путем агрегирования.|
