# ATL Typedefs
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Библиотека шаблонных классов ATL включает следующие typedef.  
  
|||  
|-|-|  
|[\_ATL\_BASE\_MODULE](../Topic/_ATL_BASE_MODULE.md)|Заданный как typedef, основанное на [\_ATL\_BASE\_MODULE70](../Topic/_ATL_BASE_MODULE70%20Structure.md).|  
|[\_ATL\_COM\_MODULE](../Topic/_ATL_COM_MODULE.md)|Заданный как typedef, основанное на [\_ATL\_COM\_MODULE70](../../atl/reference/atl-com-module70-structure.md).|  
|[\_ATL\_MODULE](../Topic/_ATL_MODULE.md)|Заданный как typedef, основанное на [\_ATL\_MODULE70](../../atl/reference/atl-module70-structure.md).|

|[ATL\_URL\_PORT](../Topic/ATL_URL_PORT.md)|Тип, используемый [скручиваемость](../../atl/reference/curl-class.md) для указания номера порта.|  
|[CComDispatchDriver](../Topic/CComDispatchDriver.md)|Этот класс управляет указатели интерфейса модели COM.|  
|[CComGlobalsThreadModel](../Topic/CComGlobalsThreadModel.md)|Вызывает соответствующие методы модели потока, независимо от используемой потоковой модели.|  
|[CComObjectThreadModel](../Topic/CComObjectThreadModel.md)|Вызывает соответствующие методы модели потока, независимо от используемой потоковой модели.|  
|[CContainedWindow](../Topic/CContainedWindow.md)|Этот класс специализация **CContainedWindowT.**|  
|[CPath](../Topic/CPath.md)|Специализация [CPathT](../../atl/reference/cpatht-class.md) с помощью `CString`.|  
|[CPathA](../Topic/CPathA.md)|Специализация [CPathT](../../atl/reference/cpatht-class.md) с помощью `CStringA`.|  
|[CPathW](../Topic/CPathW.md)|Специализация [CPathT](../../atl/reference/cpatht-class.md) с помощью `CStringW`.|  
|[CSimpleValArray](../Topic/CSimpleValArray.md)|Представляет массив для хранения простых типов.|  
|[DefaultThreadTraits](../../atl/reference/atl-typedefs.md)|Класс по умолчанию признаков потока.|  
|[LPCURL](../Topic/LPCURL.md)|Указатель на объект [скручиваемость](../../atl/reference/curl-class.md) константой.|
