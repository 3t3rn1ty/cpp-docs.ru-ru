# Category Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Эти макросы указывают сопоставления категории.  
  
|||  
|-|-|  
|[BEGIN\_CATEGORY\_MAP](../Topic/BEGIN_CATEGORY_MAP.md)|Отмечает начало сопоставления категории.|  
|[END\_CATEGORY\_MAP](../Topic/END_CATEGORY_MAP.md)|Помечает конец сопоставления категории.|  
|[IMPLEMENTED\_CATEGORY](../Topic/IMPLEMENTED_CATEGORY.md)|Отображает категории, которые реализуются COM\-объект.|  
|[REQUIRED\_CATEGORY](../Topic/REQUIRED_CATEGORY.md)|Отображает категории контейнера, должен быть com\-объектом.|
