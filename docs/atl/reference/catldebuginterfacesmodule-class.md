# CAtlDebugInterfacesModule Class
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Этот класс обеспечивает поддержку отладки интерфейсы.  
  
## Синтаксис  
  
```  
  
class CAtlDebugInterfacesModule  
  
```  
  
## Заметки  
 `CAtlDebugInterfacesModule` поддерживает интерфейсы, необходимые для отладки.  Включается в любом проекте, указывающее символ [\_ATL\_DEBUG\_QI](../Topic/_ATL_DEBUG_QI.md).  
  
## Требования  
 **Header:** atlbase.h
