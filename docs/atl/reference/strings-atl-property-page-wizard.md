# <a name="strings-atl-property-page-wizard"></a>Строки, мастер страницы свойств ATL
Предоставляет текст, связанный со страницей свойств.  
  
 **Заголовок**  
 Задает текст, отображаемый на вкладке страницы свойств.  
  
 **Строка документа**  
 Задает текстовую строку с описанием страницы. Эта строка может отображаться в диалоговом окне Свойства листа. Фрейм свойства может использовать описание в состояние строки или в подсказках. Стандартный фрейм свойства в настоящее время использует эту строку.  
  
 **Файл справки**  
 Задает имя файла справки, в которых описывается использование страницы «свойства». Это имя не должно содержать путь. Когда пользователь нажимает **справки**, фрейм открывает файл справки в каталог, указанный в значении ключа HelpDir в записях реестра страницы свойств в области идентификатора CLSID.
