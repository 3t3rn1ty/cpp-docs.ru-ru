# Строки, мастер страницы свойств ATL
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Предоставляет текст, связанный со страницей свойств.  
  
 **Заголовок**  
 Задает текст, отображаемый на вкладке страницы свойств.  
  
 **Строка документации**  
 Задает текстовую строку с описанием страницы.  Эта строка может отображаться в диалоговом окне страницы свойств.  Фрейм свойства может использовать описание в строке состояния или в подсказках.  Стандартный фрейм свойства в настоящий момент не использует эту строку.  
  
 **Файл справки**  
 Задает имя файла справки с описанием использования страницы свойств.  Данное имя не должно содержать путь.  Когда пользователь нажимает кнопку **Справка**, фрейм открывает файл справки в каталоге, имя которого указано под ключом HelpDir в записях реестра страницы свойств с соответствующим CLSID.
