# Composite Control Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Эти макросы указывают сопоставления и записи приемника событий.  
  
|||  
|-|-|  
|[BEGIN\_SINK\_MAP](../Topic/BEGIN_SINK_MAP.md)|Отмечает начало сопоставления приемника событий для составного элемента управления.|  
|[END\_SINK\_MAP](../Topic/END_SINK_MAP.md)|Помечает конец сопоставления приемника событий для составного элемента управления.|  
|[SINK\_ENTRY](../Topic/SINK_ENTRY.md)|Запись к сопоставлению приемника событий.|  
|[SINK\_ENTRY\_EX](../Topic/SINK_ENTRY_EX.md)|Запись к сопоставлению приемника событий с дополнительным параметром.|  
|[SINK\_ENTRY\_INFORMATION](../Topic/SINK_ENTRY_INFO.md)|Запись к сопоставлению приемника событий с предоставленной вручную сведений о типе для использования с [IDispEventSimpleImpl](../../atl/reference/idispeventsimpleimpl-class.md).|
