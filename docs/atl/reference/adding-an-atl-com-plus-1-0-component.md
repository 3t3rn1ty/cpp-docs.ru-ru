# Добавление компонента ATL COM+ 1.0
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Этот мастер используется для добавления в проект объекта, поддерживающего службы COM\+ 1.0, в том числе транзакции.  
  
### Добавление в проект компонента ATL СОМ \+1.0.  
  
1.  В окне **Обозреватель решений** или [Представление классов](http://msdn.microsoft.com/ru-ru/8d7430a9-3e33-454c-a9e1-a85e3d2db925) щелкните правой кнопкой мыши имя проекта, в который нужно добавить компонент ATL COM\+ 1.0.  
  
2.  В контекстном меню выберите команду **Добавить**, а затем щелкните **Добавить класс**.  
  
3.  В области "Шаблоны" диалогового окна [Добавление класса](../../ide/add-class-dialog-box.md) щелкните **Компонент ATL СОМ \+1.0**, а затем нажмите кнопку **Открыть**, чтобы открыть [Мастер компонентов ATL СОМ \+ 1.0](../../atl/reference/atl-com-plus-1-0-component-wizard.md).
