# Object Map Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Эти макросы указывают сопоставления и записи объектов.  
  
|||  
|-|-|  
|[DECLARE\_OBJECT\_DESCRIPTION](../Topic/DECLARE_OBJECT_DESCRIPTION.md)|Позволяет указать текстовое описание объекта класса, которое будет введено в сопоставление объекта.|  
|[OBJECT\_ENTRY\_AUTO](../Topic/OBJECT_ENTRY_AUTO.md)|Вставляет объект библиотеки ATL в сопоставление объекта, обновление реестра и создает экземпляр объекта.|  
|[OBJECT\_ENTRY\_NON\_CREATEABLE\_EX\_AUTO](../Topic/OBJECT_ENTRY_NON_CREATEABLE_EX_AUTO.md)|Позволяет указать, что объект должен быть зарегистрирован и инициализации, но он не должен быть создаваемыми с помощью внешнего `CoCreateInstance`.|
