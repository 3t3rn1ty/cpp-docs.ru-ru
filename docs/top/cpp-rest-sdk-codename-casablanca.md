# C++ REST SDK (Codename &quot;Casablanca&quot;)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

С помощью C\+\+ REST SDK \(кодовое название "Casablanca"\) можно легко написать современный, асинхронный кода C\+\+, который подключается с помощью облачных служб.  
  
 Проект Casablanca больше не входит в комплектацию Visual Studio.  Документация и источники хранятся и периодически обновляются в CodePlex.  Более подробные сведения о данном проекте см. на странице [C\+\+ REST SDK \(кодовое название "Casablanca"\)](http://casablanca.codeplex.com/).
