# Распространение клиентских веб-приложений
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Если приложение использует классы MFC, реализуя элемент управления WebBrowser \(например, `CHtmlView` или `CHtmlEditView`\), на компьютере должен быть установлен Microsoft Internet Explorer версии 4.0 или более поздней версии.  
  
 Установка последней версии Internet Explorer также гарантирует, что целевой компьютер имеет последнюю версию файлов общих элементов управления.  
  
 Информация об установке минимальных компонентов Internet Explorer доступна в следующей статье базы знаний Майкрософт:  
  
-   Q185375, HOWTO: Create a Single EXE Install of Internet Explorer \([http:\/\/support.microsoft.com\/support\/kb\/articles\/q185\/3\/75.asp](http://support.microsoft.com/support/kb/articles/q185/3/75.asp)\)  
  
 Статьи базы знаний можно найти в библиотеке MSDN или при помощи поиска по веб\-странице [http:\/\/support.microsoft.com\/](http://support.microsoft.com/).
