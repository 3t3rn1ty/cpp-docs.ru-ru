# мастер проекта Makefile
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Мастер проекта Makefile используется при создании командной строки для выполнения обработчика построения.  После создания проекта просмотреть и изменить свойства проекта можно на [странице свойств](../ide/working-with-project-properties.md) проекта.  
  
## Общие сведения  
 На этой странице мастера описываются текущие параметры создаваемого проекта.  По умолчанию проект создает конфигурации отладки и выпуска файла *ProjName*.exe.  
  
 Чтобы изменить заданные по умолчанию параметры, необходимо в левой колонке мастера щелкнуть вкладку [Параметры приложения](../ide/application-settings-makefile-project-wizard.md) и внести необходимые изменения.
