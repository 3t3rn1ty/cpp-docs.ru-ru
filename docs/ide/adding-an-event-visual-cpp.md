# <a name="adding-an-event-visual-c"></a>Добавление события (Visual C++)
Из представления классов можно добавить событие с помощью [мастер добавления события](../ide/add-event-wizard.md) только в класс элемента управления в вашей [управления MFC ActiveX](../mfc/reference/creating-an-mfc-activex-control.md) проекта. Если вы хотите добавить событие в проект другого типа, используйте **событий** кнопку в [окно свойств](/visualstudio/ide/reference/properties-window).  
  
### <a name="to-add-an-event-to-your-mfc-activex-control-project"></a>Чтобы добавить событие в проект элемента управления MFC ActiveX  
  
1.  В представлении класса разверните узел проекта, чтобы отобразить классы в проекте.  
  
2.  Щелкните правой кнопкой мыши проект класс элемента управления.  
  
3.  В контекстном меню щелкните **добавить**и нажмите кнопку **добавить событие** , чтобы открыть мастер добавления события.  
  
4.  Укажите сведения о событии в соответствующих полях мастера.  
  
5.  Нажмите кнопку **Готово** Чтобы добавить событие в проект.
