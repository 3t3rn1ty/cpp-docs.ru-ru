# Добавление универсального класса C++
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Универсальный класс C\+\+ можно добавить с помощью окна **Представление классов**.  Универсальным классом C\+\+ является класс, определенный или являющийся производным от определяемого класса.  
  
### Добавление в проект универсального класса C\+\+.  
  
1.  В окне **Представление классов** щелкните правой кнопкой проект, в который нужно добавить новый класс, нажмите кнопку **Добавить** и выберите **Класс**.  
  
2.  В диалоговом окне [Добавить класс](../ide/add-class-dialog-box.md) на панели шаблонов щелкните **Класс C\+\+**.  Нажмите кнопку **Добавить**, чтобы открыть [Мастер универсальных классов C\+\+](../ide/generic-cpp-class-wizard.md).  
  
3.  В мастере необходимо указать имя класса и задать параметры или принять параметры по умолчанию.  
  
4.  Чтобы закрыть мастер и просмотреть новый универсальный класс С\+\+ в проекте, нажмите кнопку **Готово**.
