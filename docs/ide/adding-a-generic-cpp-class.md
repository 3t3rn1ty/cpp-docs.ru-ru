# <a name="adding-a-generic-c-class"></a>Добавление универсального класса C++
Универсальный класс C++ можно добавить с помощью **представление классов**. Универсальный класс C++ является класс, определенный или который является производным от класса, который определяется.  
  
### <a name="to-add-a-generic-c-class-to-a-project"></a>Добавление в проект универсального класса C++  
  
1.  В **представление классов**, щелкните правой кнопкой мыши проект, к которому требуется добавить новый класс, нажмите кнопку **добавить**, а затем нажмите кнопку **класса**.  
  
2.  В [добавить класс](../ide/add-class-dialog-box.md) щелкните в области шаблонов диалогового **класс C++**. Нажмите кнопку **добавить** для отображения [мастер универсальных классов C++](../ide/generic-cpp-class-wizard.md).  
  
3.  В мастере укажите имя класса и определить параметры или принять параметры по умолчанию.  
  
4.  Чтобы закрыть мастер и просмотреть новый универсальный класс C++ в проекте, щелкните **Готово**.
