# Параметры приложения, мастер проекта Makefile
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Предоставьте данные для проекта.  Задаваемые параметры отражаются на [странице свойств](../ide/working-with-project-properties.md) проекта.  После создания проекта можно просмотреть и изменить каждый из следующих параметров на странице **Nmake** свойств проекта.  
  
 **Командная строка построения**  
 Задает запуск в командной строке команды при выборе **Построение** из меню **Построение**.  Отображается в поле **Командная строка построения** страницы **Nmake** свойств проекта.  
  
 **Output**  
 Задает имя файла, который будет содержать вывод командной строки.  По умолчанию значение этого параметра основано на имени проекта.  Отображается в поле **Вывод** страницы **Nmake** свойств проекта.  
  
 **Команды очистки**  
 Задает запуск в командной строке команды при выборе **Очистка** из меню **Построение**.  Отображается в поле **Командная строка очистки** страницы **Nmake** свойств проекта.  
  
 **Перестроить командную строку**  
 Задает запуск в командной строке команды при выборе **Повторное построение** из меню **Построение**.  Отображается в поле **Командная строка повторного построения всего** страницы **Nmake** свойств проекта.
