# &lt;code&gt; (Visual C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Тег \<code\> обеспечивает способ отображения один или несколько линий как код.  
  
## Синтаксис  
  
```  
<code>content</code>  
```  
  
#### Параметры  
 `content`  
 Текст, который необходимо пометить как код.  
  
## Заметки  
 Используйте [\<c\>](../Topic/%3Cc%3E%20\(Visual%20C++\).md) для отображения часть текста должен быть отмечен как код.  
  
 Чтобы обработать и сохранить комментарии документации в файл, при компиляции необходимо использовать параметр [\/doc](../build/reference/doc-process-documentation-comments-c-cpp.md).
