# <a name="adding-a-method--visual-c"></a>Добавление метода (Visual C++)
Можно использовать [мастер добавления метода](../ide/add-method-wizard.md) для добавления метода в интерфейс в проекте. Если проект содержит класс, связанный с интерфейсом, мастер слишком изменяет класса.  
  
### <a name="to-add-a-method-to-your-object"></a>Чтобы добавить метод в объект  
  
1.  В представлении класса разверните узел проекта, чтобы отобразить интерфейс, к которому требуется добавить метод.  
  
    > [!NOTE]
    >  Можно также добавить методы для диспетчера интерфейсов, что, если проект имеет атрибуты, находятся в узле библиотеки.  
  
2.  Щелкните правой кнопкой мыши имя интерфейса.  
  
3.  В контекстном меню выберите **добавить**, а затем нажмите кнопку **добавить метод**.  
  
4.  В мастер добавления метода укажите сведения для создания метода.  
  
5.  Укажите для этого метода в параметры языка описания интерфейса [атрибуты IDL](../ide/idl-attributes-add-method-wizard.md) странице мастера.  
  
6.  Нажмите кнопку **Готово** для добавления метода.
