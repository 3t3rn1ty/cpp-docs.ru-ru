# Добавление метода (Visual C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Чтобы добавить метод в интерфейс проекта, используйте [Мастер добавления метода](../ide/add-method-wizard.md).  Если проект содержит класс, связанный с интерфейсом, то мастер изменит также и этот класс.  
  
### Чтобы добавить метод в объект, выполните следующие действия:  
  
1.  В "Представлении классов" разверните узел проекта, чтобы отобразить интерфейс, в который необходимо добавить метод.  
  
    > [!NOTE]
    >  Методы также можно добавить в диспетчерские интерфейсы, которые находятся в узле библиотеки до тех пор, пока проект не содержит атрибуты.  
  
2.  Щелкните правой кнопкой мыши имя интерфейса.  
  
3.  В контекстном меню выберите команду **Добавить**, а затем выберите команду **Добавить метод**.  
  
4.  В мастере добавления метода укажите сведения для создания метода.  
  
5.  На странице мастера [Атрибуты IDL](../ide/idl-attributes-add-method-wizard.md) укажите для данного метода параметры языка описания интерфейса \(IDL\).  
  
6.  Нажмите кнопку **Готово**, чтобы добавить метод.
