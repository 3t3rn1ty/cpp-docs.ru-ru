# Добавление функции-члена (Visual C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

В диалоговом окне **Представление классов** можно добавлять функцию\-член к любому классу.  После этого объявления добавляется в файл заголовка и тела функции\-члена заглушки добавляется к файлу реализации класса, который затем можно изменить.  
  
### Добавление функции\-члена в класс  
  
1.  В **Представлении классов** разверните узел проекта, чтобы отобразить классы проекта. \(Чтобы открыть **Представление классов** в строке меню выберите **Представление**, **Представление классов**\).  
  
2.  Открыть контекстное меню для класса необходимо добавить функцию\-член, а затем выберите **Добавить**, **Добавить функцию**.  
  
3.  Предоставить соответствующие сведения о функции\-члене.  Для получения дополнительной информации см. [Мастер добавления функции\-члена](../ide/add-member-function-wizard.md).  
  
4.  Нажмите кнопку **Готово**, чтобы создать код функции\-члена.
