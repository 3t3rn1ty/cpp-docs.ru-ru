# <a name="adding-a-member-function-visual-c"></a>Добавление функции-члена (Visual C++)
В **представление классов**, можно добавить функцию-член для любого класса. При этом объявлении добавляется в файл заголовка и тела функции члена заглушки добавляется в файл реализации класса, который затем можно изменить.  
  
### <a name="to-add-a-member-function-to-a-class"></a>Чтобы добавить функцию-член класса  
  
1.  В **представление классов**, разверните узел проекта, чтобы отобразить классы в проекте. (Чтобы открыть **представление классов**, в строке меню выберите **представление**, **представление классов**.)  
  
2.  Откройте контекстное меню для класса, который вы хотите добавить функция-член, а затем выберите **добавить**, **добавить функцию**.  
  
3.  Укажите необходимые сведения о функции-члена. Дополнительные сведения см. в разделе [мастер добавления функции члена](../ide/add-member-function-wizard.md).  
  
4.  Выберите **Готово** кнопку, чтобы создать код функции-члена.
