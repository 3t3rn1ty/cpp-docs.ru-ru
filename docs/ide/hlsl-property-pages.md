# Страницы свойств HLSL
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

С помощью страницы свойств компилятора HLSL \(fxc.exe\) настроить, как отдельные файлы текстуры HLSL построены.  Свойства делятся на три категории: 3  
  
-   [Страницы свойств "HLSL": страница "Общие"](../ide/hlsl-property-pages-general.md)  
  
     Предоставляет свойства для определения часто используемые параметры компилятора HLSL.  
  
-   [Страницы свойств HLSL: Дополнительно](../ide/hlsl-property-pages-advanced.md)  
  
     Предоставляет свойства для указания дополнительных параметров компилятора HLSL.  
  
-   [Страницы свойств HLSL. Выходные файлы](../ide/hlsl-property-pages-output-files.md)  
  
     Предоставляет свойства для задания способа файлы остаются выводятся компилятором HLSL.  
  
 Можно указать аргументы командной строки для компилятора HLSL с помощью свойства **Дополнительные параметры** страницы свойств **Командная строка**; это включает аргументы, которые не могут быть настроены с помощью других свойств страниц свойств HLSL.  Дополнительные сведения о компиляторе HLSL см. в разделе [Средства компилятора эффектов](http://go.microsoft.com/fwlink/p/?LinkID=258285&clcid=0x409)
