# <a name="hlsl-property-pages"></a>Страницы свойств HLSL
Страницы свойств HLSL компилятор (fxc.exe) можно использовать для настройки отдельных шейдера HLSL файлы создаются. Свойства делятся на три категории:  
  
-   [Страницы свойств HLSL: "Общие"](../ide/hlsl-property-pages-general.md)  
  
     Предоставляет свойства, определяющие часто используемые параметры для компилятора HLSL.  
  
-   [Страницы свойств HLSL: "Дополнительно"](../ide/hlsl-property-pages-advanced.md)  
  
     Предоставляет свойства для указания дополнительных параметров для компилятора HLSL.  
  
-   [Страницы свойств HLSL: выходные файлы](../ide/hlsl-property-pages-output-files.md)  
  
     Предоставляет свойства, чтобы указать, каким образом файлы выводятся компилятором HLSL.  
  
 Можно указать аргументы командной строки для компилятора HLSL с помощью **Дополнительные параметры** свойство **командной строки** страницу свойств; сюда относятся аргументы, которые нельзя настроить с помощью других Свойства страницы свойств HLSL. Сведения о компиляторе HLSL см. в разделе [средства компилятора эффектов](http://go.microsoft.com/fwlink/p/?LinkID=258285&clcid=0x409)
