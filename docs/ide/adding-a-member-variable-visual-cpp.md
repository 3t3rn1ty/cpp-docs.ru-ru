# <a name="adding-a-member-variable--visual-c"></a>Добавление переменной-члена (Visual C++)
Можно добавить переменную-член для класса с помощью представления классов. Переменные-члены могут использоваться либо [обмена данными и проверки данных](../mfc/dialog-data-exchange-and-validation.md), или могут быть универсальными. Чтобы получить необходимые сведения и использовать его для вставки элементов в исходные файлы в соответствующие папки специально данных мастер переменной-члена. Можно добавить переменную-член из [редактор диалоговых окон](../windows/dialog-editor.md) в [представление ресурсов](../windows/resource-view-window.md), или из [представление классов](http://msdn.microsoft.com/en-us/8d7430a9-3e33-454c-a9e1-a85e3d2db925).  
  
> [!NOTE]
>  При проектировании и реализации диалоговое окно, может оказаться более эффективно использовать диалоговое окно Редактор, чтобы добавить элементы управления диалоговых окон, а затем реализуйте переменные-члены элементов управления.  
  
### <a name="to-add-a-member-variable-for-a-dialog-control-in-resource-view-using-the-add-member-variable-wizard"></a>Чтобы добавить переменную-член для элемента управления диалогового окна в представлении ресурсов с помощью мастера добавления переменной-члена, выполните следующие действия:  
  
1.  В представлении ресурсов разверните узел проекта и узел диалогового окна, чтобы отобразить список диалоговых окон проекта.  
  
2.  Дважды щелкните диалоговое окно, к которому требуется добавить переменную-член, чтобы открыть его в редакторе диалоговых окон.  
  
3.  В диалоговом окне отображается в редакторе диалоговых окон щелкните правой кнопкой мыши элемент управления, к которому требуется добавить переменную-член.  
  
4.  В контекстном меню щелкните **добавить переменную** для отображения [член мастер добавления переменной-](../ide/add-member-variable-wizard.md).  
  
    > [!NOTE]
    >  Значение по умолчанию уже содержится в **идентификатор элемента управления**.  
  
5.  Предоставьте информацию в соответствующих полях мастера. В разделе [элементы управления диалоговых окон и типы переменных](../ide/dialog-box-controls-and-variable-types.md) для получения дополнительной информации.  
  
6.  Нажмите кнопку **Готово** добавьте в проект код определения и реализации и закройте мастер.  
  
### <a name="to-add-a-member-variable-from-class-view-using-the-add-member-variable-wizard"></a>Чтобы добавить переменную-член из представления классов, с помощью мастера добавления переменной-члена, выполните следующие действия:  
  
1.  В [представление классов](http://msdn.microsoft.com/en-us/8d7430a9-3e33-454c-a9e1-a85e3d2db925), разверните узел проекта, чтобы отобразить классы в проекте.  
  
2.  Щелкните правой кнопкой мыши класс, к которому требуется добавить переменную.  
  
3.  В контекстном меню щелкните **добавить**и нажмите кнопку **добавить переменную** , чтобы открыть мастер добавления переменной-члена.  
  
4.  Предоставьте информацию в соответствующих полях мастера. В разделе [член мастер добавления переменной-](../ide/add-member-variable-wizard.md) подробные сведения.  
  
5.  Нажмите кнопку **Готово** добавьте в проект код определения и реализации и закройте мастер.
