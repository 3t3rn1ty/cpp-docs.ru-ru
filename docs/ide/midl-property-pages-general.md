# Страницы свойств &quot;MIDL&quot;: страница &quot;Общие&quot;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

На странице свойств **Общие** в папке **MIDL** задаются следующие параметры компилятора MIDL:  
  
-   Определения препроцессора [\(\/D](http://msdn.microsoft.com/library/windows/desktop/aa367321)\)  
  
-   Дополнительные каталоги включаемых файлов \([\/I](http://msdn.microsoft.com/library/windows/desktop/aa367328)\)  
  
-   Игнорировать стандартный путь включаемых файлов \([\/no\_def\_idir](http://msdn.microsoft.com/library/windows/desktop/aa367347)\)  
  
-   Совместимость с MkTypLib \([\/mktyplib203](http://msdn.microsoft.com/library/windows/desktop/aa367332)\)  
  
-   Порог предупреждений \([\/W](http://msdn.microsoft.com/library/windows/desktop/aa367383)\)  
  
-   Рассматривать предупреждения как ошибки \([\/WX](http://msdn.microsoft.com/library/windows/desktop/aa367387)\)  
  
-   Отмена вывода начального заголовка \([\/nologo](http://msdn.microsoft.com/library/windows/desktop/aa367341)\)  
  
-   Тип знаков MIDL \([\/char](http://msdn.microsoft.com/library/windows/desktop/aa367314)\)  
  
-   Целевая среда \([\/env](http://msdn.microsoft.com/library/windows/desktop/aa367323)\)  
  
-   Генерировать прокси без заглушек \([\/Oicf](http://msdn.microsoft.com/library/windows/desktop/aa367352)\)  
  
 Сведения о том, как открыть страницу свойств **Общие** в папке **MIDL** см. в разделе [Практическое руководство. Задание свойств проекта при помощи страниц свойств](../Topic/How%20to:%20Specify%20Project%20Properties%20with%20Property%20Pages.md).  
  
 Сведения о программном доступе к параметрам MIDL для проектов C\+\+ см. в описании объекта <xref:Microsoft.VisualStudio.VCProjectEngine.VCMidlTool>.
