# <a name="midl-property-pages-general"></a>Страницы свойств "MIDL": страница "Общие"
Страница свойств **Общие** в папке **MIDL** указывает следующие параметры компилятора MIDL:  
  
-   Определения препроцессора [(/D](http://msdn.microsoft.com/library/windows/desktop/aa367321))  
  
-   Дополнительные каталоги включаемых файлов ([/I](http://msdn.microsoft.com/library/windows/desktop/aa367328))  
  
-   Игнорировать стандартный путь поиска включаемых файлов ([/no_def_idir](http://msdn.microsoft.com/library/windows/desktop/aa367347))  
  
-   Совместимость MkTypLib ([/mktyplib203](http://msdn.microsoft.com/library/windows/desktop/aa367332))  
  
-   Уровень предупреждений ([/W](http://msdn.microsoft.com/library/windows/desktop/aa367383))  
  
-   Предупреждение как ошибка ([/WX](http://msdn.microsoft.com/library/windows/desktop/aa367387))  
  
-   Отключить загрузочное объявление ([/nologo](http://msdn.microsoft.com/library/windows/desktop/aa367341))  
  
-   Тип Char для MIDL ([/char](http://msdn.microsoft.com/library/windows/desktop/aa367314))  
  
-   Целевая среда ([/env](http://msdn.microsoft.com/library/windows/desktop/aa367323))  
  
-   Создавать прокси без заглушек ([/Oicf](http://msdn.microsoft.com/library/windows/desktop/aa367352))  
  
 Сведения о доступе к странице свойств **Общие** в папке **MIDL** см. в разделе [Работа со свойствами проектов](../ide/working-with-project-properties.md).  
  
 Сведения о программном доступе к параметрам MIDL для проектов C++ см. в описании объекта <xref:Microsoft.VisualStudio.VCProjectEngine.VCMidlTool>
