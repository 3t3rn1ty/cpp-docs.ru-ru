# <a name="midl-property-pages-general"></a>Страницы свойств "MIDL": страница "Общие"
**Общие** на странице свойств в **MIDL** папки задаются следующие параметры компилятора MIDL:  
  
-   Определения препроцессора [(/D](http://msdn.microsoft.com/library/windows/desktop/aa367321))  
  
-   Дополнительные каталоги включения ([/i](http://msdn.microsoft.com/library/windows/desktop/aa367328))  
  
-   Игнорировать стандартный путь включаемых ([/no_def_idir](http://msdn.microsoft.com/library/windows/desktop/aa367347))  
  
-   MkTypLib совместимость ([/mktyplib203](http://msdn.microsoft.com/library/windows/desktop/aa367332))  
  
-   Порог предупреждений ([/W](http://msdn.microsoft.com/library/windows/desktop/aa367383))  
  
-   Рассматривать предупреждения как ошибки ([/WX](http://msdn.microsoft.com/library/windows/desktop/aa367387))  
  
-   Отключить загрузочное объявление ([/nologo](http://msdn.microsoft.com/library/windows/desktop/aa367341))  
  
-   Тип Char MIDL ([/char](http://msdn.microsoft.com/library/windows/desktop/aa367314))  
  
-   Конечная среда ([/env](http://msdn.microsoft.com/library/windows/desktop/aa367323))  
  
-   Создавать прокси без заглушек ([/Oicf](http://msdn.microsoft.com/library/windows/desktop/aa367352))  
  
 Сведения о доступе к **Общие** на странице свойств в **MIDL** папку, в разделе [работа со свойствами проекта](../ide/working-with-project-properties.md).  
  
 Сведения о том, как получить программный доступ к параметрам MIDL для проектов C++ см. в разделе <xref:Microsoft.VisualStudio.VCProjectEngine.VCMidlTool> объекта.
