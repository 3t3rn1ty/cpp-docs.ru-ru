# <a name="walkthrough-deploying-your-program-c"></a>Пошаговое руководство. Развертывание программы (C++)
Теперь, когда вы создали приложение, выполнив ранее связанные пошаговые руководства, перечисленных в [с помощью интегрированной среды разработки Visual Studio для разработки классических приложений C++](../ide/using-the-visual-studio-ide-for-cpp-desktop-development.md), последним шагом является создание установщика, чтобы другие пользователи могли Установите это приложение на своих компьютерах. Для этого вам необходимо добавить новый проект в имеющееся решение. Выходным файлом этого проекта будет являться файл setup.exe, который служит для установки приложения на другом компьютере.  
  
 В данном пошаговом руководстве показано, как использовать установщик Windows для развертывания приложения. Кроме того, для развертывания приложения может использоваться ClickOnce. Дополнительные сведения см. в разделе [ClickOnce Deployment for Visual C++ Applications](../ide/clickonce-deployment-for-visual-cpp-applications.md). Дополнительные сведения о развертывании в целом. в разделе [развертывание приложений, служб и компонентов](/visualstudio/deployment/deploying-applications-services-and-components).  
  
## <a name="prerequisites"></a>Предварительные требования  
  
-   Это пошаговое руководство предполагает знание основ языка C++.  
  
-   Кроме того, предполагается, что вы выполнили ранее связанных пошаговых руководств, перечисленных в [с помощью интегрированной среды разработки Visual Studio для разработки классических приложений C++](../ide/using-the-visual-studio-ide-for-cpp-desktop-development.md).  
  
-   Настоящее пошаговое руководство невозможно выполнить в выпусках Express среды Visual Studio.  
  
-   Если вы еще не загрузили выпуск InstallShield Limited Edition (ISLE), сделайте это, выполнив инструкции, описанные в шагах ниже в данном разделе. Выпуск ISLE предоставляется работающим с Visual Studio разработчикам бесплатно; он заменяет собой функциональные возможности шаблонов проектов установки и развертывания из более ранних выпусков Visual Studio.  
  
### <a name="to-install-the-isle-setup-and-deployment-project-template"></a>Установка шаблона проекта установки и развертывания ISLE  
  
1.  При подключении к Интернету, в строке меню выберите **файл**, **New**, **проекта** Открытие **новый проект** диалоговое окно.  
  
2.  В левой области диалогового окна разверните **установленные**, **шаблоны**, и **другие типы проектов** узлов, а затем выберите **Установка и развертывание**. В центральной области выберите **Включение InstallShield Limited Edition** и выберите **ОК** кнопки.  
  
3.  Следуйте инструкциям, чтобы установить выпуск InstallShield Limited для Visual Studio (ISLE).  
  
4.  Загрузив, установив и активировав ISLE, закройте и снова откройте Visual Studio.  
  
5.  В строке меню выберите **файл**, **последние проекты и решения**и нажмите кнопку **игры** решение, чтобы снова открыть его.  
  
### <a name="to-create-a-setup-project-and-install-your-program"></a>Создание проекта установки и установка программы  
  
1.  Измените активную конфигурацию решения, указав значение Выпуск. В строке меню последовательно выберите пункты **Сборка**и **Диспетчер конфигураций**. В **Configuration Manager** диалогового **активная конфигурация решения** раскрывающемся списке выберите **выпуска**. Выберите **закрыть** кнопку, чтобы сохранить конфигурацию.  
  
2.  В строке меню выберите **файл**, **New**, **проекта** Открытие **новый проект** диалоговое окно.  
  
3.  В левой области диалогового окна разверните **установленные**, **шаблоны**, и **другие типы проектов** узлов, а затем выберите **Установка и развертывание**. В центральной области выберите **проект InstallShield Limited Edition**.  
  
4.  Введите имя для проекта установки в **имя** поле. Например, введите **установщик игры**. В **решения** раскрывающемся списке выберите **добавить решение**. Выберите **ОК** кнопку, чтобы создать проект установки. Объект **помощник по проектам (установщик игры)** в окне редактора откроется вкладка.  
  
5.  В нижней части **помощник по проектам (установщик игры)** выберите **сведения о приложении** ссылку.  
  
6.  На **сведения о приложении** укажите название организации, которое будет отображаться в установщике. Можно использовать название организации, или для этого примера используйте **Contoso**. Укажите название приложения; в этом примере укажите **игры**. Укажите веб-адрес вашей компании или в этом примере используется **http://www.contoso.com**.  
  
7.  В нижней части **помощник по проектам (установщик игры)** выберите **установки Interview** ссылку.  
  
8.  На **установки Interview** в разделе **вы хотите отобразить диалоговое окно лицензионного соглашения**выберите **нет** переключатель. В разделе **вы хотите предлагать пользователям вводить их название компании и имя пользователя**выберите **нет** переключатель.  
  
9. В **обозревателе решений**, разверните **установщик игры** проекта, разверните **Organize Your Setup** узел, а затем откройте **Общие сведения** страницы.  
  
10. На **Общие сведения (установщик игры)** в окне редактора, укажите **идентификатор создателя тега**, например, **regid.2012-12.com.Contoso**.  
  
11. В **обозревателе решений**в разделе **установщик игры** проекта, разверните **Specify Application Data** узел, а затем откройте **файлы** страница.  
  
12. На **файлы (установщик игры)** вкладки в окне редактора, в разделе **исходные файлы компьютера**, откройте контекстное меню для **основной вывод из игры** и выберите **Копирования**.  
  
13. Откройте контекстное меню в разделе **имя** столбца в **файлы целевого компьютера**и выберите **вставить**. Новый элемент с именем **игра.основные выходные** отображается.  
  
14. В **обозреватель решений**в разделе **Specify Application Data** открытым узлом **Redistributables** страницы.  
  
15. На **распространяемые пакеты (установщик игры)** в окне редактора, выберите **Visual C++ 11.0 CRT (x86)** флажок.  
  
16. В строке меню выберите **построения**, **построить решение** для построения проектов игры и установщик игры.  
  
17. В папке решения найдите программу setup.exe, собранную из проекта "Установщик игры", и запустите ее, чтобы установить приложение "Игра" на компьютере. Можно скопировать этот файл, чтобы установить приложение и все необходимые файлы библиотеки на другом компьютере.  
  
18. Вы можете задать целый ряд параметров в проекте установки в соответствии со своими потребностями. Дополнительные сведения см в **обозревателе решений**в разделе **установщик игры** откройте проект **Приступая к работе** страницы и нажмите клавишу F1, чтобы открыть библиотеку справки ISLE.
