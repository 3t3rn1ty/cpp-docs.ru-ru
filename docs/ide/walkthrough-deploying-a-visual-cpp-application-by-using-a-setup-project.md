# <a name="walkthrough-deploying-a-visual-c-application-by-using-a-setup-project"></a>Пошаговое руководство. Развертывание приложения Visual C++ с помощью проекта установки
Описывает использование проекта установки для развертывания приложения Visual C++.  
  
## <a name="prerequisites"></a>Предварительные требования  
 Ниже приведены компоненты, необходимые для выполнения данного пошагового руководства.  
  
-   Компьютер с установленным продуктом [!INCLUDE[vs_dev11_long](../build/includes/vs_dev11_long_md.md)].  
  
-   Дополнительный компьютер, где не установлены библиотеки Visual C++.  
  
### <a name="to-deploy-an-application-by-using-a-setup-project"></a>Развертывание приложения с помощью проекта установки  
  
1.  Используйте **мастер приложений MFC** для создания решения Visual Studio. Чтобы найти мастер, в диалоговом окне **Новый проект** разверните узел **Visual C++**, выберите **MFC** и **Приложение MFC**, введите имя проекта и нажмите кнопку **ОК**.  
  
2.  Измените активную конфигурацию решения, указав значение **Выпуск**. Из меню **Сборка** выберите элемент **Диспетчер конфигураций**. В диалоговом окне **Диспетчер конфигураций** выберите в раскрывающемся списке **Активная конфигурация решения** пункт **Выпуск**.  
  
3.  Нажмите клавишу F7, чтобы собрать приложение. Либо в меню **Сборка** выберите **Собрать решение**. Это позволяет проекту установки использовать выходные данные этого проекта приложения MFC.  
  
4.  Если вы еще не сделали это, скачайте InstallShield Limited Edition (ISLE), который предоставляется работающим с Visual Studio разработчикам бесплатно и заменяет собой функциональные возможности шаблонов проектов в Visual Studio в отношении установки и развертывания. При наличии подключения к Интернету откройте диалоговое окно **Новый проект**, выбрав **Файл**, **Создать**, **Проект** в строке меню либо щелкнув правой кнопкой мыши решение в **обозревателе решений** и выбрав **Добавить**, **Новый проект**. Разверните узел **Другие типы проектов**, выберите пункт **Включение InstallShield Limited Edition** в узле **Установка и развертывание**, а затем следуйте отображаемым инструкциям. Скачав, установив и активировав InstallShield Limited Edition, закройте и снова откройте Visual Studio.  
  
5.  Снова откройте диалоговое окно **Новый проект**, разверните узел **Другие типы проектов** и выберите пункт **Проект InstallShield Limited Edition** в узле **InstallShield Limited Edition**.  
  
6.  Следуйте инструкциям в узле **Приступая к работе** проекта установки, созданного с помощью шаблона InstallShield Limited Edition, чтобы добавить ссылку на выходные данные в проект MFC Visual Studio.  
  
7.  Выполните сборку проекта установки, чтобы создать файл установщика (setup.exe). Для этого щелкните правой кнопкой мыши узел проекта установки в **обозревателе решений** и выберите команду **Собрать**.  
  
     InstallShield Limited Edition создает файл установки в дереве проекта установки (по умолчанию он может находиться в подпапке Express\SingleImage\DiskImages\DISK1 проекта установки).  
  
8.  Запустите программу установки на втором компьютере, где не установлены библиотеки Visual C++.
