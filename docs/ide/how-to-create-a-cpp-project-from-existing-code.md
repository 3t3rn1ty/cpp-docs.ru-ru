# Практическое руководство. Создание проекта C++ из существующего кода
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

В [!INCLUDE[vsPro](../ide/includes/vspro_md.md)] и [!INCLUDE[vsUltShort](../ide/includes/vsultshort_md.md)] существующие файлы кода можно перенести в проект Visual C\+\+ с помощью мастера **Создать новый проект по существующим файлам с кодом**.  Этот мастер недоступен в выпусках Express.  
  
 Перенос существующих файлов кода в проект Visual C\+\+ позволяет использовать такие средства IDE, как [IntelliSense](../Topic/Using%20IntelliSense.md), **Обозреватель вызовов** и [Диспетчер свойств](../misc/property-manager.md).  
  
### Создание проекта C\+\+ из существующего кода  
  
1.  В меню **Файл** выберите **Создать** и щелкните пункт **Проект из существующего кода**.  
  
2.  На первой странице мастера **Создание нового проекта из файлов существующего кода**, выберите **Visual C\+\+** в списке **Выбор типа создаваемого проекта**.  
  
3.  На последующих страницах мастера задайте параметры для нового проекта, который необходимо создать.  
  
4.  Нажмите кнопку **Готово**, чтобы создать новый проект.
