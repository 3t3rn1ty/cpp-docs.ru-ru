# <a name="editing-a-com-interface"></a>Редактирование интерфейса COM
С помощью команд контекстного меню представления классов в проектах Visual C++ можно определить новые методы и свойства для COM-интерфейсов. Кроме того из области элементов можно определить события элементов управления ActiveX.  
  
 Для классов объектов ATL и MFC-основе COM можно изменить реализацию класса одновременно изменять интерфейс.  
  
> [!NOTE]
>  Для интерфейсов, которые определены за пределами **добавить класс** диалоговом Visual C++ добавляет методы или свойства в IDL-файл, а также добавлять заглушки в классы, реализующие методы, даже в том случае, когда интерфейсы добавляются вручную.  
  
 Ниже приведены три мастера помогают настроить существующие интерфейсы. Они доступны из представления классов.  
  
|Мастер|Тип проекта|  
|------------|------------------|  
|[Мастер добавления свойства](../ide/names-add-property-wizard.md)|Проекты ATL и MFC, поддержка ATL. Щелкните правой кнопкой мыши интерфейс, к которому требуется добавить свойство.<br /><br /> Visual C++ определяет тип проекта и соответствующим образом изменяет параметры мастера добавления свойства.<br /><br /> — Для диспетчерских в проектах, созданных с помощью [мастер приложений MFC](../mfc/reference/mfc-application-wizard.md), вызов мастер добавления свойства предоставляет параметры, характерные для MFC.<br />-Для интерфейсов элемента управления ActiveX в MFC мастер добавления свойства предоставляет список стандартных методов и свойств, которые можно использовать в качестве или настроить для элемента управления.<br />-Для других интерфейсов мастера добавления свойств предоставляют параметры для большинства ситуаций.|  
|[Мастер добавления метода](../ide/add-method-wizard.md)|Проекты ATL и MFC, поддержка ATL. Щелкните правой кнопкой мыши интерфейс, к которому требуется добавить метод.<br /><br /> Visual C++ определяет тип проекта и соответствующим образом изменяет параметры мастера добавления метода.<br /><br /> — Для диспетчерских в проектах, созданных с помощью [мастер приложений MFC](../mfc/reference/mfc-application-wizard.md), мастер добавления метода вызова предоставляет параметры, характерные для MFC.<br />-Для интерфейсов элемента управления ActiveX в MFC мастер добавления метода предоставляет список стандартных методов и свойств, которые можно использовать в качестве или настроить для элемента управления.<br />-Для других интерфейсов **добавить метод** мастера предоставляют параметры наиболее значимые.|  
  
 Кроме того, можно реализовать новые интерфейсы для своего элемента управления COM, щелкнув правой кнопкой мыши класс объекта элемента управления в представлении классов команду [реализовать интерфейс](../ide/implement-interface-wizard.md).
