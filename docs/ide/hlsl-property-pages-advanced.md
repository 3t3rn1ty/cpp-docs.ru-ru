# <a name="hlsl-property-pages-advanced"></a>Страницы свойств HLSL: Дополнительно
Чтобы настроить следующие свойства компилятор HLSL (fxc.exe), используйте его **Дополнительно** страницу свойств. Сведения о доступе к **Дополнительно** см. на странице свойств в папке HLSL [работа со свойствами проекта](../ide/working-with-project-properties.md).  
  
## <a name="uielement-list"></a>Список элементов пользовательского интерфейса  
 **Отключить загрузочное объявление**  
 **Да (/ nologo)** для подавления вывода сообщения объявления, а также сведения о запуске; в противном случае **нет**. Значением по умолчанию является **Да (/ nologo)**.  
  
 **Обрабатывать предупреждения как ошибки**  
 **Да (/WX)** обрабатывать все предупреждения компилятора как ошибки; в противном случае **нет**.
