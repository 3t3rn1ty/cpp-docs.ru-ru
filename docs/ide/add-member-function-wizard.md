# <a name="add-member-function-wizard"></a>Мастер добавления функции-члена
Этот мастер добавляет объявления функции-члена в файл заголовка и заглушки реализации функции-члена в файл реализации для выбранного класса.  
  
 После добавления функции-члена с помощью мастера можно редактировать код в среде разработки.  
  
 **Return Type** (Возвращаемый тип)  
 Задает тип возвращаемого значения для добавляемой функции-члена. Можно указать собственный тип возвращаемого значения, или можно выбрать из списка доступных типов. Сведения о типах см. в разделе [базовые типы](../cpp/fundamental-types-cpp.md).  
  
||||  
|-|-|-|  
|`char`|`int`|`unsigned int`|  
|**double**|**long**|`unsigned long`|  
|**float**|**short**|`void`|  
|`HRESULT`|`unsigned char`||  
  
 **Имя функции**  
 Задает имя добавляемой функции-члена.  
  
 **Тип параметра**  
 Задает тип параметра, добавляемого для функции-члена, если функция-член имеет параметры. Вы можете предоставить свой собственный тип параметра или можно выбрать из списка доступных типов.  
  
||||  
|-|-|-|  
|`char`|`int`|`unsigned char`|  
|**double**|**long**|`unsigned int`|  
|**float**|**short**|`unsigned long`|  
  
 **Имя параметра**  
 Задает имя параметра, добавляемого для функции-члена, если функция-член имеет параметры.  
  
 **Список параметров**  
 Отображает список параметров, добавляемых к функции-члену. Чтобы добавить параметр в список, указать тип и имя в **тип параметра** и **имя параметра** и нажмите кнопку **добавить**. Чтобы удалить параметр из списка, выберите параметр и нажмите кнопку **удалить**.  
  
 **Доступ**  
 Задает уровень доступа к функции-члену. Модификаторы доступа указаны ключевые слова, указывающие уровень доступа других классов к функции-члену. В разделе [управления доступом к членам](../cpp/member-access-control-cpp.md) Дополнительные сведения об определении доступа. Имеет значение уровень доступа члена функции **открытый** по умолчанию.  
  
-   [public](../cpp/public-cpp.md)  
  
-   [protected](../cpp/protected-cpp.md)  
  
-   [private](../cpp/private-cpp.md)  
  
 Проверьте ли новый функция-член является статическим или виртуальный и является встроенной или чистого. Если значение быть чистыми, функция-член `Virtual` установлен флажок и **встроенного** станет недоступным. Значение по умолчанию — это функция невиртуальные нестатического члена.  
  
|Параметр|Описание:|  
|------------|-----------------|  
|[Статические](../cpp/storage-classes-cpp.md)|Указывает, что функция работает как глобальная и может быть вызван вне класса — даже без создания экземпляра класса. Функция-член не имеет доступа к нестатическим членам. Функция-член, указанный как `Static` не может быть виртуальным.|  
|[Виртуальный](../cpp/virtual-cpp.md)|Гарантирует, что правильный элемент функция вызывается для объекта, независимо от того, выражение, используемое для вызова функции члена. Функция-член, указанный как `Virtual` не может быть статическим.|  
|**Чистые**|Указывает, что нет реализация предоставлена для виртуальной функции-члена; Таким образом **чистый** может быть указан только для виртуальных функций-членов. Класс, содержащий хотя бы одну чисто виртуальной функции-члена, считается абстрактным классом. Классы, производные от абстрактного класса, должны реализовывать чисто виртуальной функции-члена или они, также будут абстрактными классами.|  
|[Встроенная](../cpp/inline-functions-cpp.md)|Предписывает компилятору вставить копию тела функции-члена в каждое место, где вызывается функция-член. Функция-член, указанный как **встроенного** не может быть чистым.|  
  
 **CPP-файл**  
 Задает расположение файла, куда будут записываться заглушки реализация функции-члена. По умолчанию она записывается в CPP-файл для класса, к которому добавляется функция-член. Нажмите кнопку с многоточием, чтобы изменить имя файла. Реализация функции-члена добавляется содержимое выбранного файла.  
  
 **Комментарий**  
 Предоставляет комментарий в файле заголовка для функции-члена.  
  
 **Сигнатура функции**  
 Отображает функцию-член, отображенное в коде при нажатии кнопки **Готово**. Текст в этом поле нельзя изменить. Чтобы изменить функцию-член, измените соответствующие поля в мастере.
