# Страница свойств веб-ссылок
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

На странице свойств **Веб\-ссылки** указывается, каким образом будет создан прокси\-класс веб\-службы XML.  Прокси\-класс веб\-службы XML будет создан, если добавить веб\-ссылку в проект.  Дополнительные сведения см. в разделе [Добавление веб\-ссылки](http://msdn.microsoft.com/ru-ru/bdf05776-c591-40af-bfd7-e1e2aa1e87b5).  
  
 Страница свойств **Веб\-ссылки** содержит следующие свойства:  
  
 **Выходной файл**  
 Имя файла, в котором содержится прокси\-класс веб\-службы XML.  
  
 **Отключить отображение приветствия при загрузке**  
 Запрет на отображение объявления для средства языка WSDL \(Wsdl.exe\).  
  
 **Пространство имен**  
 Задает имя создаваемого веб\-прокси.  
  
 **Дополнительные ссылки**  
 Задает дополнительные библиотеки DLL, на которые ссылается библиотека DLL прокси.  
  
 Сведения о том, как получить доступ к странице свойств **Веб\-ссылки**, см. в разделе [Практическое руководство. Задание свойств проекта при помощи страниц свойств](../Topic/How%20to:%20Specify%20Project%20Properties%20with%20Property%20Pages.md).
