# <a name="web-references-property-page"></a>Страница свойств веб-ссылок
**Веб-ссылки** страница свойств задает, каким образом будет создан прокси-класс XML-веб-службы. Класс прокси XML-веб-службы создается при добавлении веб-ссылки в проект. В разделе [Добавление веб-ссылки](http://msdn.microsoft.com/en-us/bdf05776-c591-40af-bfd7-e1e2aa1e87b5) Дополнительные сведения.  
  
 **Веб-ссылки** страница свойств содержит следующие свойства:  
  
 **Выходной файл**  
 Имя файла, который содержит класс XML-веб-службы прокси-сервера.  
  
 **Отключить загрузочное объявление**  
 Отображение объявления для средства языка описания веб-служб (Wsdl.exe).  
  
 **Пространство имен**  
 Задает имя создаваемого веб-прокси.  
  
 **Дополнительные ссылки**  
 Задает дополнительные библиотеки DLL, ссылаться на DLL прокси-сервера.  
  
 Сведения о доступе к **веб-ссылки** страницу свойств в разделе [работа со свойствами проекта](../ide/working-with-project-properties.md).
