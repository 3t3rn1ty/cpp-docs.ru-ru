# Страницы свойств командной строки
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Большинство папок страниц свойств содержит страницу свойств **Командная строка**.  На этой странице отображаются сведения, какие свойства установлены в папке.  Страница свойств **Командная строка** содержит также поле **Дополнительные параметры**, в котором можно задать свойства, действительные для инструмента, для которого нет свойства в папке.  
  
 Команда, которая вводится в поле редактирования, будут передана в инструмент для папки.  На входе не будет проводиться никаких проверок, здесь также не будет выполняться проверка зависимости.  
  
 Сведения о том, как получить доступ на страницы свойств **Командная строка**, см. в разделе [Практическое руководство. Задание свойств проекта при помощи страниц свойств](../Topic/How%20to:%20Specify%20Project%20Properties%20with%20Property%20Pages.md).
