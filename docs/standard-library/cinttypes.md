# &lt;cinttypes&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Включает заголовок стандартной библиотеки C \<inttypes.h\>.  При включении этого заголовка также включается \<cstdint\>.  
  
## Синтаксис  
  
```  
  
#include <cinttypes>  
  
```  
  
## Заметки  
 Включение этого заголовка гарантирует, что имена, объявленные путем использования внешней компоновки в заголовке стандартной библиотеки C, объявляются в пространстве имен `std`.
