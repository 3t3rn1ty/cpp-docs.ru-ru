# <a name="container-classconstiterator"></a>Класс контейнера::const_iterator
> [!NOTE]
>  Данный раздел включен в документацию Visual C++ в качестве нефункционального примера контейнеров, используемых в стандартной библиотеке C++. Дополнительные сведения см. в разделе [Контейнеры стандартной библиотеки C++](../standard-library/stl-containers.md).  
  
 Описывает объект, который можно использовать в качестве постоянного итератора для управляемой последовательности.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
 
typedef T6 const_iterator;  
```  
  
## <a name="remarks"></a>Примечания  
 Он описан здесь как синоним для неопределенного типа **T6**.
