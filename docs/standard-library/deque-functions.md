## <a name="swap"></a>  swap

Меняет местами элементы двух объектов deque.

```cpp
void swap(
    deque<Type, Allocator>& left,
    deque<Type, Allocator>& right,);
```

### <a name="parameters"></a>Параметры

*слева* объект типа `deque`.

*правом* объект типа `deque`.
