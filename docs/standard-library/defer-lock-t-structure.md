# <a name="deferlockt-structure"></a>Структура defer_lock_t

Представляет тип, который определяет объект [defer_lock](../standard-library/mutex-functions.md#defer_lock), используемый для выбора одного из перегруженных конструкторов [unique_lock](../standard-library/unique-lock-class.md).

## <a name="syntax"></a>Синтаксис

```cpp
struct defer_lock_t;
```

## <a name="requirements"></a>Требования

**Заголовок:** \<мьютекс >

**Пространство имен:** std
