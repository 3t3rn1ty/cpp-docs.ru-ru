# Структура defer_lock_t
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Представляет тип, определяющий объект [defer\_lock](../Topic/defer_lock%20Variable.md), который используется для выбора одного из перегруженных конструкторов [unique\_lock](../standard-library/unique-lock-class.md).  
  
## Синтаксис  
  
```cpp  
struct defer_lock_t;  
```  
  
## Требования  
 **Заголовок:** mutex  
  
 **Пространство имен:** std
