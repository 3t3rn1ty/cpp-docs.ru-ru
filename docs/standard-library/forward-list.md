# &lt;forward_list&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Определяет forward\_list шаблона класса контейнера и поддерживает несколько шаблонов.  
  
## Синтаксис  
  
```  
#include <forward_list>  
```  
  
### Операторы  
  
|||  
|-|-|  
|[operator\=\= \(\<forward\_list\>\)](../Topic/operator==%20\(%3Cforward_list%3E\).md)|Если тесты передний объекта списка в левой части оператора равно переднему объекту списка в правой части.|  
|[оператор\! \= \(\<forward\_list\>\)](../Topic/operator!=%20\(%3Cforward_list%3E\).md)|Если тесты передний объекта списка в левой части оператора не равен переднему объекту списка в правой части.|  
|[оператор\< \(\<forward\_list\>\)](../Topic/operator%3C%20\(%3Cforward_list%3E\).md)|Если тесты передний объекта списка слева от оператора, чем передний объекта списка в правой части.|  
|[operator\=\<\(\<forward\_list\>\)](../Topic/operator%3C=%20\(%3Cforward_list%3E\).md)|Если тесты передний объекта списка в левой части оператора меньше или равно переднему объекту списка в правой части.|  
|[оператор\> \(\<forward\_list\>\)](../Topic/operator%3E%20\(%3Cforward_list%3E\).md)|Если тесты передний объекта списка в левой части оператора больше передний объекта списка в правой части.|  
|[operator\=\>\(\<forward\_list\>\)](../Topic/operator%3E=%20\(%3Cforward_list%3E\).md)|Если тесты передний объекта списка в левой части оператора больше или равна переднему объекту списка в правой части.|
