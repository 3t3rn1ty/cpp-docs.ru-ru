# Класс regex_traits&lt;wchar_t&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Специализация regex\_traits для wchar\_t.  
  
## Синтаксис  
  
```  
template <>  
    class regex_traits<wchar_t>  
```  
  
## Заметки  
 Этот класс является явной специализацией шаблона класса [Класс regex\_traits](../standard-library/regex-traits-class.md) для элементов типа `wchar_t` \(благодаря чему он может использовать преимущества библиотечных функций, манипулирующих объектами данного типа\).  
  
## Требования  
 **Заголовок:** \<regex\>  
  
 **Пространство имен:** std
