# Класс контейнера::reverse_iterator
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

> [!NOTE]
>  В этом разделе в документации Visual C\+\+ как нефункциональный пример контейнеров, используемых в стандартной библиотеке C\+\+.  Дополнительные сведения см. в разделе [Контейнеров STL](../standard-library/stl-containers.md).  
  
 Описание объекта, который можно использовать в качестве обратного итератор для контролируемой последовательности.  
  
## Синтаксис  
  
```  
  
typedef T7 reverse_iterator;  
  
```  
  
## Заметки  
 Он описанный здесь синонимом неспецифицированного типа **T7** \(обычно `reverse_iterator` **\<**[iterator](../standard-library/container-class-iterator.md)**\>**\).
