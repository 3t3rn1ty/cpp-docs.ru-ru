# <a name="ltcvtwbuffergt"></a>&lt;cvt/wbuffer&gt;

Заголовок `<cvt/wstring>` в предыдущих версиях Visual Studio определял класс шаблона [wbuffer_convert](../standard-library/wbuffer-convert-class.md) в пространстве имен stdext::cvt. Заголовок сохраняется для обеспечения обратной совместимости. В новом коде следует использовать версию класса, определенную в параметре [\<locale>](../standard-library/locale.md) в пространстве имен `std`.

## <a name="syntax"></a>Синтаксис

```cpp
#include <cvt/wbuffer>

```
