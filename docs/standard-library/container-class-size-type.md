# Класс контейнера::size_type
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

> [!NOTE]
>  В этом разделе в документации Visual C\+\+ как нефункциональный пример контейнеров, используемых в стандартной библиотеке C\+\+.  Дополнительные сведения см. в разделе [Контейнеров STL](../standard-library/stl-containers.md).  
  
 Описывает объект, который может представлять любой длины контролируемой последовательности.  
  
## Синтаксис  
  
```  
  
typedef T0 size_type;  
  
```  
  
## Заметки  
 Оно описана ниже синонимом типа обычно неспецифицированного **T0** \( **Alloc::size\_type**\).
