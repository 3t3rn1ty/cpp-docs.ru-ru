# Класс контейнера::const_reference
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

> [!NOTE]
>  В этом разделе в документации Visual C\+\+ как нефункциональный пример контейнеров, используемых в стандартной библиотеке C\+\+.  Дополнительные сведения см. в разделе [Контейнеров STL](../standard-library/stl-containers.md).  
  
 Описание объекта, который можно использовать в качестве константной ссылки на элемент контролируемой последовательности.  
  
## Синтаксис  
  
```  
  
typedef T3 const_reference;  
  
```  
  
## Заметки  
 Оно описана ниже синонимом типа обычно неспецифицированного **T3** \( **Alloc::const\_reference**\).
