# Структура char_traits&lt;char32_t&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Структура, которая специализация шаблона структуры **char\_traits\<CharType\>** с элементом типа `char32_t`.  
  
## Синтаксис  
  
```  
template<> struct char_traits<char32_t>;  
```  
  
## Заметки  
 Специализация позволяет структура для использования функций библиотеки, управляющих объектов этого типа `char32_t`.  
  
## Требования  
 **Заголовок:**\<string\>  
  
 **Пространство имен:** std
