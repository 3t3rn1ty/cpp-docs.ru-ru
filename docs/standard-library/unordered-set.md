# &lt;unordered_set&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Определяет шаблонные классы [unordered\_multiset](../standard-library/unordered-multiset-class.md) и [unordered\_set](../standard-library/unordered-set-class.md) контейнера и их поддерживает шаблоны.  
  
## Синтаксис  
  
```  
#include <unordered_set>  
```  
  
### Классы  
  
|||  
|-|-|  
|[Класс unordered\_multiset](../standard-library/unordered-multiset-class.md)|Хранит хэш\-таблица ключей.|  
|[Класс unordered\_set](../standard-library/unordered-set-class.md)|Хранит хэш\-таблица ключей.|  
  
### Функции  
  
|||  
|-|-|  
|[\!\= \- оператор \(unordered\_multiset\)](../Topic/operator!=%20\(unordered_multiset\).md)|Тесты, если объект unordered\_multiset слева от оператора не равен объекту unordered\_multiset в правой части.|  
|[\=\= \- оператор \(unordered\_multiset\)](../Topic/operator==%20\(unordered_multiset\).md)|Тесты, если объект unordered\_multiset слева от оператора равно объекту unordered\_multiset в правой части.|  
|[Функция swap \(unordered\_multiset\)](../Topic/swap%20Function%20\(unordered_multiset\).md)|Обмен multisets 2.|  
|[\!\= \- оператор \(unordered\_set\)](../Topic/operator!=%20\(unordered_set\).md)|Тесты, если объект unordered\_set слева от оператора не равен объекту unordered\_set в правой части.|  
|[\=\= \- оператор \(unordered\_set\)](../Topic/operator==%20\(unordered_set\).md)|Тесты, если объект unordered\_set слева от оператора равно объекту unordered\_set в правой части.|  
|[Функция swap \(unordered\_set\)](../Topic/swap%20Function%20\(unordered_set\).md)|Обмен 2 набора.|
