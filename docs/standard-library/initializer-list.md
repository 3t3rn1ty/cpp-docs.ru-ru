# &lt;initializer_list&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Определяет класс шаблона контейнера `initializer_list` и несколько вспомогательных шаблонов.  
  
## Синтаксис  
  
```  
#include <initializer_list>  
```  
  
### Классы  
  
|||  
|-|-|  
|[initializer\_list](../standard-library/initializer-list-class.md)|Предоставляет доступ к массиву элементов, в которых каждый элемент указанного типа.|
