# <a name="lttypeinfogt"></a>&lt;typeInfo&gt;

Включите стандартный заголовок \<typeinfo > для определения нескольких типов, связанной с оператором код типа [оператор typeid](../cpp/typeid-operator.md), какая из них обеспечивает сведения о статических и динамических типах.

## <a name="syntax"></a>Синтаксис

```cpp
#include <typeinfo>

```
