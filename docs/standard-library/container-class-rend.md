# Класс контейнера::rend
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

> [!NOTE]
>  В этом разделе в документации Visual C\+\+ как нефункциональный пример контейнеров, используемых в стандартной библиотеке C\+\+.  Дополнительные сведения см. в разделе [Контейнеров STL](../standard-library/stl-containers.md).  
  
 Функция\-член возвращает обратный итератор, указывающим на первый элемент последовательности \(или просто за пределы пустой последовательности\), обозначая обратной конец последовательности.  
  
## Синтаксис  
  
```  
  
      const_reverse_iterator rend( ) const;   
reverse_iterator rend( );  
```
