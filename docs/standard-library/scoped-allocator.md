# &lt;scoped_allocator&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Определяет scoped\_allocator шаблона класса контейнера.  
  
## Синтаксис  
  
```cpp  
#include <scoped_allocator>  
```  
  
### Операторы  
  
|||  
|-|-|  
|[operator\!\=](../Topic/operator!=%20Operator%20\(%3Cscoped_allocator%3E\).md)|Тесты, если объект scoped\_allocator слева от оператора не равен объекту списка в правой части.|  
|[operator\=\=](../Topic/operator==%20Operator%20\(%3Cscoped_allocator%3E\).md)|Тесты, если объект scoped\_allocator слева от оператора равно объекту списка в правой части.|  
  
### Классы  
  
|||  
|-|-|
