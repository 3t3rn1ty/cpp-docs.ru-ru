# <a name="ltcvtwstringgt"></a>&lt;cvt/wstring&gt;
Заголовок `<cvt/wstring>` в предыдущих версиях Visual Studio определял класс шаблона [wstring_convert Class](../standard-library/wstring-convert-class.md) в пространстве имен stdext::cvt. Заголовок сохраняется для обеспечения обратной совместимости. В новом коде следует использовать версию класса, определенную в параметре [\<locale>](../standard-library/locale.md) в пространстве имен `std`.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
#include <cvt/wstring>  
  
```
