# &lt;cvt/wstring&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Заголовок `<cvt/wstring>` в предыдущих версиях Visual Studio определял класс шаблона [Класс wstring\_convert](../standard-library/wstring-convert-class.md) в пространстве имен stdext::cvt.  Заголовок сохраняется для обеспечения обратной совместимости.  В новом коде следует использовать версию класса, определенную в параметре [\<locale\>](../standard-library/locale.md) в пространстве имен `std`.  
  
## Синтаксис  
  
```  
  
#include <cvt/wstring>  
  
```
