# regex_traits &lt; char &gt; класса
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Специализация regex\_traits для char.  
  
## Синтаксис  
  
```  
template <>  
    class regex_traits<char>  
```  
  
## Заметки  
 Класс является явной специализации шаблона класса [Класс regex\_traits](../standard-library/regex-traits-class.md) для элементов типа `char` \(так, что он может использовать преимущества библиотечных функций, управлять объектами данного типа\).  
  
## Требования  
 **Заголовок:** \<regex\>  
  
 **Пространство имен:** std
