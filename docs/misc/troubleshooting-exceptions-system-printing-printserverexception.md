# Разрешение вопросов, связанных с исключениями: System.Printing.PrintServerException
Исключение <xref:System.Printing.PrintServerException> возникает, когда из–за ошибочного условия запрещены доступ к <xref:System.Printing.PrintServer> или его создание.
