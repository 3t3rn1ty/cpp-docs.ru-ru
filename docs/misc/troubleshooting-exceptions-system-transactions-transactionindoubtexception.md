# Разрешение вопросов, связанных с исключениями: System.Transactions.TransactionInDoubtException
Исключение вызывается при попытке выполнить "сомнительную" транзакцию.  
  
## Примечания  
 Транзакция находится в состоянии "сомнительная", если нельзя связаться с назначенным диспетчером ресурсов.
