# Разрешение вопросов, связанных с исключениями: System.ServiceModel.ActionNotSupportedException
Исключение <xref:System.ServiceModel.ActionNotSupportedException> обычно генерируется на стороне клиента, когда действие, связанное с вызываемой операцией не соответствует никакому действию операций на сервере.
