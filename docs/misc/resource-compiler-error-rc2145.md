# Ошибка компилятора ресурсов RC2145
PRIMARY LANGUAGE ID слишком велик  
  
 PRIMARY LANGUAGE ID должен быть допустимым шестнадцатеричным идентификатором языка. Допустимые идентификаторы языков см. в разделе [Языки и настройки для стран и регионов](../c-runtime-library/locale-names-languages-and-country-region-strings.md) в *справочнике по библиотеке среды выполнения*.
