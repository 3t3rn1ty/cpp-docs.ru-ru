# Разрешение вопросов, связанных с исключениями: System.Windows.Controls.PrintDialogException
Исключение <xref:System.Windows.Controls.PrintDialogException> вызывается, если ошибка возникает при открытии, использовании или обращении к <xref:System.Windows.Controls.PrintDialog>.
