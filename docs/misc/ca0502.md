# CA0502
Неподдерживаемый язык  
  
 Эта ошибка возникает, если пользователь пытается выполнить анализ проекта ASP.NET, содержащего одну или несколько страниц, написанных на языках, отличных от C\# и Visual Basic.  Чтобы устранить эту ошибку, удалите файл из проекта или отключите анализ кода.
