# Разрешение вопросов, связанных с исключениями: System.ComponentModel.Design.Serialization.CodeDomSerializerException
Исключение, которое возникает, если в преобразователе CodeDom произошла ошибка.  
  
## Примечания  
 Это исключение позволяет сериализатору возвращать сведения о расположении и типе ошибки.
