# Разрешение вопросов, связанных с исключениями: System.Printing.PrintCommitAttributesException
Исключение <xref:System.Printing.PrintCommitAttributesException> возникает, когда ошибочное условие не позволяет некоторым атрибутам быть переданными <xref:System.Printing.PrintSystemObject> фактическому компьютеру, принтеру или другому устройству, которое представляет объект.
