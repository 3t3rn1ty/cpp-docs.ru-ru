# Разрешение вопросов, связанных с исключениями: System.IdentityModel.Selectors.IdentityValidationException
Исключение <xref:System.IdentityModel.Selectors.IdentityValidationException> вызывается, когда сертификат получателя недействителен. Это может произойти, если не существует цепочки от этого сертификата до корневого.
