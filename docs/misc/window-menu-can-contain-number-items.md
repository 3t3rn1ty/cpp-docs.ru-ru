# Window menu can contain &lt;number&gt; items.
Список окон в меню "Окно" может отображать не более двадцати четырех имен.  Эта ошибка обычно возникает, когда в поле **Пунктов в меню "Окно: \<число\>** панели "Общие" параметров "Среда" диалогового окна "Параметры" было введено число меньше единицы или больше двадцати четырех.  
  
### Чтобы исправить эту ошибку  
  
1.  Введите в этом поле число от единицы до двадцати четырех.
