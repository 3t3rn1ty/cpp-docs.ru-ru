# Most recently used list can contain &lt;number&gt; items.
Списки MRU могут содержать до двадцати четырех элементов.  Эта ошибка обычно возникает, когда в поле **Число пунктов в списке недавно использованных элементов: \<число\>** области "Общие" в параметрах "Среда" диалогового окна "Параметры" было введено число меньше единицы или больше двадцати четырех.  
  
### Чтобы исправить эту ошибку  
  
1.  Введите в этом поле число от единицы до двадцати четырех.
