# CA0069
UsingAlternatePlatform.  Не удалось найти *FrameworkVersion1*.  Использование *FrameworkVersion2* вместо этого.  Для получения оптимальных результатов анализа убедитесь, что на компьютере установлена правильная версия платформы .NET Framework.
