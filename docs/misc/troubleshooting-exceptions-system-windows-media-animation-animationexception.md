# Разрешение вопросов, связанных с исключениями: System.Windows.Media.Animation.AnimationException
Исключение <xref:System.Windows.Media.Animation.AnimationException> вызывается при возникновении ошибки во время анимации свойства.  
  
## Примечания  
 Этот класс не используется для ошибок, связанных с временными значениями, таких, как создание анимации с отрицательным значением <xref:System.Windows.Media.Animation.Timeline.Duration%2A> или недопустимыми значениями параметров <xref:System.Windows.Media.Animation.Timeline.AccelerationRatio%2A> и <xref:System.Windows.Media.Animation.Timeline.DecelerationRatio%2A>.
