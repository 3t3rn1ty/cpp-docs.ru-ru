# Command allows at most one argument.
Эта ошибка обычно возникает, когда для команды, которой не нужны аргументы или нужен только один аргумент, было введено слишком много аргументов.  
  
### Чтобы исправить эту ошибку  
  
1.  Проверьте в документации правильность синтаксиса команды и повторите попытку.
