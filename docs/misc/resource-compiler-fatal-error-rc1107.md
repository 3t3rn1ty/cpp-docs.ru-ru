# Неустранимая ошибка компилятора ресурсов RC1107
недопустимое использование; используйте RC \/? для справки  
  
 Указан недопустимый параметр компилятора ресурсов. Допустимые параметры: \/r, \/v, \/d, \/fo, \/l, \/i, \/x и \/c.
