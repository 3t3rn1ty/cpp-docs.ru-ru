# Ошибка MSBuild MSB8012
*{0}\(\\{1\\}\)* не соответствует значению свойства OutputFile \(*\\{3\\}*\) объекта *\\{2\\}*. Это может привести к неправильной сборке проекта. Чтобы устранить ошибку, убедитесь, что значения свойств $\(OutDir\), $\(TargetName\) и $\(TargetExt\) соответствуют значению, указанному в %\(\\{4\\}.OutputFile\).
