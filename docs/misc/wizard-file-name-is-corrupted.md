# Wizard file &lt;name&gt; is corrupted.
Неправильный формат файла VSZ для мастера.  Эта ошибка возникает, если файл VSZ поврежден или не относится к Visual Studio.  
  
### Чтобы исправить эту ошибку  
  
1.  Убедитесь, что этот мастер является мастером Visual Studio.  
  
     либо  
  
     Переустановите Visual Studio, чтобы заменить поврежденный файл.
