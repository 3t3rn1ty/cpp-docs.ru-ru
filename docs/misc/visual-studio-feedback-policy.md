# Политика отзывов о Visual Studio
Благодарим вас за отзыв.  Группы разработчиков документации используют отзывы для повышения качества своей работы.  В ходе рассмотрения вашего отзыва вы можете получить от нас по электронной почте просьбу уточнить отзыв или прокомментировать решение.  Ваш адрес электронной почты не используется для каких\-либо других целей. После рассмотрения отзыва он будет удален.  Дополнительные сведения о политиках конфиденциальности корпорации Майкрософт см. в разделе [http:\/\/privacy.microsoft.com\/default.aspx](http://privacy.microsoft.com/default.aspx).
