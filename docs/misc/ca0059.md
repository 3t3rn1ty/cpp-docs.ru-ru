# CA0059
Ошибка параметра командной строки.  
  
 Данная ошибка возникает в том случае, если отсутствует обязательный параметр командной строки, например  
  
 не указан параметр \/out или \/console.  Эта ошибка также может происходить при наличии недопустимого параметра, например неверного значения времени ожидания.
