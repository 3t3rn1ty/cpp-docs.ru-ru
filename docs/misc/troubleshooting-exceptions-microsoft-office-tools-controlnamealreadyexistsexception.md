# Разрешение вопросов, связанных с исключениями: Microsoft.Office.Tools.ControlNameAlreadyExistsException
Исключение <xref:Microsoft.Office.Tools.ControlNameAlreadyExistsException> генерируется при попытке программного создания элемента управления с именем, которое уже было присвоено другому элементу управления в проекте.  
  
## Полезные советы  
 **Используйте другое имя для одного из элементов управления.**  
 Имена элементов управления должны быть уникальными.
