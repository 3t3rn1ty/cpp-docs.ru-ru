# Разрешение вопросов, связанных с исключениями: Microsoft.VisualStudio.Tools.Applications.Runtime.UpdateFailedException
Исключение вызывается, когда манифест развертывания ссылается на манифест приложения иной версии, чем в документе, но при этом манифест приложения в документе не может быть обновлен.
