# Разрешение вопросов, связанных с исключениями: System.Xml.Xsl.XsltCompileException
Исключение вызывается методом <xref:System.Xml.Xsl.XslCompiledTransform.Load%2A>, если в таблице стилей Extensible Stylesheet Language Transformation \(XSLT\) обнаружена ошибка.
