# Неустранимая ошибка компилятора ресурсов RC1049
Недопустимый числовой аргумент "строка"  
  
 Компилятор ресурсов ожидал числовой аргумент, но получил указанную строку.  
  
 Эта ошибка может быть вызвана предоставлением шестнадцатеричного числа без нужного префикса \\x или числом с плавающей запятой в неправильном формате.
