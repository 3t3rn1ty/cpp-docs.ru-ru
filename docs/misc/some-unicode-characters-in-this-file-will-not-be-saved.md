# Some Unicode characters in this file will not be saved.
Эта ошибка обычно возникает, если файл содержит символы, которые могут быть сохранены только с помощью кодовой страницы Юникода.  
  
### Сохранение файла с символами Юникода  
  
1.  Нажмите кнопку **Отмена**.  
  
2.  В меню **Файл** выберите команду **Дополнительные параметры сохранения**.  
  
3.  В диалоговом окне **Дополнительные параметры сохранения** выберите кодировку, соответствующую вашему файлу.
