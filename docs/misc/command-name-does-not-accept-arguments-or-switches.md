# Command &lt;name&gt; does not accept arguments or switches.
Эта ошибка обычно возникает, когда с командой или псевдонимом, не использующим ни переключателей, ни аргументов, был введен переключатель или аргумент.  Если был введен псевдоним, в сообщении об ошибке появится полное имя команды, а не имя псевдонима.  Список допустимых команд, использующих переключатели и аргументы, см. в разделе [Команды Visual Studio](../Topic/Visual%20Studio%20Commands.md).  
  
### Чтобы исправить эту ошибку  
  
1.  Повторно введите команду и аргументы или введите новую команду.
