# Ошибка MSBuild MSB3110
**MSB3110: в сборке "\<сборка\>" имеется несогласованное удостоверение "\<удостоверение\_сборки\>", ожидаемое имя файла: "\<файл\>".**  
  
 Эта ошибка появляется, когда имя файла, полученное из удостоверения сборки, не соответствует реальному имени файла.  
  
### Чтобы исправить эту ошибку  
  
-   Исправьте имя файла сборки, чтобы оно соответствовало удостоверению сборки.
