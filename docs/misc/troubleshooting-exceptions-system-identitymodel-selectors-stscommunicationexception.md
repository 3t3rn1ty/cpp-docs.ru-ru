# Разрешение вопросов, связанных с исключениями: System.IdentityModel.Selectors.StsCommunicationException
Исключение <xref:System.IdentityModel.Selectors.StsCommunicationException> генерируется в случае ошибки связи со службой маркеров безопасности.  
  
 Причины проблемы могут быть следующими:  
  
-   Сбой в сети.  
  
-   Не удалось достичь конечной точки MEX.  
  
-   У службы маркеров безопасности отсутствует допустимый идентификатор конечной точки.
