# Разрешение вопросов, связанных с исключениями: System.IO.Log.ReservationNotFoundException
Исключение <xref:System.IO.Log.ReservationNotFoundException> возникает, когда резервирование конкретного места в <xref:System.IO.Log.LogRecordSequence> не было найдено.
