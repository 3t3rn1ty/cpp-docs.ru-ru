# Directory or file specified in Look In is not available or does not exist.
Эта ошибка обычно возникает, если не удается найти путь или имя, указанные в списке обзора.  
  
### Чтобы исправить эту ошибку  
  
1.  Заново введите путь и имя файла.  
  
     —или—  
  
2.  Перейдите к правильному файлу или папке, используя список "Обзор" и поле **Доступные элементы**.
