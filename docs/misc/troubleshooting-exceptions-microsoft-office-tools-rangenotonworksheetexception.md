# Разрешение вопросов, связанных с исключениями: Microsoft.Office.Tools.RangeNotOnWorksheetException
Исключение <xref:Microsoft.Office.Tools.RangeNotOnWorksheetException> генерируется, если при программном создании элемента представления используется диапазон, не соответствующий листу коллекции элементов управления.  
  
## Полезные советы  
 Убедитесь, что заданный диапазон соответствует правильному листу.  
 Исключение генерируется, когда элемент представления создается программным способом, и при этом связывается с диапазоном не из того листа, который соответствует коллекции элементов управления.
