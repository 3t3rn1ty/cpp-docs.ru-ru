# Ошибка компилятора ресурсов RC2126
Ожидается строка меню  
  
 Каждый оператор **MENUITEM** и **POPUP** должен содержать поле *text*, которое является строкой, заключенной в кавычки и указывающей имя пункта основного или всплывающего меню. Для оператора **MENUITEM SEPARATOR** требуется строка без кавычек.
