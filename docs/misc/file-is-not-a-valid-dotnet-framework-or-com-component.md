# File is not a valid .NET Framework or COM component
Предпринята попытка просмотреть в обозревателе объектов файл, который не содержит допустимого объекта.  В обозревателе объектов можно просматривать только компоненты [!INCLUDE[dnprdnshort](../error-messages/tool-errors/includes/dnprdnshort_md.md)] и COM.  
  
### Чтобы исправить эту ошибку  
  
1.  Если был выбран не тот файл, выберите нужный файл и повторите попытку.  
  
2.  Если обозреватель объектов был ошибочно выбран в диалоговом окне **Открыть с помощью**, выберите другую программу для открытия файла.
