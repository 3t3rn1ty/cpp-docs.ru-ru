# Ошибка сборки проекта PRJ0051
Не удалось скопировать "файл" в целевой каталог.  Убедитесь, что целевой файл не используется.  
  
 Системе сборки не удалось скопировать файл.  
  
 Убедитесь, что файл не открыт на вашем компьютере и не используется другой программой.
