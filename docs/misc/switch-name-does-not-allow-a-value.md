# Switch &lt;name&gt; does not allow a value.
Эта ошибка обычно возникает при присвоении значения параметру, которому нельзя присваивать значение.  
  
### Чтобы исправить эту ошибку  
  
1.  Введите команду еще раз, опустив значение параметра.
