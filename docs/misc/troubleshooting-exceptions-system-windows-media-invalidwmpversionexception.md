# Разрешение вопросов, связанных с исключениями: System.Windows.Media.InvalidWmpVersionException
Исключение <xref:System.Windows.Media.InvalidWmpVersionException> возникает, когда установленная версия проигрывателя Windows Media не поддерживается.
