# Ошибка вычислителя выражений CXX0003
внутренняя ошибка при вычислении выражения  
  
 В процессе вычисления выражения произошла внутренняя ошибка.  
  
 Соберите сведения об обстоятельствах появления ошибки и обратитесь в корпорацию Майкрософт. Вы можете сообщить об этой ошибке в Интернете по адресу [http:\/\/support.microsoft.com\/support\/visualc\/report\/](http://support.microsoft.com/support/visualc/report/) или позвонить в службу технической поддержки корпорации Майкрософт. Чтобы получить дополнительные сведения, выберите пункт **Техническая поддержка** в меню **Справка** в Visual C\+\+.  
  
 Эта ошибка идентична ошибке CAN0003.
