# Load of property &#39;property&#39; failed. &lt;reason&gt;
Значение свойства находится вне диапазона допустимых значений.  
  
 Эта ошибка часто возникает при редактировании файла проекта вручную.  
  
 **Чтобы исправить эту ошибку**  
  
-   Отредактируйте файл проекта и вручную устраните ошибку.  
  
     Указанные свойства примут значения по умолчанию — те значения, которые они получили при создании проекта.
