# Не удается создать папку, поскольку уже существует файл, имеющий такой же путь
Предпринята попытка создать каталог при наличии файла, уже имеющего такой же путь. Это может быть результатом неправильного анализа пути, особенно при работе с относительными путями.  
  
### Исправление ошибки  
  
-   Убедитесь в том, что вы указали правильный путь.  
  
-   Удалите лишний файл, если он больше не нужен.
