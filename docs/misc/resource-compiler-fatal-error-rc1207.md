# Неустранимая ошибка компилятора ресурсов RC1207
недопустимая кодовая страница по умолчанию  
  
 Кодовая страница, указанная переменной среды **RCCODEPAGE**, является недопустимой. Дополнительные сведения см. в разделе [Кодовые страницы](../c-runtime-library/code-pages.md)*справочника по библиотеке времени выполнения*.
