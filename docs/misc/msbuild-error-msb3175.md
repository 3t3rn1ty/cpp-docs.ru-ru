# Ошибка MSBuild MSB3175
**MSB3175: недопустимое значение для "\<файл\>" элемента "\<задача построения\>".**  
  
 Это предупреждение появляется, когда процесс построения не распознает значение различных перечислимых свойств элемента задачи, таких как `AssemblyType`, `DependencyType`, `FileType` или `TargetZone`.  Это справедливо как для манифестов приложений, так и для манифестов развертывания.
