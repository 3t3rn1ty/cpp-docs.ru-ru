# Разрешение вопросов, связанных с исключениями: System.ServiceModel.MsmqException
Исключение <xref:System.ServiceModel.MsmqException> инкапсулирует ошибки, возвращаемые службой очередей сообщений. Это исключение генерируется транспортом службы очередей сообщений и каналом интеграции службы очередей сообщений.
