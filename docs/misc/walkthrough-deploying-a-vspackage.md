# Пошаговое руководство. Развертывание пакета VSPackage
Начиная с Visual Studio 2010 для развертывания пакета VSPackage следует использовать технологию VSIX. Дополнительные сведения о VSIX см. в разделе [Расширения Visual Studio доставки](../Topic/Shipping%20Visual%20Studio%20Extensions.md).
