# CA0504
Каталог правил пропущен, поскольку он не существует  
  
 Предупреждение CA504 создается, если не удается найти каталог, созданный в свойстве **CodeAnalysisRuleDirectories**, проекте или файле конфигурации построения.  Для устранения этой проблемы, введите действительный путь к каталогу.
