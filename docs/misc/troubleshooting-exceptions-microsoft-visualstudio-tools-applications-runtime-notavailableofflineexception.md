# Разрешение вопросов, связанных с исключениями: Microsoft.VisualStudio.Tools.Applications.Runtime.NotAvailableOfflineException
Исключение генерируется, когда предпринимается попытка вызвать сборку, но сборка в кэше или в сети недоступна.
