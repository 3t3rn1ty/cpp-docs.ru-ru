# Разрешение вопросов, связанных с исключениями: System.Workflow.Activities.Rules.RuleSetValidationException
Исключение <xref:System.Workflow.Activities.Rules.RuleSetValidationException> возникает, когда обработка не может продолжаться, так как набор правил не может быть проверен. Исключение вызывается конструктором <xref:System.Workflow.Activities.Rules.RuleEngine>, если используемый <xref:System.Workflow.Activities.Rules.RuleSet> имеет ошибки проверки.
