# Разрешение вопросов, связанных с исключениями: System.ServiceModel.Dispatcher.MultipleFilterMatchesException
Исключение <xref:System.ServiceModel.Dispatcher.MultipleFilterMatchesException> создается в случае, если найдено несколько подходящих фильтров, а ожидался только один.
