# Package &lt;name&gt; failed to load.
Не удается открыть пакет.  Эта ошибка обычно возникает, если пакет не был правильно зарегистрирован, в пакете отсутствует DLL, DLL повреждена, отсутствует DLL, которую данному пакету должен предоставить другой компонент.  
  
### Чтобы исправить эту ошибку  
  
1.  Переустановите продукт, предоставляющий пакет.
