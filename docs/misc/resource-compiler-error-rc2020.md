# Ошибка компилятора ресурсов RC2020
недопустимая цифра для основания системы счисления  
  
 Указанная `digit` не является допустимой цифрой для основания, указанного `radix`. Возможно, неправильно указана цифра или основание.  
  
 Восьмеричные цифры должны быть числами от 0 до 7, а шестнадцатеричные цифры должны быть числами от 0 до 9 или буквами от A до E.
