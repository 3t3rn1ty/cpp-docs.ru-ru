# Ошибка MSBuild MSB3168
**MSB3168: повторяющийся элемент "\<пакет\>" будет игнорироваться.**  
  
 Это предупреждение появляется, если указано, что загрузчик должен установить два одинаковых пакета загрузчика.  В этом случае загрузчик установит только один экземпляр.

-   Дополнительные сведения о причинах и способах устранения ошибки MSBuild см. в разделе [Дополнительные ресурсы](../Topic/Additional%20MSBuild%20Resources.md).
