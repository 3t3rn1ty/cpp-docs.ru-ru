# Ошибка MSBuild MSB3160
**MSB3160: Предупреждение проверки Xml в файле "\<файл\>": \<ошибка\>**  
  
 Это предупреждение появляется, когда XML\-файл продукта или пакета не согласуется со схемой XML.  В последней части этого предупреждения указано конкретное предупреждающее сообщение.  
  
### Чтобы исправить эту ошибку  
  
-   Обновите XML\-файл, чтобы устранить ошибку проверки.
