# Add-in &lt;name&gt; caused an access violation. Remove it from the Add-in Manager?
Произошел сбой надстройки, который мог привести к другим проблемам интегрированной среды разработки.  
  
### Чтобы исправить эту ошибку  
  
1.  Нажмите кнопку "Да", чтобы удалить надстройку из списка доступных надстроек.  
  
     \-или\-  
  
     Нажмите кнопку "Нет".  
  
    > [!NOTE]
    >  Свяжитесь с автором надстройки, чтобы сообщить ему о проблеме.
