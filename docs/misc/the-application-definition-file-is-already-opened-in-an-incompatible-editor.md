# Файл определения приложения уже открыт в несовместимом редакторе
**Файл определения приложения уже открыт в несовместимом редакторе.  Закройте другой редактор и перезагрузите страницу свойств проекта.**  
  
 Эта ошибка появляется при попытке открыть файл определения приложения, который уже открыт в другом редакторе.  
  
### Чтобы исправить эту ошибку  
  
1.  Закройте другой редактор.  
  
2.  Закройте конструктор проектов.  
  
3.  В меню **Проект** выберите команду **Свойства**, чтобы снова открыть конструктор проектов.
