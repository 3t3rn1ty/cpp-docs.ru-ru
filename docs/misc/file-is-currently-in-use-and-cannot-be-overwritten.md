# File is currently in use and cannot be overwritten.
Не удается сохранить текущий файл.  Эта ошибка обычно возникает при попытке сохранить один из файлов под именем другого файла, также уже открытого.  
  
### Чтобы исправить эту ошибку  
  
1.  Закройте файл с таким же именем и повторите попытку.  
  
     —или—  
  
     Переименуйте файл и сохраните его.
