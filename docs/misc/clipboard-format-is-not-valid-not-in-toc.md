# Clipboard format is not valid.
В буфер обмена данные могут быть помещены только в определенных заданных форматах.  Эта ошибка обычно возникает при попытке поместить в буфер обмена данные из компонента, формат которых несовместим с буфером обмена.  
  
### Чтобы исправить эту ошибку  
  
1.  Просмотрите документацию для компонента, чтобы определить, можно ли использовать буфер обмена для передачи данных из компонента.  
  
     —или—  
  
2.  Выберите другой элемент и повторите попытку.
