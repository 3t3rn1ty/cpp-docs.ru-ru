# Ошибка MSBuild MSB1022
**Файл ответов не существует.**  
  
 Файл ответов в указанном расположении не существует.  
  
### Чтобы исправить эту ошибку  
  
1.  убедитесь в том, что файл ответов существует в указанном каталоге, если было задано абсолютное или относительное местоположение этого файла.
