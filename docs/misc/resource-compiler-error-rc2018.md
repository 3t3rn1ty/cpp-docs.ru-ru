# Ошибка компилятора ресурсов RC2018
неизвестный знак "шестнадцатеричное\_число"  
  
 Символ ASCII, соответствующий заданному шестнадцатеричному числу, присутствует в файле исходного кода, но является недопустимым символом.  
  
 Причиной этой ошибки может быть повреждение файла исходного кода.
