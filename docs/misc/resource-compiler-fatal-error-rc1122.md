# Неустранимая ошибка компилятора ресурсов RC1122
Ошибка ввода\-вывода при записи файла  
  
 Компилятору ресурсов не удалось выполнить запись в файл.  
  
### Чтобы устранить ошибку, проверьте указанные ниже возможные причины ее возникновения.  
  
1.  Недостаточно места на диске. Свободного места должно быть примерно вдвое больше, чем размер создаваемого выполняемого файла.  
  
2.  Том доступен только для чтения.  
  
3.  Поврежденный сектор.  
  
4.  Нарушение общего доступа.
