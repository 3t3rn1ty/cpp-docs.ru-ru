# Project type is not valid.
Эта ошибка обычно возникает, если для аргумента *имя\_шаблона* команды **Добавить новый проект** было введено неверное значение.  
  
### Чтобы исправить эту ошибку  
  
1.  Проверьте в документации правильность синтаксиса команды **Добавить новый проект** и повторите попытку.
