# CA0067
Каталог не найден.  
  
 Значение параметра **\/directory** не найдено в командной строке FxCopCmd.  
  
 Это предупреждение может возникать, если при использовании свойства **CodeAnalysisDependentAssemblyPaths** [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)] для указания дополнительных путей поиска базовой сборки один из этих путей не существует.  
  
-   Если появится предупреждение CA0067 с ошибками [CA0058](../misc/ca0058.md) или предупреждения [CA0060](../Topic/CA0060.md), разрешение других ошибок обычно разрешает CA0067.  
  
-   Если появится предупреждение CA0067 без других ошибок или предупреждений, обычно предупреждение можно игнорировать, потому что каталог не требовался.
