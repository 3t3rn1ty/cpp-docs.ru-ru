# Look In list must contain valid files or folders, separated by semicolons.
Эта ошибка обычно возникает, когда каталог или файлы, перечисленные в списке **Искать в**, содержат недопустимые знаки.  
  
### Чтобы исправить эту ошибку  
  
1.  Вместо ввода пути или имени файла используйте диалоговое окно **Поиск** для указания каталога или имени файла.  Доступ к диалоговому окну **Поиск** осуществляется с помощью кнопки с троеточием, расположенной рядом со списком **Искать в**.
