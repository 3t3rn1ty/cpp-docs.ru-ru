# Разрешение вопросов, связанных с исключениями: System.Reflection.TargetException
Исключение, которое возникает при попытке вызвать недопустимый адресат.  
  
## Примечания  
 Исключение <xref:System.Reflection.TargetException> возникает при попытке вызвать для пустого объекта метод, который не является статическим. Это может случиться, если вызывающая процедура не имеет доступа к элементу или адресат не определяет элемент, и по подобным причинам.
