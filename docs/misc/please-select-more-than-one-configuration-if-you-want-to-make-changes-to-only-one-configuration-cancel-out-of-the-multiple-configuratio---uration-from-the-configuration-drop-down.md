# Выберите несколько конфигураций. Если требуется внести изменения только в одну конфигурацию, закройте диалоговое окно &quot;Несколько конфигураций&quot;, не сохраняя параметры, а затем выберите необходимый вариант в списке конфигураций.
Эта ошибка обычно возникает при выборе одной конфигурации решения в диалоговом окне **Несколько конфигураций**.  
  
### Исправление ошибки  
  
1.  Выберите дополнительные конфигурации решения.  
  
     —или—  
  
2.  Если вы хотите обновить одну конфигурацию, нажмите кнопку "Отмена" и выберите правильную конфигурацию решения в списке "Конфигурация" в диалоговом окне **Свойства решения**.
