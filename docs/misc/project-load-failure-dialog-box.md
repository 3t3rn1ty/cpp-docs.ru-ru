# Диалоговое окно &quot;Ошибка загрузки проекта&quot;
Это диалоговое окно отображается в случае, если загрузка файла проекта оказывается невозможной.  
  
 **Исправьте указанные ошибки и перегрузите проект.**  
 Сведения о возникшей ошибке.  Сведения, приводимые в этом диалоговом окне, могут использоваться для выявления и устранения причины ошибки в файле проекта.  Сохраните проект и перезагрузите его в интегрированной среде разработки.  
  
 **Не показывать больше этот диалог в течение данной сессии**  
 При выборе данной настройки, диалоговое окно **Ошибка загрузки проекта** не будет появляться до следующего открытия интегрированной среды разработки.
