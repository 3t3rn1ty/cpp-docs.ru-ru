# Ошибка MSBuild MSB2011
**Недопустимый элемент \<{0}\>.**  
  
 Указан недопустимый элемент.  
  
### Чтобы исправить эту ошибку  
  
-   Проверьте правильность написания указанного элемента.  
  
-   Проверьте, не был ли файл проекта изменен или поврежден.  Если он изменен или поврежден, откройте проект в той версии [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)], в которой он был создан, сохраните его и попробуйте преобразовать еще раз.
