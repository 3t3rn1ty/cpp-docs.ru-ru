# Разрешение вопросов, связанных с исключениями: System.IO.DirectoryNotFoundException
Исключение <xref:System.IO.DirectoryNotFoundException> создается, когда часть пути к файлу или каталогу не найдена.  
  
## Полезные советы  
 **Убедитесь, что каталог находится в указанном месте.**  
 Проверьте, что папка существует в указанном месте.  
  
 **При использовании относительных путей убедитесь, что текущий каталог правилен.**  
 Путь может быть неправильным, если указан неправильный текущий каталог.
