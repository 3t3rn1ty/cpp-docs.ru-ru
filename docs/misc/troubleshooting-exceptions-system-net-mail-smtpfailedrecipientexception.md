# Разрешение вопросов, связанных с исключениями: System.Net.Mail.SmtpFailedRecipientException
Исключение возникает, когда электронная почта, отправленная с помощью <xref:System.Net.Mail.SmtpClient>, не может быть доставлена адресату.
