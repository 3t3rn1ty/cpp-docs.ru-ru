# Выберите несколько платформ. Если требуется внести изменения только в одну платформу, закройте диалоговое окно &quot;Несколько платформ&quot;, не сохраняя параметры, а затем в списке платформ выберите необходимый вариант.
Эта ошибка обычно возникает при выборе нескольких платформ в диалоговом окне **Несколько платформ**.  
  
### Исправление ошибки  
  
1.  Выберите дополнительные платформы.  
  
     —или—  
  
2.  Если вы хотите обновить одну платформу, нажмите кнопку "Отмена" и выберите нужную платформу в списке платформ.
