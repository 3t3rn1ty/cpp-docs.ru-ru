# Разрешение вопросов, связанных с исключениями: System.Windows.Automation.ElementNotAvailableException
Исключение <xref:System.Windows.Automation.ElementNotAvailableException> возникает при попытке доступа к <xref:System.Windows.Automation.AutomationElement>, соответствующему более недоступной части интерфейса пользователя.  
  
## Примечания  
 Это исключение может возникнуть, если этот элемент находился в диалоговом окне, которое было закрыто, или в приложении, которое было закрыто или прервано.
