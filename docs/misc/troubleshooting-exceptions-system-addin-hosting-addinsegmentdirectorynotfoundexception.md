# Разрешение вопросов, связанных с исключениями: System.AddIn.Hosting.AddInSegmentDirectoryNotFoundException
Исключение <xref:System.AddIn.Hosting.AddInSegmentDirectoryNotFoundException> возникает, когда каталог сегмента отсутствует в структуре каталогов конвейера.  
  
## Примечания  
 При активизации Add\-In, создается канал к узлу. Система надстройки требует, чтобы сегменты конвейера были помещены в указанную структуру каталогов с указанными именами каталогов, как описано в разделе [Требования к разработке конвейера](http://msdn.microsoft.com/ru-ru/ef9fa986-e80b-43e1-868b-247f4c1d9da5).
