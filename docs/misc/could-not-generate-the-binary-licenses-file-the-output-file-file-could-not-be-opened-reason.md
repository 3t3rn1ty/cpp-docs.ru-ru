# Не удалось создать двоичный файл лицензий. Не удалось открыть выходной файл. &lt;причина&gt;
Системе проектов не удалось удалить устаревшую копию файла лицензий. В процессе подготовки лицензий система проектов преобразует текстовые LICX\-файлы в двоичные файлы лицензий, которые распознаются системой лицензирования .NET.  
  
 **Исправление ошибки**  
  
-   Перезапустите [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)].  
  
     Процесс сборки завершится сбоем, если возникает эта ошибка.
