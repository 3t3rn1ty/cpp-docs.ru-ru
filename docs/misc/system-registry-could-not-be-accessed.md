# System registry could not be accessed.
Эта ошибка обычно возникает, когда операционная система не может найти файл реестра, когда файл поврежден либо попытка чтения или записи в файл заканчивается неудачей.  
  
### Чтобы исправить эту ошибку  
  
1.  Переустановите Visual Studio.
