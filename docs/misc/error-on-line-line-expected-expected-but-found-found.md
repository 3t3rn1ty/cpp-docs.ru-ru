# Error on line &lt;line&gt;. Expected &#39;expected&#39; but found &#39;found&#39;
Это ошибка синтаксического анализа XML общего характера, возникающая при чтении файла проекта.  Ошибка может произойти как для файлов проекта \(VBPROJ\/CSPROJ\), так и для файлов параметров пользователя \(.vbproj.user\/.csproj.user\).  Дополнительные сведения будут представлены системой работы с проектами в параметрах `expected` и `found`.  
  
 Эта ошибка часто возникает при редактировании файла проекта \(или параметров пользователя\) вручную.  
  
 **Чтобы исправить эту ошибку**  
  
-   Отредактируйте файл проекта еще раз, исправив ошибку.  
  
     В зависимости от серьезности ошибки, возможно, проект нельзя будет открыть.
