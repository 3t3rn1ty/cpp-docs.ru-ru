# An invalid file was specified for /resetsettings: &lt;filename&gt;.  /resetsettings will be ignored.
Эта ошибка обычно возникает, если указанный файл недоступен для чтения, не существует или содержит некорректный XML.  
  
### Чтобы исправить эту ошибку  
  
1.  Проверьте правильность имени файла.  
  
2.  Проверьте правильность формата файла.  
  
3.  Проверьте наличие в файле корректного XML.
