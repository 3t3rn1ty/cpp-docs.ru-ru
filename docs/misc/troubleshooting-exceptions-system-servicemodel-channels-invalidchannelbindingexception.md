# Разрешение вопросов, связанных с исключениями: System.ServiceModel.Channels.InvalidChannelBindingException
Исключение <xref:System.ServiceModel.Channels.InvalidChannelBindingException> возникает, когда указанная привязка не согласована с контрактом требований к службе.
