# Устранение неполадок: System.Data.SqlServerCe.SqlCeException
Исключение `System.Data.SqlServerCe.SqlCeException` возникает, когда базовый поставщик возвращает предупреждение или ошибку от источника данных SQL Server Compact. Дополнительные сведения о причине этой ошибки см. в разделе [Ошибки SQL Server Compact Edition](http://msdn2.microsoft.com/library/ms171849.aspx) на веб\-сайте MSDN.  
  
## Заметки  
 Исключение `System.Data.SqlServerCe.SqlCeException` всегда содержит по крайней мере один экземпляр объекта `System.Data.SqlServerCe.SqlCeError`. Этот класс не может наследоваться.  
  
 Для получения дополнительной информации см. [Библиотека классов SQL Server Compact 3.5](http://go.microsoft.com/fwlink/?LinkID=102595).
