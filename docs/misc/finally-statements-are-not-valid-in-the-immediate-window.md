# Операторы Finally недопустимы в окне интерпретации
Операторы `Finally` можно использовать только в исходном коде.  
  
 **Идентификатор ошибки:** BC30716  
  
### Исправление ошибки  
  
-   Удалите операторы `Finally` из отлаживаемого кода.
