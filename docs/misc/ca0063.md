# CA0063
Не удалось загрузить файл набора правил или один из его зависимых файлов наборов правил.  
  
 Не удалось загрузить файл набора правил или один из его зависимых файлов наборов правил.  
  
 Не удается найти указанный набора правил или один из файлов набора правил, включенных в ваш набор правил, не удалось найти.  Убедитесь, что все наборы правил в вашем наборе правил существуют на диске и соответствующие каталоги наборов правил указаны в проекте в свойстве **CodeAnalysisRuleSetDirectories** [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)].  
  
 Для отладки ошибки просмотрите файл правил в текстовом редакторе.  Чтобы найти путь к файлу набора правил, щелкните правой кнопкой мыши соответствующий проект в обозревателе решений, выберите пункт **Свойства** и затем щелкните **Анализ кода**.  Убедитесь, что файл набора правил выбран в **Выполнить этот набор правил**.  В поле описания отображается путь к набору правил.  
  
 Проверьте значение атрибута **Path** для всех элементы **Include**.  Пути включения могут использовать относительные пути к файлу родительского и текущего набора правил, переменные среды и абсолютные пути.  Примеры.  
  
```  
<Include Path="%PUBLIC%\Documents\RuleSets\alldesign.ruleset" Action="Default" />  
<Include Path="..\alldesign.ruleset" Action="Default" />  
<Include Path="C:\Rulesets\alldesign.ruleset" Action="Default" />  
```  
  
 Проверьте корректность каждого из путей включения.  
  
 В некоторых случаях расположение набор правил может зависеть от свойства [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)].  На свойства [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)] нельзя ссылаться из набора правил.  Для решения этой проблемы следует указать дополнительные пути поиска в свойстве **CodeAnalysisRuleSetDirectories** [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)].  В этом случае укажите только имя набора правил в атрибуте **Path** элемента **Include**.
