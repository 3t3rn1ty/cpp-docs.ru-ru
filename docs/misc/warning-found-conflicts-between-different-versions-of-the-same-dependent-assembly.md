# Предупреждение: обнаружены конфликты между различными версиями одной зависимой сборки
Это предупреждение отображается, если в графе зависимостей проекта содержатся ссылки на другие версии той же сборки.  
  
 Если имеется файл app.config, то [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] позволяет добавить в него перенаправление привязки.  Перенаправление привязки позволяет принудительно перенаправить все ссылки сборки в самую последнюю версию сборки. [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] сохраняет информацию о перенаправлении в файле app.config.  Если используется перенаправление привязки, это предупреждение больше не появится.  
  
 Если вы не хотите добавлять перенаправление привязки, в проекте и дальше будут использоваться ссылки на ту же версию сборки, что и раньше.  Но будет снова появляться это предупреждение.  
  
### Чтобы исправить эту ошибку  
  
-   Дважды щелкните предупреждение и выберите "Да" при получении запроса "Устранить эти конфликты при помощи добавления связывающих перенаправляющих записей в файл app.config?".
