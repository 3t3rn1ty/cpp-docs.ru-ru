# Предупреждение о соответствии спецификации CLS CLS00404
Идентификаторы членов должны соответствовать Приложению 7 к документу "Технический отчет по стандарту Юникода версии 3.0".  
  
 Убедитесь, что идентификаторы членов соответствуют Приложению 7 к документу "Технический отчет по стандарту Юникода версии 3.0".
