# Неустранимая ошибка компилятора ресурсов RC1106
недопустимый параметр: параметр  
  
 Параметр не является допустимым параметром компилятора ресурсов.  
  
 Допустимые параметры:  
  
 \/r  
  
 Создание RES\-файла  
  
 \/v  
  
 Вывод подробных сведений \(печать сообщений о ходе выполнения\)  
  
 \/d  
  
 Определение символа  
  
 \/fo  
  
 Переименование RES\-файла  
  
 \/l  
  
 Идентификатор языка по умолчанию в шестнадцатеричном формате  
  
 \/I  
  
 Добавление пути для операций поиска **INCLUDE**  
  
 \/x  
  
 Игнорирование переменной среды **INCLUDE**  
  
 \/c  
  
 Определение кодовой страницы, используемой при преобразовании NLS
