# Ошибка компилятора ресурсов RC2132
требуется ключевое слово VALUE, BLOCK или END  
  
 Неправильное завершение или начало блока в ресурсе **VERSION**. Это приводит к тому, что оператор **VERSION** не является допустимым блоком.
