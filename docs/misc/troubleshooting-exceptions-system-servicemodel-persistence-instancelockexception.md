# Разрешение вопросов, связанных с исключениями: System.ServiceModel.Persistence.InstanceLockException
Исключение <xref:System.ServiceModel.Persistence.InstanceLockException> предназначено для использования классами–наследниками <xref:System.ServiceModel.Persistence.LockingPersistenceProvider>, если операция не может быть выполнена из–за состояния экземпляра блокировки.
