# Разрешение вопросов, связанных с исключениями: System.ServiceModel.MessageHeaderException
Исключение <xref:System.ServiceModel.MessageHeaderException> вызывается, когда ожидания относительно заголовков сообщения SOAP не удовлетворены после обработки сообщения.
