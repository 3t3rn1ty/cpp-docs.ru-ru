# Property value is not valid. (Visual Studio) (1)
Эта ошибка обычно возникает, если введено недопустимое значение свойства.  Например, если свойство является логическим, принимая допустимые значения TRUE и FALSE, значение MAYBE для этого свойства будет недопустимым.  
  
### Чтобы исправить эту ошибку  
  
1.  Введите значение, допустимое для этого свойства.  
  
    > [!TIP]
    >  Обратитесь к интерактивной документации для имени свойства, чтобы установить, какие значения являются допустимыми.
