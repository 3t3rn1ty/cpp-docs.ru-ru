# Command &lt;name&gt; is not available.
Эта ошибка обычно возникает, если в окне **Команда** или поле **Найти\/Команда** введена недопустимая команда \(в зависимости от текущего состояния Visual Studio\).  Например, команда `File.CloseSolution` недопустима, если решение еще не открыто в Visual Studio.  
  
### Чтобы исправить эту ошибку  
  
1.  Выберите другую команду.  
  
     —или—  
  
     Выберите для команды другой объект.
