# Ошибка MSBuild MSB1040
**MSB1040: недопустимое значение ToolsVersion.**  
  
 С помощью ключа **\/ToolsVersion** задано недопустимое значение `ToolsVersion`.  
  
### Чтобы исправить эту ошибку  
  
-   В процессе сборки укажите допустимое значение `ToolsVersion`, используя ключ **\/ToolsVersion** в командной строке.
