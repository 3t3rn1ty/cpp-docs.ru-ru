# Ошибка MSBuild MSB1026
**Файл схемы не существует.**  
  
 Файл схемы в указанном расположении не существует.  
  
### Чтобы исправить эту ошибку  
  
1.  убедитесь в том, что файл схемы существует в указанном каталоге, если было задано абсолютное или относительное местоположение этого файла.
