# Разрешение вопросов, связанных с исключениями: System.MulticastNotSupportedException
Исключение возникает при попытке объединить два экземпляра не объединяемого типа делегата. Исключение не возникает, если один из операндов является пустой ссылкой \(`Nothing` в Visual Basic\).
