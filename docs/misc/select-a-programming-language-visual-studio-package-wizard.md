# Выбор языка программирования, мастер пакетов Visual Studio
Этот шаг позволяет:  
  
-   выбрать язык программирования, который будет использоваться для создания VSPackage;  
  
-   создать или использовать существующий файл ключа для подписи сборки VSPackage.  
  
 **Visual C\#**  
 VSPackage создается в C\#.  
  
 **Visual Basic**  
 VSPackage создается в Visual Basic.  
  
 **Visual C\+\+**  
 VSPackage создается в Visual C\+\+.  
  
## Получить ключи для подписывания сборки  
 **Создать новый файл ключа для подписывания сборки**  
 Приводит к созданию мастером файла ключа для вас. Созданный файл ключа называется key.snk.  
  
 **Использовать следующий файл ключа для подписывания сборки**  
 Позволяет ввести путь к существующему файлу ключа или нажать кнопку **Обзор** для получения файла. Мастер создает копию файла и вызывает его key.snk.  
  
 **Cancel**  
 Отменяет создание VSPackage.  
  
 **Назад**  
 Переход на предыдущую страницу мастера.  
  
 **Далее**  
 Переход на следующую страницу мастера.  
  
 **Готово**  
 Создает пакет VSPackage.
