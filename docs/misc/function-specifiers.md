# Спецификаторы функций
Ключевые слова [inline](../misc/inline-inline-forceinline.md) и [virtual](../cpp/virtual-specifier.md) можно использовать как спецификаторы в объявлениях функций. Такое применение ключевого слова **virtual** отличается от его использования в спецификаторе базового класса определения класса.
