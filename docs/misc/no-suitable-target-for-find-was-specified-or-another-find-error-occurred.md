# No suitable target for find was specified or another find error occurred.
Эта ошибка обычно возникает при попытке выполнения поиска, когда для области поиска установлено "Текущий документ", а ни один из файлов не открыт.  
  
### Чтобы исправить эту ошибку  
  
1.  Откройте файл и снова выполните поиск или измените область поиска.
