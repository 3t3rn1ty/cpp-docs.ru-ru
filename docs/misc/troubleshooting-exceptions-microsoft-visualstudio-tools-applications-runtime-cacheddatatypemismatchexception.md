# Разрешение вопросов, связанных с исключениями: Microsoft.VisualStudio.Tools.Applications.Runtime.CachedDataTypeMismatchException
Исключение вызывается, когда сериализованное XML состояние объекта в кэше не соответствует информации о типе в кэшированном манифесте данных.
