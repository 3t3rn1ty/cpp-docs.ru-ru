# Разрешение вопросов, связанных с исключениями: System.ServiceModel.Dispatcher.NavigatorInvalidBodyAccessException
Исключение <xref:System.ServiceModel.Dispatcher.NavigatorInvalidBodyAccessException> вызывается, когда <xref:System.Xml.XPath.XPathNavigator> направляется для проверки текста небуферизованного сообщения.
