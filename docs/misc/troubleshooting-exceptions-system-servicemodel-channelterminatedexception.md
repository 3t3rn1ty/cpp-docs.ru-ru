# Разрешение вопросов, связанных с исключениями: System.ServiceModel.ChannelTerminatedException
Исключение <xref:System.ServiceModel.ChannelTerminatedException> вызывается обычно на стороне клиента, если канал закрывается из–за закрытия подключения сервером.
