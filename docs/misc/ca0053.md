# CA0053
Не удалось загрузить сборку правил.  
  
 Данное предупреждение указывает, что сборку правил не удалось загрузить посредством отражения.  Такая ситуация можно происходить, если пользовательские правила не были корректно построены или средству анализа не удалось загрузить сборку по другим причинам, например из\-за отказа в доступе к файловой системе.
