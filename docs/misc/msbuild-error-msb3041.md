# Ошибка MSBuild MSB3041
**Не удается создать имя ресурса манифеста для "'\<file\>'".**  
  
 Задача `CreateManifestResourceName` не может создать имя ресурса манифеста для указанного файла.  
  
### Чтобы исправить эту ошибку
