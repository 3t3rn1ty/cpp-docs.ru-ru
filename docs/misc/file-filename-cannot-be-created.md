# File &lt;filename&gt; cannot be created.
Эта ошибка обычно возникает при попытке создать веб\-отчет с комментариями кода, однако текущий пользователь не имеет прав на запись.  
  
### Чтобы исправить эту ошибку  
  
1.  Убедитесь, что у вас есть доступ для записи.  
  
2.  Проверьте, достаточно ли места для сохранения страниц.
