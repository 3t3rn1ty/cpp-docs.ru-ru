# &lt;name&gt; is not a valid project name.
Эта ошибка обычно возникает при вводе команды `File.NewProject` в окне **Команды** или в поле **Найти\/Команда**, а также если для *имя\_проекта* введено неверное имя.  
  
### Чтобы исправить эту ошибку  
  
1.  Убедитесь, что имя проекта не содержит лишних пробелов и не является зарезервированным словом, например NUL, CON или AUX.  
  
     —или—  
  
     Введите команду повторно, опустив имя проекта.
