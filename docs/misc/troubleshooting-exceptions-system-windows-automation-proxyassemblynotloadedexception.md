# Разрешение вопросов, связанных с исключениями: System.Windows.Automation.ProxyAssemblyNotLoadedException
Исключение <xref:System.Windows.Automation.ProxyAssemblyNotLoadedException> содержит сведения об исключении, которое выводится при проблемах с загрузкой сборки, содержащей поставщиков стороны клиента.
