# Разрешение вопросов, связанных с исключениями: System.Workflow.Activities.Rules.RuleEvaluationIncompatibleTypesException
Исключение <xref:System.Workflow.Activities.Rules.RuleEvaluationIncompatibleTypesException> вызывается, когда при оценке правила происходит попытка применения оператора к несовместимым операндам.
