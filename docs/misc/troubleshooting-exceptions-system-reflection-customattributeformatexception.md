# Разрешение вопросов, связанных с исключениями: System.Reflection.CustomAttributeFormatException
Это исключение выдается при неправильном двоичном формате настраиваемого атрибута.  
  
## Примечания  
 Это может быть результатом ошибки в средстве, использованном для создания типа данных, с которым связана ошибка.
