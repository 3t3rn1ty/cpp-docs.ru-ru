# Error getting Codezone Community sites.
Эта ошибка возникает в случае, если не удается найти подключение к интерактивной веб\-службе.  Веб\-службы вносят сведения на страницу **Интернет** в разделы **Среда** и **Справка**, диалогового окна **Параметры**.  
  
### Чтобы исправить эту ошибку  
  
1.  Проверьте наличие подключения к Интернету.  
  
     —или—  
  
     При подключении к Интернету через прокси\-сервер проверьте его параметры.  
  
2.  Повторите попытку доступа к веб\-странице.
