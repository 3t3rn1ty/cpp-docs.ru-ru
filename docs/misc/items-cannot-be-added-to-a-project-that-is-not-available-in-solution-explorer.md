# Нельзя добавить элементы в проект, который недоступен в обозревателе решений.
Эта ошибка обычно возникает, когда указанный проект не открыт в обозревателе решений. Visual Studio не может добавить элементы в проект, который сейчас недоступен в обозревателе решений.  
  
### Исправление ошибки  
  
1.  Добавьте указанный проект в текущее решение или откройте решение, содержащее указанный проект.
