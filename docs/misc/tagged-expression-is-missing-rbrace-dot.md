# Tagged expression is missing &#39;}&#39;.
Эта ошибка обычно возникает, когда в строке поиска была введена открывающая фигурная скобка \(`{`\), а закрывающая фигурная скобка \(`}`\) была пропущена.  
  
### Чтобы исправить эту ошибку  
  
1.  Чтобы найти знак `{`, используйте `\{`.  
  
2.  Чтобы пометить выражение, введите пропущенный знак `}`.
