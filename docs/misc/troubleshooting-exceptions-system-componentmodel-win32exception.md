# Разрешение вопросов, связанных с исключениями: System.ComponentModel.Win32Exception
Исключение, вызываемое кодом ошибки Win32.  
  
## Примечания  
 Коды ошибок типа Win32 при отображении преобразуются из их числового представления в системное сообщение. Используйте свойство <xref:System.ComponentModel.Win32Exception.NativeErrorCode%2A> для доступа к числовому представлению кода ошибки.
