# Component is not correctly registered.
Эта ошибка обычно возникает, когда реестр поврежден либо недоступен для системы или когда элемент был записан в реестр в неверном формате.  
  
### Чтобы исправить эту ошибку  
  
1.  Переустановите компонент.
