# Разрешение вопросов, связанных с исключениями: Microsoft.VisualStudio.Tools.Applications.Runtime.CannotFindStartupObjectAssemblyException
Исключение вызывается, когда в основной сборке решения отсутствует автоматически запускаемый объект, указанный в манифесте приложения.
