# Command &lt;name&gt; is not valid.
Эта ошибка обычно возникает, если в окне **Команда** или поле **Найти\/Команда**  были введены команда или псевдоним, не распознанные Visual Studio.  Если был введен псевдоним, в сообщении об ошибке появится полное имя команды, а не имя псевдонима.  
  
### Чтобы исправить эту ошибку  
  
1.  Повторно введите или выберите правильное имя команды.
