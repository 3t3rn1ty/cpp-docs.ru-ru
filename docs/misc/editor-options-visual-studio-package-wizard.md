# Параметры редактора, мастер создания пакетов Visual Studio
На этом шаге можно определить расширение имени файла, связанное с редактором, имя файла, которое используется по умолчанию для документов, создаваемых с помощью редактора, и значок, связанный с редактором и типом файлов.  
  
 **Имя редактора**  
 Введите имя нового редактора. Это имя используется для редактора в диалоговом окне **Открыть с помощью**.  
  
 **Расширение файла**  
 Определите расширение имени файла, связанное с редактором. Можно указать несколько расширений, введя их через запятую. Ваш редактор будет редактором [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] по умолчанию для файлов, имеющих указанное расширение.  
  
 **Имя файла по умолчанию**  
 Введите имя, которое будет использоваться для формирования имен по умолчанию для документов, создаваемых редактором.  
  
 **Изменить значок**  
 Нажмите эту кнопку, чтобы открыть диалоговое окно **Смена значка**. Вы можете выбрать значок для редактора в этом диалоговом окне или из другого файла. Файл должен иметь расширение ICO. Выбранный значок отображается в окне редактора и связывается с файлами, создаваемыми в нем.  
  
 **Cancel**  
 Останавливает работу мастера.  
  
 **Назад**  
 Переход на предыдущую страницу мастера.  
  
 **Готово**  
 Создает пакет VSPackage.
