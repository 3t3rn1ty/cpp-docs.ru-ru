# Top-level menu names cannot contain separators.
Эта ошибка обычно возникает при попытке перетащить в заголовок меню текст, содержащий разделитель, например дефис \(\-\).  Меню верхнего уровня не могут содержать этот знак, так как он зарезервирован для разделения наборов пунктов меню.  
  
### Чтобы исправить эту ошибку  
  
1.  Введите текст, не используя разделитель.
