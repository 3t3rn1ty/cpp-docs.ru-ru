# Ошибка MSBuild MSB1015
**MSBuild не работает в этой версии операционной системы.  Он может работать только в Windows 2000, Windows XP и более поздних версиях.**  
  
 [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)] не поддерживается в этой версии операционной системы.  
  
### Чтобы исправить эту ошибку  
  
1.  Установите и запустите [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)] в одной из операционных систем, поддерживающих работу продукта.
