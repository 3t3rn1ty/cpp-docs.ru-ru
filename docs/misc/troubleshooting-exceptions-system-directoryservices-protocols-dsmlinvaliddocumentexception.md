# Разрешение вопросов, связанных с исключениями: System.DirectoryServices.Protocols.DsmlInvalidDocumentException
Исключение возникает, если запрос или ответ языка разметки служб каталогов \(DSML\) не является XML\-документом правильного формата или не может быть проверен схемой DSMLV2.
