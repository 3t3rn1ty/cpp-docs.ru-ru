# Разрешение вопросов, связанных с исключениями: System.Workflow.Runtime.WorkflowOwnershipException
Исключение <xref:System.Workflow.Runtime.WorkflowOwnershipException> вызывается, когда механизм среды выполнения для рабочих процессов пытается загрузить экземпляр рабочего процесса, который в данный момент загружен другим экземпляром механизма среды выполнения для рабочих процессов. Кроме того, это исключение вызывается, когда механизм среды выполнения для рабочих процессов пытается сохранить рабочий процесс по истечении времени владения, указанного во время загрузки рабочего процесса.
