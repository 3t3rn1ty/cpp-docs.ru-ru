# Разрешение вопросов, связанных с исключениями: System.ArrayTypeMismatchException
Исключение <xref:System.ArrayTypeMismatchException> вызывается, когда система не может преобразовать элемент в тип, объявленный для массива.  
  
## Полезные советы  
 **Убедитесь, что тип объекта может быть преобразован в тип массива.**  
 Между этими двумя типами должно быть возможно преобразование.  
  
 Дополнительные сведения для пользователей Visual Basic см. в разделе [Массивы](../Topic/Arrays%20in%20Visual%20Basic.md).
