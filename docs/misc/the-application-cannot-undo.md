# The application cannot undo.
Эта ошибка обычно возникает при попытке выполнить операцию отмены для элемента, для которого существует операция отмены, связанная с другим файлом.  Если после связанного изменения было выполнено какое\-то действие, Visual Studio не сможет отменить изменение.  
  
### Чтобы исправить эту ошибку  
  
1.  Исправьте изменение вручную.
