# Разрешение вопросов, связанных с исключениями: Microsoft.VisualStudio.Tools.Applications.Runtime.NonCacheableTypeException
Это исключение возникает при попытке кэширования типа, который не является <xref:System.Data.DataSet>, <xref:System.Data.DataTable> или типом, удовлетворяющим условиям, описанным в разделе [Caching Data](http://msdn.microsoft.com/ru-ru/e60dc731-2cb0-4de6-a861-0f0d131d780a).
