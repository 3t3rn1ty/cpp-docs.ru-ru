# new
Кратко описывает использование ключевого слова `new` и предоставляет ссылки на дополнительные сведения.  
  
|Название|Описание|  
|--------------|--------------|  
|[оператор new](../c-runtime-library/operator-new-crt.md)|Выделяет блок памяти из кучи.|  
|[оператор new](../Topic/operator%20new%20\(%3Cnew%3E\).md)|Функция, вызываемая с помощью выражения new для выделения хранилища для отдельных объектов.|  
|[Оператор new](../cpp/new-operator-cpp.md)|Выделяет память для объекта или массива объектов из свободного хранилища и возвращает подходящим образом типизированный ненулевой указатель на объект.|  
|[new \(новый слот в vtable\)](../windows/new-new-slot-in-vtable-cpp-component-extensions.md)|В проекте C\+\+\/CLI указывает, что виртуальный член получает новую ячейку в таблице vtable.  Другими словами, функция не переопределяет метод базового класса.|  
|[ref new, gcnew](../windows/ref-new-gcnew-cpp-component-extensions.md)|В [!INCLUDE[cppwrt_short](../Token/cppwrt_short_md.md)] \(`ref new`\) или проекте C\+\+\/CLI \(`gcnew`\) это составное ключевое слово создает объект, для которого осуществляется учет ссылок.|
