# Оператор ^ (C++)
В зависимости от контекста оператор `^` в Visual C\+\+ может иметь два значения.  Дополнительные сведения см. в разделе:  
  
-   [Битовый оператор ИЛИ \(исключительное\): ^](../cpp/bitwise-exclusive-or-operator-hat.md) — для использования во всех вариантах C\+\+.  
  
-   [Оператор дескриптора объекта \(^\)](../windows/handle-to-object-operator-hat-cpp-component-extensions.md) для использования в [!INCLUDE[cppwrt_short](../Token/cppwrt_short_md.md)] и [!INCLUDE[cppcli](../build/reference/includes/cppcli_md.md)]
