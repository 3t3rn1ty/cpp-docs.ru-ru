# Разрешение вопросов, связанных с исключениями: Microsoft.VisualStudio.Tools.Applications.Runtime.ControlNotFoundException
Исключение <xref:Microsoft.VisualStudio.Tools.Applications.Runtime.ControlNotFoundException> возникает, когда код пытается получить доступ к элементу управления, которого нет в документе. Элемент управления может быть удален конечным пользователем или каким\-то другим кодом. Не существует способа для проверки наличия элемента до попытки получить доступ; если элемент был удален, решение не будет работать.
