# Разрешение вопросов, связанных с исключениями: System.ServiceModel.PoisonMessageException
Исключение <xref:System.ServiceModel.PoisonMessageException> возникает, когда сообщение считается "подозрительным". Сообщение считается подозрительным, если множественные попытки его доставить провалились.
