# You must supply a title for this tool.
Эта ошибка обычно возникает, если поле **Название** диалогового окна **Внешние инструменты** оставлено пустым.  
  
### Чтобы исправить эту ошибку  
  
1.  В текстовом поле **Название** введите имя нового внешнего средства.
