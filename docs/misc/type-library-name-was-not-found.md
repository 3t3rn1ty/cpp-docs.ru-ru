# Type library &lt;name&gt; was not found.
Visual Studio не может выполняться без библиотек типов DTE.olb и vsext.tlb.  Эта ошибка обычно возникает, если эти два файла были удалены.  
  
### Чтобы исправить эту ошибку  
  
1.  Переустановите Visual Studio.
