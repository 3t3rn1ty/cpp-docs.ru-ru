# Основные сведения о пакетах VSPackage
Пакеты VSPackage — это программные модули, которые образуют и расширяют среду Visual Studio с помощью элементов пользовательского интерфейса, служб, проектов, редакторов и конструкторов. Пакеты VSPackage являются центральным элементом архитектуры [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)]. Сама среда [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] представляет собой в основном коллекцию пакетов VSPackage.  
  
 Чтобы создать приложение расширяемости [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)], сначала нужно создать пакет VSPackage. Затем можно добавить команды меню, окна инструментов, окна документов или другой код для расширения интегрированной среды разработки [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)].  
  
 Для использования пакетов VSPackage необходимо установить пакет SDK для Visual Studio. Для выполнения этого пошагового руководства необходимо установить пакет SDK для Visual Studio. Дополнительные сведения см. в разделе [SDK для Visual Studio](../Topic/Visual%20Studio%20SDK.md).
