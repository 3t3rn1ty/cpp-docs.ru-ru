# Ошибка MSBuild MSB3189
**MSB3189: местом обновления для этого приложения является локальный путь.**  
  
 Это предупреждение применяется только к манифестам развертывания.  
  
### Чтобы исправить эту ошибку  
  
-   Убедитесь, что в качестве места обновления указан действительный URL\-адрес.
