# Разрешение вопросов, связанных с исключениями: System.IdentityModel.Selectors.UntrustedRecipientException
Исключение <xref:System.IdentityModel.Selectors.UntrustedRecipientException> возникает, когда пользователь просматривает информацию объекта, запрашивающего маркер, и решает не доверять объекту. Т.е. пользователь щелкает **Отключить Windows CardSpace** в интерфейсе пользователя CardSpace.
