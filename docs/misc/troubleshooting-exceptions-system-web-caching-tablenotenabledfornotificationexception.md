# Разрешение вопросов, связанных с исключениями: System.Web.Caching.TableNotEnabledForNotificationException
Исключение, которое порождается при использовании класса <xref:System.Web.Caching.SqlCacheDependency> для таблицы базы данных, не поддерживающей уведомления об изменениях.
