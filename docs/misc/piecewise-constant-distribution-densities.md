# piecewise_constant_distribution::densities
Возвращает хранимые плотности распределения.  
  
## Синтаксис  
  
```  
std::vector<double> densities() const;  
```  
  
## Заметки  
 Функция\-член возвращает хранимые плотности, вычисленные на основе параметров веса.  
  
## Требования  
 **Заголовок:** \<random\>  
  
 **Пространство имен:** std
