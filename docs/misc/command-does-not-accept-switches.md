# Command does not accept switches.
Эта ошибка обычно возникает, если для команды, воспринимающей только аргументы, указан параметр.  
  
### Чтобы исправить эту ошибку  
  
1.  Просмотрите требования к синтаксису для команды.  Дополнительные сведения см. в разделе [Команды Visual Studio с аргументами](../Topic/Visual%20Studio%20Commands.md).
