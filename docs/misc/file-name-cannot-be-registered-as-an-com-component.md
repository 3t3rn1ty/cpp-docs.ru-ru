# File &lt;name&gt; cannot be registered as an COM component.
Файл нельзя добавить в число ссылок проекта.  Эта ошибка возникает, если файл был поврежден, не содержит библиотек типов, использует несовместимый тип файла или если у системы недостаточно ресурсов для завершения операции.  
  
### Чтобы исправить эту ошибку  
  
1.  Убедитесь, что файл содержит допустимую библиотеку типов и повторите попытку.  
  
     —или—  
  
     Переустановите продукт, установивший файл.  
  
    > [!NOTE]
    >  Файл может бытm установлен не Visual Studio.
