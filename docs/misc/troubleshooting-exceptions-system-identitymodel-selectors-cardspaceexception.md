# Разрешение вопросов, связанных с исключениями: System.IdentityModel.Selectors.CardSpaceException
Исключение <xref:System.IdentityModel.Selectors.CardSpaceException> вызывается, когда одно или несколько исключений произошло в службе CardSpace. Причина ошибки записана в журнал событий.  
  
## Примечания  
 Условия возникновения этого исключения включают время ошибки — при загрузке хранилища или книги.
