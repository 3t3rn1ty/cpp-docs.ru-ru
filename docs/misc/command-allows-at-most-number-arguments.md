# Command allows at most &lt;number&gt; arguments.
Эта ошибка обычно возникает, если для команды введено слишком много аргументов или было использовано неверное сочетание аргументов.  
  
### Чтобы исправить эту ошибку  
  
1.  Проверьте в документации правильность синтаксиса команды и повторите попытку.
