# The &lt;name&gt; switch requires a value.
Эта ошибка обычно возникает, если параметр задан без обязательного значения.  
  
### Чтобы исправить эту ошибку  
  
1.  Введите команду еще раз, указав значение параметра.  
  
     —или—  
  
     Введите команду еще раз, опустив параметр.
