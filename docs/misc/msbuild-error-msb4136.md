# Ошибка MSBuild MSB4136
**MSB4136: ошибка чтения информации о конфигурации.**  
  
 В [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)] получена ошибка при попытке чтения информации о наборе инструментов в файле конфигурации [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)] \(msbuild.exe.config\).  
  
### Чтобы исправить эту ошибку  
  
-   Убедитесь, что файл конфигурации сформирован правильно.  Например, если вы настраивали файл конфигурации, добавляя в него набор инструментов, проверьте определение набора инструментов.
