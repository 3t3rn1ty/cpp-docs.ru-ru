# Property value is not valid. (Visual Studio) (2)
Эта ошибка обычно возникает, если введено недопустимое значение свойства.  Например, использование неправильного формата для целого числа или ввод слишком большого значения вместо короткого.  
  
### Чтобы исправить эту ошибку  
  
1.  Введите значение, допустимое для этого свойства.  
  
    > [!TIP]
    >  Обратитесь к интерактивной документации, чтобы установить, какие значения являются допустимыми для этого свойства.
