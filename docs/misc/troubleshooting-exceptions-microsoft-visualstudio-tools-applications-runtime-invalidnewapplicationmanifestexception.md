# Разрешение вопросов, связанных с исключениями: Microsoft.VisualStudio.Tools.Applications.Runtime.InvalidNewApplicationManifestException
Это исключение генерируется в случае, если манифест приложения, на который ссылается манифест развертывания, недопустим.
