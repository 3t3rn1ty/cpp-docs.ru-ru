# Разрешение вопросов, связанных с исключениями: System.Runtime.InteropServices.MarshalDirectiveException
Исключение, которое создается модулем упаковки и передачи, когда он встречает неподдерживаемый атрибут <xref:System.Runtime.InteropServices.MarshalAsAttribute>.
