# Ошибка MSBuild MSB3178
**MSB3178: сборка "\<сборка\>" неверно задана как файл.**  
  
 Это предупреждение появляется во время создания манифеста приложения, когда в процессе построения обнаруживается, что ссылка на файл в действительности указывает на \(управляемую или машинную\) сборку.
