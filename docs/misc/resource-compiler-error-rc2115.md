# Ошибка компилятора ресурсов RC2115
В элементе управления ожидалась текстовая строка или порядковый номер  
  
 Поле *text* оператора CONTROL в операторе **DIALOG** должно быть текстовой строкой или порядковым номером типа элемента управления. При использовании порядкового номера убедитесь, что у вас есть оператор `#define` для элемента управления.
