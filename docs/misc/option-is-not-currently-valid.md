# Option is not currently valid.
Эта ошибка обычно возникает, когда аргумент или параметр, указанные для команды, в настоящий момент недоступны.  Например, если вводится `>Edit.Find test /sel`, а в файле нет выделенного текста, параметр \/sel будет недоступен.  
  
### Чтобы исправить эту ошибку  
  
1.  Проверьте, что введенные аргументы и параметры доступны в настоящий момент.  Дополнительные сведения см. в разделе [Команды Visual Studio с аргументами](../Topic/Visual%20Studio%20Commands.md).
