# Выбор параметров проекта теста, мастер создания пакетов Visual Studio
Этот шаг позволяет:  
  
-   создать проект интеграционного теста для VSPackage;  
  
-   создать проект модульного теста для VSPackage.  
  
 **Проект интеграционного теста**  
 Создает проект интеграционного теста. Интеграционные тесты работают в экспериментальном экземпляре Visual Studio и используют модель автоматизации для тестирования интеграции пакета в Visual Studio.  
  
 **Проект модульного теста**  
 Создает проект модульного теста. Модульные тесты — это небольшие экземпляры MSTest, которые можно использовать для тестирования отдельных классов и членов в пакете.  
  
 **Cancel**  
 Отменяет и останавливает команду создания проекта.  
  
 **Назад**  
 Возврат к предыдущему шагу.  
  
 **Готово**  
 Создает пакет VSPackage.
