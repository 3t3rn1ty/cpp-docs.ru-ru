# Разрешение вопросов, связанных с исключениями: Microsoft.VisualStudio.Tools.Applications.Runtime.CannotGetCachedDataManifestException
Исключение возникает, если манифест, перечисляющий все кэшированные объекты данных, поврежден или отсутствует.
