# Ошибка MSBuild MSB3183
**MSB3183: ссылка "\<сборка\>" является сборкой взаимодействия, требующей полного доверия.**  
  
 Это предупреждение появляется при обнаружении сборки взаимодействия среди ссылок во время создания манифеста для приложения с неполным доверием.
