# Object with program ID &lt;number&gt; cannot be created.
Строка класса, которую вы пытаетесь использовать, не является допустимой.  Данное сообщение — это ошибка операционной системы.  
  
### Чтобы исправить эту ошибку  
  
1.  Переустановите Visual Studio или соответствующее приложение и повторите попытку.
