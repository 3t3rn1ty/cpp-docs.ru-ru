# The following components could not be browsed:
В данном окне сообщений содержится список ошибок, относящихся к диалоговому окну обозревателя объектов или выбора компонентов.  Наиболее часто встречаются следующие ошибки.  
  
## Недоступный для просмотра в обозревателе объектов тип компонентов или файлов.  
 Эта ошибка обычно возникает, когда указывается несуществующее или недопустимое имя файла или файл, который нельзя просмотреть через обозреватель объектов.  
  
#### Чтобы исправить эту ошибку  
  
-   Убедитесь, что указанное имя файла существует и является компонентом, который можно просмотреть в браузере, например сборкой .NET, библиотекой типов COM или файлом обозревателя исходного кода.  
  
## Файл не является допустимым компонентом .NET Framework или COM.  
 Эта ошибка обычно возникает, когда указанный компонент не является сборкой .NET Framework или библиотекой типов COM.  
  
#### Чтобы исправить эту ошибку  
  
-   Выберите другой компонент для просмотра.  
  
## Не удается зарегистрировать файл в качестве компонента COM.  
 Эта ошибка обычно возникает, когда библиотека типов для компонента COM не может быть зарегистрирована.  Библиотека типов, возможно, не существует или повреждена.  
  
#### Чтобы исправить эту ошибку  
  
-   Переустановите компонент и повторите попытку.  
  
## Соответствующая библиотека типов не зарегистрирована или зарегистрированные данные недопустимы.  
 Указанная библиотека типов, возможно, больше не существует или не содержит допустимые данные.  
  
#### Чтобы исправить эту ошибку  
  
-   Убедитесь, что данная библиотека типов существует и зарегистрирована правильно.  
  
## Компоненты, необходимые для просмотра библиотек типов COM, отсутствуют или неправильно зарегистрированы.  
 Файл vstlbinf.dll отсутствует или зарегистрирован неправильно.  
  
#### Исправление ошибки  
  
1.  Найдите на компьютере файл vstlbinf.dll и зарегистрируйте его, используя regsvr32.exe.  
  
     —или—  
  
2.  Если файл vstlbinf.dll не найден на компьютере, переустановите данный продукт.
