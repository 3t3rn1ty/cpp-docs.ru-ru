# Ошибка MSBuild MSB3176
**MSB3176: указанная минимальная требуемая версия старше текущей версии публикации.  Укажите минимальную требуемую версию, младше или равную текущей версии публикации.**  
  
 Эта ошибка появляется, когда требуемая минимальная версия приложения больше текущей версии манифеста развертывания.  
  
### Чтобы исправить эту ошибку  
  
-   Укажите минимальную требуемую версию, меньшую или равную текущей версии публикации.
