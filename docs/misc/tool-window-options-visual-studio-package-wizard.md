# Параметры окна инструментов, мастер создания пакетов Visual Studio
Этот шаг позволяет определить имя и идентификатор команды для вашего окна инструментов.  
  
 **Имя окна**  
 Позволяет вводить имя, которое будет отображаться в строке заголовка окна инструментов. Это имя также будет именем команды меню, которая открывает это окно инструментов.  
  
 Команда меню для окна инструментов добавляется в подменю **Другие окна** меню **Вид**.  
  
 **Идентификатор команды**  
 Позволяет определить идентификатор команды для вашего окна инструментов. Этот идентификатор является константой, которая используется в создаваемом коде для ссылки на этот пункт меню.  
  
 **Cancel**  
 Отменяет создание VSPackage.  
  
 **Назад**  
 Переход на предыдущую страницу мастера.  
  
 **Готово**  
 Создает пакет VSPackage.
