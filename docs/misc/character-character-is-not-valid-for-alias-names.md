# Character &lt;character&gt; is not valid for alias names.
Эта ошибка обычно возникает, если в имени псевдонима использован недопустимый знак.  Имена псевдонимов не могут содержать пробел, "\=", "\[, \]", »\/", "\-" и """.  
  
### Чтобы исправить эту ошибку  
  
1.  Повторно введите имя псевдонима, исключая неверные знаки.
