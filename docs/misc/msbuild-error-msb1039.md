# Ошибка MSBuild MSB1039
**MSB1039: Укажите версию данного набора инструментов.**  
  
 При использовании ключа **\/ToolsVersion** не было указано значение.  
  
### Чтобы исправить эту ошибку  
  
-   При использовании ключа **\/ToolsVersion** убедитесь в том, что указано допустимое непустое значение для `ToolsVersion`.
