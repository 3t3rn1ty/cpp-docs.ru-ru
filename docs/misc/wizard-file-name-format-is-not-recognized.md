# Wizard file &lt;name&gt; format is not recognized.
Не удается открыть мастер.  Эта ошибка обычно вызывается поврежденным файлом мастера или тем, что мастер не соответствует номеру версии Visual Studio.  
  
### Чтобы исправить эту ошибку  
  
1.  Убедитесь, что номер версии файла мастера совпадает с версией Visual Studio, указанной как значение VSWIZARD, например `VSWIZARD 7.0`.  
  
     —или—  
  
     Переустановите Visual Studio, чтобы восстановить поврежденный файл.
