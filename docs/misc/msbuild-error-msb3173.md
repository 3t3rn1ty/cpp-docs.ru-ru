# Ошибка MSBuild MSB3173
**MSB3173: не удалось записать манифест "\<файл\>". '  "\<ошибка\>"**  
  
 Эта ошибка появляется, если в процессе построения обнаружена общая проблема при записи файла манифеста.  
  
### Чтобы исправить эту ошибку
