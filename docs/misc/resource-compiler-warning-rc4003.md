# Предупреждение компилятора ресурсов RC4003
не хватает фактических параметров для макроса "идентификатор"  
  
 Число фактических параметров, определенных с помощью данного идентификатора, меньше, чем число формальных параметров, заданных в макроопределении идентификатора.  
  
 Когда в определении имеется ссылка на формальный параметр, а соответствующий фактический параметр не предоставлен, в расширение макроса подставляется пустой текст.
