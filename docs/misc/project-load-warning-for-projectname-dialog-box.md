# Предупреждение при загрузке проектов для диалогового окна &lt;имя_проекта&gt;
Данное диалоговое окно появляется, если параметры для проектов, открытых в интегрированной среде разработки, загружены некорректно.  
  
 **Исправьте указанные ошибки и перегрузите проект**  
 Предоставляет сведения об осложнениях, таких как неверный путь или имя файла.  Для исправления ошибок удалите или замените недопустимые ссылки на странице свойств проекта.  Сохраните проект и перезагрузите его в интегрированной среде разработки.  
  
 **Не показывать больше этот диалог в течение данной сессии**  
 Если этот параметр выбран, предупреждение загрузки проекта для \<диалогового окна Projectname\> больше не отображается до тех пор, пока при следующем открытии интегрированной среды разработки.
