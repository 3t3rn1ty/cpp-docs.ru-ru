# Для средства требуется допустимый исполняемый файл. Если этот инструмент не нужен, нажмите кнопку &quot;Удалить&quot; в диалоговом окне &quot;Внешние инструменты&quot;.
Эта ошибка обычно возникает, когда в диалоговое окно "Внешние инструменты" был добавлен новый инструмент, но не было указан исполняемый файл.  
  
### Чтобы исправить эту ошибку  
  
1.  Если требуется сохранить инструмент, нажмите кнопку **ОК** и введите имя файла в поле **Команда** для инструмента или укажите файл, нажав кнопку с троеточием.  
  
2.  Чтобы не сохранять этот инструмент, нажмите кнопку **ОК**, а затем — кнопку **Удалить** в диалоговом окне **Внешние инструменты**.
