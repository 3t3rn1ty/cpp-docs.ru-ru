# The project &#39;project&#39; cannot be referenced. &lt;reason&gt;
Ссылка между проектами на определенный проект не может быть создана.  
  
 Наиболее распространенной причиной этой ошибки является ссылка на проект, который создает EXE\-файл.  Эта ошибка также указывает на то, что проект ссылается на другой проект, результат которого не является сборкой \(например, собственное приложение на C\+\+\).
