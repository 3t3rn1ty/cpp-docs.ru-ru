# The project file is missing the &#39;subsection&#39; sub-section of the &#39;section&#39; section
Параметр "SubSection" \(подраздел\) имеет значение "Include", а параметр "section" \(раздел\) — значение "Files".  
  
 Эта ошибка часто возникает при редактировании файла проекта вручную.  
  
 **Чтобы исправить эту ошибку**  
  
-   Перейдите в режим **Показать все файлы**, нажав соответствующую кнопку в панели инструментов обозревателя решений, и включите необходимые файлы.  
  
     Если появляется эта ошибка, проект откроется, но к нему не будут добавлены файлы.
