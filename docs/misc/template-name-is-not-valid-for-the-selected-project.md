# Template &lt;name&gt; is not valid for the selected project.
Эта ошибка обычно возникает, если для команды **Создать файл**, `File.NewFile`, или для команды **Добавить новый элемент решения**, `File.AddNewSolutionItem`, введено неизвестное имя шаблона.  
  
### Чтобы исправить эту ошибку  
  
1.  Повторно введите команду и ее аргументы.  
  
     —или—  
  
2.  Проверьте правильность синтаксиса в разделах «Команда "Создать файл"» или «Команда "Добавить новый элемент"».
