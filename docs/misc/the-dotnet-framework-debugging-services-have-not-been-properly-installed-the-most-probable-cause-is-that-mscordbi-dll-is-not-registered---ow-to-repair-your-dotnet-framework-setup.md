# Службы отладки .NET Framework установлены неправильно. Наиболее вероятная причина: неправильная регистрация библиотеки mscordbi.dll. Для получения дополнительных сведений о том, как исправить установку .NET Framework, нажмите кнопку &quot;Справка&quot;.
Эта ошибка обычно возникает, когда Visual Studio не удается найти файл mscordbi.dll, установленный со службами отладки среды CLR.  
  
### Исправление ошибки  
  
-   Восстановите установку .NET Framework на компьютере.
