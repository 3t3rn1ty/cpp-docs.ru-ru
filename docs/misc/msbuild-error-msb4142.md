# Ошибка MSBuild MSB4142
**MSB4142: MSBuildToolsPath не совпадает с MSBuildBinPath для ToolsVersion "\<x.x\>".**  
  
 В определении набора инструментов указаны разные значения для атрибутов `MSBuildBinPath` и `MSBuildToolsPath`.  
  
### Чтобы исправить эту ошибку  
  
-   Убедитесь, что в определении набора инструментов значения `MSBuildBinPath` и `MSBuildToolsPath` совпадают.
