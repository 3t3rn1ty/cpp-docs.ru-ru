# Unable to remove the last generated output
Последние созданные выходные данные не были удалены с диска и\/или из проекта.  
  
 Если в пользовательском средстве произошел сбой, его последние выходные данные \(если таковые имеются\) удаляются с диска и из проекта.  
  
 Если эти данные заблокированы на диске, может произойти ошибка.  
  
 **Чтобы исправить эту ошибку**  
  
-   Закройте и перезапустите среду [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)].
