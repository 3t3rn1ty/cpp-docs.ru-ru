# Разрешение вопросов, связанных с исключениями: System.Data.SqlServerCe.SqlCeLockTimeoutException
Исключение `System.Data.SqlServerCe.SqlCeLockTimeoutException` возникает по прошествии времени ожидания блокировки, если в SQL Server Compact истекло время ожидания блокировки. По умолчанию время ожидания блокировки равно 2 000 миллисекунд. Дополнительные сведения см. в разделе [Библиотека классов SQL Server Compact 3.5](http://go.microsoft.com/fwlink/?LinkID=102595).
