# Разрешение вопросов, связанных с исключениями: System.ComponentModel.WarningException
Указывает исключение, которое будет обрабатываться как предупреждение, а не как ошибка.  
  
## Примечания  
 Если такое исключение не обработано, оно выводится пользователю как предупреждение.
