# Диалоговое окно &quot;Очистить переопределенные свойства&quot;
> [!NOTE]
>  Это диалоговое окно не поддерживается в [!INCLUDE[vs_dev10_long](../build/includes/vs_dev10_long_md.md)].  
  
 При добавлении определенной пользователем страницы свойств \(PROPS\-файла\) в конфигурацию проекта эта конфигурация наследует все свойства, определенные на этой странице по умолчанию.  Это диалоговое окно служит для очистки нежелательных унаследованных свойств в конфигурации проекта.
