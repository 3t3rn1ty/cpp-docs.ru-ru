# This tool could not be executed. Please ensure that it points to a valid executable.
Эта ошибка обычно возникает при неудачной попытке запустить внешний инструмент.  Причина этой ошибки может заключаться в том, что указанный файл не является допустимым исполняемым или пакетным файлом.  
  
### Чтобы исправить эту ошибку  
  
1.  Убедитесь, что внешний инструмент является допустимым исполняемым или пакетным файлом, используя диалоговое окно **Внешние инструменты**, доступное из меню **Сервис**.
