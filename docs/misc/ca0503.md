# CA0503
Свойство не рекомендуется использовать.  Используйте более приоритетное свойство  
  
 Предупреждение СА503 создается, если в нерекомендуемом свойстве анализа кода содержится значение из файла конфигурации проекта или построения.  Для устранения проблемы, ставшей причиной этого предупреждения, воспользуйтесь заданным свойством с более высоким приоритетом.
