# Tab name &lt;name&gt; already exists.
Имя каждой вкладки в панели элементов должно быть уникальным.  Эта ошибка обычно возникает, когда имя, введенное для вкладки панели элементов, совпадает с существующей вкладкой панели элементов.  
  
### Чтобы исправить эту ошибку  
  
1.  Введите для вкладки уникальное имя.
