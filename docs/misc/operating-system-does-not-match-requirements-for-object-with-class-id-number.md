# Operating system does not match requirements for object with class ID &lt;number&gt;.
Эта ошибка обычно возникает при обнаружении несовместимой операционной системы или несовместимой версии операционной системы.  
  
### Чтобы исправить эту ошибку  
  
1.  Проверьте системные требования для приложения.  
  
2.  Установите одну из совместимых операционных систем и повторите попытку.
