# Ошибка MSBuild MSB4137
**MSB4137: недопустимое значение указано в файле конфигурации в "\<x\>".  Имя свойства и ToolsVersion не могут быть пустыми строками.**  
  
 Определение набора инструментов содержит недопустимые имена или значения свойств.  
  
### Чтобы исправить эту ошибку  
  
-   Убедитесь, что определение набора инструментов не содержит недопустимых или пустых имен и значений свойств.
