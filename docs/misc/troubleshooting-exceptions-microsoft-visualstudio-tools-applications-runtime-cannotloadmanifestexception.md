# Разрешение вопросов, связанных с исключениями: Microsoft.VisualStudio.Tools.Applications.Runtime.CannotLoadManifestException
Исключение возникает, когда манифест не может быть загружен, поскольку он не находится в документе, поврежден, или недопустим.
