# Ошибка MSBuild MSB3162
**MSB3162: элемент "\<пакет\>" не смог найти зависимый элемент "\<пакет\>".**  
  
 Эта ошибка появляется, когда один пакет загрузчика зависит от другого, но не удалось найти манифест, необходимый для зависимого загрузчика.
