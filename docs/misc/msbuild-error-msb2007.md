# Ошибка MSBuild MSB2007
**Visual Studio не может найти файл проекта "{0}".**  
  
 [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)] не удалось найти указанный файл проекта.  
  
### Чтобы исправить эту ошибку  
  
1.  Проверьте правильность написания имени файла проекта.
