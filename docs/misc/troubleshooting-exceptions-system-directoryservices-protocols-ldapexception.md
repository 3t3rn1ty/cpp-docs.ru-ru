# Разрешение вопросов, связанных с исключениями: System.DirectoryServices.Protocols.LdapException
Это исключение создается, если протокол LDAP возвращает код ошибки, отсутствующий в <xref:System.DirectoryServices.Protocols.ResultCode>.
