# Ошибка MSBuild MSB8020
Не удалось найти средства сборки для *\<платформа\>* \(набор средств платформы \= "*\<версия\>*"\). Чтобы выполнить сборку с помощью версии *\<версия\>* средств сборки, установите средства сборки *\<платформа\>*.  Также можно выполнить обновление до текущих средств Visual Studio, выбрав меню "Проект" или щелкнув правой кнопкой мыши решение, а затем выбрав "Изменить целевую платформу решения".
