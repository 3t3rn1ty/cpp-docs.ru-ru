# &lt;name&gt; is not a valid switch for this command.
Эта ошибка обычно возникает, если для команды были введены неверный параметр или сочетание однозначных параметров.  
  
### Чтобы исправить эту ошибку  
  
1.  Проверьте в документации правильность синтаксиса команды и повторите попытку.
