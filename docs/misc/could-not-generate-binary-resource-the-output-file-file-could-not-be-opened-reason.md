# Не удалось создать двоичный файл ресурса. Не удалось открыть выходной файл. &lt;причина&gt;
Системе проектов не удалось удалить устаревшую копию RESOURCES\-файла. В процессе подготовки ресурсов система проектов преобразует XML\-файлы с расширением RESX в двоичные RESOURCES\-файлы для диспетчера ресурсов .NET.  
  
 Все необходимые сведения содержатся в связанной с операционной системой причине, которая указана в конце сообщения об ошибке.  
  
 **Исправление ошибки**  
  
-   Перезапустите [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)].  
  
     Эта ошибка приведет к сбою процесса сборки.
