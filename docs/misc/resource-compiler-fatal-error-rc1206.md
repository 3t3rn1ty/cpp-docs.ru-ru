# Неустранимая ошибка компилятора ресурсов RC1206
кодовая страница, заданная в командной строке, отсутствует в реестре  
  
 Вслед за параметром "Указать кодовую страницу" \(\/c\) указана кодовая страница, местоположение которой не было задано в реестре. Запустите программу установки, чтобы установить указанную кодовую страницу.
