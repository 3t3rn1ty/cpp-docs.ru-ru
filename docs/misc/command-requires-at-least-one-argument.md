# Command requires at least one argument.
Эта ошибка обычно возникает, когда для команды, которой требуется как минимум один аргумент, не было введено ни одного аргумента.  
  
### Чтобы исправить эту ошибку  
  
1.  Проверьте в документации правильность синтаксиса команды и повторите попытку.
