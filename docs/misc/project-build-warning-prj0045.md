# Предупреждение сборки проекта PRJ0045
Не задано свойство Outputs для настраиваемого правила сборки "правило", назначенного файлу "файл". Выполнение правила будет пропущено.  
  
 Свойство Outputs не определено для настраиваемого правила сборки. Настраиваемым правилам сборки требуются выходные данные действия сборки.
