# File name &lt;name&gt; cannot contain character device names.
Эта ошибка обычно возникает, когда введенное имя файла совпадает с буквой устройства.  Имена устройств включают CON, PRN, AUX, CLOCK$, NUL, LPT1, LPT2, COM1, COM2, COM3 и COM4, но не ограничиваются ими.  
  
### Чтобы исправить эту ошибку  
  
1.  Введите другое имя файла.
