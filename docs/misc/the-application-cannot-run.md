# The application cannot run.
Недостаточно памяти для выполнения Visual Studio.  
  
### Чтобы исправить эту ошибку  
  
1.  Убедитесь, что компьютер соответствует минимальным требованиям для выполнения Visual Studio.  
  
2.  Чтобы освободить память, закройте другие программы, работающие на этом компьютере.  
  
     —или—  
  
     Переустановите Visual Studio.
