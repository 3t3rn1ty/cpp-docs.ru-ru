# Ошибка вычислителя выражений CXX0042
неправильный номер строки  
  
 Эта ошибка никогда не должна возникать.  
  
 Соберите сведения об обстоятельствах появления ошибки и обратитесь в корпорацию Майкрософт. Вы можете сообщить об этой ошибке в Интернете на веб\-сайте [http:\/\/support.microsoft.com\/support\/visualc\/report\/](http://support.microsoft.com/support/visualc/report/) или позвонить в службу технической поддержки корпорации Майкрософт. Чтобы получить дополнительные сведения, выберите пункт **Техническая поддержка** в меню **Справка** в Visual C\+\+.  
  
 Эта ошибка идентична ошибке CAN0042.
