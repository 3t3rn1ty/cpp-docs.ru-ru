# Неустранимая ошибка компилятора ресурсов RC1023
не удается открыть файл с исходным кодом "имя\_файла"  
  
 Указанный файл не существует, не может быть открыт либо не найден.  
  
 Убедитесь в том, что параметры среды являются допустимыми и путь к файлу указан верно.  
  
 Если эта ошибка возникает без сообщения об ошибке, компилятору ресурсов не хватает дескрипторов файлов.
