# CA0057
Не удается сопоставить нарушения с текущим набором целевых объектов и правил  
  
 Эта ошибка возникает, если процесс FxCopCmd.exe вызывается вручную с файлом проекта или импортированным XML\-отчетом с нарушениями, которые не удается сопоставить с текущим набором целевых объектов и правил.  
  
 Эта ошибка возникает только в случае, если для анализа используется версия данной программы, предназначенная для командной строки.
