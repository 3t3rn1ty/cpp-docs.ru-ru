# A failure occurred writing to the resources file
Система работы с проектами не может записать выходной файл ресурсов.  В процессе подготовки ресурсов система работы с проектами преобразует XML\-файлы с расширением RESX в двоичные файлы для диспетчера ресурсов .NET.  
  
 Зависящая от операционной системы причина \(если таковая имеется\) будет указана в конце этого сообщения.  Будет указано также имя RESX\-файла.  
  
 При возникновении этой ошибки процесс построения завершается неудачей.
