# Разрешение вопросов, связанных с исключениями: System.Net.Mail.SmtpFailedRecipientsException
Исключение вызывается, когда электронная почта отправляется с использованием <xref:System.Net.Mail.SmtpClient> и не может быть доставлена всем адресатам.
