# Общие сведения об определениях для грамматики
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Терминальные слова — это конечные точки в определении синтаксиса.  Никакое другое разрешение невозможно.  Терминальные слова включают в себя набор зарезервированных ключевых слов и определенные пользователем идентификаторы.  
  
 Нетерминальные слова — это местозаполнители в синтаксисе.  Большая их часть определена в других местах этой сводки синтаксиса.  Определения могут быть рекурсивными.  В [Сводке по грамматике](../misc/grammar-summary-cpp.md) *Справочника по языку C\+\+* определены следующие нетерминальные слова:  
  
 `constant`, *константное\-выражение*, *идентификатор*, *ключевое слово*, `operator`, `punctuator`  
  
 Необязательный компонент указывается нижним индексом "необ.".  Например, ниже приведено необязательное выражение, заключенное в фигурные скобки:  
  
 **{** *выражение*необ. **}**
