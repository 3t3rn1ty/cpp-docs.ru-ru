# push_macro
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Сохраняет значение *имени\-макроса* на вершине стека этого макроса.  
  
## Синтаксис  
  
```  
  
#pragma push_macro("  
macro_name  
")  
  
```  
  
## Заметки  
 Можно извлечь значение *имени\-макроса* с **pop\_macro**.  
  
 Пример см. в разделе [pop\_macro](../Topic/pop_macro.md).
