# <a name="compiler-warnings-that-are-off-by-default"></a>Выключенные по умолчанию предупреждения компилятора

Компилятор включает в себя предупреждения, отключенные по умолчанию, поскольку большинство пользователей не хотят видеть их. Однако такие предупреждения можно включить с помощью одного из следующих параметров.

`#pragma warning(default : warning_number )`  
Указанное предупреждение (`warning_number`) включается на соответствующем уровне по умолчанию. Документация по предупреждениям содержит уровень предупреждения по умолчанию.

`#pragma warning( warning_level : warning_number )`  
Указанное предупреждение (`warning_number`) включается на соответствующем уровне (`warning_level`).

[/ Wall](../build/reference/compiler-option-warning-level.md)  
**/ Wall** включает все предупреждения, отключенные по умолчанию.
