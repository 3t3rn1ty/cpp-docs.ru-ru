# <a name="null-directive"></a>Директива NULL
Директива null препроцессора представляет знак номера (**#**) отдельно в строке. Она не имеет никакого влияния.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
#  
  
```
