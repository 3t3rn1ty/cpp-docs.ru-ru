# no_search_namespace
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Блок, относящийся только к языку C\+\+**  
  
 Имеет ту же функцию, что и атрибут [no\_namespace](../Topic/no_namespace.md), однако применяется для библиотек типов, для которых вы использовали директиву \#import с атрибутом [auto\_search](../preprocessor/auto-search.md).  
  
## Синтаксис  
  
```  
no_search_namespace  
```  
  
## Заметки  
 **Завершение блока, относящегося только к языку C\+\+**
