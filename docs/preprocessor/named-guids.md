# named_guids
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Блок, относящийся только к языку C\+\+**  
  
 Указывает компилятору определять и инициализировать переменные GUID в старом стиле, в форме **LIBID\_MyLib**, **CLSID\_MyCoClass**, **IID\_MyInterface** и **DIID\_MyDispInterface**.  
  
## Синтаксис  
  
```  
named_guids  
```  
  
## Заметки  
 **Завершение блока, относящегося только к языку C\+\+**
