# Флаг направления
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Флаг направления — это флаг ЦП, относящийся к процессорам 80x86.  Он применяется ко всем инструкциям ассемблера, которые используют префикс REP \(повтор\), например MOVS, MOVSD, MOVSW и другие.  Адреса, предоставленные применимым инструкциям увеличены, если флаг направления не установлен.  
  
 Подпрограммы времени выполнения языка C предполагают, что флаг направления не установлен.  При использовании других функций совместно с функциями среды выполнения языка C, необходимо убедиться в том, что другие функции не изменяют флаг направления или устанавливают его в исходное состояние.  Ожидание не установленного флага направления после запуска делает код среды выполнения более быстрым и более эффективным.  
  
 Функции библиотеки времени выполнения языка С, например управления строками и буферами, предполагают что флаг направления не установлен.
