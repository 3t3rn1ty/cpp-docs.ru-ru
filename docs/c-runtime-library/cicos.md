# <a name="cicos"></a>_CIcos
Вычисляет косинус верхнего значения в стеке.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
void __cdecl _CIcos();  
```  
  
## <a name="remarks"></a>Примечания  
 Эта версия функции `cos` включает специальные соглашения о вызовах, распознаваемые компилятором. Это ускоряет выполнение, поскольку исключает создание копий и помогает распределять регистры.  
  
 Полученное значение помещается в верхнюю часть стека.  
  
## <a name="requirements"></a>Требования  
 **Платформа:** x86
