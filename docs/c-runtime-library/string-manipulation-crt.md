# Управление строками (CRT)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Эти процедуры работают с завершающимися символом null строками из однобайтовых, расширенных и многобайтовых символов.  Используйте процедуры управления буфером, описанные в разделе [Управления буфером](../Topic/Buffer%20Manipulation.md), для работы с массивами символов, которые не заканчиваются нуль\-символом.
