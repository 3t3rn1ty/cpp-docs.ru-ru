# _FREEENTRY, _USEDENTRY
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

## Синтаксис  
  
```  
#include <malloc.h>  
```  
  
## Заметки  
 Эти константы представляют значения, присвоенные процедурами `_heapwalk` элементу **\_useflag** структуры **\_HEAPINFO**.  Они показывают состояние записи кучи.
