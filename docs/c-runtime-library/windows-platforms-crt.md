# <a name="windows-platforms-crt"></a>Платформы Windows (CRT)
Библиотеки времени выполнения C для [!INCLUDE[vs2012_upd01](../c-runtime-library/includes/vs2012_upd01_md.md)] поддерживают [!INCLUDE[win8](../build/reference/includes/win8_md.md)], [!INCLUDE[winserver8](../build/reference/includes/winserver8_md.md)], [!INCLUDE[win7](../build/includes/win7_md.md)], [!INCLUDE[winsvr08](../build/reference/includes/winsvr08_md.md)], Windows Vista, [!INCLUDE[winxp](../build/includes/winxp_md.md)] с пакетом обновления 3 (SP3) для x86, [!INCLUDE[winxp](../build/includes/winxp_md.md)] с пакетом обновления 2 (SP2) для x64 и [!INCLUDE[WinXPSvr](../build/includes/winxpsvr_md.md)] c пакетом обновления 2 (SP2) как для x86, так и для x64. Все эти операционные системы поддерживают интерфейс API Windows и обеспечивают поддержку Юникода. Кроме того, любое приложение Win32 может использовать многобайтовую кодировку (MBCS).  
  
> [!NOTE]
>  [!INCLUDE[vs_dev11_long](../build/includes/vs_dev11_long_md.md)] не поддерживает разработку для [!INCLUDE[winxp](../build/includes/winxp_md.md)] и [!INCLUDE[WinXPSvr](../build/includes/winxpsvr_md.md)]. Необходимо установить [!INCLUDE[vs2012_upd01](../c-runtime-library/includes/vs2012_upd01_md.md)], чтобы добавить поддержку [!INCLUDE[winxp](../build/includes/winxp_md.md)] и [!INCLUDE[WinXPSvr](../build/includes/winxpsvr_md.md)].
