# FOPEN_MAX, _SYS_OPEN
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

## Синтаксис  
  
```  
  
#include <stdio.h>  
  
```  
  
## Заметки  
 Это максимальное количество файлов, которые можно открыть одновременно.  `FOPEN_MAX` — ANSI\-совместимое имя.  `_SYS_OPEN` предоставлена для обеспечения совместимости с существующим кодом.
