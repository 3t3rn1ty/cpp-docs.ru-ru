# _iob
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Массив структур элементов управления stdio.  
  
## Синтаксис  
  
```  
FILE _iob[_IOB_ENTRIES];  
```  
  
## Заметки  
 `IOB_ENTRIES` определено как 20 в stdio.h.
