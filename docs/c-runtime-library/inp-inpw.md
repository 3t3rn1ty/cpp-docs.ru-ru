# <a name="inp-inpw"></a>inp, inpw
Эти функции POSIX устарели. Используйте вместо нее соответствующую стандарту ISO C++ функцию [_inp, _inpw, _inpd](../c-runtime-library/inp-inpw-inpd.md) .  
  
> [!IMPORTANT]
>  Эти функции устарели. Начиная с Visual Studio 2015 они недоступны в CRT.  
  
> [!IMPORTANT]
>  Этот API нельзя использовать в приложениях, выполняемых в среде выполнения Windows. Дополнительные сведения см. в статье [Функции CRT, которые не поддерживаются с ключом /ZW](http://msdn.microsoft.com/library/windows/apps/jj606124.aspx).
