# <a name="carg-cargf-cargl"></a>carg, cargf, cargl

Извлекает аргумент комплексного числа, ветви которого заканчиваются в отрицательной части реальной оси.

## <a name="syntax"></a>Синтаксис

```C
double carg(
   _Dcomplex z
);
float carg(
   _Fcomplex z
);  // C++ only
long double carg(
   _Lcomplex z
);  // C++ only
float cargf(
   _Fcomplex z
);
long double cargl(
   _Lcomplex z
);
```

### <a name="parameters"></a>Параметры

*z*<br/>
Комплексное число.

## <a name="return-value"></a>Возвращаемое значение

Аргумент (этап) *z*. Результатом является в интервале [-π, + π].

## <a name="remarks"></a>Примечания

Поскольку C++ допускает перегрузку, можно вызывать перегрузки **carg** , которые принимают **_Fcomplex** или **_Lcomplex** значений и возвращают **float** или **длинные** **двойные** значения. В программе на языке C **carg** всегда принимает **_Dcomplex** значения и возвращает **двойные** значение.
