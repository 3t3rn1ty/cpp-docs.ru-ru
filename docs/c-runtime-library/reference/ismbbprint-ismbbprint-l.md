# <a name="ismbbprint-ismbbprintl"></a>_ismbbprint, _ismbbprint_l
Определяет, является ли указанный многобайтовой символ печатным символом.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
int _ismbbprint(  
   unsigned int c   
);  
int _ismbbprint_l(  
   unsigned int c,  
   _locale_t locale  
);  
```  
  
#### <a name="parameters"></a>Параметры  
 `c`  
 Целое число, которое требуется проверить.  
  
 `locale`  
 Используемый языковой стандарт.  
  
## <a name="return-value"></a>Возвращаемое значение  
 `_ismbbprint` возвращает ненулевое значение, если выражение:  
  
```  
isprint || _ismbbkprint  
```  
  
 имеет ненулевое значение для `c`, или значение 0, если это не так. Функция`_ismbbprint` использует текущий языковой стандарт для любого поведения, зависящего от языкового стандарта. Функция`_ismbbprint_l` идентична за исключением того, что использует переданный языковой стандарт. Для получения дополнительной информации см. [Locale](../../c-runtime-library/locale.md).
