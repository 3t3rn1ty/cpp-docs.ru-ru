# Глобальные переменные и стандартные типы
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Библиотека времени выполнения Microsoft содержит определения для [глобальных переменных](../c-runtime-library/global-variables.md), [управляющих флагов](../c-runtime-library/control-flags.md) и [стандартных типов](../c-runtime-library/standard-types.md) используемых библиотечными процедурами.  Получите доступ к этим переменным, флагам и типам, объявив их в программе или включив необходимые заголовочные файлы.
