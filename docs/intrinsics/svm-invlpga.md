# __svm_invlpga
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Только для систем Microsoft**  
  
 Делает недействительной запись сопоставления адреса в буфере взгляда\-в сторону компьютерного перевода.  Параметры определяют идентификатор виртуального адресного пространства и полей адреса страниц, чтобы сделать недействительными.  
  
## Синтаксис  
  
```  
void __svm_invlpga(  
     void *Va,  
     int ASID);  
```  
  
#### Параметры  
  
|Параметр|Описание|  
|--------------|--------------|  
|\[входящий\] `Va`|Виртуальный адрес веб\-страницы, который требуется сделать недействительным.|  
|\[входящий\] `ASID`|Идентификатор адресного пространства \(ASID\) страницы, который требуется сделать недействительным.|  
  
## Заметки  
 Функция `__svm_invlpga` эквивалентна инструкцие на компьютере `INVLPGA`.  Эта функция поддерживает взаимодействие монитора виртуальной машины узла с операционной системой для виртуальной машине и своими приложениями.  Дополнительные сведения см. в документе «этом 2 программиста архитектуры AMD64 ручного: Программирование системы,» документ 24593, 3,11, на [корпорация AMD](http://go.microsoft.com/fwlink/?LinkId=23746) сайте.
