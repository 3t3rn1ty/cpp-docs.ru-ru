# <a name="int2c"></a>__int2c
**Блок, относящийся только к системам Microsoft**  
  
 Приводит к возникновению ошибки `int 2c` инструкции, которая запускает `2c` прерываний.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
void __int2c(void);  
```
