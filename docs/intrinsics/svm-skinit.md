# __svm_skinit
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Только для систем Microsoft**  
  
 Начинает загрузку verifiably безопасного программного обеспечения, например монитор виртуальной машины.  
  
## Синтаксис  
  
```  
void __svm_skinit(  
   int SLB  
);  
```  
  
#### Параметры  
  
|Параметр|Описание|  
|--------------|--------------|  
|`SLB`|32 Байта 64K предоставляет физическим адресом блока загрузчика \(SLB\).|  
  
## Заметки  
 Функция `__svm_skinit` эквивалентна инструкцие на компьютере `SKINIT`.  Эта функция является частью системы безопасности, которая использует процессор и доверенный платформенный модуль \(надежный платформенный модуль\), который требуется проверить и загрузить доверило именем программного обеспечения средством безопасности \(SK\).  Монитор виртуальной машины примере ядра безопасности.  Система безопасности проверяет, в котором компоненты программы, загруженные в процессе инициализации и обеспечивает защиту от подделки прерываниями компоненты, доступом устройства или другой программой, если компьютер мультипроцессор.  
  
 Параметр `SLB` определяет физический адрес блока памяти называется 64K *защищенный блок загрузчика* \(SLB\).  Содержит программы SLB называемую безопасный загрузчиком, который устанавливает операционную среду для компьютера, и затем загружает ядро безопасности.  
  
 Эта функция поддерживает взаимодействие монитора виртуальной машины узла с операционной системой для виртуальной машине и своими приложениями.  Дополнительные сведения см. в документе «этом 2 программиста архитектуры AMD64 ручного: Программирование системы,» документ 24593, 3,11, на [корпорация AMD](http://go.microsoft.com/fwlink/?LinkId=23746) сайте.
