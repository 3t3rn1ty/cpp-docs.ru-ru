# Обработка сообщений Windows для диалогового окна
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Диалоговые окна, поэтому они могут обрабатывать сообщения Windows, если указываются соответствующие функции обработчика.  При создании класса диалогового окна с помощью класса, добавив мастер добавит пустую схему сообщений в класс.  Используйте окно свойств для сопоставления все сообщения или команды Windows необходимо класс обработки.  
  
 Дополнительные сведения см. в разделе [Сообщения Windows сопоставления к классу диалогового окна](../mfc/mapping-windows-messages-to-your-class.md).
