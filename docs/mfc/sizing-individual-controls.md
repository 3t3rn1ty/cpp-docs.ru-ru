# Sizing Individual Controls
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Для изменения размеров элементов управления используются маркеры размера.  Когда указатель находится над маркером размера, он изменяется, показывая направления, в которых может быть изменен размер элемента управления.  Активные маркеры размеров являются сплошными, если маркер пуст, размер элемента управления по этой оси изменить нельзя.  
  
 Размер элемента управления также можно изменить, привязывая элемент управления к направляющим или полям, либо перемещая один привязанный элемент управления и направляющую прочь от другого элемента.  
  
### Изменение размера элемента управления  
  
1.  Выберите элемент управления.  
  
2.  Перетащите маркеры размера, чтобы изменить размер элемента управления:  
  
    -   маркеры размера вверху и по сторонам изменяют размер по горизонтали или по вертикали.  
  
    -   маркеры размера по углам изменяют размеры и по вертикали, и по горизонтали.  
  
    > [!TIP]
    >  За один раз можно изменить одну единицу диалогового окна, удерживая нажатой клавишу SHIFT и используя клавиши стрелок вверх и вниз.  
  
### Автоматическое изменение размера элемента управления в соответствии с текстом в нем  
  
1.  Выберите пункт **Установить размер по содержимому** в меню **Формат**.  
  
 \- или \-  
  
-   Щелкните элемент управления правой кнопкой мыши и выберите в контекстном меню **Установить размер по содержимому**.
