# <a name="building-on-the-framework"></a>Сборка в платформе

Используйте стандартные технологии C++ и язык C++ для формирования собственные классы конкретного приложения с теми, у библиотеки классов, а также для переопределить и расширить поведение базового класса.

 В большинстве случаев вы можете использовать эти таблицы как последовательность шагов для создания приложения MFC, несмотря на то, что некоторые действия, альтернативные варианты. Например большинство приложений использует один тип класса представления из нескольких доступных типов.
