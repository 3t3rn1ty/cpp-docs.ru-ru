# Использование шаблона диалогового окна в памяти
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Вместо уступанных методов [Таблица создания диалогового окна](../mfc/creating-a-dialog-class-with-code-wizards.md) можно создать любой тип диалогового окна непосредственно из шаблона диалоговых окон в памяти.  Класс [CDialog](../mfc/reference/cdialog-class.md). Дополнительные сведения см. в *справочнике по MFC*.
