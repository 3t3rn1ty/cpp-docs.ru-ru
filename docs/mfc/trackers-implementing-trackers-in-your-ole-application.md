# Средства отслеживания. Реализация средств отслеживания в приложении OLE
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Отслежыватели предоставляют графический интерфейс, чтобы позволить пользователям взаимодействовать с элементами OLE клиента.  С помощью различных стилей отслежывателя, OLE клиентскими можно отобразить с насиженными границами, маркеров изменения размера, или различных другими визуальными эффектами.  В этом разделе описываются:  
  
-   [Реализация отслеживания в исправление кода](../mfc/how-to-implement-tracking-in-your-code.md).  
  
-   [Эластичное соединения и отслежыватели](../Topic/Rubber-Banding%20and%20Trackers.md).  
  
 Статья также описываются с помощью стилей отслежывателями.  Кроме того, создается несколько ссылок в пример [OCLIENT](../top/visual-cpp-samples.md) MFC OLE.
