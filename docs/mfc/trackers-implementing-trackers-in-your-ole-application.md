# <a name="trackers-implementing-trackers-in-your-ole-application"></a>Средства отслеживания. Реализация средств отслеживания в приложении OLE
Средства отслеживания предоставляет графический интерфейс для предоставления пользователям возможности взаимодействия с клиентом OLE-элементы. С помощью отслеживания различных стилей OLE клиентские элементы отображаются со штриховой границей, изменение размера дескрипторы или ряд других визуальных эффектов. В этой статье описываются.  
  
-   [Как реализовать отслеживания в коде](../mfc/how-to-implement-tracking-in-your-code.md).  
  
-   [Эластичные соединения и средства отслеживания](../mfc/rubber-banding-and-trackers.md).  
  
 В статье также описывается использование стилей с помощью средства отслеживания. Кроме того, он выполняет несколько ссылок, пример MFC OLE [OCLIENT](../visual-cpp-samples.md).
