# <a name="trackers-implementing-trackers-in-your-ole-application"></a>Средства отслеживания. Реализация средств отслеживания в приложении OLE

Средства отслеживания предоставляют графический интерфейс, чтобы пользователи могли взаимодействовать с элементами клиента OLE. С помощью различных tracker стилей OLE клиентские элементы, которые могут отображаться со штриховой границей, изменение размера дескрипторы и множество других визуальных эффектов. В этой статье описывается:

- [Как реализовать отслеживания в коде](../mfc/how-to-implement-tracking-in-your-code.md).

- [Эластичные соединения и средства отслеживания](../mfc/rubber-banding-and-trackers.md).

Также рассматривается использование стилей с помощью средства отслеживания. Кроме того, он делает несколько ссылок, пример MFC OLE [OCLIENT](../visual-cpp-samples.md).
