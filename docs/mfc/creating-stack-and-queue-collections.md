# <a name="creating-stack-and-queue-collections"></a>Создание коллекций стеков и очередей
В этой статье описывается создание других структур данных, таких как [стеки](#_core_stacks) и [очереди](#_core_queues), списке классы MFC. В примерах используются классы, производные от `CList`, но можно использовать `CList` напрямую, если необходимо добавить функциональные возможности.  
  
##  <a name="_core_stacks"></a>Стеки  
 Так как стандартный список коллекции головы и хвоста, несложно создать производный список коллекцию, которая имитирует поведение стек последним пришел первым вышел. Стек — например, в стеке лотков в кафе. Лотков добавляются в стек, переходят на вершине стека. Последний добавлен в панели задач является первым для удаления. Функции-члены коллекции списка `AddHead` и `RemoveHead` можно использовать для добавления и удаления элементов в частности из заголовка списка; таким образом, наиболее недавно был добавлен элемент является первым для удаления.

##  <a name="_core_queues"></a>Очереди  
 Так как стандартный список коллекции головы и хвоста, его можно легко создать производный список коллекцию, которая имитирует поведение очереди первым пришел первым вышел. Очередь не является как обычная людей в кафе. В строке первого пользователя является первым к обработке. Как приходят большее количество людей, переходят в конец строки в очередь ожидания. Функции-члены коллекции списка `AddTail` и `RemoveHead` можно использовать для добавления и удаления элементов в частности из head или конца списка; таким образом, наиболее недавно был добавлен элемент всегда является последней для удаления.
