# <a name="win32-internet-extensions-wininet"></a>Расширения Интернета Win32 (WinInet)
Веб-приложение клиента — это программа, получает доступ к сведениям из сетевого источника данных (сервер), используя протоколы Интернета, например gopher, HTTP или FTP. Клиент веб-приложение может получить доступ к серверу для получения данных, например карты погоды, цен на акции или газете, например. Интернет-клиента можно управлять через внешней сети (Интернет) или во внутренней сети (также называемые интрасети).  
  
 MFC включает расширения Интернета Win32 или WinInet, для создания клиентских приложений в Интернете. MFC инкапсулирует эти расширения Интернета в набор классов, стандартные, простые в использовании. Можно написать клиентское приложение WinInet путем вызова функции Win32 напрямую или с помощью классов MFC WinInet.  
  
 Функции Microsoft Win32 Internet (WinInet), чтобы помочь в создании Интернет неотъемлемой частью любого приложения. Новые функции, содержащиеся в WININET. Библиотеки DLL, упростить доступ к Интернету, с помощью HTTP (протокол HTTPS), FTP (File Transfer Protocol) и gopher.
