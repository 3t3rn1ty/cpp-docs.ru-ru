# Сокеты Windows
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Это семейство статей охватывает реализацию MFC Windows SSL.  MFC предоставляет 2 класса для поддержки программирования приложений сети с помощью API Windows SSL.  Класс [CAsyncSocket](../Topic/CAsyncSocket%20Class.md) инкапсулирует API одно Windows SSL для одного, предоставляя расширенные большинство программистов сети и степень гибкости.  Класс [CSocket](../mfc/reference/csocket-class.md) предоставляет упрощенный интерфейс для сериализации данных и обратное преобразование объекта [CArchive](../mfc/reference/carchive-class.md).
