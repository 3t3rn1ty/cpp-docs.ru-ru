# Aligning Groups of Controls
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

В следующей процедуре показано выравнивание группы элементов управления.  
  
### Выравнивание группы элементов управления  
  
1.  [Выберите элементы управления](../mfc/selecting-multiple-controls.md), которые необходимо выровнять.  Главный элемент управления должен быть выбран первым в группе. Главный элемент также можно назначить непосредственно перед выполнением групповой операции выравнивания или изменения размеров.  
  
     Последняя позиция в группе зависит от позиции главного элемента управления.  Дополнительные сведения о выборе главного элемента управления см. в разделе [Выбор главного элемента управления](../mfc/specifying-the-dominant-control.md).  
  
2.  В меню **Формат** выберите **Выровнять**, а затем выберите один из следующих вариантов выравнивания:  
  
    -   **Левые границы**. Выравнивание выбранных элементов управления по их левому краю.  
  
    -   **Центры**. Выравнивание выбранных элементов управления вдоль горизонтальной оси, проходящей через их центральные точки.  
  
    -   **Правые границы**. Выравнивание выбранных элементов управления по их правому краю.  
  
    -   **Верхние границы**. Выравнивание выбранных элементов управления по их верхнему краю.  
  
    -   **Середины**. Выравнивание выбранных элементов управления вдоль вертикальной оси, проходящей через их центральные точки.  
  
    -   **Нижние границы**. Выравнивание выбранных элементов управления по их нижнему краю.
