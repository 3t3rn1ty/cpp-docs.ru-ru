# <a name="template-classes-for-arrays-lists-and-maps"></a>Классы шаблонов для массивов, списков и схем
Эти классы коллекций — это шаблоны, параметры которого определения типов объектов, хранящихся в статистические выражения. `CArray`, `CMap`, И `CList` классы используют глобальный вспомогательные функции, которые обычно должны быть настроены. Дополнительные сведения об этих функциях вспомогательный см. в разделе [вспомогательные функции классов коллекции](../mfc/reference/collection-class-helpers.md). Типизированный указатель классы являются оболочками для другие классы в библиотеке классов. Используя эти оболочки, прикрепите компилятора проверку типов для предотвращения ошибок. Дополнительные сведения об использовании этих классов см. в разделе [коллекции](../mfc/collections.md).  
  
 Эти классы предоставляют шаблоны, которые можно использовать для создания массивов, списков и схем, с помощью любого типа, как вам удобно.  
  
 [CArray](../mfc/reference/carray-class.md)  
 Класс шаблона для создания произвольных типов массивов.  
  
 [CList](../mfc/reference/clist-class.md)  
 Класс шаблона для создания списков произвольных типов.  
  
 [CMap](../mfc/reference/cmap-class.md)  
 Класс шаблона для создания карты с произвольных типов ключей и значений.  
  
 [CTypedPtrArray](../mfc/reference/ctypedptrarray-class.md)  
 Шаблон класса для строго типизированные массивы указателей.  
  
 [CTypedPtrList](../mfc/reference/ctypedptrlist-class.md)  
 Шаблонный класс для строго типизированный список указателей.  
  
 [CTypedPtrMap](../mfc/reference/ctypedptrmap-class.md)  
 Класс шаблона для карт строго типизированным с указателями.
