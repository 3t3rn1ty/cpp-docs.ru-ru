# <a name="writing-mfc-applications"></a>Создание приложений MFC

В этой статье описывается начальной предпринять для разработки приложения. Во-первых необходимо решить, какие приложения вы создаете. Некоторые из вариантов обсуждались в [приложения проектных](../mfc/application-design-choices.md). Будет ли приложение:

- Выполнение в Интернете или интрасети

- Выполнение на клиенте или на сервере

- Запуск в браузере или как изолированное приложение

- С помощью технологии COM "или" активный "

- Загрузка данных с помощью WinInet или асинхронных моникеров

Решения определения классов, которые подходят для вашего приложения. Ответы также помогают определить, выбранные при запуске мастера приложения, чтобы начать построение приложения.
