# <a name="application-framework"></a>Исполняющая среда
Основные библиотеки Microsoft Foundation Class (MFC) — это инкапсуляция большую часть API Windows в C++ форме. Библиотека классов представляют windows, диалоговые окна, контексты устройств GDI общие объекты, например кисти, перья, элементы управления и другие стандартные элементы Windows. Эти классы обеспечивают удобный интерфейс функции C++ члена для структуры в Windows, они инкапсулируют. Дополнительные сведения об использовании этих классов см. в разделе [разделы объекта окна](../mfc/window-objects.md).  
  
 Но библиотеки MFC также предоставляет слой функциональности дополнительные приложения, построенные на C++ инкапсулируют Windows API. Этот уровень находится на платформу рабочего приложения для Windows, которое предоставляет большую часть общего пользовательского интерфейса для Windows, включая панели инструментов, строки состояния, печать, предварительный просмотр, ожидалось программ базы данных поддержки и поддержки ActiveX. [Использование классов для написания приложений для Windows](../mfc/using-the-classes-to-write-applications-for-windows.md) framework подробно объясняется.
