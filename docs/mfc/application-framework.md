# <a name="application-framework"></a>Исполняющая среда

Основные библиотеки Microsoft Foundation Class (MFC) — это инкапсуляция большая часть API Windows в C++ форме. Библиотека классов представляют собой windows, диалоговые окна, контексты устройств, объекты распространенных GDI, такие как кисти, перья, элементы управления и другие стандартные элементы Windows. Эти классы предоставляют удобный интерфейс функция члена C++ к структурам в Windows, они инкапсулируют. Дополнительные сведения об использовании этих классов см. в разделе [разделы объекта окна](../mfc/window-objects.md).

Но библиотеки MFC также предоставляет уровень возможности дополнительные приложения, основанные на C++ инкапсулируют Windows API. Этот уровень — это платформа приложений работы для Windows, которая предоставляет большую часть общего интерфейса пользователя для Windows, включая панели инструментов, строки состояния, печать, предварительный просмотр, ожидалось программ базы данных поддержка и поддержка ActiveX. [Использование классов для записи приложений для Windows](../mfc/using-the-classes-to-write-applications-for-windows.md) этой статье описана структура подробно.
