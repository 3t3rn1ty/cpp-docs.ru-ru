# TN050. Общие диалоговые окна MFC/OLE (MFCUIx32)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Эта заметка исходного покрыла некоторые возможные проблемы и OLE общих диалоговых окон, предоставленное и использует MFC.  OLE стандартные диалоги теперь реализованы в виде компонента встроенной системы \(OLEDLG.DLL\) и полностью описаны в документации по продукту.
