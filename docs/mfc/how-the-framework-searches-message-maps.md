# Выполнение платформой поиска по схемам сообщений
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Платформа в таблице сопоставления сообщений для совпадений с входящими сообщениями.  После записи запись сопоставления сообщений для каждого сообщения требуется класс обработки и записи соответствующих обработчиков, платформа вызывает обработчики автоматически.  В следующих разделах описан поиск схемы сообщений:  
  
-   [, Где найти сопоставления сообщений](../mfc/where-to-find-message-maps.md)  
  
-   [Производные сопоставления сообщений](../mfc/derived-message-maps.md)  
  
-   [Диапазоны сопоставления сообщений, идентификаторы команд, или идентификатор элемента управления в один обработчик](../mfc/handlers-for-message-map-ranges.md)
