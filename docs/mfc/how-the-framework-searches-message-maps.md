# <a name="how-the-framework-searches-message-maps"></a>Выполнение платформой поиска по схемам сообщений
Платформа в таблице схемы сообщений ищет совпадения с входящих сообщений. После записи схемы сообщений для каждого сообщения, требуется класс для обработки и создания соответствующих обработчиков, платформа автоматически вызывает обработчиков. В следующих разделах описаны в схеме сообщений поиска:  
  
-   [Расположение схем сообщений](../mfc/where-to-find-message-maps.md)  
  
-   [Производные схемы сообщений](../mfc/derived-message-maps.md)  
  
-   [Сопоставляется один обработчик диапазоны сообщения, идентификаторы команд и идентификаторы элементов управления](../mfc/handlers-for-message-map-ranges.md)
