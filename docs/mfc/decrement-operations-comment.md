# <a name="-operations-comment"></a>// Комментарий операций
`// Operations` Раздел объявления класса MFC содержит функции-члены могут быть вызваны для объекта, чтобы сделать его выполнять действия или выполнения действий (выполнения операций). Эти функции являются обычно отличных**const** так, как они обычно имеют побочные эффекты. Они могут быть виртуальным или невиртуальные в зависимости от потребностей класса. Как правило эти члены являются открытыми.  
  
 В образце вывод из класса `CStdioFile`в [пример комментариев](../mfc/an-example-of-the-comments.md), список включает две функции-члена в этот комментарий: `ReadString` и `WriteString`.  
  
 С помощью атрибутов, операций можно дополнительно разделить.
