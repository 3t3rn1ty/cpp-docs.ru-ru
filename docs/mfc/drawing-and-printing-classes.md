# Классы рисования и печати
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

В Windows, полностью графический вывод строится на виртуальной системной области рисования контекстом устройства \(DC\).  MFC предоставляет классы для инкапсуляции различные контексты устройства, так и для инкапсуляции инструментов для рисования Windows как растровые изображения, кисти палитры и перья.
