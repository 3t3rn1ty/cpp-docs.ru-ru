# TN036. Использование CFormView с AppWizard и ClassWizard
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Это описанное техническое примечание об изменении AppWizard созданный приложение так, чтобы он использовал `CFormView` вместо по умолчанию `CView` в качестве основной класс представления.  Это поддерживается непосредственно с данной версией Visual C\+\+.
