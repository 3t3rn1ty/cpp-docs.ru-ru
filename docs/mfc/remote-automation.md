# <a name="remote-automation"></a>Удаленная автоматизация
> [!NOTE]
>  Удаленная автоматизация — это технология устаревшим и не должны использоваться для новых приложений. Visual C++ не поддерживает Windows 95. Описываются случаи, в которых будет необходим для поддержки удаленной автоматизации в [где Does удаленной автоматизации помещаются в](where-does-remote-automation-fit-in-q.md).  
  
 Удаленная автоматизация — это тип [автоматизации](../mfc/automation.md) , позволяющий потребителя интерфейс для выполнения интерфейса поставщика, который находится на другом компьютере, например, в сети.  
  
 В этой статье объясняется, как создать объекты автоматизации, может быть вызван и выполнена удаленно и способ создания контроллеров автоматизации, которые могут использовать эти объекты удаленной автоматизации. Также проверяет параметры конфигурации и указывает, основные различия между удаленной автоматизации и DCOM (распределенные версия COM и OLE, позволяющий интерфейсы, отличные от, связанные с автоматизации вызывается и выполнять их удаленно).
