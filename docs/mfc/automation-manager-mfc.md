# <a name="automation-manager-mfc"></a>Диспетчер автоматизации (MFC)
AUTMGR32. EXE-файла должны копироваться в системном каталоге Windows каждого компьютера, планирует предоставляют объекты удаленной автоматизации. Обычно это C:\WINDOWS\SYSTEM32.  
  
 Если вы хотите включить обратные вызовы от сервера клиенту, этот исполняемый файл должен также копируются в системном каталоге каждого клиентского компьютера.  
  
 При запуске диспетчер автоматизации, на сервере, содержащий сведения о состоянии краткое отображается небольшое окно. Если вы хотите скрыть его, обратитесь к статье Q138067 базы знаний Майкрософт.
