# Использование редактора диалоговых окон для добавления элементов управления
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

При создании этот ресурс шаблона диалоговых окон с помощью [редактор диалоговых окон](../mfc/dialog-editor.md), при перетаскивании элементов управления в палитре элементов управления и удалять в диалоговое окно.  Это добавляет спецификации для данного типа элемента управления в ресурс шаблона диалоговых окон.  При создании объекта диалогового окна и вызовите функцию\-член **Создать** объекта или `DoModal` платформа создает элемент управления Windows и разместить его в окне диалогового окна на экране.  
  
 Вместо этого можно [создайте элементы управления вручную](../mfc/adding-controls-by-hand.md), если необходимо.  Это больше работы.
