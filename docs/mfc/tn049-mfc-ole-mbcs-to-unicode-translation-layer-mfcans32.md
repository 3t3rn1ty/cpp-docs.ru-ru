# TN049. Слой трансляции MFC/OLE MBCS в Юникод (MFCANS32)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Следует заметить, что эта изначально, описанная MFCANS32.DLL предоставляет интерфейсы ANSI в главном мире юникод 32 разрядного OLE.  Эта библиотека DLL MFC больше не используется.
