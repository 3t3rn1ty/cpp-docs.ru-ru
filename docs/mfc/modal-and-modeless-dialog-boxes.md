# Модальные и немодальные диалоговые окна
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Можно использовать класс [CDialog](../mfc/reference/cdialog-class.md) для управления 2 типа диалоговых окон:  
  
-   *Модальные диалоговые окна*, которые требуют, чтобы пользователь отвечает перед продолжением программы  
  
-   *Немодальные диалоговые окна*, которые остаются на экране и доступным для использования в любое время, но позволяют другие действия пользователя  
  
 Изменение и процедуры ресурса для создания шаблона диалоговых окон одинаковы для режимного и немодальных диалоговых окон.  
  
 Создание диалогового окна для программы необходимо выполнить следующие действия.  
  
1.  Используйте [редактор диалоговых окон](../mfc/dialog-editor.md) для разработки диалоговое окно и создать его ресурс шаблона диалоговых окон.  
  
2.  Создайте класс диалогового окна.  
  
3.  В [управления ресурсов диалоговых окон в обработчик сообщений](../mfc/adding-event-handlers-for-dialog-box-controls.md) в классе диалогового окна.  
  
4.  Добавьте элементы данных, связанные с элементами управления диалогового окна и указать [обмена данными с диалоговым окном](../mfc/dialog-data-exchange.md) и [проверки данных диалогового окна](../mfc/dialog-data-validation.md) для элементов управления.
