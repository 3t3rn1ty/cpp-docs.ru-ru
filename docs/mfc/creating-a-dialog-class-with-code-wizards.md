# Создание класса диалогового окна с помощью мастеров кода
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Следующие задачи таблицы диалоговых окон, связанных списков, что мастера кода помогают управлять.  
  
### Связанные задачи диалоговых окон  
  
|Задача|Применить к. . .|  
|------------|----------------------|  
|[Создайте новый CDialog\-производный класс](../Topic/Creating%20Your%20Dialog%20Class.md) управление диалоговом окне.|Каждое диалоговое окно.|  
|[Сообщения Windows сопоставления](../mfc/handling-windows-messages-in-your-dialog-box.md) к классу диалогового окна.|Каждое сообщение необходимо как обработанные.|  
|Объявите переменные членов класса для представления элементов управления в диалоговом окне.|Каждый элемент управления, который генерирует текст или числовое значение следует получить из программы.|  
|Определите [хранения данных, передаваемых](../mfc/dialog-data-exchange-and-validation.md) между элементами управления клиентской стороны и переменными\-членами.|Каждый элемент управления, который необходимо получить из программы.|  
|Определите [правила проверки](../mfc/dialog-data-exchange-and-validation.md) для переменных\-членов.|Каждый элемент управления, который генерирует текст или числовое значение, при необходимости.|
