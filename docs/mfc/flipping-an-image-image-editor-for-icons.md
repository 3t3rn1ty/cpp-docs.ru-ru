# Flipping an Image (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Функции поворота и отражения изображения позволяют выполнять следующие задачи:  
  
-   Создать зеркальное отражение исходного рисунка.  
  
-   Перевернуть изображение вверх ногами.  
  
-   Поворачивать изображение с шагом ровно 90 градусов.  
  
### Зеркальное отражение по горизонтали  
  
1.  В меню **Изображение** выберите команду **Отразить слева направо**.  
  
### Зеркальное отражение по вертикали  
  
1.  В меню **Изображение** выберите команду **Отразить сверху вниз**.  
  
### Поворот изображения на 90 градусов  
  
1.  В меню **Изображение** выберите команду **Повернуть на 90 градусов**.  
  
    > [!NOTE]
    >  Также можно вызвать эти команды с помощью [сочетаний клавиш](../mfc/accelerator-keys-image-editor-for-icons.md) или из контекстного меню \(для этого нужно щелкнуть за пределами изображения в области редактора изображений\).
