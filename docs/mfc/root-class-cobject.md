# Корневой класс: CObject
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Большинство классов в библиотеке Microsoft Foundation Class \(MFC\) является производным от одного базового класса в корне иерархии классов.  `CObject` предоставляет несколько полезных возможностей все классы, производные от него с очень низкой нагрузкой.  Дополнительные сведения о `CObject` и его возможности см. в разделе [Использование CObject](../mfc/using-cobject.md).  
  
 [CObject](../Topic/CObject%20Class.md)  
 Обычно базовый класс большинство классов MFC.  Поддерживает сериализацию данных и получение сведений о среде выполнения о классе.  
  
 [CRuntimeClass](../Topic/CRuntimeClass%20Structure.md)  
 Структура, используемая для определения точный класс объекта во время выполнения.
