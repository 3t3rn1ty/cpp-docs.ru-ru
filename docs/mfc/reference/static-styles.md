# <a name="static-styles"></a>Статические стили
-   **SS_BITMAP** указывает растровое изображение для отображения в статический элемент управления. Заданный текст является имя растрового изображения (не имя файла), определенный в файле ресурсов. Стиль игнорирует параметры nWidth и nHeight; элемент управления автоматически изменяет свой размер в соответствии с битовой карты.  
  
-   **SS_BLACKFRAME** указывает окно с фреймом, рисуемых при помощи цвет рамки окна. Значением по умолчанию является Black.  
  
-   **SS_BLACKRECT** задает прямоугольник, закрашенный цвет, используемый для рисования рамки окон. Значением по умолчанию является Black.  
  
-   **SS_CENTER** обозначает простой прямоугольник и отображает заданный текст по центру прямоугольника. Перед отображением форматирования текста. Слова, которые бы расширить после конца строки, автоматически переносятся в начало следующей строки по центру.  
  
-   **SS_CENTERIMAGE** указывает, что в случае, если точечный рисунок или значок меньше клиентской области статического элемента управления, остальная часть клиентской области заполняется цвет пикселя в левом верхнем углу точечного рисунка или значок. Если статический элемент управления содержит одну строку текста, текст выравнивается по центру по вертикали в клиентской области элемента управления.  
  
-   **SS_ENDELLIPSIS** или **SS_PATHELLIPSIS** заменяет часть заданной строки эллипсы, при необходимости, чтобы результат соответствовал в заданном прямоугольнике.  
  
     Можно указать **SS_END_ELLIPSIS** для замены символов в конце строки или **SS_PATHELLIPSIS** для замены символов в середине строки. Если строка содержит обратную косую черту (\\) символов, **SS_PATHELLIPSIS** сохраняет максимально возможную часть текста после последней обратной косой черты можно.  
  
-   **SS_ENHMETAFILE** указывает расширенный метафайл, отображаемый в элементе управления статический. Заданный текст называется метафайл. Расширенный метафайл статический элемент управления имеет фиксированный размер; метафайл масштабируется по размерам клиентской области статического элемента управления.  
  
-   **SS_ETCHEDFRAME** рисует рамки статический элемент управления с помощью **EDGE_ETCHED** edge стиля.  
  
-   **SS_ETCHEDHORZ** верхней и нижней границ, используя статический элемент управления рисует **EDGE_ETCHED** edge стиля.  
  
-   **SS_ETCHEDVERT** рисует левой и правой границ статического элемента управления с использованием стиля границы EDGE_ETCHED.  
  
-   **SS_GRAYFRAME** указывает окно с фреймом, рисуемых при помощи цвет фона экрана (рабочий стол). По умолчанию задано значение Gray.  
  
-   **SS_GRAYRECT** задает прямоугольник, закрашенный цвет, используемый для заливки фона экрана. По умолчанию задано значение Gray.  
  
-   **SS_ICON** определяет значок, отображающийся в диалоговом окне. Заданный текст является имя значка (не имя файла), определенный в файле ресурсов. `nWidth` И `nHeight` игнорируются; значок автоматически изменяет свой размер.  
  
-   **SS_LEFT** обозначает простой прямоугольник и отображает заданный текст flush левого в прямоугольнике. Перед отображением форматирования текста. Слова, которые бы расширить после конца строки, автоматически переносятся в начало следующей строки flush слева.  
  
-   **SS_LEFTNOWORDWRAP** обозначает простой прямоугольник и отображает заданный текст flush левого в прямоугольнике. Вкладки расширяются, но слова не заключаются в оболочку. Текст, который выходит за пределы строки обрезается.  
  
-   **SS_NOPREFIX** Если задан этот стиль, Windows будет интерпретировать знаки амперсанда (&) в тексте элемента управления символами префикса сочетаний клавиш. В этом случае амперсанд удаляется и следующий символ в строке подчеркивается. Если статический элемент управления должен содержать текст где эта возможность не требуется, **SS_NOPREFIX** могут быть добавлены. Этот стиль элемента управления static могут быть включены в любой из определенных статических элементов управления. Можно объединить **SS_NOPREFIX** с другим стилем с помощью побитового оператора OR. Это наиболее часто используется при имена файлов или других строк, которые могут содержать знак амперсанда необходимо отображать в статический элемент управления в диалоговом окне.  
  
-   **SS_NOTIFY** отправляет родительского окна **STN_CLICKED**, **STN_DBLCLK**, **STN_DISABLE**, и **STN_ENABLE** сообщений уведомлений, когда пользователь нажимает кнопку или дважды щелкает элемент управления.  
  
-   **SS_OWNERDRAW** указывает, что владелец статический элемент управления отвечает за рисования элемента управления. Получает окна-владельца `WM_DRAWITEM` сообщение каждый раз, когда элемент управления должен быть нарисован.  
  
-   **SS_REALSIZEIMAGE** предотвращает статический элемент управления значок или растровое изображение (то есть, статические элементы управления, имеющие **SS_ICON** или **SS_BITMAP** стиль) изменения размеров при загрузке или рисования. Если значок или растровое изображение больше, чем место назначения, изображение обрезается.  
  
-   **SS_RIGHT** обозначает простой прямоугольник и отображает заданный текст flush справа в прямоугольнике. Перед отображением форматирования текста. Слова, которые бы расширить после конца строки, автоматически переносятся в начало следующей строки flush справа.  
  
-   **SS_RIGHTJUST** указывает, что нижний правый угол статический элемент управления с **SS_BITMAP** или **SS_ICON** стиль фиксируется оставаться при изменении размера элемента управления. Только верхней и левой сторон подгоняются в соответствии нового растрового изображения или значка.  
  
-   **SS_SIMPLE** обозначает простого прямоугольника и отображается одна строка текста flush левого в прямоугольнике. Строка текста не может укорачивать или изменен каким-либо образом. (Поле окне или диалоговом окне родительский элемент управления не должен обрабатывать `WM_CTLCOLOR` сообщения.)  
  
-   **SS_SUNKEN** рисует утопленная граница вокруг статического элемента управления.  
  
-   **SS_USERITEM** указывает элемент, определяемых пользователем.  
  
-   **SS_WHITEFRAME** указывает окно с фреймом, рисуемых при помощи совпадает с цветом фона окна. По умолчанию цвет белый.  
  
-   **SS_WHITERECT** задает прямоугольник, закрашенный цвет, используемый для заполнения фона окна. По умолчанию цвет белый.  
  
-   **SS_WORDELLIPSIS** усекает текст, который не соответствует и добавляет эллипсов.
