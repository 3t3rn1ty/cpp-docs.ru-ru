# <a name="edit-styles"></a>Изменение стилей
-   **ES_AUTOHSCROLL** автоматически прокручивает текст справа на 10 символов при вводе символа в конце строки. Когда пользователь нажимает клавишу ВВОД, элемент управления прокручивается весь текст в позиции 0.  
  
-   **ES_AUTOVSCROLL** автоматическая прокрутка текста вверх на одну страницу при нажатии клавиши ВВОД в последней строке.  
  
-   **ES_CENTER** центров текст в однострочный и многострочный элемент управления.  
  
-   **ES_LEFT** выравниваются по левому краю текст в однострочный и многострочный элемент управления.  
  
-   **ES_LOWERCASE** преобразует все символы в нижний регистр при вводе в элемент управления.  
  
-   **ES_MULTILINE** обозначает элемент управления редактирование несколько строк. (Значение по умолчанию представляет одну строку). Если **ES_AUTOVSCROLL** указанный стиль, элемент управления для редактирования показано, как можно столько строк и вертикальной прокрутке, когда пользователь нажимает клавишу ВВОД. Если **ES_AUTOVSCROLL** — не указан, элемент управления для редактирования показывает столько строк как возможные и звуковых сигналов Если нажатия клавиши ВВОД, если строки не могут быть отображены. Если **ES_AUTOHSCROLL** указанный стиль, элемент управления редактирование многострочных автоматически прокручивается горизонтально когда курсор выходит за правый край элемента управления. Чтобы начать новую строку, пользователь должен нажмите клавишу ВВОД. Если **ES_AUTOHSCROLL** — не указан, элемент управления автоматический перенос по словам в начало следующей строки, при необходимости; Если нажатия клавиши ВВОД новая строка также запущена. Положение wordwrap определяется размер окна. При изменении размера окна отобразится изменения положения переноса слов и текста. Элементы управления редактирования несколько строк можно использовать полосы прокрутки. Элемент управления редактированием с полосами прокрутки обрабатывает сообщения полосы прокрутки. Редактировать элементы управления без прокрутки полосы прокрутки, как описано выше и обрабатывать любые прокрутки сообщения, отправленные родительского окна.  
  
-   **ES_NOHIDESEL** обычно элемент управления редактированием скрывает выбор, когда элемент управления теряет фокус ввода и Инвертирует выбор, когда элемент управления получает фокус ввода. Указание **ES_NOHIDESEL** удаляет это действие по умолчанию.  
  
-   **ES_NUMBER** позволяет только цифры для ввода в элемент управления.  
  
-   **ES_OEMCONVERT** текста, введенного в элемент управления поля ввода преобразуется из набора знаков ANSI в набор символов OEM и обратно в формат ANSI. Это гарантирует правильности преобразования текста, когда приложение вызывает `AnsiToOem` функции Windows для преобразования строки ANSI в элементе управления редактированием в символы OEM. Этот стиль полезен для изменения настроек содержащих имена файлов.  
  
-   **ES_PASSWORD** отображает все символы в виде звездочки (**\***) при вводе в элемент управления. Приложение может использовать `SetPasswordChar` измените символ, который отображается функция-член.  
  
-   **ES_READONLY** запрещает пользователю ввод и изменение текста в элементе управления.  
  
-   **ES_RIGHT** по правому краю текст в однострочный и многострочный элемент управления.  
  
-   **ES_UPPERCASE** преобразует все символы в верхний регистр при вводе в элемент управления.  
  
-   **ES_WANTRETURN** указывает вставить символ возврата каретки, когда пользователь нажимает клавишу ВВОД при вводе текста в элемент управления редактирование многострочных в диалоговом окне. Без этого стиля нажатия клавиши ВВОД действует так же, как нажатие кнопки по умолчанию поля диалогового окна. Этот стиль не оказывает влияния на однострочный элемент управления.
