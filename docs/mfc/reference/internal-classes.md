# Внутренние классы
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Следующие классы используются внутри в MFC.  В данном разделе эти внутренние классы описываются с целью предоставления полной информации, но они не предназначены для непосредственного использования в коде.  
  
## В этом подразделе  
  
|Класс|Описание|
