# Инициализация OLE
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Прежде, чем приложение может использовать OLE системные службы, он должен инициализировать OLE системные библиотеки DLL и проверить, что библиотека DLL нужная версия.  Функция **AfxOleInit** инициализирует OLE системные библиотеки DLL.
