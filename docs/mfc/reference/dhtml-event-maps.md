# Схемы событий DHTML
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Следующие макросы можно использовать для обработки событий DHTML.  
  
## Макросы сопоставления событий DHTML  
 Следующие макросы можно использовать для обработки событий в [CDHtmlDialog](../Topic/CDHtmlDialog%20Class.md)\- производные классы DHTML.  
  
|||  
|-|-|  
|[BEGIN\_DHTML\_EVENT\_MAP](../Topic/BEGIN_DHTML_EVENT_MAP.md)|Отмечает начало сопоставления событий DHTML.|  
|[BEGIN\_DHTML\_EVENT\_MAP\_INLINE](../Topic/BEGIN_DHTML_EVENT_MAP_INLINE.md)|Отмечает начало сопоставления событий DHTML.|  
|[DECLARE\_DHTML\_EVENT\_MAP](../Topic/DECLARE_DHTML_EVENT_MAP.md)|Объявляет сопоставление событий DHTML.|  
|[DHTML\_EVENT](../Topic/DHTML_EVENT.md)|Используется для обработки события на уровне документа для отдельного элемента HTML.|  
|[DHTML\_EVENT\_AXCONTROL](../Topic/DHTML_EVENT_AXCONTROL.md)|Используется для обработки события увольнянное элемента управления ActiveX.|  
|[DHTML\_EVENT\_CLASS](../Topic/DHTML_EVENT_CLASS.md)|Используется для обработки события на уровне документа для всех элементов HTML с указанным классом CSS.|  
|[DHTML\_EVENT\_ELEMENT](../Topic/DHTML_EVENT_ELEMENT.md)|Используется для обработки события на уровне элемента.|  
|[DHTML\_EVENT\_ONAFTERUPDATE](../Topic/DHTML_EVENT_ONAFTERUPDATE.md)|Используется для обработки события **onafterupdate** из элемента HTML.|  
|[DHTML\_EVENT\_ONBEFOREUPDATE](../Topic/DHTML_EVENT_ONBEFOREUPDATE.md)|Используется для обработки события **onbeforeupdate** из элемента HTML.|  
|[DHTML\_EVENT\_ONBLUR](../Topic/DHTML_EVENT_ONBLUR.md)|Используется для обработки события **onblur** из элемента HTML.|  
|[DHTML\_EVENT\_ONCHANGE](../Topic/DHTML_EVENT_ONCHANGE.md)|Используется для обработки события `onchange` из элемента HTML.|

## Макросы сопоставления событий URL\-адреса

|||  
|-|-|  
|[BEGIN\_DHTML\_URL\_EVENT\_MAP](../Topic/BEGIN_DHTML_URL_EVENT_MAP.md)|Отмечает начало multipage сопоставления событий DHTML и URL\-адресов.|

|[END\_URL\_ENTRIES](../Topic/END_URL_ENTRIES.md)|Отмечает конец сопоставления записи события URL\-адреса.|
