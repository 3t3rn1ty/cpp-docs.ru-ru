# Редактирование обработчика сообщений
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

После определения обработчика сообщений можно перейти в определение функции\-члена, чтобы добавить или изменить код.  
  
 Чтобы перейти в определение функции\-члена из [редактора диалоговых окон](../../mfc/dialog-editor.md), дважды щелкните элемент управления, для которого обработчик уже определен.  Откроется файл, в котором определен обработчик сообщений выбранного элемента управления.  
  
 Чтобы перейти в определение функции\-члена из [представления классов](http://msdn.microsoft.com/ru-ru/8d7430a9-3e33-454c-a9e1-a85e3d2db925), дважды щелкните имя функции в представлении классов.
