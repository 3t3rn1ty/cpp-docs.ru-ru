# <a name="dialog-data-exchange-functions-for-ole-controls"></a>Функции обмена данными диалоговых окон для элементов управления OLE
В этом разделе перечислены функции DDX_OC, используемые для обмена данными между свойством элемента управления OLE в диалоговое окно, представление формы или объекте представления элемента управления и элемент данных диалоговое окно, представление формы или объекте представления элемента управления.  
  
### <a name="ddxoc-functions"></a>Функции DDX_OC  
  
|||  
|-|-|  
|[DDX_OCBool](#ddx_ocbool)|Управляет передачей **BOOL** данные между свойством элемента управления OLE и **BOOL** члена данных.|  
|[DDX_OCBoolRO](#ddx_ocboolro)|Управляет передачей **BOOL** данных между только для чтения свойство элемента управления OLE и **BOOL** члена данных.|  
|[DDX_OCColor](#ddx_occolor)|Управляет передачей **OLE_COLOR** данные между свойством элемента управления OLE и **OLE_COLOR** члена данных.|  
|[DDX_OCColorRO](#ddx_occolorro)|Управляет передачей **OLE_COLOR** данных между только для чтения свойство элемента управления OLE и **OLE_COLOR** члена данных.|  
|[DDX_OCFloat](#ddx_ocfloat)|Управляет передачей **float** (или **двойные**) данные между свойством элемента управления OLE и **float** (или **двойные**) члена данных.|  
|[DDX_OCFloatRO](#ddx_ocfloatro)|Управляет передачей **float** (или **двойные**) данные между только для чтения свойство элемента управления OLE и **float** (или **двойные**) данных член.|  
|[DDX_OCInt](#ddx_ocint)|Управляет передачей **int** (или **длинные**) данные между свойством элемента управления OLE и **int** (или **длинные**) члена данных.|  
|[DDX_OCIntRO](#ddx_ocintro)|Управляет передачей **int** (или **длинные**) данные между только для чтения свойство элемента управления OLE и **int** (или **длинные**) члена данных.|  
|[DDX_OCShort](#ddx_ocshort)|Управляет передачей **короткие** данных между свойством элемента управления OLE и **короткие** члена данных.|  
|[DDX_OCShortRO](#ddx_ocshortro)|Управляет передачей **короткие** данных между только для чтения свойство элемента управления OLE и **короткие** члена данных.|  
|[DDX_OCText](#ddx_octext)|Управляет передачей **CString** данные между свойством элемента управления OLE и **CString** члена данных.|  
|[DDX_OCTextRO](#ddx_octextro)|Управляет передачей **CString** данных между только для чтения свойство элемента управления OLE и **CString** члена данных.|  
  
##  <a name="ddx_ocbool"></a>  DDX_OCBool  
 `DDX_OCBool` Функция управляет передачей **BOOL** данные между свойством элемента управления OLE в диалоговом окне, представлении формы или объекте представления элемента управления и **BOOL** членом данных в диалоговом окне, представлении формы или Объект представления элемента управления.  
  
```   
void AFXAPI DDX_OCBool(
    CDataExchange* pDX,  
    int nIDC,  
    DISPID dispid,  
    BOOL& value);
```  
  
### <a name="parameters"></a>Параметры  
 *pDX*  
 Указатель на объект `CDataExchange` . Структура предоставляет этот объект для формирования контекста обмена данными, включая его направление.  
  
 *nIDC*  
 Идентификатор элемента управления OLE в диалоговом окне, представлении формы или объекте представления элемента управления.  
  
 *DISPID*  
 Идентификатор диспетчера свойства элемента управления.  
  
 *значение*  
 Ссылка на переменную-член диалогового окна, представления формы или объекта представления элемента управления, с которой осуществляется обмен данными.  
  
### <a name="remarks"></a>Примечания  
 Дополнительные сведения об DDX см. в разделе [обмен данными диалоговых окон и проверка](../../mfc/dialog-data-exchange-and-validation.md).  
  
### <a name="requirements"></a>Требования  
  **Заголовок:** afxdisp.h

 Указатель на **CDataExchange** объекта. Структура предоставляет этот объект для формирования контекста обмена данными, включая его направление.  
  
 *nIDC*  
 Идентификатор элемента управления OLE в диалоговом окне, представлении формы или объекте представления элемента управления.  
  
 *DISPID*  
 Идентификатор диспетчера свойства элемента управления.  
  
 *значение*  
 Ссылка на переменную-член диалогового окна, представления формы или объекта представления элемента управления, с которой осуществляется обмен данными.
