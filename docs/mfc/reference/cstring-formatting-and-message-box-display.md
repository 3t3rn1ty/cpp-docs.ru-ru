# Форматирование CString и отображение окна сообщения
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Предоставляются для формате и анализируют несколько функций объекты `CString`.  Можно использовать эти функции при необходимости управления объектами `CString`, но они особенно полезны для форматирования строк, которые отображаются в текст сообщения окна.  
  
 Эта команда в составе функции также включает глобальную процедуру для отображения окна сообщения.  
  
### Функции CString  
  
|||  
|-|-|  
|[AfxExtractSubString](../Topic/AfxExtractSubString.md)|Извлекает подстроки, разделенные одным символом из заданной строки источника.|  
|[AfxFormatString1](../Topic/AfxFormatString1.md)|Замените заданную строку символов для формата «%1 " в строке, содержащиеся в таблице строк.|  
|[AfxFormatString2](../Topic/AfxFormatString2.md)|Замещает 2 символов для строки формата «%1 " и «%2 " в строке, содержащиеся в таблице строк.|  
|[AfxMessageBox](../Topic/AfxMessageBox.md)|Отображает окно сообщения.|
