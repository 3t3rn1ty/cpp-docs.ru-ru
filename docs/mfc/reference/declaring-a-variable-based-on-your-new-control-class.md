# Объявление переменной на основании нового класса элемента управления
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

После создания класса элемента управления MFC, на его основании можно определить переменную.  Чтобы обеспечить контекст для создаваемой переменной, необходимо открыть редактор диалоговых окон и отредактировать диалоговое окно, в котором будет использоваться элемент управления.  Кроме того, с диалоговым окном предварительно должен быть связан класс.  Сведения об использовании редактора диалоговых окон см. в разделе [Редактор диалоговых окон](../../mfc/dialog-editor.md).  
  
### Объявление переменной на основании повторно используемого класса  
  
1.  В процессе редактирования диалогового окна перетащите элемент управления того же типа, что и базовый класс нового элемента управления из панели элементов в диалоговое окно.  
  
2.  Наведите указатель мыши на перетащенный элемент управления.  
  
3.  Дважды щелкните элемент управления, удерживая нажатой клавишу CTRL.  
  
     Откроется диалоговое окно [Добавление переменной\-члена](../../ide/add-member-variable-wizard.md).  
  
4.  В поле **Доступ** выберите для элемента управления соответствующий тип доступа.  
  
5.  Щелкните флажок **Переменная элемента управления**.  
  
6.  В поле **Имя переменной** введите имя.  
  
7.  В разделе **Категория** выберите **Элемент управления**.  
  
8.  В списке **Идентификатор элемента управления** выберите добавленный элемент управления.  В списке **Тип переменной** должен отобразиться соответствующий тип переменной, а в поле **Тип элемента управления** — соответствующий тип элемента управления.  
  
9. В поле **Комментарий** можно добавить необходимый комментарий, который будет отображаться в коде.  
  
10. Нажмите кнопку **ОК**.
