# Инициализация и прекращение работы ядра базы данных DAO
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

При использовании объектов MFC DAO, ядро СУБД DAO необходимо инициализировать перед использованием и затем выполнена до приложения или DLL прекращают.  2 Функции, `AfxDaoInit` и `AfxDaoTerm`, выполнять эти задачи.  
  
### Инициализации и завершения ядра СУБД DAO  
  
|||  
|-|-|  
|[AfxDaoInit](../Topic/AfxDaoInit.md)|Инициализирует ядра СУБД DAO.|  
|[AfxDaoTerm](../Topic/AfxDaoTerm.md)|Завершает ядра СУБД DAO.|
