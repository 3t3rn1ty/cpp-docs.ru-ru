# Функции серого цвета и сглаживания точечного рисунка
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**Функции серого растрового изображения**  
  
 MFC предоставляет две функции для придания растровому изображению вида отключенного элемента управления.  
  
 ![Сравнение серых и оригинальных версий значков](../../mfc/reference/media/vcgraybitmap.png "vcGrayBitmap")  
  
|||  
|-|-|  
|[AfxDrawGrayBitmap](../Topic/AfxDrawGrayBitmap.md)|Рисует серую версию растрового изображения.|  
|[AfxGetGrayBitmap](../Topic/AfxGetGrayBitmap.md)|Копирует серую версию растрового изображения.|  
  
 **Функции сглаживания растрового изображения**  
  
 MFC также предоставляет две функции для замены фона растрового изображения сглаженным шаблоном.  
  
 ![Сравнение сглаженных и оригинальных версий значков](../../mfc/reference/media/vcditheredbitmap.png "vcDitheredBitmap")  
  
|||  
|-|-|  
|[AfxDrawDitheredBitmap](../Topic/AfxDrawDitheredBitmap.md)|Рисует растровое изображение со сглаженным фоном.|  
|[AfxGetDitheredBitmap](../Topic/AfxGetDitheredBitmap.md)|Копирует растровое изображение со сглаженным фоном.|
