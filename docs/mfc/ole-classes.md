# <a name="ole-classes"></a>Классы OLE
Классы OLE работать с другими классами framework приложения обеспечивают легкий доступ к API ActiveX, предоставляя простой способ предоставления пользователям возможности ActiveX программах. ActiveX можно:  
  
-   Создание составных документов, которые позволяют пользователям создавать и редактировать документы, содержащие данные, созданные несколько приложений, включая текст, графики, электронные таблицы, звук, или другие типы данных.  
  
-   Создайте OLE-объекты, которые могут быть внедрены в составных документов.  
  
-   Используйте перетаскивание OLE для копирования данных между приложениями.  
  
-   Использование автоматизации для управления одной программы с другим.  
  
-   Создайте элементы управления ActiveX и контейнеры элементов управления ActiveX (ранее — элементы управления OLE и контейнеры элементов управления OLE, соответственно).
