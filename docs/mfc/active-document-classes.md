# <a name="active-document-classes"></a>Классы активных документов
Активные документы могут отображаться в окне всю клиентскую веб-браузера, например Internet Explorer 5.5 или в активных контейнеров, таких как Microsoft Office Binder, которая поддерживает активные документы.  
  
 [CDocObjectServer](../mfc/reference/cdocobjectserver-class.md)  
 Сопоставляет интерфейсы активного документа и инициализирует и активирует объект активного документа.  
  
 [CDocObjectServerItem](../mfc/reference/cdocobjectserveritem-class.md)  
 Реализует команды OLE-сервера специально для серверов в активный документ.  
  
 [COleDocObjectItem](../mfc/reference/coledocobjectitem-class.md)  
 Реализует хранение активных документов.
