# <a name="menus-mfc"></a>Меню (MFC)
MFC предоставляет два элемента, которые помогают работать с меню:  
  
-   Класс [CMenu](../mfc/reference/cmenu-class.md) для работы программы меню во время выполнения. См. в документации для `CMenu` и образец, чтобы узнать, как использовать `CMenu` эффективно.  
  
-   Механизм для обновления меню и кнопки панели инструментов: Включение или отключение их в режиме реального времени в соответствии с текущего состояния программы.  
  
 Visual C++ также предоставляет [редактор меню](../windows/menu-editor.md) для создания и редактирования ресурсов меню программы.
