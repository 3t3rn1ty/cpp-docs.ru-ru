# Оркестрация других действий окна
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Фреймовое окно оркеструет semimodal состояния, такие как контекстная справка и предварительный просмотр.  Описание роли фреймового окна в режиме предварительного просмотра см. в разделе [Печать и предварительный просмотр](../mfc/printing-and-print-preview.md).
