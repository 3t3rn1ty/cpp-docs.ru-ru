# Классы поддержки отладки
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

MFC предоставляет следующие классы для облегчения отладки динамические проблемы выделения памяти.  
  
 [CDumpContext](../mfc/reference/cdumpcontext-class.md)  
 Предоставляет назначение для диагностических дамп.  
  
 [CMemoryState](../mfc/reference/cmemorystate-structure.md)  
 Структура, которая предоставляет моментальные снимки использования памяти.  Также используется для сравнения двух и более ранние версии более поздние снимки памяти.
