# <a name="debugging-support-classes"></a>Классы поддержки отладки
MFC предоставляет следующие классы для отладки проблем с динамической памятью.  
  
 [CDumpContext](../mfc/reference/cdumpcontext-class.md)  
 Предоставляет место назначения для диагностики дампов.  
  
 [CMemoryState](../mfc/reference/cmemorystate-structure.md)  
 Структура, предоставляющей показатели использования памяти. Также используется для сравнения снимки памяти более ранних и более поздних версий.
