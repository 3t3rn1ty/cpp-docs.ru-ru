# <a name="using-csliderctrl"></a>Использование CSliderCtrl
[CSliderCtrl](../mfc/reference/csliderctrl-class.md) класс представляет элемент управления "ползунок", который также называется trackbar. «Ползунок» является окно, содержащее ползунка и дополнительну метки. При перемещении ползунка, с помощью мыши или клавиши со стрелками, элемент управления "ползунок" отправляет сообщения уведомления для указания изменения.  
  
 Элементы управления "ползунок" полезны в тех случаях, когда пользователю необходимо выбрать дискретное значение или набор последовательных значений в диапазоне. Например может использовать элемент управления "ползунок" позволяет пользователю устанавливать частота возврата клавиатуры с помощью ползунка метки данной шкалы.  
  
 Ползунок в элементе управления "ползунок" Перемещение на, указываемые при ее создании. Например, если указать, что элемент управления "ползунок" должен иметь диапазон 5, ползунок можно только занимают шесть позиции: позиции с левой стороны элемента управления "ползунок" и на одну позицию для каждого фрагмента в диапазоне. Как правило каждый из этих позиций идентифицируется деления.
