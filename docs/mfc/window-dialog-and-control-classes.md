# <a name="window-dialog-and-control-classes"></a>Классы окон, диалоговых окон и элементов управления

Класс `CWnd` и его производные классы инкапсулируют `HWND`, дескриптор окна Windows. `CWnd` можно использовать самостоятельно или в качестве основы для создания новых классов. Производные классы, предоставляемые библиотекой классов, представляющих различные виды окон.

[CWnd](../mfc/reference/cwnd-class.md)<br/>
Базовый класс для всех окон. Можно использовать один из классов, производных от `CWnd` или создавать собственные производные классы напрямую из него.
