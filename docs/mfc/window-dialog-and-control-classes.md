# Классы окон, диалоговых окон и элементов управления
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Класс `CWnd` и производные классы инкапсулируют `HWND`, дескриптором окна Windows.  `CWnd` может использоваться сама по себе или в качестве основы для создания производного новые классы.  Производные классы, предоставляемые библиотекой классов представляют различные типы окон.  
  
 [CWnd](../Topic/CWnd%20Class.md)  
 Базовый класс для всех окон.  Можно использовать один из классов, унаследованных от `CWnd` или создать собственные классы непосредственно из него.
