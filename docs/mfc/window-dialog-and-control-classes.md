# <a name="window-dialog-and-control-classes"></a>Классы окон, диалоговых окон и элементов управления
Класс `CWnd` и его производные классы инкапсулируют `HWND`, дескриптор окна Windows. `CWnd` можно использовать отдельно или как основа для создания новых классов. Производные классы, предоставляемые библиотека классов представляют различные виды окон.  
  
 [CWnd](../mfc/reference/cwnd-class.md)  
 Базовый класс для всех окон. Можно использовать один из классов, производных от `CWnd` или создавать собственные производные классы непосредственно из него.
