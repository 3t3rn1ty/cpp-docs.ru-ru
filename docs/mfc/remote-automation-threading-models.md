# <a name="remote-automation-threading-models"></a>Модели потоков удаленной автоматизации
Удаленная автоматизация поддерживает одним потоком и COM объектов модели COM. На 32-разрядных и 64-разрядной платформы, которые являются все, что Visual C++ поддерживает модель потоков должно быть присвоено «подразделение». Дополнительные сведения о том, как и где для этого см. в разделе [создание программы, использование удаленной автоматизации](../mfc/creating-programs-that-use-remote-automation.md).
