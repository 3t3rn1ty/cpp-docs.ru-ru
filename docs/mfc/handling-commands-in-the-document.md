# Обработка команд в документе
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Класс документа может также обрабатывать некоторые команды, создаваемые пунктами меню, кнопки панели инструментов или сочетаниями клавиш.  По умолчанию **CDocument** обрабатывает и команды "Сохранить как" в меню " Файл, используя сериализацию.  Другие команды, влияющие на данные также могут обрабатываться функции\-членами создаваемого документа.  Например, в программе Scribble, класс `CScribDoc` предоставляет обработчик для ясности правки всю команду, которая удаляет все данные, в данный момент хранящиеся в документе.  Документы могут иметь сопоставления сообщений, но в отличие от представления, документы не могут обрабатывать стандартные сообщения Windows — только сообщения **WM\_COMMAND** или «команды».
