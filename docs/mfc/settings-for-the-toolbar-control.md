# Параметры панели инструментов
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Кнопки на панели инструментов могут отображать растровое изображение, строки или оба.  По умолчанию размер образа пикселями измерения присвоено значение 16 15.  Все кнопки ту же ширину, по умолчанию 24 22 пикселями.  Определяется высотой кнопок панели инструментов высота и ширина инструмента совпадает с ширина клиентской области родительского окна, по умолчанию.  
  
 Панель инструментов может иметь встроенные функции настройки, включая системой конкретного диалогового окна настройки, которые позволяют пользователю для вставки, удаления или переставляют кнопки панели инструментов.  Приложение определяет, доступны ли функция настройки пользователя и элементы управления, в которой пользователь сможет настраивать панель инструментов.  Дополнительные сведения о панели инструментов в ходе класс см. в разделе [CToolBarCtrl](../mfc/reference/ctoolbarctrl-class.md) в *справочнике по MFC*.
