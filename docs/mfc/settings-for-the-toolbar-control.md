# <a name="settings-for-the-toolbar-control"></a>Параметры панели инструментов

Кнопки на панели инструментов можно отобразить точечный рисунок и строка. По умолчанию размер изображения будет присвоено размером 16, 15 пикселей. Все кнопки имеют одинаковую ширину, на 24 x 22 пикселя по умолчанию. Высота элемента управления панель инструментов определяется высота кнопки и ширины элемента управления панель инструментов является таким же, как Ширина клиентской области родительского окна, также по умолчанию.

Панель инструментов может иметь встроенную функциональность настройки, включая диалогового окна системные настройки, которые позволяют пользователю вставлять, удалять и переупорядочивать кнопки панели инструментов. Приложение определяет возможности настройки, доступны ли для пользователя и контролирует степень, в который пользователь может настраивать панели инструментов. Дополнительные сведения о настройке панели инструментов см. в разделе класса [CToolBarCtrl](../mfc/reference/ctoolbarctrl-class.md) в *Справочник по библиотеке MFC*.
