# <a name="customization-for-mfc"></a>Настройка для MFC
В этом разделе содержатся советы по настройке приложения MFC.  
  
## <a name="general-customizations"></a>Общие настройки  
 Можно сохранить и загрузить состояние приложения в реестр. При включении этого параметра приложение будет загружать исходное состояние из реестра. При изменении исходного макета закрепления для вашего приложения, необходимо очистить данные реестра для приложения. В противном случае данные в реестре переопределяет любые изменения, внесенные в исходный макет.
