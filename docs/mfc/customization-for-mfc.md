# Настройка для MFC
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

В данном разделе содержатся советы по настраивать приложение MFC.  
  
## Стандартные настройки  
 Можно сохранять и загружать состояние приложения в реестр.  При включении этого параметра, приложение загружает его начальное состояние из реестра.  При изменении начальный макет закрепления для приложения, необходимо очистить данные о регистрации для приложения.  В противном случае данные в реестре переопределяют любые изменения, внесенные в первоначальный макет.
