# Использование списков изображений с элементами управления &quot;Заголовок&quot;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Элементы заголовка имеют возможность отображать изображение в элементе заголовка.  Этот изображение, хранящееся в связанном списке изображений, 16 пикселей x16 и имеет те же характеристики, изображения Значка используется в элементе управления списка.  Чтобы реализовать это расширение функциональности успешно, необходимо создать и инициализировать список изображений можно связать список с элементом управления " Заголовок ", а затем изменить атрибуты элемента заголовка, который выводит изображение.  
  
 В следующей процедуре показано сведения, используя указатель на элемент управления " Заголовок " \(`m_pHdrCtrl`\) и указатель на список изображений \(`m_pHdrImages`\).  
  
### Отображение изображение в элементе заголовка  
  
1.  Создайте новый список изображений \(или использовать существующий объект списка изображений\) с помощью конструктора [CImageList](../Topic/CImageList%20Class.md), храните результирующий указатель.  
  
2.  Инициализируйте новый объект списка изображений с помощью метода [CImageList::Create](../Topic/CImageList::Create.md).  Следующий код является примером этого вызова.

5.  Измените элемент заголовка для отображения образа из связанного списка изображений.  Следующий пример присваивает первый изображение, из `m_phdrImages`, на первый элемент заголовка, `m_pHdrCtrl`.

> [!NOTE]
>  Можно иметь несколько элементов управления, используя тот же список изображений.  Например, в элементе управления стандартном списке, может быть список изображений \(образов 16 пикселя x16\), используемый обоими небольшое представление Значка список элементов управления и заголовка элемента управления "список".
