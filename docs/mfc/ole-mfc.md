# <a name="ole-mfc"></a>OLE (MFC)
Реализация функции OLE в программе влияет пользовательского интерфейса несколькими способами:  
  
-   Визуального редактирования (Активация на месте) отображает интерфейс пользователя из другой программы в вашей программе windows и изменяет меню программы с элементами из другой программы.  
  
-   Перетаскивание позволяет перетаскивать объекты внутри и между окнами и даже между программами.  
  
-   Средства отслеживания визуально отображают состояние объектов во время визуального редактирования и перетаскивания.
