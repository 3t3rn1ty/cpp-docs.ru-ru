# <a name="html-basics"></a>Основы HTML
Большинство браузеров поддерживает возможность проверки исходного кода HTML страницы, которые вы просматриваете. При просмотре исходного кода, вы увидите несколько HTML-тегов (язык HTML), заключенных в угловые скобки (<>), смешивается с текстом.  
  
 HTML-теги используйте указанные ниже действия для создания простой веб-страницы. В этом пошаговом руководстве будет введите обычный текст в файл в блокноте, внести некоторые изменения, сохраните файл и перезагрузить страницу в браузере, чтобы увидеть изменения.  
  
#### <a name="to-create-an-html-file"></a>Для создания HTML-файл  
  
1.  Откройте «Блокнот» или любой текстовый редактор.  
  
2.  Из **файл** меню, выберите **New**.  
  
3.  Введите следующие строки:  
  
 ```  
 <HTML>  
 <HEAD>  
 <TITLE>Top HTML Tags</TITLE>  
 </HEAD>  
 </HTML>  
 ```  
  
4.  Из **файл** меню, выберите **Сохранить**и сохраните файл как c:\webpages\First.htm. Оставьте файл открыт в редакторе.  
  
5.  Перейдите в браузере, а также из **файл** меню, выберите **откройте**, или тип *file://C:/webpages/first.htm* в поле URL-адрес в браузере. Появится пустая страница с заголовком окна «Top HTML-теги».  
  
     Обратите внимание, объединены в пару тегов и включаются в угловые скобки. Теги не учитывают регистр, но регистр часто используются, чтобы сделать выделялись тегов.  
  
     Тег \<HTML > запускает документа и тег  \< /HTML > завершает его. Завершение тегов (требуется не всегда) аналогичны начальный тег, но иметь косой черты (/) перед тегом. Должно быть пробелов между угловая скобка (<) и началом тег.  
  
6.  Перейдите обратно в «Блокнот», а после  \< /HEAD > строки, введите:  
  
 ```  
 <BODY>  
    HTML is swell.  
    Life is good.  
 </BODY>  
 ```  
  
7.  Из **файл** меню, выберите **Сохранить**.  
  
8.  Вернитесь в браузере и обновите страницу.  
  
     Слова будут отображаться в клиентской области окна браузера. Обратите внимание, что учитывается, возврата каретки. Если вы хотите иметь разрывы строк, необходимо включить `<BR>` тег после первой строки.  
  
     Для всех шагов, следует выполнить, вставка текста в любом месте между \<BODY > и  \< /BODY > Добавление в текст документа.  
  
9. Добавьте верхний колонтитул:  
  
 ```  
 <H3>Here's the big picture</H3>  
 ```  
  
10. Добавление изображения, с помощью GIF-файла, сохраненные в том же каталоге, что и страница:  
  
 ```  
 <IMG src="yourfile.gif">  
 ```  
  
11. Добавление списка:  
  
 ```  
 <UL>Make me an unordered list.  
 <LI>One programmer</LI>  
 <LI>Ten SDKs</LI>  
 <LI>Great Internet Apps</LI>  
 </UL>  
 ```  
  
12. Вместо этого номер списка, используйте пару \<OL > и \</OL > теги вместо \<UL > и \</UL > тегов.  
  
 Что следует приступить к работе. Если вы видите компонент на веб-странице, можно выяснить как был создан с помощью проверки исходного кода HTML. Редакторов HTML, например Microsoft Front Page может использоваться для создания простых и дополнительных страниц.  
  
 Ниже приведен файл, который построение весь HTML-код.  
  
```  
<HTML>  
<HEAD>  
<TITLE>Top HTML Tags</TITLE>  
</HEAD>  
<BODY>  
HTML is swell.<BR>  
Life is good.  
<H3>Here's the big picture</H3>  
<IMG src="yourfile.gif">  
<UL>Make me an unordered list.  
<LI>One programmer</LI>  
<LI>Ten SDKs</LI>  
<LI>Great Internet Apps</LI>  
</UL>  
</BODY>  
</HTML>  
```  
  
 Полное описание тегов атрибутами и расширениями, см. в спецификации языка (HTML):  
  
 [http://www.w3.org/pub/WWW/MarkUp/](http://www.w3.org/pub/www/markup/)
