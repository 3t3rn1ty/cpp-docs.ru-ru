# Уведомляющие сообщения древовидного элемента управления
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Элемент управления "Дерево" \([CTreeCtrl](../mfc/reference/ctreectrl-class.md)\) отправляет следующие сообщения уведомления как сообщения **WM\_NOTIFY**:  
  
|Сообщение уведомления|Описание|  
|---------------------------|--------------|  
|**TVN\_BEGINDRAG**|Сигнализирует начала операции перетаскивания|  
|**TVN\_BEGINLABELEDIT**|Сигнализирует начала редактирования на месте метки|  
|**TVN\_BEGINRDRAG**|Сигнализирует начала операции перетаскивания, используя правой кнопки мыши|  
|**TVN\_DELETEITEM**|Сигнализирует удаление определенного элемента|  
|**TVN\_ENDLABELEDIT**|Окончание означает изменение метки|  
|**TVN\_GETDISPINFO**|Данные запросов, элемент управления "Дерево" требуется для отображения элемента|  
|**TVN\_ITEMEXPANDED**|Означает, развернул или свернул список дочерних элементов родительского элемента|  
|**TVN\_ITEMEXPANDING**|Означает, список родительского элемента дочерних элементов или быть примерно развернутым свернутым|  
|**TVN\_KEYDOWN**|Сигнализирует событие клавиатуры|  
|**TVN\_SELCHANGED**|Означает, что выделение меняется от одного элемента к другому|  
|**TVN\_SELCHANGING**|Означает, что выделение около несанкционированного изменения от одного элемента к другому|  
|**TVN\_SETDISPINFO**|Уведомление для обновления сведений поддерживаемые для элемента|
