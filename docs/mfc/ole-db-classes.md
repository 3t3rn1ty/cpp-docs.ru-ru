# Классы OLE DB
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Поддержка OLE DB в MFC в данный момент состоит из класса [COLEDBRecordView](../mfc/reference/coledbrecordview-class.md).  `COleDBRecordView` отображает записи базы данных в элементах управления, через представление формы напрямую подключенное к объекту [CRowset](../Topic/CRowset%20Class.md).  Дополнительные сведения о шаблонах объект\-получателя OLE DB см. в разделе [Список шаблонов OLE DB объект\-получателя](../data/oledb/ole-db-consumer-templates-reference.md).
