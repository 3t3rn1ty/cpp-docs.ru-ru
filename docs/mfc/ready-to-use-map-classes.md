# Готовые классы сопоставления
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Следующие классы готовые к использованию классы сопоставления.  
  
 [CMapPtrToPtr](../mfc/reference/cmapptrtoptr-class.md)  
 Указатели используются `void` в качестве ключей для поиска других указателей `void`.  
  
 [CMapPtrToWord](../mfc/reference/cmapptrtoword-class.md)  
 Указатели используются `void` в качестве ключей для поиска данных типа **word**.  
  
 [CMapStringToOb](../mfc/reference/cmapstringtoob-class.md)  
 Использует объекты `CString` в качестве ключей для поиска указатели `CObject`.  
  
 [CMapStringToPtr](../Topic/CMapStringToPtr%20Class.md)  
 Использует объекты `CString` в качестве ключей для поиска указатели `void`.  
  
 [CMapStringToString](../mfc/reference/cmapstringtostring-class.md)  
 Использует объекты `CString` в качестве ключей для найти другие объекты `CString`.  
  
 [CMapWordToOb](../mfc/reference/cmapwordtoob-class.md)  
 Использует данные типа **word** для поиска указатели `CObject`.  
  
 [CMapWordToPtr](../mfc/reference/cmapwordtoptr-class.md)  
 Использует данные типа **word** для поиска указатели `void`.
