# <a name="ready-to-use-map-classes"></a>Готовые классы сопоставления

Следующие классы являются классы сопоставления готовые к использованию.

[CMapPtrToPtr](../mfc/reference/cmapptrtoptr-class.md)<br/>
Использует **void** указатели в качестве ключей поиска другой **void** указатели.

[CMapPtrToWord](../mfc/reference/cmapptrtoword-class.md)<br/>
Использует **void** указатели в качестве ключей поиска данных типа **WORD**.

[CMapStringToOb](../mfc/reference/cmapstringtoob-class.md)<br/>
Использует `CString` объектов в качестве ключей поиска `CObject` указатели.

[CMapStringToPtr](../mfc/reference/cmapstringtoptr-class.md)<br/>
Использует `CString` объектов в качестве ключей поиска **void** указатели.

[CMapStringToString](../mfc/reference/cmapstringtostring-class.md)<br/>
Использует `CString` объектов в качестве ключей для поиска других `CString` объектов.

[CMapWordToOb](../mfc/reference/cmapwordtoob-class.md)<br/>
Использует данные типа **WORD** найти `CObject` указатели.

[CMapWordToPtr](../mfc/reference/cmapwordtoptr-class.md)<br/>
Использует данные типа **WORD** найти **void** указатели.
