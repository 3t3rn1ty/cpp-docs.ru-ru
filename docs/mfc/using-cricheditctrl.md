# <a name="using-cricheditctrl"></a>Использование CRichEditCtrl
Элемента управления rich edit — это окно, в котором пользователь может вводить и редактировать текст. Текст можно назначить форматирование знаков и абзацев и может содержать внедренные объекты OLE. Элемент управления форматированием представляется в MFC, [CRichEditCtrl](../mfc/reference/cricheditctrl-class.md) класса.
