# <a name="remote-automation-user-components"></a>Пользовательские компоненты удаленной автоматизации
Убедитесь, что каждый клиентский компьютер содержит клиент программы, а также обеспечивает поддержку библиотек DLL, необходимо будет необходимо. Будет необходимо также обеспечить серверного приложения, а также обеспечивает поддержку библиотек DLL, необходимо наличие на компьютере сервера. Наконец необходимо зарегистрировать сервер программы на каждом клиентском компьютере перед выполнением RAC Manager для настройки подключения. Если программа автоматической регистрацией (как и большинство будет), требуется только выполнить программу сервера на клиентском компьютере, чтобы зарегистрировать его. После неудачной может потребоваться выполнить файл регистрации, который предоставляется или вручную изменить реестр.
