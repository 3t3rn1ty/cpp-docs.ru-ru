# Практическое руководство. Добавление элементов управления и обработчиков событий ленты
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Элементы управления ленты элементы, такие как кнопки и поля со списком, добавляемые в панели.  Панель рабочей области панели ленты, указывающие группу в составе связанные элементы управления.  
  
 В этом разделе будет открыт конструктор лент, добавьте кнопки и затем связать событие отобразится сообщение «Hello World».  
  
### Открыть конструктор лент  
  
1.  В меню Visual Studio **Вид** выберите пункт **Ресурсы**.  
  
2.  В **Ресурсы** дважды щелкните ресурс ленты для отображения в области конструктора.  
  
### Добавление кнопки и обработчик событий  
  
1.  Из **Панель инструментов**, щелкните **Кнопка** и перетащите его на значение " в области конструктора.  
  
2.  Щелкните правой кнопкой мыши и нажмите кнопку  **Добавить обработчик события**.  
  
3.  В поле **Мастер обработчика события** убедитесь, что параметры по умолчанию и нажмите **Добавить\/править**.  Для получения дополнительной информации см. [Мастер обработчиков событий](../ide/event-handler-wizard.md).  
  
4.  В редакторе кода добавьте следующий код в функцию обработчика.  
  
    ```  
    MessageBox((LPCTSTR)L"Hello World");  
    ```
