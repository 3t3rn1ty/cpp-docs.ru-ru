# <a name="ready-to-use-list-classes"></a>Готовые классы списков

Следующие классы являются классами список готовых к использованию.

[CObList](../mfc/reference/coblist-class.md)<br/>
Содержит указатели на объекты класса `CObject` или на объекты классов, производных от `CObject` в связанном списке.

[CPtrList](../mfc/reference/cptrlist-class.md)<br/>
Содержит указатели на **void** (универсальных указателей) в связанном списке.

[CStringList](../mfc/reference/cstringlist-class.md)<br/>
Магазины `CString` объектов в связанном списке.
