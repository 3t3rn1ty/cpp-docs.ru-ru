# <a name="using-ccomboboxex"></a>Использование CComboBoxEx
Элемент управления «Расширенное поле со списком» является расширением стандартный элемент управления ComboBox, обеспечивает встроенную поддержку для изображения элемента. Эти образы можно использовать для указания состояния отдельных элементов в поле со списком, такие как элементы в настоящее время в выбранном и невыбранном. Чтобы сделать элемент изображения становятся доступны, элемент управления поддерживает изображения список.  
  
 Используйте этот элемент управления для обеспечения функциональности списком без необходимости вручную добавлять графические объекты элемента.
