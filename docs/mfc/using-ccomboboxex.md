# <a name="using-ccomboboxex"></a>Использование CComboBoxEx

Расширенные поле со списком является расширением стандартной поле со списком, обеспечивает встроенную поддержку для изображения элемента. Эти образы можно использовать для указания состояния отдельных элементов в поле со списком, например в настоящее время выбранном и невыбранном элементы. Чтобы сделать элемент изображения легко доступной, элемент управления поддерживает изображение списка.

Используйте этот элемент управления для обеспечения функциональности списком без необходимости вручную добавлять графические объекты элемента.
