# Установка цвета фона диалогового окна
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Можно задать цвет фона для диалоговых окон, обработка сообщений `WM_CTLCOLOR` для окна диалогового окна.  Цвета, набор используются только для указанного диалогового окна.
