# <a name="drawing-tool-classes"></a>Классы инструмента рисования
Эти классы инкапсулируют графических средств, используемых для отрисовки на контекст устройства.  
  
 [CGdiObject](../mfc/reference/cgdiobject-class.md)  
 Базовый класс для инструментов рисования GDI.  
  
 [CBrush](../mfc/reference/cbrush-class.md)  
 Инкапсулирует кисть GDI, которая может быть выбран в качестве текущей кисти в контексте устройства. Кисти используются для заполнения внутренние отрисовывается объектов.  
  
 [CPen](../mfc/reference/cpen-class.md)  
 Инкапсулирует перо GDI, которая может быть выбран в качестве текущего пера в контекст устройства. Перья используются для рисования линий границы объектов.  
  
 [CFont](../mfc/reference/cfont-class.md)  
 Инкапсулирует шрифта GDI, которая может быть выбран в качестве текущего шрифта в контексте устройства.  
  
 [CBitmap](../mfc/reference/cbitmap-class.md)  
 Инкапсулирует растровое GDI, обеспечивая интерфейс для управления растровые изображения.  
  
 [CPalette](../mfc/reference/cpalette-class.md)  
 Инкапсулирует цветовую палитру GDI для использования в качестве интерфейс между приложением и устройством выходной цвет, например экрана.  
  
 [CRectTracker](../mfc/reference/crecttracker-class.md)  
 Отображает и обрабатывает пользовательский интерфейс для изменения размера и перемещения прямоугольной объектов.
