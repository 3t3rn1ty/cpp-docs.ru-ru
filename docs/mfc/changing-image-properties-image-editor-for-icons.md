# Changing Image Properties (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

С помощью [окна свойств](../Topic/Properties%20Window.md) можно задавать и изменять свойства изображения.  
  
### Изменение свойств изображения  
  
1.  Откройте изображение в **редакторе изображений**.  
  
2.  В **окне свойств** измените нужные свойства изображения.  
  
    |Свойство.|Описание|  
    |---------------|--------------|  
    |**Цвета**|Задает цветовую схему изображения.  Выберите один из вариантов схемы: монохромный, 16 цветов, 256 цветов или "True Color".  Если для изображения, созданного с использованием 16\-цветной палитры, выбрать монохромный вариант, все цвета будут заменены белым и черным.  При этом контраст может сохраняться не всегда: например, смежные области красного и зеленого цветов будут переданы черным цветом.|  
    |**Имя файла**|Позволяет задать имя файла изображения.  По умолчанию Visual Studio присваивает файлу стандартное имя, которое создается на основе стандартного идентификатора ресурса \(IDB\_BITMAP1\) путем удаления первых четырех символов \("IDB\_"\) и добавления соответствующего расширения.  В данном примере файлу изображения присваивается имя BITMAP1.bmp.  Имя файла можно изменить на MYBITMAP1.bmp.|  
    |**Высота**|Задает высоту изображения в пикселях.  Значением по умолчанию является 48.  Изображение может быть обрезано или же к нему снизу может быть добавлено пустое пространство.|  
    |**Идентификатор**|Задает идентификатор ресурса.  По умолчанию Microsoft Visual Studio присваивает изображениям идентификаторы, которые образуются по одной схеме: IDB\_BITMAP1, IDB\_BITMAP2 и так далее.  Аналогичные имена используются для значков и курсоров.|  
    |**Палитра**|Позволяет изменять свойства цветов.  Щелкните дважды, чтобы выбрать цвет и открыть диалоговое окно [Выбор цвета](../windows/custom-color-selector-dialog-box-image-editor-for-icons.md).  Чтобы определить цвет, введите в соответствующих полях значения RGB и HSL.|  
    |**SaveCompressed**|Указывает, сохранено ли изображение в сжатом формате.  Это свойство доступно только для чтения.  Visual Studio не позволяет сохранять изображения в сжатом формате, поэтому у всех изображений, созданных в среде Visual Studio значение этого свойства будет **ЛОЖЬ**.  Если в среде Visual Studio открыть сжатое изображение, созданное с помощью другого приложения, у такого изображения данное свойство будет иметь значение **ИСТИНА**.  При сохранении сжатого изображения с помощью Visual Studio изображение будет сохранено без сжатия, а для этого свойства будет снова задано значение **ЛОЖЬ**.|  
    |**Ширина**|Задает ширину изображения в пикселях.  Значением по умолчанию для растровых изображений является 48.  Изображение может быть обрезано или же к нему справа может быть добавлено пустое пространство.|  
  
 Сведения о добавлении ресурсов в управляемые проекты см. в разделе [Ресурсы приложений](../Topic/Resources%20in%20Desktop%20Apps.md) *Руководства разработчика .NET Framework*. Сведения о том, как вручную добавлять файлы ресурсов в управляемые проекты, осуществлять доступ к ресурсам, отображать статические ресурсы и присваивать строки ресурсов свойствам см. в разделах [Пошаговое руководство. Локализация приложений Windows Forms](http://msdn.microsoft.com/ru-ru/9a96220d-a19b-4de0-9f48-01e5d82679e5) и [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md).
