# Классы файлов и баз данных
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Эти классы позволяют к данным магазина в базу данных или дисковому файлу.  3 — Набора классов базы данных OLE DB, ODBC и DAO —, предоставляют аналогичную функциональность.  Команда OLE DB реализуется с использованием OLE DB и работает с шаблонами объект\-получателя OLE DB, команда DAO реализуется с помощью объекта доступа к данным, и команда ODBC реализована с помощью ODBC.  Также набор классов для управления являются стандартными файлами, активных потоков и потоки HTML.
