# <a name="dialog-data-exchange-and-validation"></a>Обмен данными диалоговых окон и их проверка
Обмен данными (диалоговых окон DDX) является простым способом для инициализации диалогового окна элементы управления и для сбора данных, вводимых пользователем. Проверка данных диалогового окна (DDV) является простой способ проверки элемента данных в диалоговом окне. Чтобы воспользоваться преимуществами DDX и DDV к элементам Управления в диалоговые окна, используйте [член мастер добавления переменной-](../ide/add-member-variable-wizard.md) для создания элементов данных и задайте их типам данных и задавать правила проверки.
