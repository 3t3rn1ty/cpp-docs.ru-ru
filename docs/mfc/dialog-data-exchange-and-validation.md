# Обмен данными диалоговых окон и их проверка
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Данные диалогового окна \(DDX\) простой способ инициализации элементов управления в данном диалоговом окне и сбора ввод данных пользователем.  Проверка данных диалогового окна \(DDV\) простой способ для записи данных в диалоговом окне.  Чтобы воспользоваться преимуществами DDX и DDV на диалоговых окнах, используйте [Мастер добавления переменной\-члена](../ide/add-member-variable-wizard.md) для создания элементов данных и задать их типы данных и указания правил проверки.
