# Converting an Image From One Format to Another (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Можно открыть изображение GIF или JPEG в редакторе изображений и сохранить его как растровое изображение.  Также можно открыть файл растрового изображения и сохранить изображение как GIF или JPEG.  Редактируемые в среде разработки изображения не обязаны быть частью проекта \(см. [редактирование отдельных изображений](../Topic/Editing%20an%20Image%20Outside%20of%20a%20Project%20\(Image%20Editor%20for%20Icons\).md)\).  
  
### Преобразование изображений из одного формата в другой  
  
1.  Откройте изображение в **редакторе изображений**.  
  
2.  В меню **Файл** выберите **Сохранить *имя\_файла*как**.  
  
3.  В диалоговом окне **Сохранение файла**, в поле **Имя файла**, введите имя файла и расширение, обозначающее нужный формат.  
  
4.  Нажмите кнопку **Сохранить**.  
  
 Сведения о добавлении ресурсов в управляемые проекты см. в разделе [Ресурсы приложений](../Topic/Resources%20in%20Desktop%20Apps.md) *Руководства разработчика .NET Framework*. Сведения о том, как вручную добавлять файлы ресурсов в управляемые проекты, осуществлять доступ к ресурсам, отображать статические ресурсы и присваивать строки ресурсов свойствам см. в разделах [Пошаговое руководство. Локализация приложений Windows Forms](http://msdn.microsoft.com/ru-ru/9a96220d-a19b-4de0-9f48-01e5d82679e5) и [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md).
