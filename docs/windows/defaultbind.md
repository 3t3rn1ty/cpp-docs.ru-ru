# defaultbind
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Отображает одно свойство, связываемое лучше всего представляет объект.  
  
## Синтаксис  
  
```  
  
[defaultbind]  
  
```  
  
## Заметки  
 **defaultbind** Атрибут C\+\+ имеет ту же функциональность, что и  [defaultbind](http://msdn.microsoft.com/library/windows/desktop/aa366790) атрибут MIDL.  
  
## Пример  
 См. пример [bindable](../windows/bindable.md) пример использования  **defaultbind**.  
  
## Требования

|||  
|-|-|

|**Repeatable**|Нет|  
|**Обязательные атрибуты**|None|  
|**Недопустимые атрибуты**|None|  
  
 Дополнительные сведения см. в разделе [Контексты атрибута](../windows/attribute-contexts.md).
