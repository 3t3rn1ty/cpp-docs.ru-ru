# <a name="changing-image-properties-image-editor-for-icons"></a>Изменение свойств изображения (редактор изображений для значков)

Можно задать или изменить свойства изображения с помощью [окно "Свойства"](/visualstudio/ide/reference/properties-window).

### <a name="to-change-an-images-properties"></a>Изменение свойств изображения

1. Открывать изображение в **изображение** редактора.

2. В **свойства** окна, измените любой или все свойства изображения.

   |Свойство.|Описание:|
   |--------------|-----------------|
   |**Цвета**|Задает цветовую схему для образа. Выберите **монохромная**, **16**, или **256**, или **True Color**. Если изображение с 16-цветовую палитру уже создана, выбрав **монохромная** вызывает подстановок черного и белого цвета изображения. Контрастность не всегда сохраняются: например, смежные области красного и зеленого цветов преобразуются на черный.|
   |**Имя файла**|Задает имя файла изображения. По умолчанию Visual Studio назначает базовое имя файла, созданные путем удаления первые четыре символа («IDB_») из стандартного идентификатора ресурса (IDB_BITMAP1) и добавления соответствующего расширения. Имя файла для изображения в этом примере будет `BITMAP1.bmp`. Можно переименовать его `MYBITMAP1.bmp`.|
   |**Высота**|Задает высоту изображения (в пикселях). Значение по умолчанию — 48. Изображение обрезается или пробел добавляется под существующий образ.|
   |**ID**|Задает идентификатор ресурса. Для изображения, Microsoft Visual Studio, по умолчанию назначается следующий доступный идентификатор в ряде: IDB_BITMAP1, IDB_BITMAP2, и т. д. Аналогичные имена используются для значков и курсоров.|
   |**Палитра**|Изменения цвета свойства. Дважды щелкните, чтобы выбрать цвет и отобразить [диалоговое окно «Выбор цвета»](../windows/custom-color-selector-dialog-box-image-editor-for-icons.md). Определите цвет путем ввода значений RGB или HSL в соответствующие текстовые поля.|
   |**SaveCompressed**|Указывает, является ли изображение в сжатом формате. Это свойство доступно только для чтения. Visual Studio не позволяет сохранять изображения в сжатом формате, поэтому все образы, созданные в Visual Studio, это свойство будет иметь **False**. При открытии сжатого образа (созданного в другом приложении) в Visual Studio, это свойство будет иметь **True**. При сохранении сжатого образа с помощью Visual Studio, будут сжиматься и будет возвращено это свойство к **False**.|
   |**Ширина**|Задает ширину изображения (в пикселях). Значение по умолчанию для растровых изображений — 48. Изображение обрезается или справа от существующего образа может быть добавлена.|

Сведения о добавлении ресурсов в управляемые проекты см. в разделе [ресурсы в приложениях для настольных систем](/dotnet/framework/resources/index) в *руководства разработчика .NET Framework*. Сведения о вручную добавлять файлы ресурсов в управляемые проекты, осуществлять доступ к ресурсам, отображать статические ресурсы и присваивать строки ресурсов свойствам, см. в разделе [Создание файлов ресурсов для приложений рабочего стола](/dotnet/framework/resources/creating-resource-files-for-desktop-apps). Сведения о глобализации и локализации ресурсов в управляемых приложениях, см. в разделе [Globalizing and Localizing .NET Framework Applications](/dotnet/standard/globalization-localization/index).
