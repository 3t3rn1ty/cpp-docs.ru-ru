# Универсальные типы и методы (расширения компонентов C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Универсальные шаблоны \- это параметризованные типы и методы.  В этом разделе можно узнать, какие универсальные функции поддерживаются и средой выполнения Windows, и средой CLR, а какие поддерживаются только средой CLR.  Кроме того, можно узнать, как создавать собственные универсальные методы и типы в Visual C\+\+ и как использовать универсальные типы, созданные на языке платформы .NET Framework в Visual С\+\+.  Наконец, в этом разделе приводится сравнение универсальных шаблонов и шаблонов C\+\+.
