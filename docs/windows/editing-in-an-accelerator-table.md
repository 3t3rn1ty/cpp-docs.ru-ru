# Редактирование в таблице сочетаний клавиш
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

### <a name="to-edit-in-an-accelerator-table"></a>Редактирование в таблице сочетаний клавиш  
  
1.  Откройте таблицу сочетаний клавиш, дважды щелкнув его значок в [ресурсов](../windows/resource-view-window.md).  
  
    > [!NOTE]
    >  Если в проекте еще нет RC-файла, см. раздел [Создание нового файла описания ресурсов](../windows/how-to-create-a-resource-script-file.md).  
  
2.  Выберите запись в таблице и щелкните ее для активации режима редактирования "на месте".  
  
3.  Выберите значение из раскрывающегося списка или введите вручную нужное изменение.  
  
    -   Для [идентификатор](Id%20Property.xml), выберите из списка или впечатайте для редактирования.  
  
    -   Для [модификатор](../windows/accelerator-modifier-property.md), выберите из списка.  
  
    -   Для [ключ](../Topic/Accelerator%20Key%20Property.md), выберите из списка или впечатайте для редактирования.  
  
    -   Для [типа](../windows/accelerator-type-property.md), выберите из списка в ASCII или VIRTKEY.  
  
 Сведения о добавлении ресурсов в проекты управляемого кода см. в разделе [Ресурсы приложений](../Topic/Resources%20in%20Desktop%20Apps.md)  *Руководства разработчика .NET Framework*  
  
 **Requirements**  
  
 Win32
