# Setting the Size of the Combo Box and Its Drop-Down List
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Задать размер поля со списком можно при добавлении его в диалоговое окно.  Можно также задать размер поля раскрывающегося списка.  
  
### Задание размера поля со списком  
  
1.  Выделите в диалоговом окне элемент управления поля со списком в диалоговом окне.  
  
     Сначала активны только правый и левый маркеры размера.  
  
2.  Эти маркеры размера используются для задания ширины поля со списком.  
  
 Можно также задать вертикальный размер раскрывающейся части поля со списком.  
  
#### Задание размера раскрывающегося списка поля со списком  
  
1.  Щелкните кнопку стрелки раскрытия в правой части поля со списком.  
  
     ![Стрелка поля со списком проекта MFC](../Image/vcComboBoxArrow.gif "vcComboBoxArrow")  
  
     Контур элемента управления меняется, чтобы показать размер поля со списком с развернутой областью раскрывающегося списка.  
  
2.  Для изменения первоначального размера области раскрывающегося списка используется нижний маркер размера.  
  
     ![Размеры поля со списком в проекте MFC](../mfc/media/vccomboboxsizing.png "vcComboBoxSizing")  
  
3.  Щелкните стрелку раскрытия еще раз, чтобы закрыть раскрывающуюся часть поля со списком.
