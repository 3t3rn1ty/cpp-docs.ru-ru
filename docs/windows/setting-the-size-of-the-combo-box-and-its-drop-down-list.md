# <a name="setting-the-size-of-the-combo-box-and-its-drop-down-list"></a>Задание размера для поля со списком и соответствующего раскрывающегося списка
При добавлении его в диалоговом окне можно изменить размеры поле со списком. Также можно указать размер поля раскрывающегося списка.  
  
### <a name="to-size-a-combo-box"></a>Чтобы изменить размер поля со списком  
  
1.  Выберите элемент управления списком диалогового окна.  
  
     Изначально активны только маркеры изменения размера вправо и влево.  
  
2.  Используйте маркеры, чтобы задать ширину поля со списком.  
  
 Можно также задать вертикальный размер раскрывающаяся часть поля со списком.  
  
#### <a name="to-set-the-size-of-the-combo-box-drop-down-list"></a>Чтобы задать размер поля со списком поле раскрывающегося списка  
  
1.  Нажмите кнопку со стрелкой раскрывающегося списка в правом поле со списком.  
  
     ![Стрелка на поле со списком в проекте MFC](../mfc/media/vccomboboxarrow.gif "vcComboBoxArrow")  
  
     Контур элемента управления меняется, чтобы показать размер поля со списком с развернутой областью раскрывающегося списка.  
  
2.  Чтобы изменить исходный размер области раскрывающегося списка, используйте нижний маркер изменения размера.  
  
     ![Поле со списком&#45;изменение размера в проекте MFC](../mfc/media/vccomboboxsizing.gif "vcComboBoxSizing")  
  
3.  Щелкните стрелку раскрывающегося списка, чтобы закрыть часть раскрывающегося списка в поле со списком.
