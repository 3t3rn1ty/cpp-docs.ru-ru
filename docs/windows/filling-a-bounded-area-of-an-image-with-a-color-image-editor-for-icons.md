# <a name="filling-a-bounded-area-of-an-image-with-a-color-image-editor-for-icons"></a>Заливка цветом выделенной области изображения (редактор изображений для значков)

Редактор изображений предоставляет **заполнения** средство для заполнения любой заключены области изображения с помощью текущего цвета или текущий цвет фона.

> [!TIP]
> Всплывающие подсказки отображаются при наведении курсора на кнопку панели инструментов. Эти советы помогут определить назначение каждой кнопки.

### <a name="to-use-the-fill-tool"></a>Чтобы использовать средство заполнения

1. На **редактор изображений** панели инструментов (или **изображение** меню **средства** команды), нажмите кнопку **заполнения** средство.

2. При необходимости выберите рисования цветов: В [палитра цветов](../windows/colors-window-image-editor-for-icons.md), щелкните левой кнопкой мыши выберите цвет переднего плана или правой кнопки мыши выберите цвет фона.

3. Переместить **заливки** инструментов в область, которую требуется заполнить.

4. Нажмите кнопку мыши влево или вправо, чтобы заполнить цвет переднего плана или фона, соответственно.

Сведения о добавлении ресурсов в управляемые проекты см. в разделе [ресурсы в приложениях для настольных систем](/dotnet/framework/resources/index) в *руководства разработчика .NET Framework*. Сведения о вручную добавлять файлы ресурсов в управляемые проекты, осуществлять доступ к ресурсам, отображать статические ресурсы и присваивать строки ресурсов свойствам, см. в разделе [Создание файлов ресурсов для приложений рабочего стола](/dotnet/framework/resources/creating-resource-files-for-desktop-apps). Сведения о глобализации и локализации ресурсов в управляемых приложениях, см. в разделе [Globalizing and Localizing .NET Framework Applications](/dotnet/standard/globalization-localization/index).
