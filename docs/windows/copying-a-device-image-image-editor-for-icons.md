# <a name="copying-a-device-image-image-editor-for-icons"></a>Копирование изображения устройства (редактор изображений для значков)
### <a name="to-copy-a-device-image"></a>Копирование изображения устройства  
  
1.  На **изображения** меню, нажмите кнопку **открыть изображение устройства** и выберите изображение из текущего списка изображений. Например можно выберите 32 × 32, 16-цветную версию значка.  
  
2.  Скопировать изображение в настоящее время значка (**CTRL + C**).  
  
3.  Откройте другое изображение значка в другом **редактора изображений** окна. Например откройте 16 х 16, 16 цветов версии значка.  
  
4.  Вставьте изображение значка (**CTRL + V**) из одного **редактора изображений** окна в другой. При вставке большего размера в меньшего размера можно использовать маркеры значка для изменения размера изображения.  
  
 Сведения о добавлении ресурсов в управляемые проекты см. в разделе [ресурсы в классических приложениях](/dotnet/framework/resources/index) в *руководства разработчика .NET Framework.* Сведения о вручную добавлять файлы ресурсов в управляемые проекты, осуществлять доступ к ресурсам, отображать статические ресурсы и присваивать строки ресурсов свойствам см. в разделе [Создание файлов ресурсов для приложений рабочего стола](/dotnet/framework/resources/creating-resource-files-for-desktop-apps). Сведения о глобализации и локализации ресурсов в управляемых приложениях см. в разделе [Globalizing и локализация приложений .NET Framework](/dotnet/standard/globalization-localization/index).
