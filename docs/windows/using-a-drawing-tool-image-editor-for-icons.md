# <a name="using-a-drawing-tool-image-editor-for-icons"></a>Использование инструмента рисования (редактор изображений для значков)
В редакторе изображений от руки Рисование и стирание средства работают одинаково: выберите средство и, при необходимости, [выберите цвет текста и фона](../windows/selecting-foreground-or-background-colors-image-editor-for-icons.md) и параметры размера и формы. Затем наведите указатель на изображения и щелкните или перетащите для рисования и стирания.  
  
 При выборе **ластика** средство, **кисти** средство, или **распылителем** средство селектор параметров отображаются параметры этого инструмента.  
  
> [!TIP]
>  Вместо использования **ластика** средство, может оказаться более удобным для рисования, используя один из инструментов рисования цвет фона.  
  
 Инструменты рисования можно выбрать либо из **редактора изображений** панели инструментов или **изображения** меню.  
  
### <a name="to-select-and-use-a-drawing-tool-from-the-image-editor-toolbar"></a>Выбор и использование инструмента рисования в панели инструментов редактора изображений  
  
1.  Нажмите кнопку **редактора изображений** инструментов.  
  
    -   **Ластика** средства рисование поверх изображения, используя текущий цвет фона при нажатии левой кнопки мыши.  
  
    -   **Карандаша** средство позволяет рисовать от руки с постоянной шириной один пиксель.  
  
    -   **Селектор вариантов определяет форму и размер инструмента Кисть**.  
  
    -   **Распылителем** инструмент случайным образом распределяет цветные точки вокруг центра кисти.  
  
        > [!TIP]
        >  Отображаются всплывающие подсказки при наведении курсора на кнопки на [панель инструментов редактора изображений](../windows/toolbar-image-editor-for-icons.md). Следующие советы помогут определить конкретные из вышеупомянутых кнопок.  
  
2.  При необходимости выберите цвета и кисть:  
  
    -   В [палитра цветов](../windows/colors-window-image-editor-for-icons.md), щелкните левой кнопки мыши выберите цвет переднего плана, или правой кнопкой мыши выберите цвет фона.  
  
    -   В [селектор параметров](../windows/toolbar-image-editor-for-icons.md), щелкните форму, представляющую кисть, которая будет использоваться.  
  
3.  Укажите место, на которой вы хотите начать рисование изображения или рисования. Указатель изменится в соответствии с выбранным инструментом.  
  
4.  Нажмите левую кнопку мыши (для основного цвета) или правую кнопку мыши (для фонового цвета) и удерживайте его в процессе рисования.  
  
### <a name="to-select-and-use-a-drawing-tool-from-the-image-menu"></a>Выбор и использование инструмента рисования в меню "изображение"  
  
1.  Нажмите кнопку **изображения** и выбрать пункт **средства** команды.  
  
2.  Каскадные подменю выберите средство, которое вы хотите использовать.  
  
 Сведения о добавлении ресурсов в управляемые проекты см. в разделе [ресурсы в классических приложениях](/dotnet/framework/resources/index) в *руководства разработчика .NET Framework.* Сведения о вручную добавлять файлы ресурсов в управляемые проекты, осуществлять доступ к ресурсам, отображать статические ресурсы и присваивать строки ресурсов свойствам см. в разделе [Создание файлов ресурсов для приложений рабочего стола](/dotnet/framework/resources/creating-resource-files-for-desktop-apps). Сведения о глобализации и локализации ресурсов в управляемых приложениях см. в разделе [Globalizing и локализация приложений .NET Framework](/dotnet/standard/globalization-localization/index).
