# Using a Drawing Tool (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

В редакторе изображений все инструменты рисования от руки и удаления работают одинаково: разработчик выбирает инструмент и при необходимости [выбирает основной цвет и цвет фона](../windows/selecting-foreground-or-background-colors-image-editor-for-icons.md), а также параметры размера и формы.  Затем разработчик перемещает указатель на изображение и щелкает мышью или перетаскивает указатель для рисования и стирания.  
  
 При выборе инструмента **Ластик**, **Кисть** или **Распылитель**, селектор вариантов содержит возможные варианты для этого инструмента.  
  
> [!TIP]
>  Вместо использования инструмента **Ластик** разработчик может выбрать один из инструментов рисования, используя цвет фона.  
  
 Инструменты рисования можно выбрать либо в панели инструментов **редактора изображений**, либо в меню **Изображение**.  
  
### Выбор инструмента рисования в панели редактора изображений и его использование  
  
1.  Нажмите кнопку в панели инструментов **редактора изображений**.  
  
    -   Инструмент **Ластик**, при нажатой левой кнопке мыши, рисует поверх изображения, используя текущий цвет фона.  
  
    -   Инструмент **Карандаш** позволяет рисовать от руки с постоянной шириной в один пиксель.  
  
    -   Селектор вариантов определяет форму и размер инструмента **Кисть**.  
  
    -   Инструмент **Распылитель** в случайном порядке распределяет цветные пиксели вокруг центра кисти.  
  
        > [!TIP]
        >  При помещении указателя над кнопками [панели инструментов редактора изображений](../mfc/toolbar-image-editor-for-icons.md) появляются всплывающие подсказки.  Эти подсказки помогут определить конкретные из вышеупомянутых кнопок.  
  
2.  При необходимости выберите цвета и кисть:  
  
    -   В области [Палитра цветов](../Topic/Colors%20Window%20\(Image%20Editor%20for%20Icons\).md) щелкните левой кнопкой мыши, чтобы выбрать основной цвет, или правой кнопкой мыши, чтобы выбрать цвет фона.  
  
    -   В области [Селектор вариантов](../mfc/toolbar-image-editor-for-icons.md) щелкните форму, представляющую нужную кисть.  
  
3.  Укажите мышью место изображения, с которого нужно начать рисование.  Указатель изменится в соответствии с выбранным инструментом.  
  
4.  Нажмите левую кнопку мыши \(для основного цвета\) или правую кнопку мыши \(для фонового цвета\), и удерживайте ее нажатой во время рисования.  
  
### Выбор инструмента рисования в меню "Изображение" и его использование  
  
1.  Щелкните меню **Изображение** и выберите команду **Инструменты**.  
  
2.  В каскадно раскрывающемся подменю выберите нужный инструмент.
