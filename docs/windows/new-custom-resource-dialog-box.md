# Диалоговое окно &quot;Создание настраиваемого ресурса&quot;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Позволяет создавать новый настраиваемый ресурс.  
  
 **Тип ресурса**  
 Предоставляет текстовое поле для ввода имени типа настраиваемого ресурса.  Visual C\+\+ автоматически делает первую букву имени прописной при выходе из диалогового окна **Создание настраиваемого ресурса**.  
  
## Заметки  
 Не указывайте имена ресурсов или [!INCLUDE[TLA2#tla_id](../windows/includes/tla2sharptla_id_md.md)], зарезервированные для [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)].  Зарезервированы имена DESIGNINFO, HWB и TEXTINCLUDE, а также идентификатор [!INCLUDE[TLA2#tla_id](../windows/includes/tla2sharptla_id_md.md)] 255.
