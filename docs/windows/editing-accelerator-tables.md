# <a name="editing-accelerator-tables"></a>Редактирование в таблицах сочетаний клавиш
В Visual C++ можно изменить таблицу сочетаний клавиш непосредственно с помощью редактирования прямо в редакторе сочетаний клавиш.  
  
 Описанные ниже процедуры относятся к страницам стандартных свойств, тем не менее, редактирования по месту и метод страницы свойств обеспечивают одинаковый результат. Изменения, внесенные с помощью страниц свойств или с помощью редактирования прямо немедленно отражаются в таблице сочетаний клавиш.

-   [Добавление записи в таблице сочетаний клавиш](../windows/adding-an-entry-to-an-accelerator-table.md)
