# propget
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Определяет функции метода доступа свойства.  
  
## Синтаксис  
  
```  
  
[propget]  
  
```  
  
## Заметки  
 **propget** Атрибут C\+\+ имеет ту же функциональность, что и  [propget](http://msdn.microsoft.com/library/windows/desktop/aa367145) атрибут MIDL.  
  
## Пример  
 См. пример [bindable](../windows/bindable.md) для использования образцы  **propget**.  
  
## Требования  
  
### Контекст атрибута  
  
|||  
|-|-|  
|**Применение**|Метод|  
|**Repeatable**|Нет|  
|**Обязательные атрибуты**|None|  
|**Недопустимые атрибуты**|**propput**"  **propputref**|  
  
 Дополнительные сведения о контекстах атрибута см. в разделе [Контексты атрибута](../windows/attribute-contexts.md).
