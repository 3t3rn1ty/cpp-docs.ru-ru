# <a name="custom-image-dialog-box-image-editor-for-icons"></a>Диалоговое окно "Настраиваемое изображение" (редактор изображений для значков)
Позволяет создать новый образ с пользовательский размер и количество цветов.
 **Ширина**  
 Поле ввода значения ширины настраиваемого изображения в точках (1-512, предельное значение 2048).  
  
 **Высота**  
 Поле для ввода значения высоты настраиваемого изображения в точках (1-512, предельное значение 2048).  
  
 **Цвета**  
 Предоставляет место для выбора количества цветов для настраиваемого образа: 2, 16 или 256.  
  
## <a name="requirements"></a>Требования  
 Нет
