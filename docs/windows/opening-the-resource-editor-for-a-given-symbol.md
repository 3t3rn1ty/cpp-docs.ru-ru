# <a name="opening-the-resource-editor-for-a-given-symbol"></a>Доступ к редактору ресурсов заданного символа
При просмотре символов в [символы ресурсов-диалоговое окно](../windows/resource-symbols-dialog-box.md), могут получить дополнительные сведения о том, как используется конкретный символ. **Просмотреть** кнопки позволяет быстро получить эту информацию.  
  
### <a name="to-move-to-the-resource-editor-where-a-symbol-is-being-used"></a>Перемещение к редактору ресурсов, где используется символ  
  
1.  Выберите символ в **имя** поле **символы ресурсов** диалоговое окно.  
  
2.  В **используется** выберите тип ресурса, который нужно просмотреть.  
  
3.  Нажмите кнопку **просмотреть** кнопки.  
  
     Ресурс будет показан в соответствующем окне редактора.  
  
 Сведения о добавлении ресурсов в управляемые проекты см. в разделе [ресурсы в классических приложениях](/dotnet/framework/resources/index) в *руководства разработчика .NET Framework.*  
  
 Требования  
  
 Win32
