# Making Controls the Same Width, Height, or Size
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Группу элементов управления можно выровнять по размеру главного элемента управления.  Также можно [изменить размеры элемента управления, исходя из размеров шрифта в его заголовке](../mfc/sizing-individual-controls.md).  
  
### Выравнивание элементов управления по размеру  
  
1.  [Выберите элементы управления](../mfc/selecting-multiple-controls.md), размеры которых необходимо изменить.  
  
     Первый выбранный элемент управления в группе считается главным.  Размер остальных элементов управления в группе в конечном счете будет зависеть от размеров главного элемента управления.  Дополнительные сведения о выборе главного элемента управления см. в разделе [Выбор главного элемента управления](../mfc/specifying-the-dominant-control.md).  
  
2.  В меню **Формат** выберите **Установить тот же размер**, а затем выберите одну из следующих команд:  
  
    -   **Оба**  
  
    -   **Высота**  
  
    -   **Ширина**

Локализация приложений Windows Forms](http://msdn.microsoft.com/ru-ru/9a96220d-a19b-4de0-9f48-01e5d82679e5) и [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md).
