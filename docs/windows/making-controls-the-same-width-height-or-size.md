# <a name="making-controls-the-same-width-height-or-size"></a>Выравнивание элементов управления по размеру
Можно изменить размер группы элементов управления, исходя из размера главного элемента управления. Вы также можете [размера элемента управления на основе аналитик его заголовке](../windows/sizing-individual-controls.md).  
  
### <a name="to-make-controls-the-same-width-height-or-size"></a>Выравнивание элементов управления ширину, высоту и размер  
  
1.  [Выберите элементы управления](../windows/selecting-multiple-controls.md) которого требуется изменить.  
  
     Элемент, выделенный в первую очередь в ряду — главного элемента управления. Окончательный размер элементов управления в группе зависит от размера главного элемента управления. Дополнительные сведения о выборе главного элемента управления см. в разделе [Задание главного элемента управления](../windows/specifying-the-dominant-control.md).  
  
2.  Из **формат** меню, выберите **установить тот же размер**, а затем выберите одну из следующих команд:  
  
    -   **Оба**  
  
    -   **Высота**  
  
    -   **Ширина**  
  
 Сведения о добавлении ресурсов в управляемые проекты см. в разделе [ресурсы в классических приложениях](/dotnet/framework/resources/index) в *руководства разработчика .NET Framework.* Сведения о вручную добавлять файлы ресурсов в управляемые проекты, осуществлять доступ к ресурсам, отображать статические ресурсы и присваивать строки ресурсов свойствам см. в разделе [Создание файлов ресурсов для приложений рабочего стола](/dotnet/framework/resources/creating-resource-files-for-desktop-apps). Сведения о глобализации и локализации ресурсов в управляемых приложениях см. в разделе [Globalizing и локализация приложений .NET Framework](/dotnet/standard/globalization-localization/index).
