# <a name="disabling-guides"></a>Отключение направляющих
Отключение эффекта привязки к направляющим, можно использовать специальные ключи в сочетании с помощью мыши. С помощью клавиши ALT отключает эффект привязки для выбранной направляющей. Перемещение направляющей с клавишей SHIFT предотвращает привязанные элементы перемещение по направляющей.  
  
### <a name="to-disable-the-snapping-effect-of-the-guides"></a>Отключение эффекта привязки к направляющим  
  
1.  Перетащите элемент управления, удерживая нажатой **ALT** ключа.  
  
### <a name="to-move-guides-without-moving-the-snapped-controls"></a>Перемещение направляющих без перемещения привязанных элементов управления  
  
1.  Перетащите направляющую, удерживая нажатой **SHIFT** ключа.  
  
### <a name="to-turn-off-the-guides"></a>Отключение направляющих  
  
1.  Из **формат** меню, выберите **параметры направляющих**.  
  
2.  В [диалоговое окно "Параметры направляющих"](../windows/guide-settings-dialog-box.md)в разделе **направляющие**выберите **нет**.  
  
    > [!NOTE]
    >  Можно также дважды щелкнуть строку линейки, чтобы получить доступ к **параметры направляющих** диалоговое окно.  
  
 \- или -  
  
-   На **формат** меню, нажмите кнопку **направляющие**.
