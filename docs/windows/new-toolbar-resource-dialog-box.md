# Диалоговое окно &quot;Создание ресурса панели инструментов&quot;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Диалоговое окно "Создание ресурса панели инструментов" позволяет задать ширину и высоту кнопок, добавляемых в ресурс панели инструментов.  Значение по умолчанию составляет 16 × 15 пикселей.  
  
 Ширина растрового изображения, используемого для создания панели инструментов, не может превышать значение 2048.  Поэтому, если значение **Ширина кнопки** задано равным 512, на панели можно разместить только четыре кнопки.  Если ширина задана равной 513, можно использовать только три кнопки.  
  
 **Ширина кнопки**  
 Предоставляет место ввода ширины кнопок панели инструментов, преобразуемых из ресурса растрового изображения в ресурс панели инструментов.  Изображения обрезаются по заданным значениям ширины и высоты, а цвета подгоняются под использование стандартных цветов панели инструментов \(16 цветов\).  
  
 **Высота кнопки**  
 Предоставляет место ввода высоты кнопок панели инструментов, преобразуемых из ресурса растрового изображения в ресурс панели инструментов.  Изображения обрезаются по заданным значениям ширины и высоты, а цвета подгоняются под использование стандартных цветов панели инструментов \(16 цветов\).  
  
 Сведения о добавлении ресурсов в управляемые проекты см. в разделе [Ресурсы приложений](../Topic/Resources%20in%20Desktop%20Apps.md) *Руководства разработчика .NET Framework*. Сведения о том, как вручную добавлять файлы ресурсов в управляемые проекты, осуществлять доступ к ресурсам, отображать статические ресурсы и присваивать строки ресурсов свойствам см. в разделах [Пошаговое руководство. Локализация приложений Windows Forms](http://msdn.microsoft.com/ru-ru/9a96220d-a19b-4de0-9f48-01e5d82679e5) и [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md).
