# <a name="new-toolbar-resource-dialog-box"></a>Создание ресурса панели инструментов - диалоговое окно
Диалоговое окно «Создание ресурса панели инструментов» позволяет задать ширину и высоту кнопок, добавляемых в ресурс панели инструментов. Значение по умолчанию — 16 x 15 пикселов.  
  
 Точечный рисунок, который используется для создания панели инструментов имеет максимальную ширину 2048. Поэтому, если задать **Ширина кнопки** 512, можно установить только четыре кнопки. Если задать ширину для 513, можно установить только три кнопки.  
  
 **Ширина кнопки**  
 Поле ввода ширины кнопок панели инструментов, преобразуемых из ресурса точечного рисунка панели инструментов. Изображения обрезаются к ширине и высоте и цвета корректируются для использования стандартной палитры панели инструментов (16 цветов).  
  
 **Высота кнопки**  
 Поле ввода высоты кнопок панели инструментов, преобразуемых из ресурса точечного рисунка панели инструментов. Изображения обрезаются к ширине и высоте и цвета корректируются для использования стандартной палитры панели инструментов (16 цветов).  
  
 Сведения о добавлении ресурсов в управляемые проекты см. в разделе [ресурсы в классических приложениях](/dotnet/framework/resources/index) в *руководства разработчика .NET Framework.* Сведения о вручную добавлять файлы ресурсов в управляемые проекты, осуществлять доступ к ресурсам, отображать статические ресурсы и присваивать строки ресурсов свойствам см. в разделе [Создание файлов ресурсов для приложений рабочего стола](/dotnet/framework/resources/creating-resource-files-for-desktop-apps). Сведения о глобализации и локализации ресурсов в управляемых приложениях см. в разделе [Globalizing и локализация приложений .NET Framework](/dotnet/standard/globalization-localization/index).  
  
## <a name="requirements"></a>Требования  
 MFC или ATL
