# <a name="new-toolbar-resource-dialog-box"></a>Создание ресурса панели инструментов - диалоговое окно
**Создание ресурса панели инструментов** диалоговое окно позволяет указать ширину и высоту кнопки, при добавлении панели инструментов. Значение по умолчанию — 16 х 15 пикселей.  
  
 Битовая карта, используется для создания панели инструментов имеет максимальную ширину 2048. Поэтому, если задать **Ширина кнопки** 512, может быть только четыре кнопки. Если вам для настройки ширины 513, может иметь только три кнопки.  
  
### <a name="button-width"></a>Ширина кнопки  
 Место для ввода ширину для кнопок панели инструментов, при преобразовании ресурса точечного рисунка панели инструментов. Изображения обрезаются до ширины и высоты, заданной и цвета корректируются для использования стандартной палитры панели инструментов (16 цветов).  
  
### <a name="button-height"></a>Высота кнопки  
 Место для ввода высоту для кнопок панели инструментов, при преобразовании ресурса точечного рисунка панели инструментов. Изображения обрезаются до ширины и высоты, заданной и цвета корректируются для использования стандартной палитры панели инструментов (16 цветов).  
  
 Сведения о добавлении ресурсов в управляемые проекты см. в разделе [ресурсы в приложениях для настольных систем](/dotnet/framework/resources/index) в *руководства разработчика .NET Framework*. Сведения о вручную добавлять файлы ресурсов в управляемые проекты, осуществлять доступ к ресурсам, отображать статические ресурсы и присваивать строки ресурсов свойствам, см. в разделе [Создание файлов ресурсов для приложений рабочего стола](/dotnet/framework/resources/creating-resource-files-for-desktop-apps). Сведения о глобализации и локализации ресурсов в управляемых приложениях, см. в разделе [Globalizing and Localizing .NET Framework Applications](/dotnet/standard/globalization-localization/index).  
  
## <a name="requirements"></a>Требования  
 MFC или ATL
