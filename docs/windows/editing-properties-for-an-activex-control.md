# <a name="editing-properties-for-an-activex-control"></a>Изменение свойств элемента управления ActiveX
Элементы управления ActiveX, предоставляемые независимыми поставщиками могут оснащены собственные свойства и характеристики. В окне «Свойства» отображаются свойства элементов управления ActiveX. Кроме того, отображаются все страницы свойств, созданные изготовителем элемента управления ActiveX в **страницы свойств** диалоговое окно (Чтобы просмотреть страницу свойств для конкретного элемента управления ActiveX, нажмите кнопку страницы свойств в [Окно свойств](/visualstudio/ide/reference/properties-window)).  
  
 Различные вкладки отображаются на странице свойств для элемента управления ActiveX, в зависимости от страницы свойств, которые поставляются в составе элемента управления ActiveX.  
  
> [!NOTE]
>  Следующая процедура применяется с помощью страницы свойств для редактирования элементов управления ActiveX. Можно также просмотреть и изменить свойства ActiveX в окне «Свойства».  
  
### <a name="to-edit-properties-for-an-activex-control"></a>Чтобы изменить свойства для элемента управления ActiveX  
  
1.  Выберите **ActiveX** элемента управления.  
  
2.  На **представление** меню, нажмите кнопку **страницу свойств** , чтобы просмотреть свойства.  
  
3.  При необходимости внесите изменения на странице свойств.  
  
 Сведения о добавлении ресурсов в управляемые проекты см. в разделе [ресурсы в классических приложениях](/dotnet/framework/resources/index) в *руководства разработчика .NET Framework.* Сведения о вручную добавлять файлы ресурсов в управляемые проекты, осуществлять доступ к ресурсам, отображать статические ресурсы и присваивать строки ресурсов свойствам см. в разделе [Создание файлов ресурсов для приложений рабочего стола](/dotnet/framework/resources/creating-resource-files-for-desktop-apps). Сведения о глобализации и локализации ресурсов в управляемых приложениях см. в разделе [Globalizing и локализация приложений .NET Framework](/dotnet/standard/globalization-localization/index).  
  
 Требования  
  
 Win32
