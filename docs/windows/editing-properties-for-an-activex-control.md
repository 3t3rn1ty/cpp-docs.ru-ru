# Editing Properties for an ActiveX Control
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Элементы управления ActiveX, предоставляемые независимыми поставщиками, могут поставляться с предварительно установленными свойствами и характеристиками.  Свойства элементов управления ActiveX отображаются в окне свойств.  Кроме того страницы свойств, созданные изготовителем элемента управления ActiveX отображаются в диалоговом окне **Страницы свойств** \(чтобы просмотреть страницу свойств для конкретного элемента управления ActiveX, нажмите кнопку "Страница свойств" в [окне свойств](../Topic/Properties%20Window.md)\).  
  
 На странице свойств элемента управления ActiveX отображаются разные закладки в зависимости от того, какие вкладки свойств были заданы изготовителем элемента управления ActiveX.  
  
> [!NOTE]
>  В следующем примере кода показано, каким образом можно изменить элемент управления ActiveX с помощью его страницы свойств.  Окно свойств также позволяет просматривать и изменять свойства элементов управления ActiveX.  
  
### Изменение свойств элемента управления ActiveX  
  
1.  Выберите элемент управления **ActiveX**.  
  
2.  В меню **Вид** выберите пункт **Страница свойств**, чтобы просмотреть свойства.  
  
3.  На странице свойств внесите необходимые изменения.  
  
 Сведения о добавлении ресурсов в управляемые проекты см. в разделе [Ресурсы приложений](../Topic/Resources%20in%20Desktop%20Apps.md) *Руководства разработчика .NET Framework*. Сведения о том, как вручную добавлять файлы ресурсов в управляемые проекты, осуществлять доступ к ресурсам, отображать статические ресурсы и присваивать строки ресурсов свойствам см. в разделах [Пошаговое руководство. Локализация приложений Windows Forms](http://msdn.microsoft.com/ru-ru/9a96220d-a19b-4de0-9f48-01e5d82679e5) и [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md).
