# C++ Attributes Reference
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Атрибуты разработаны для упрощения программирования COM и.NET Framework общего языка среды разработки.  При добавлении атрибутов в файлах исходного кода компилятор работает с поставщик библиотек DLL для вставки кода или изменения кода в файлах создаваемый объект.
