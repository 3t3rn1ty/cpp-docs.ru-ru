# <a name="changing-the-properties-of-a-resource"></a>Изменение свойств ресурса
### <a name="to-edit-the-properties-of-a-resource"></a>Изменение свойств ресурса  
  
1.  В [представление ресурсов](../windows/resource-view-window.md), щелкните правой кнопкой мыши ресурс, который нужно изменить и выберите **свойства** в контекстном меню.  
  
     **Примечание** Если проект еще не содержит RC-файл, см. статью [Создание нового файла описания ресурсов](../windows/how-to-create-a-resource-script-file.md).  
  
2.  В [окно свойств](/visualstudio/ide/reference/properties-window), измените свойства ресурса.  
  
### <a name="to-undo-a-change-youve-made-to-the-properties-of-a-resource"></a>Отмена изменений, внесенных в свойства ресурса  
  
1.  Убедитесь, что ресурс имеет фокус в представлении ресурсов.  
  
2.  Выберите **отменить** из **изменить** меню.  
  

  
 **Требования**  
  
 Нет
