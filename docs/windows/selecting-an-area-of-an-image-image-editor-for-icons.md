# <a name="selecting-an-area-of-an-image-image-editor-for-icons"></a>Выбор области изображения (редактор изображений для значков)
Выбор средства можно использовать для определения области изображения, который требуется очистить, Вырезать, копировать, изменения размеров, Инверсия или переместить. С **прямоугольник выделения** средство можно определить и выбрать прямоугольной области изображения. С **произвольное выделение** средство можно нарисовать произвольной структуры области, нужно вырезать, копировать и другие операции.  
  
> [!NOTE]
>  В разделе **прямоугольник выделения** и **произвольное выделение** средства на рисунке в [панель инструментов редактора изображений](../windows/toolbar-image-editor-for-icons.md) или просмотр всплывающие подсказки, связанный с каждой кнопкой **Редактора изображений** инструментов.  
  
 Можно также создать настраиваемую кисть из выделенной области. Дополнительные сведения см. в разделе [Создание настраиваемых кисти](../windows/creating-a-custom-brush-image-editor-for-icons.md).  
  
### <a name="to-select-an-area-of-an-image"></a>Чтобы выбрать область изображения  
  
1.  На **редактора изображений** панель инструментов (или из **изображения** меню **средства** команды), щелкните нужный инструмент выделения.  
  
2.  Переместите курсор на угол области изображения, которое вы хотите выбрать. Перекрестие отображаются, когда курсор находится над изображением.  
  
3.  Перетащите курсор противоположного угла области, которую вы хотите выбрать. Прямоугольник показывает, будут выбраны какие пикселей. Все точки внутри прямоугольника, включая под прямоугольником, включенные в выделенном фрагменте.  
  
4.  Отпустите кнопку мыши. Граница выделения заключает выбранной области.  
  
### <a name="to-select-an-entire-image"></a>Чтобы выбрать всего изображения  
  
1.  Щелкните изображение за пределами текущего выбора. Граница выделения перемещении фокуса и охватывает все изображение в целом еще раз.
