# <a name="converting-bitmaps-to-toolbars"></a>Преобразование растровых изображений в панель инструментов
Можно создать новую панель инструментов путем преобразования растрового изображения. График из растрового изображения преобразует изображения кнопок для панели инструментов. Обычно растровое изображение содержит несколько изображений кнопок отдельной изображением для каждой кнопки. Изображения могут быть любого размера; значение по умолчанию — 16 пикселей в ширину и высоту изображения. Можно указать размер изображения кнопок в [Создание ресурса панели инструментов-диалоговое окно](../windows/new-toolbar-resource-dialog-box.md) при выборе редактор панелей инструментов из **изображения** меню в редакторе изображений.  
  
### <a name="to-convert-bitmaps-to-a-toolbar"></a>Преобразование растровых изображений в панель инструментов  
  
1.  Откройте существующий ресурс растрового изображения в [редактора изображений](../windows/image-editor-for-icons.md). (Если в RC-файле еще нет растрового изображения, щелкните правой кнопкой мыши RC-файл, выберите **импорта** в контекстном меню, перейдите к растровому изображению, нужно добавить в RC-файл, а затем нажмите кнопку **откройте**.)  
  
2.  Из **изображения** меню, выберите **редактор панелей инструментов**.  
  
     [Создание ресурса панели инструментов-диалоговое окно](../windows/new-toolbar-resource-dialog-box.md) отображается. Можно изменить ширину и высоту изображения значков для сопоставления растрового изображения. Изображение кнопки панели инструментов отображается в редакторе панелей инструментов.  
  
3.  Чтобы завершить преобразование, измените команду **идентификатор** кнопки с помощью [окно свойств](/visualstudio/ide/reference/properties-window). Введите новое **идентификатор** или выберите **идентификатор** из раскрывающегося списка.  
  
    > [!TIP]
    >  Окно свойств содержится кнопка в строке заголовка. При нажатии этой кнопки включает или отключает автоматическое скрытие окна. Чтобы быстро пролистать все свойства кнопок панели инструментов без необходимости повторно открыть отдельное свойство windows, отключите автоматическое скрытие, окно свойств остается неподвижным.  
  
 Можно также изменять идентификаторы команд кнопок на новую панель инструментов с помощью [окно свойств](/visualstudio/ide/reference/properties-window). Сведения о редактировании новой панели инструментов в разделе [создание, перемещение и редактирование кнопок панели инструментов](../windows/creating-moving-and-editing-toolbar-buttons.md).  
  
 Сведения о добавлении ресурсов в управляемые проекты см. в разделе [ресурсы в классических приложениях](/dotnet/framework/resources/index) в *руководства разработчика .NET Framework.* Сведения о вручную добавлять файлы ресурсов в управляемые проекты, осуществлять доступ к ресурсам, отображать статические ресурсы и присваивать строки ресурсов свойствам см. в разделе [Создание файлов ресурсов для приложений рабочего стола](/dotnet/framework/resources/creating-resource-files-for-desktop-apps). Сведения о глобализации и локализации ресурсов в управляемых приложениях см. в разделе [Globalizing и локализация приложений .NET Framework](/dotnet/standard/globalization-localization/index).  
  
 Требования  
  
 MFC или ATL
