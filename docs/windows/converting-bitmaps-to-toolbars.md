# Converting Bitmaps to Toolbars
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Панели инструментов можно создавать путем преобразования растровых изображений.  Рисунки на растровом изображении становятся изображениями кнопок в панели инструментов.  Как правило, растровое изображение содержит несколько изображений кнопок, каждое из которых становится изображением отдельной кнопки.  Размер изображения может быть любым. Стандартными считаются размеры 16 пикселей в высоту на 16 пикселей в ширину.  Размер изображения кнопок можно задать в диалоговом окне [Создание ресурса панели инструментов](../mfc/new-toolbar-resource-dialog-box.md) после того, как в меню **Изображение** в редакторе изображений был выбран редактор панелей инструментов.  
  
### Преобразование растровых изображений в панель инструментов  
  
1.  Откройте существующий ресурс растрового изображения в [редакторе изображений](../mfc/image-editor-for-icons.md).  \(Если в RC\-файле еще нет растрового изображения, щелкните RC\-файл правой кнопкой мыши, выберите команду **Импорт** в контекстном меню, перейдите к растровому изображению, которое необходимо добавить в RC\-файл, а затем щелкните **Открыть**.\)  
  
2.  В меню **Изображение** выберите пункт **Редактор панелей инструментов**.  
  
     Откроется диалоговое окно [Создание ресурса панели инструментов](../mfc/new-toolbar-resource-dialog-box.md).  Ширину и высоту значка можно изменить таким образом, чтобы размеры значка соответствовали размерам растрового изображения.  В редакторе панелей инструментов появится изображение панели инструментов.  
  
3.  Чтобы завершить преобразование, измените в кнопке **идентификатор** команды с помощью [окна свойств](../Topic/Properties%20Window.md).  Введите новый **идентификатор** или выберите **идентификатор** из раскрывающегося списка.  
  
    > [!TIP]
    >  В заголовке окна свойств содержится кнопка.  С помощью этой кнопки можно включить или отключить функцию автоматического скрытия окна.  Чтобы ускорить работу со свойствами кнопки, отключите функцию автоматического скрытия. Тогда окно свойств будет отображаться постоянно, и вам не придется открывать его заново каждый раз, когда потребуется изменить то или иное свойство.  
  
 С помощью [окна свойств](../Topic/Properties%20Window.md) можно также изменять идентификаторы команд в кнопках.  Дополнительные сведения о редактировании новой панели инструментов см. в разделе [Создание, перемещение и редактирование кнопок в панели инструментов](../mfc/creating-moving-and-editing-toolbar-buttons.md).  
  
 Сведения о добавлении ресурсов в управляемые проекты см. в разделе [Ресурсы приложений](../Topic/Resources%20in%20Desktop%20Apps.md) *Руководства разработчика .NET Framework*. Сведения о том, как вручную добавлять файлы ресурсов в управляемые проекты, осуществлять доступ к ресурсам, отображать статические ресурсы и присваивать строки ресурсов свойствам см. в разделах [Пошаговое руководство. Локализация приложений Windows Forms](http://msdn.microsoft.com/ru-ru/9a96220d-a19b-4de0-9f48-01e5d82679e5) и [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md).
