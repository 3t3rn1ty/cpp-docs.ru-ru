# Adding Values to a Combo Box Control
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Пока открыт редактор диалоговых окон, можно добавить значения в элемент управления поля со списком.  
  
> [!TIP]
>  Рекомендуется добавлять все значения в поле со списком *до* задания размера поля в редакторе диалоговых окон, либо часть текста, которая должна появляться в этом элементе, может быть обрезана.  
  
#### Ввод значений в элемент управления поля со списком  
  
1.  Выберите элемент управления поля со списком, щелкнув его.  
  
2.  В [окне "Свойства"](../Topic/Properties%20Window.md) выполните прокрутку вниз до свойства **Данные**.  
  
    > [!NOTE]
    >  При отображении свойств, сгруппированных по типу, свойство **Данные** появляется в свойствах **Прочие**.  
  
3.  Щелкните в области значений для свойства **Данные** и введите нужные значения данных, разделенные точками с запятой.  
  
    > [!NOTE]
    >  Не вставляйте пробелы между значениями, так как пробелы влияют на сортировку в алфавитном порядке в раскрывающемся списке.  
  
4.  Закончив добавление значений, нажмите кнопку **Ввести**.  
  
 Сведения об увеличении раскрывающейся части поля со списком см. в разделе [Задание размера поля со списком и его раскрывающегося списка](../mfc/setting-the-size-of-the-combo-box-and-its-drop-down-list.md).  
  
> [!NOTE]
>  С помощью этой процедуры нельзя добавлять значения в проекты Win32 \(свойство **Данные** является недоступным для проектов Win32\).  Так как для проектов Win32 отсутствуют библиотеки, поддерживающие эту возможность, в проекте Win32 разработчик дожжен добавлять значения в поле со списком программным образом.  
  
#### Проверка внешнего вида значений в поле со списком  
  
1.  Введя значения в свойство **Данные**, нажмите кнопку **Проверить** на [панели инструментов редактора диалоговых окон](../mfc/showing-or-hiding-the-dialog-editor-toolbar.md).  
  
     Попробуйте прокрутить весь список значений.  Значения появляются в точности в том виде, как они были введены в свойстве **Данные** в окне "Свойства".  Проверка правописания и написание прописными буквами не используются.  
  
     Чтобы вернуться в редактор диалоговых окон нажмите клавишу ESC.
