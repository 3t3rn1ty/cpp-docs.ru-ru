# Selecting Controls
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Выделите элементы управления, с которыми нужно выполнить операции изменения размера, выравнивания, перемещения, копирования или удаления, а затем выполните нужные действия.  Чтобы воспользоваться средствами изменения размера и выравнивания на [панели инструментов редактора диалоговых окон](../mfc/showing-or-hiding-the-dialog-editor-toolbar.md), в большинстве случаев требуется выбрать несколько элементов управления.  
  
 При выделении элемента управления вокруг него появляется затененная граница со сплошными \(активными\) или пустыми \(неактивными\) "маркерами размера", небольшими квадратиками, появляющимися на границе выделения.  При выделении нескольких элементов управления маркеры размеров главного элемента управления оказываются сплошными, а маркеры размеров всех остальных выделенных элементов управления — пустыми.  
  
 При изменении размеров или выравнивании нескольких элементов управления в редакторе диалоговых окон для определения способа изменения размеров или выравнивания других элементов управления используется "главный элемент управления".  По умолчанию главным элементом управления становится первый выделенный элемент управления.
