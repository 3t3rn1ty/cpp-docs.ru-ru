# <a name="modifying-the-layout-grid"></a>Изменение сетки макета
При размещение и упорядочение элементов управления в диалоговом окне, можно использовать для более точного размещения сетки макета. При включении сетки элементы управления отображаются «привязки для» пунктирными линиями сетки как намагниченный. Можно включать эту функцию «привязать к сетке» и отключать и изменение размера ячеек сетки.  
  
### <a name="to-turn-the-layout-grid-on-or-off"></a>Чтобы выключать сетки макета  
  
1.  Из **формат** меню, выберите **параметры направляющих**.  
  
2.  В [диалоговое окно "Параметры направляющих"](../windows/guide-settings-dialog-box.md)установите или снимите **сетки** кнопки.  
  
     Можно управлять в сетку в отдельных окнах редактора окон **сетка** кнопку [инструментов редактора диалоговых окон](../windows/showing-or-hiding-the-dialog-editor-toolbar.md).  
  
### <a name="to-change-the-size-of-the-layout-grid"></a>Чтобы изменить размер сетки макета  
  
1.  Из **формат** меню, выберите **параметры направляющих**.  
  
2.  В [диалоговое окно "Параметры направляющих"](../windows/guide-settings-dialog-box.md), введите значения высоты и ширины в DLU ячеек в сетке. Минимальная высота или ширина — 4 DLU. Дополнительные сведения о единицах DLU см. в разделе [упорядочение элементов управления в диалоговых окнах](../windows/arrangement-of-controls-on-dialog-boxes.md).  
  
 Сведения о добавлении ресурсов в управляемые проекты см. в разделе [ресурсы в классических приложениях](/dotnet/framework/resources/index) в *руководства разработчика .NET Framework.* Сведения о вручную добавлять файлы ресурсов в управляемые проекты, осуществлять доступ к ресурсам, отображать статические ресурсы и присваивать строки ресурсов свойствам см. в разделе [Создание файлов ресурсов для приложений рабочего стола](/dotnet/framework/resources/creating-resource-files-for-desktop-apps). Сведения о глобализации и локализации ресурсов в управляемых приложениях см. в разделе [Globalizing и локализация приложений .NET Framework](/dotnet/standard/globalization-localization/index).
