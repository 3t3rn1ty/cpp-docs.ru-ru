# Creating a Submenu
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

### Создание подменю  
  
1.  Выберите команду меню, для которой требуется создать подменю.  
  
2.  В поле **Новый элемент**, которое появится справа, введите имя новой команды меню. Эта новая команда будет первой в подменю.  
  
3.  Добавьте остальные команды в подменю.  
  
 Сведения о добавлении ресурсов в проекты управляемого кода см. в разделе [Ресурсы приложений](../Topic/Resources%20in%20Desktop%20Apps.md)*Руководства разработчика .NET Framework*.  
  
 **Требования**  
  
 Win32
