# Finding an Entry in an Accelerator Table
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

### Поиск записи в открытой таблице сочетаний клавиш  
  
1.  Откройте таблицу сочетаний клавиш, дважды щелкнув ее значок в [представлении ресурсов](../windows/resource-view-window.md).  
  
    > [!NOTE]
    >  Если в проекте еще нет RC\-файлов, см. статью [Создание нового файла описания ресурсов](../windows/how-to-create-a-resource-script-file.md).  
  
2.  Щелкните заголовок столбца, чтобы отсортировать содержимое столбца по алфавиту.  Например, щелкните столбец **Идентификатор**, чтобы просмотреть все идентификаторы в таблице в алфавитном порядке.  
  
 Затем просмотрите список и найдите запись.  
  
 Сведения о добавлении ресурсов в управляемые проекты см. в разделе [Ресурсы приложений](../Topic/Resources%20in%20Desktop%20Apps.md) *Руководства разработчика .NET Framework*.  
  
 **Требования**  
  
 Win32
