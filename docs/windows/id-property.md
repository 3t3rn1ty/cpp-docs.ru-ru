# <a name="id-property"></a>Свойство "Идентификатор"
Свойство идентификатора ссылается на каждой записи таблицы сочетаний клавиш в программном коде. Это значение команды для программы, когда пользователь нажимает сочетание клавиш или сочетание клавиш. Чтобы сочетание клавиш таким же, как элемент меню, сделайте их идентификаторы одинаковыми (при условии, что идентификатор таблицы сочетаний клавиш является таким же, как идентификатор ресурса меню).  
  
## <a name="requirements"></a>Требования  
 Win32
