# Finding Binary Data
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Можно выполнять поиск либо строк ASCII, либо шестнадцатеричных байтов.  Например, чтобы найти строку "Hello" можно выполнить поиск либо строки "Hello", либо "48 65 6C 6C 6F" \(шестнадцатеричный эквивалент\).  
  
### Поиск двоичных данных  
  
1.  В меню **Правка** выберите пункт [Найти](../Topic/Find%20Command.md).  
  
2.  В поле **Найти** выберите предыдущую строку поиска из раскрывающегося списка или введите данные, которые нужно найти.  
  
3.  Выберите любой из параметров команды **Найти**.  
  
4.  Нажмите кнопку **Найти далее**.
