# <a name="creating-and-setting-guides-and-margins"></a>Создание и настройка полей и направляющих
Перемещение элементов управления, добавление элементов управления или изменять текущий макет, руководства помогут выравнивания элементов управления, точно в диалоговое окно. Направляющие представляют собой синие пунктирные линии через окно отображается в редакторе и соответствующий стрелки в линейки (в верхней и левой части редактора диалоговых окон).  
  
 При создании диалоговое окно приведены четыре поля. Поля являются разновидность направляющих, они синими пунктирными линиями.  
  
### <a name="to-create-a-guide"></a>Создание структуры  
  
1.  В области линейки щелкните один раз для создания структуры. (Одним щелчком мыши создает новое руководство, дважды щелкнув запускает [диалоговое окно "Параметры направляющих"](../windows/guide-settings-dialog-box.md) в котором можно указать параметры направляющих.)  
  
### <a name="to-set-a-guide"></a>Настройка направляющей  
  
1.  В диалоговом окне щелкните направляющую и перетащите его на новое место. (Также можно щелкнуть стрелку на линейке перетащите связанный руководства.)  
  
     Координаты структуры, отображаются в строке состояния в нижней части окна и на линейке. Переместите указатель над стрелкой в линейки, точное расположение этого руководства.  
  
### <a name="to-delete-a-guide"></a>Удаление направляющей  
  
1.  Перетащите направляющую в соответствующем диалоговом окне.  
  
 \- или -  
  
-   Перетащите соответствующую стрелку за пределы линейки.  
  
#### <a name="to-move-margins"></a>Перемещение полей  
  
1.  Перетащите поле в новое положение.  
  
     Чтобы удалить поле, перетащите его в нулевую позицию. Чтобы вернуть поля, наведите указатель на поле нулевой позиции и перетащите его в нужное место.  
  
 Сведения о добавлении ресурсов в управляемые проекты см. в разделе [ресурсы в классических приложениях](https://msdn.microsoft.com/library/f45fce5x.aspx) в *руководства разработчика .NET Framework.* Сведения о вручную добавлять файлы ресурсов в управляемые проекты, осуществлять доступ к ресурсам, отображать статические ресурсы и присваивать строки ресурсов свойствам см. в разделе [Создание файлов ресурсов для приложений рабочего стола](https://msdn.microsoft.com/library/xbx3z216.aspx). Сведения о глобализации и локализации ресурсов в управляемых приложениях см. в разделе [Globalizing и локализация приложений .NET Framework](https://msdn.microsoft.com/library/h6270d0z.aspx).
