# <a name="load-palette-colors-dialog-box-image-editor-for-icons"></a>Диалоговое окно "Загрузка цветов палитры" (редактор изображений для значков)
Используйте это диалоговое окно предназначено для загрузки специальные цветовые палитры для использования в проекте. Дополнительные сведения см. в разделе [сохранение и загрузка различных цветовых палитр](../windows/saving-and-loading-different-color-palettes-image-editor-for-icons.md).  
  
 **Искать в**  
 Указывает расположение, где вы хотите найти файл или папку. Нажмите кнопку со стрелкой, чтобы выбрать другое расположение, или щелкните значок папки на панели инструментов для перемещения вверх уровней.  
  
 **Имя файла**  
 Поле для ввода имени файла, который требуется открыть. Чтобы быстро найти файл, который ранее открыт, щелкните имя файла, в раскрывающемся списке, если он доступен.  
  
 Если для поиска файла, может служить подстановочные знаки звездочки (*). Например, можно ввести \*.\* для просмотра списка всех файлов. Можно также ввести полный путь файла, например, C:\My Documents\MyColorPalette.pal или \\\NetworkServer\MyFolder\MyColorPalette.pal.  
  
 **Тип файлов**  
 Список типов файлов для отображения. Палитры (* .pal) является типом файла по умолчанию для палитры цветов.  
  
## <a name="requirements"></a>Требования  
 Нет
