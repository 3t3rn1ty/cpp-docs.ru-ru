# <a name="flipping-an-image-image-editor-for-icons"></a>Отражение изображения (редактор изображений для значков)
Можно отразить или повернуть изображение:  
  
-   Создание зеркального отображения исходного.  
  
-   Включите изображение сверху вниз.  
  
-   Поворот изображения вправо на 90 градусов по одному.  
  
### <a name="to-flip-the-image-horizontally-mirror-image"></a>Отражение изображения по горизонтали (зеркальное отображение)  
  
1.  Из **изображения** меню, выберите **Отразить слева направо**.  
  
### <a name="to-flip-the-image-vertically-turn-upside-down"></a>Отражение по вертикали (включить сверху вниз)  
  
1.  Из **изображения** меню, выберите **Отразить сверху**.  
  
### <a name="to-rotate-the-image-90-degrees"></a>Поворот изображения на 90 градусов  
  
1.  Из **изображения** меню, выберите **Повернуть на 90 градусов**.  
  
    > [!NOTE]
    >  Можно также использовать [сочетаний клавиш](../windows/accelerator-keys-image-editor-for-icons.md) для этих команд или получить доступ к командам контекстного меню (щелкните за пределами изображения в редакторе изображений).
