# <a name="flipping-an-image-image-editor-for-icons"></a>Отражение изображения (редактор изображений для значков)

Можно перевернуть или смены к изображению:

- Создания зеркального изображения исходного.

- Включите изображение сверху вниз.

- Повернуть на 90 градусов изображение справа за раз.

### <a name="to-flip-the-image-horizontally-mirror-image"></a>Перевернуть изображение по горизонтали (Зеркальное изображение)

1. Из **изображение** меню, выберите **Отразить слева направо**.

### <a name="to-flip-the-image-vertically-turn-upside-down"></a>Перевернуть изображение по вертикали (перевернуть вверх ногами)

1. Из **изображение** меню, выберите **Отразить сверху**.

### <a name="to-rotate-the-image-90-degrees"></a>Поворот изображения на 90 градусов

1. Из **изображение** меню, выберите **Повернуть на 90 градусов**.

   > [!NOTE]
   > Можно также использовать [сочетаний клавиш](../windows/accelerator-keys-image-editor-for-icons.md) для этих команд или получить доступ к командам контекстного меню (щелкните за пределами изображения в редакторе изображений).

Сведения о добавлении ресурсов в управляемые проекты см. в разделе [ресурсы в приложениях для настольных систем](/dotnet/framework/resources/index) в *руководства разработчика .NET Framework*. Сведения о вручную добавлять файлы ресурсов в управляемые проекты, осуществлять доступ к ресурсам, отображать статические ресурсы и присваивать строки ресурсов свойствам, см. в разделе [Создание файлов ресурсов для приложений рабочего стола](/dotnet/framework/resources/creating-resource-files-for-desktop-apps). Сведения о глобализации и локализации ресурсов в управляемых приложениях, см. в разделе [Globalizing and Localizing .NET Framework Applications](/dotnet/standard/globalization-localization/index).
