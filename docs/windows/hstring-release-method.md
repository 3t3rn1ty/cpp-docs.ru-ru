# Метод HString::Release
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Удаляет базовое строковое значение и инициализирует текущий объект HString пустым значением.  
  
## Синтаксис  
  
```  
void Release() throw()  
```  
  
## Требования  
 **Заголовок:** corewrappers.h  
  
 **Пространство имен:** Microsoft::WRL::Wrappers
