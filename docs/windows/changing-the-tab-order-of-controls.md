# Changing the Tab Order of Controls
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Последовательность табуляции — это последовательность, в которой фокус ввода при нажатии клавиши TAB перемещается от одного элемента управления к другому в рамках диалогового окна.  Как правило, последовательность табуляции обеспечивает перемещение в направлении слева направо и сверху вниз.  У каждого элемента управления есть свойство **Tabstop**, которое определяет, получает ли данный элемент управления фокус.  
  
### Установка фокуса ввода для элемента управления  
  
1.  В [окне свойств](../Topic/Properties%20Window.md) выберите значение **ИСТИНА** или **ЛОЖЬ** для свойства **Tabstop**.  
  
 Даже элементы управления, в которых не нужно устанавливать значение ИСТИНА для свойства "Tabstop", должны занимать определенную позицию в последовательности табуляции.  Это может понадобиться например, в процессе [определения клавиш быстрого доступа](../mfc/defining-mnemonics-access-keys.md) для элементов управления, у которых нет заголовков.  Статический текст, в котором содержится клавиша быстрого доступа, связанная с элементом управления, должен быть расположен непосредственно перед данным элементом управления в последовательности табуляции.  
  
> [!NOTE]
>  Если в диалоговом окне есть элементы управления, перекрывающие друг друга, изменение последовательности табуляции может повлиять на отображение таких элементов управления.  Элементы управления, которые занимают позицию в конце последовательности, всегда отображаются поверх элементов управления, которые предшествуют им в последовательности табуляции.  
  
#### Просмотр последовательности табуляции для всех элементов управления в диалоговом окне  
  
1.  В меню **Формат** выберите пункт **Последовательность табуляции**.  
  
 \- или \-  
  
-   Нажмите сочетание клавиш CTRL\+D.  
  
#### Изменение последовательности табуляции для всех элементов управления в диалоговом окне  
  
1.  В меню **Формат** выберите пункт **Последовательность табуляции**.  
  
     В левом верхнем углу каждого элемента управления отображается число, обозначающее его позицию в последовательности табуляции.  
  
2.  Чтобы задать последовательность табуляции, щелкните все элементы управления в том порядке, в котором должен перемещаться фокус при нажатии клавиши TAB.  
  
3.  Нажмите клавишу **ВВОД**, чтобы выйти из режима **Последовательности табуляции**.  
  
    > [!TIP]
    >  Отключить возможность изменения последовательности табуляции в режиме "Последовательность табуляции" можно как с помощью клавиши ВВОД, так и с помощью клавиши ESC.  
  
#### Изменение последовательности табуляции для нескольких элементов управления  
  
1.  В меню **Формат** выберите пункт **Последовательность табуляции**.  
  
2.  Укажите позицию, с которой необходимо начать изменение.  Для этого нажмите клавишу **CTRL** и, удерживая ее, щелкните элемент управления непосредственно перед тем элементом управления, позицию которого необходимо изменить.  
  
     Например, если в последовательности табуляции требуется изменить позиции элементов управления с 7 по 9, то при нажатой клавише CTRL необходимо будет щелкнуть элемент управления 6.  
  
    > [!NOTE]
    >  Чтобы присвоить определенному элементу управления в первую позицию в последовательности табуляции, необходимо дважды щелкнуть этот элемент управления.  
  
3.  Отпустите клавишу CTRL, а затем щелкните все элементы управления в том порядке, в котором должен перемещаться фокус при нажатии клавиши TAB.  
  
4.  Нажмите клавишу **ВВОД**, чтобы выйти из режима **Последовательности табуляции**.
