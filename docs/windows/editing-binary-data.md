# Editing Binary Data
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

### Редактирование ресурса в двоичном редакторе  
  
1.  Выберите байт, который нужно изменить.  
  
     Клавиша TAB перемещает фокус между шестнадцатеричным разделом и ASCII\-разделом двоичного редактора.  Для перемещения по ресурсу на один экран можно использовать клавиши PAGE UP и PAGE DOWN.  
  
2.  Введите новое значение.  
  
     Значение немедленно изменяется как в шестнадцатеричном, так и в ASCII\-разделе, и фокус перемещается к следующему значению в строке.  
  
    > [!NOTE]
    >  Двоичный редактор автоматически применяет изменения при закрытии редактора.
