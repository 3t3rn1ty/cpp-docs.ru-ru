# <a name="drawing-lines-or-closed-figures-image-editor-for-icons"></a>Рисование линий и замкнутых фигур (редактор изображений для значков)
Редактор изображений средства для рисования линий и замкнутых фигур все работать так же: Поместите курсор в одной точке и перетащите в другую. Для строк эти точки являются конечными точками. Эти точки замкнутых фигур: противоположные углы прямоугольника, ограничивающего прямоугольника.  
  
 По ширине, определяется в текущем выделенном фрагменте кисти рисуются линии и пустых замкнутых фигур рисуются с шириной, определяется в текущем выделенном фрагменте ширины. Линии и фигуры, как в рамке и заполняется, отображаются в текущий цвет переднего плана при нажатии левой кнопки мыши, или текущий цвет фона при нажатии правой кнопки мыши.  
  
### <a name="to-draw-a-line"></a>Чтобы нарисовать линию  
  
1.  На [панель инструментов редактора изображений](../windows/toolbar-image-editor-for-icons.md) (или из **изображения** меню **средства** команды), нажмите кнопку **строки** средства.  
  
2.  При необходимости выберите цвета и кисть:  
  
    -   В [палитра цветов](../windows/colors-window-image-editor-for-icons.md), щелкните левой кнопки мыши выберите цвет переднего плана, или правой кнопкой мыши выберите цвет фона.  
  
    -   В [селектор параметров](../windows/toolbar-image-editor-for-icons.md), щелкните форму, представляющую кисть, которая будет использоваться.  
  
3.  Поместите указатель в начале строки.  
  
4.  Перетащите конечную точку линии.  
  
### <a name="to-draw-a-closed-figure"></a>Чтобы нарисовать замкнутую фигуру  
  
1.  На **редактора изображений** инструментов (или из **изображения** меню **средства** команды), нажмите кнопку **Рисование замкнутой фигуры** средства.  
  
     **Рисование замкнутой фигуры** инструменты создания фигур, как указано на соответствующих им кнопках.  
  
2.  При необходимости выберите нужный цвет и толщину линии.  
  
3.  Указатель на один из углов прямоугольной области, в котором нужно нарисовать фигуру.  
  
4.  Перетащите указатель мыши по диагонали противоположного угла.
