# <a name="open-ltdevicegt-image-dialog-box-image-editor-for-icons"></a>Откройте &lt;устройства&gt; изображения диалоговое окно (редактор изображений для значков)
Используйте это диалоговое окно предназначено для открытия изображений устройств. В ней перечисляются существующие изображения устройств в текущем ресурсе (изображения, которые являются частью текущего ресурса).  
  
 **Текущие изображения**  
 Список изображений, включенных в ресурс. Выберите тип образа, который требуется открыть.  
  
## <a name="requirements"></a>Требования  
 Нет
