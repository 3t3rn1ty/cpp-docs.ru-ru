# Классические приложения Windows (C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Ознакомившись с интегрированной средой разработки Visual Studio и приложениями командной строки, перейдем к рассмотрению способов создания приложений Windows.  Благодаря Visual C\+\+ можно создавать приложения Windows с использованием различных технологий, например [Windows API](https://msdn.microsoft.com/en-us/library/cc433218.aspx) \(называемой также Win32 API\) и .NET Framework.  
  
 В этом разделе будет рассмотрено создание двух простых приложений Windows с использованием Win32 API и .NET Framework.  Также будет рассмотрено создание элемента управления Windows Forms с помощью .NET Framework, и в завершение будет создано простейшее игровое приложение с использованием DirectX.  
  
## Обязательные компоненты  
 Эти разделы предполагают знание основ языка C\+\+.
