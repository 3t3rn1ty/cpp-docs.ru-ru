# Editing Graphical Resources (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Графические ресурсы — это изображения, определяемые разработчиком в своем приложении.  Можно рисовать от руки или с помощью фигур.  Можно выделить часть изображения для редактирования, отражения или изменения размера либо можно создать настраиваемую кисть из выделенной части изображения и использовать эту кисть для рисования.  Можно определять свойства изображения, сохранять изображения в различных форматах и преобразовывать изображения из одного формата в другой.  
  
 Кроме создания новых графических ресурсов, можно для редактирования [импортировать существующие изображения](../windows/how-to-import-and-export-resources.md), а затем использовать их в своем проекте.  Можно также открыть и изменить изображения, не являющееся частью проекта, чтобы [редактировать отдельное изображение](../Topic/Editing%20an%20Image%20Outside%20of%20a%20Project%20\(Image%20Editor%20for%20Icons\).md).
