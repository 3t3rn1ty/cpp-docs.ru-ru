# Отображение или скрытие окна &quot;Цвета&quot; (редактор изображений для значков)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

### Отображение окна "Цвета"  
  
1.  Щелкните правой кнопкой мыши область **редактора изображений** и выберите в контекстном меню команду **Показать окно "Цвета"**.  
  
     \- или \-  
  
2.  Щелкните команду **Показать окно "Цвета"** в [меню "Изображение"](../mfc/image-menu-image-editor-for-icons.md).  
  
### Скрытие окна "Цвета"  
  
1.  Снимите закрепление окна.  Это позволит окну автоматически скрываться, когда оно не используется.  Дополнительные сведения см. в разделе [Упорядочивание окон](../misc/how-to-arrange-and-dock-windows.md).  
  
 \- или \-  
  
-   Нажмите кнопку **Закрыть**.  
  
 Требования  
  
 None
