# Accelerator Keys for the Dialog Editor
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Ниже перечислены стандартные сочетания клавиш для команд редактора диалоговых окон.  Чтобы изменить сочетания клавиш, в меню **Сервис** щелкните **Параметры**, а затем щелкните **Клавиатура** в папке **Среда**.  Дополнительные сведения см. в разделе [Определение и настройка сочетаний клавиш](../Topic/Identifying%20and%20Customizing%20Keyboard%20Shortcuts%20in%20Visual%20Studio.md).  
  
> [!NOTE]
>  Доступные в диалоговых окнах параметры, а также названия и расположение команд меню могут отличаться от описанных в справке в зависимости от текущих параметров или версии.  Чтобы изменить параметры, в меню **Сервис** выберите команду **Импорт и экспорт параметров**.  Дополнительные сведения см. в разделе [Customizing Development Settings in Visual Studio](http://msdn.microsoft.com/ru-ru/22c4debb-4e31-47a8-8f19-16f328d7dcd3).  
  
|Command|Разделы|Описание|  
|-------------|-------------|--------------|  
|Format.AlignBottoms|CTRL \+ SHIFT \+ СТРЕЛКА ВНИЗ|Выравнивание нижних границ элементов управления по нижней границе главного элемента управления|  
|Format.AlignCenters|SHIFT \+ F9|Выравнивание элементов управления вдоль вертикальной оси, проходящей через центральную точку главного элемента управления|  
|Format.AlignLefts|CTRL \+ SHIFT \+ СТРЕЛКА ВЛЕВО|Выравнивание левых границ элементов управления по левой границе главного элемента управления|  
|Format.AlignMiddles|F9|Выравнивание элементов управления вдоль горизонтальной оси, проходящей через центральную точку главного элемента управления|  
|Format.AlignRights|CTRL \+ SHIFT \+ СТРЕЛКА ВПРАВО|Выравнивание правых границ элементов управления по правой границе главного элемента управления|  
|Format.AlignTops|CTRL \+ SHIFT \+ СТРЕЛКА ВВЕРХ|Выравнивание верхних границ элементов управления по верхней границе главного элемента управления|  
|Format.ButtonBottom|CTRL \+ B|Размещение выбранных кнопок вдоль нижнего края по центру диалогового окна|  
|Format.ButtonRight|CTRL \+ R|Размещение выбранных кнопок в правом верхнем углу диалогового окна|  
|Format.CenterHorizontal|CTRL \+ SHIFT \+ F9|Выравнивание элементов управления по горизонтали относительно центра диалогового окна|  
|Format.CenterVertical|CTRL \+ F9|Выравнивание элементов управления по вертикали относительно центра диалогового окна|  
|Format.CheckMnemonics|CTRL \+ M|Проверка уникальности назначенных клавиш быстрого доступа|  
|Format.SizeToContent|SHIFT \+ F7|Изменение размеров выбранных элементов управления, чтобы уместился текст заголовков|  
|Format.SpaceAcross|ALT \+ СТРЕЛКА ВЛЕВО|Выравнивание интервалов между выбранными элементами управления, размещенными по горизонтали|  
|Format.SpaceDown|ALT \+ СТРЕЛКА ВНИЗ|Выравнивание интервалов между выбранными элементами управления, размещенными по вертикали|  
|Format.TabOrder|CTRL \+ D|Определение порядка следования элементов управления в диалоговом окне|  
|Format.TestDialog|CTRL \+ T|Запуск диалогового окна для оценки внешнего вида и проверки поведения|  
|Format.ToggleGuides|CTRL \+ G|Переключение между режимами разметки в редакторе диалоговых окон: сетка выключена, направляющие, сетка включена|
