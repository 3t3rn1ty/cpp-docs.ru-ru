# Selecting Foreground or Background Colors (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Все кроме **ластика** инструменты на панели инструментов **редактора изображений** рисуют с использованием текущего цвета переднего плана или фонового цвета при нажатии левой или правой кнопки мыши соответственно.  
  
### Выбор цвета переднего плана  
  
1.  Щелкните левой кнопки мыши нужный цвет в [палитре цветов](../Topic/Colors%20Window%20\(Image%20Editor%20for%20Icons\).md).  
  
### Выбор цвета фона  
  
1.  Щелкните правой кнопки мыши нужный цвет в **палитре цветов**.  
  
 Сведения о добавлении ресурсов в управляемые проекты см. в разделе [Ресурсы приложений](../Topic/Resources%20in%20Desktop%20Apps.md) *Руководства разработчика .NET Framework*.

 Нет
