# Adding, Editing, or Deleting Controls
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

С помощью редактора диалоговых окон можно добавлять, изменять размер, редактировать и удалять элементы управления в диалоговых окнах.  Можно также изменять свойства элемента управления, такие как его идентификатор или возможность его отображения во время выполнения.  
  
 Можно использовать контекстное меню редактора диалоговых окон для быстрого добавления зарегистрированного элемента управления ActiveX в диалоговое окно; можно также добавить элементы управления ActiveX на панель элементов для осуществления быстрого доступа \(дополнительные сведения см. в разделе [Настройка диалогового окна "Панель элементов"](http://msdn.microsoft.com/ru-ru/bd07835f-18a8-433e-bccc-7141f65263bb)\).
