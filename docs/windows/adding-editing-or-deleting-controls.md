# <a name="adding-editing-or-deleting-controls"></a>Добавление, изменение и удаление элементов управления
С помощью редактора диалоговых окон, можно добавить, изменить размер, изменить и удалить элементы управления в диалоговых окнах. Можно также изменить свойства элемента управления, например его идентификатор, или это изначально является видимым во время выполнения.  
  
 Для быстрого добавления зарегистрированных элементов управления ActiveX в диалоговое окно можно использовать контекстное меню в редакторе диалоговых окон и элементов управления ActiveX можно добавить на панель инструментов для быстрого доступа (Дополнительные сведения см. в разделе [элементов](http://msdn.microsoft.com/bd07835f-18a8-433e-bccc-7141f65263bb)).
