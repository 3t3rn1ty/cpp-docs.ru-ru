# <a name="creating-a-console-application"></a>Создание консольного приложения
С помощью мастера приложений Win32 является самым простым способом создания консольного приложения Win32.  
  
### <a name="to-create-a-win32-console-application"></a>Создание консольного приложения Win32  
  
1.  Следуйте инструкциям в разделе справки [Создание проекта с использованием мастера приложений Visual C++](../ide/creating-desktop-projects-by-using-application-wizards.md).  
  
2.  В **новый проект** выберите **проект Win32** в области шаблонов, чтобы открыть мастер.  
  
3.  Определение вашей [параметры приложения](../windows/application-settings-win-32-project-wizard.md) с помощью [мастер приложений Win32](../windows/win32-application-wizard.md).  
  
    > [!NOTE]
    >  Для сохранения параметров, заданных в мастере по умолчанию, пропустите этот шаг.  
  
4.  Нажмите кнопку **Готово** , чтобы закрыть мастер и только что созданный проект откроется в **обозревателе решений**.
