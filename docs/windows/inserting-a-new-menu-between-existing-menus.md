# <a name="inserting-a-new-menu-between-existing-menus"></a>Вставка нового меню между существующими меню
### <a name="to-insert-a-new-menu-between-existing-menus"></a>Вставка нового меню между существующими меню  
  
1.  Выберите имя существующего меню и нажмите клавишу **вставить** ключа. **Новый элемент** перед выбранным элементом будет вставлено поле.  
  
     \- или -  
  
2.  Щелкните правой кнопкой мыши в строке меню и выберите **вставить новый** в контекстном меню.  
  
 Сведения о добавлении ресурсов в управляемые проекты см. в разделе [ресурсы в классических приложениях](/dotnet/framework/resources/index) в *руководства разработчика .NET Framework.*  
  
 Требования  
  
 Win32
