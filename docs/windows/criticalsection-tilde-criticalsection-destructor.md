# <a name="criticalsectioncriticalsection-destructor"></a>Деструктор CriticalSection::~CriticalSection
Отменяет инициализацию и удаляет текущий объект CriticalSection.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
WRL_NOTHROW ~CriticalSection();  
```  
  
## <a name="requirements"></a>Требования  
 **Заголовок:** corewrappers.h  
  
 **Пространство имен:** Microsoft::wrl:: wrappers
