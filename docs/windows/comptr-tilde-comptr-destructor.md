# <a name="comptrcomptr-destructor"></a>Деструктор ComPtr::~ComPtr
Отменяет инициализацию экземпляра **ComPtr**.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
WRL_NOTHROW ~ComPtr();  
```  
  
## <a name="requirements"></a>Требования  
 **Заголовок:** client.h  
  
 **Пространство имен:** Microsoft::WRL
