# <a name="attribute-requirements"></a>Требования к атрибутам
Указанные для атрибутов C++ требования описаны типы проектов, параметры компилятора и другие сведения, необходимые для атрибута для работы. Ниже описаны категории сведений.  
  
> [!NOTE]
>  Использование атрибутов для класса, производного от класса, который также использует атрибуты не поддерживается.  
  
## <a name="header"></a>Header  
 Это поле содержит файлы заголовков, которые должны быть включены, перед тем как использовать атрибут.  
  
## <a name="project"></a>Проект  
 Это поле описывает типы проектов, в которых используется атрибут.  
  
## <a name="compiler"></a>Компилятор  
 Это поле включает параметры компилятора, которые должны быть созданы для этого атрибута для использования.
