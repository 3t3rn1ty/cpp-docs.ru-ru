# <a name="attribute-requirements"></a>Требования к атрибутам
Требования для атрибуты C++ описывают типы проектов, параметры компилятора и другие сведения, необходимые для атрибута для работы. Ниже описаны категории сведений.
  
> [!NOTE]
> С помощью атрибутов в класс, производный от класса, который также использует атрибуты не поддерживается.
  
## <a name="header"></a>Header
 В этом поле перечислены файлы заголовков, которые должны быть включены, прежде чем можно будет использовать атрибут.
  
## <a name="project"></a>Проект
 Это поле описывает типы проектов, в которых используется атрибут.
  
## <a name="compiler"></a>Компилятор
 Это поле содержит параметры компилятора, которые должны быть созданы для этого атрибута для использования.
