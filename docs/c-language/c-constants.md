# Константы в C
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

"Константа" — это число, символ или символьная строка, которую можно использовать в программе в качестве значения.  Константы служат для представления значений с плавающей запятой, целых чисел, перечислений или символов, изменение которых невозможно.  
  
## Синтаксис  
 `constant`:  
 *константа\-с\-плавающей запятой*  
  
 *целочисленная\-константа*  
  
 *константа\-перечисления*  
  
 *символьная\-константа*  
  
 Константы имеют значение и тип.  [Константы с плавающей запятой](../c-language/c-floating-point-constants.md), [целые константы](../Topic/C%20Character%20Constants.md) и [символьные константы](../Topic/C%20Integer%20Constants.md) рассматриваются в следующих трех разделах.  Константы перечисления рассматриваются в разделе [Объявления перечислений](../c-language/c-enumeration-declarations.md).
