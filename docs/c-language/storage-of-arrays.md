# Хранение массивов
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Хранилище, связанное с типом массива, является хранилищем, необходимым для всех его элементов.  Элементы массива хранятся в смежных адресах памяти в порядке увеличения от первого элемента к последнему.
