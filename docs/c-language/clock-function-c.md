# Функция clock (C)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.12.2.1** Эра для функции `clock`  
  
 Эра функции `clock` начинается \(со значения 0\) при запуске программы C на выполнение.  Она возвращает значения времени, измеренные в следующих единицах: 1\/**CLOCKS\_PER\_SEC** \(1\/1000 для Microsoft C\).
