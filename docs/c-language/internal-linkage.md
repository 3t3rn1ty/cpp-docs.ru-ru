# Внутренняя компоновка
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Если объявление идентификатора области видимости файла для объекта или функции содержит *спецификатор\-класса\-хранения* типа **static**, идентификатор имеет внутреннюю компоновку.  В противном случае идентификатор имеет внешнюю компоновку.  См. обсуждение нетерминального параметра [спецификатор\-класса\-хранения](../c-language/c-storage-classes.md) в разделе *Классы хранения*.  
  
 В пределах одной записи преобразования каждый экземпляр идентификатора с внутренней компоновкой обозначает тот же идентификатор или функцию.  Идентификаторы с внутренней компоновкой уникальны для конкретной записи преобразования.
