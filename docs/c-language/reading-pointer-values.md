# Считывание значений указателя
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.9.6.2** Входные данные для преобразования **%p** в функции `fscanf`  
  
 Когда символ формата **%p** определен, функция `fscanf` преобразует указатели из шестнадцатеричных значений ASCII в соответствующий адрес.
