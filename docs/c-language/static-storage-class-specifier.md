# Спецификатор класса хранения static
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Переменная, объявленная на внутреннем уровне с помощью описателя класса хранения **static**, имеет глобальное время жизни, но является видимой только внутри блока, в котором она объявлена.  В случае константных строк использование **static** полезно, поскольку при этом снимается нагрузка частой инициализации в часто вызываемых функциях.  
  
## Заметки  
 Если переменная **static** не инициализирована явно, она инициализируется значением 0 по умолчанию.  Внутри функции **static** выделяет хранилище и служит определением.  Внутренние статические переменные представляют закрытое постоянное хранилище, видимое только одной функции.
