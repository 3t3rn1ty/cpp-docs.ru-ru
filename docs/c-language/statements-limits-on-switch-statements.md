# Операторы: пределы операторов выбора
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.6.4.2** Максимальное число значений **case** в операторе `switch`  
  
 В Microsoft C количество значений **case** в операторе **switch** не ограничено.  Это число ограничивается только объемом доступной памяти.
