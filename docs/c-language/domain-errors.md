# Ошибки домена
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.5.1** Значения, возвращаемые математическими функциями в случае ошибок домена  
  
 Файл ERRNO.H определяет константу доменной ошибки `EDOM` как 33. См. сведения о возвращаемом значении в разделе справки, посвященном конкретной функции, вызвавшей ошибку.
