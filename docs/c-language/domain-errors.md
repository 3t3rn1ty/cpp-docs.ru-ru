# <a name="domain-errors"></a>Ошибки домена
**ANSI 4.5.1** Значения, возвращаемые математическими функциями в случае ошибок домена  
  
 Файл ERRNO.H определяет для константы ошибки домена `EDOM` значение 33.  См. сведения о возвращаемом значении в разделе справки, посвященном конкретной функции, вызвавшей ошибку.
