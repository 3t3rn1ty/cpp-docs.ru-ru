# Не представленные константы символов
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.1.3.4** Значение целочисленной символьной константы, которая содержит символ или escape\-последовательность, не представленную в базовой кодировке выполнения или расширенной кодировке для константы расширенных символов  
  
 Все символьные константы или escape\-последовательности можно представить в виде расширенной кодировки.
