# Операторы (C)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Операторы программы на языке C\+\+ управляют потоком выполнения программы.  В языке C, как и в других языках программирования, предусмотрено несколько типов операторов для выполнения циклов, для выбора других выполняемых операторов и для передачи управления.
