# Дата и время по умолчанию
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.8.8** Определения для \_DATE\_ и \_TIME\_ в случае недоступности даты и времени трансляции соответственно  
  
 Если операционная система не предоставляет дату и время трансляции, по умолчанию для \_DATE\_ и \_TIME\_ получают значения `May 03 1957` и `17:00:00"`.
