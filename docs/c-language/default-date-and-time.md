# <a name="default-date-and-time"></a>Дата и время по умолчанию
**ANSI 3.8.8** Определения для _DATE\_ и _TIME\_ в случае недоступности даты и времени преобразования  
  
 Если операционная система не предоставляет сведений о дате и времени преобразования _DATE\_ и _TIME\_ по умолчанию получают значения `May 03 1957` и `17:00:00"`.
