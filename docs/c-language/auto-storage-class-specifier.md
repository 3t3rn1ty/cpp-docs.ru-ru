# Спецификатор классов хранения auto
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Описатель класса хранения **auto** объявляет автоматическую переменную, то есть переменную с локальным временем жизни.  Переменная **auto** является видимой только в блоке, в котором она объявлена.  Объявления переменных **auto** могут включать инициализаторы, как описано в разделе [Инициализация](../c-language/initialization.md).  Поскольку переменные с классом хранения **auto** не инициализируются автоматически, необходимо явно инициализировать переменные при их объявлении или присвоить им начальные значения в операторах в блоке.  Значения неинициализированных переменных **auto** не определены. \(Локальная переменная класса хранения **auto** или **register** инициализируется при каждом вступлении в область, если указан инициализатор.\)  
  
 Внутреннюю переменную **static** \(статическую переменную с локальной областью или областью видимости блока\) можно инициализировать с адресом любого внешнего элемента или элемента **static**, но не с адресом другого элемента **auto**, поскольку адрес элемента **auto** не является константой.
