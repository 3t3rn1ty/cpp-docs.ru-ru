# <a name="default-signals"></a>Сигналы по умолчанию
**ANSI 4.7.1.1** Если эквивалент **signal (**<em>sig</em>**, SIG_DFL)** не выполняется до вызова обработчика сигнала, выполняется блокировка этого сигнала  
  
 В начале выполнения программы для сигналов задаются состояния по умолчанию.
