# Унарные операторы C
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Унарные операторы располагаются до операнда, и связь устанавливается справа налево.  
  
## Синтаксис  
 *унарное\-выражение*:  
 *постфиксное\-выражение*  
  
 **\+\+**  *унарное\-выражение*  
  
 `--`  *унарное\-выражение*  
  
 *унарный\-оператор выражение\-приведения*  
  
 **размер**  *унарного\-выражения*  
  
 **размер \(**  *имя\-типа*  **\)**  
  
 *унарный\-оператор*: один из  
 **&**  
 **\* \+ –** `~` **\!**
