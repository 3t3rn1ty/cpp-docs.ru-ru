# Спецификаторы классов хранения для объявлений внутреннего уровня
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Любой из четырех терминальных *спецификаторов\-класса\-хранения* можно использовать для объявления переменных на внутреннем уровне.  Если в таком объявлении не указан *спецификатор\-класса\-хранения*, по умолчанию используется класс хранения **auto**.  Поэтому ключевое слово **auto** редко встречается в программах на языке C.
