# Первичные выражения в C
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Операнды в выражениях называются основными выражениями.  
  
## Синтаксис  
 *основное\-выражение*:  
 *identifier*  
  
 *константа*  
  
 *строковый\-литерал*  
  
 **\(**  *выражение*  **\)**  
  
 *выражение*:  
 *выражение\-присваивания*  
  
 *выражение*  **,**  *выражение\-присваивания*
