# Общие сведения о строковых литералах
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

*строковый\-литерал*:  
 **'** *последовательность\-символов\-s*  необ. **'**  
  
 **L'** *последовательность\-символов\-s*  необ.             **'**  
  
 *последовательность\-символов\-s*:  
 *символ\-s*  
  
 *последовательность\-символов\-s символ\-s*  
  
 *символ\-s*:  
 любой член исходной кодировки, кроме *escape\-последовательности* двойной кавычки \("\), обратной косой черты \(\\\) и символа новой строки
