# Положение файла в режиме дозаписи
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.9.3** Исходное положение индикатора позиции файла потока в режиме добавления — в начале или в конце файла  
  
 Когда файл открывается в режиме добавления, исходное положение индикатора позиции файла указывает на конец файла.
