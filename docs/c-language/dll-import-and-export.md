# Импорт и экспорт DLL
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Блок, относящийся только к системам Microsoft**  
  
 Модификаторы класса хранения dllimport и dllexport — это расширения языка С, характерные для систем Microsoft.  Эти модификаторы определяют интерфейс DLL клиента \(исполняемый файл или другая библиотека DLL\).  Дополнительные сведения об использовании этих модификаторов см. в разделе [dllexport, dllimport](../cpp/dllexport-dllimport.md).
