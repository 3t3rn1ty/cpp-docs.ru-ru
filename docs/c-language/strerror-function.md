# <a name="strerror-function"></a>Функция strerror
**ANSI 4.11.6.2** Содержимое строк сообщения об ошибке, возвращаемых функцией `strerror`  
  
 Функция `strerror` создает следующие сообщения.  
  
```  
0   Error 0  
1    
2   No such file or directory  
3

5    
6    
7   Arg list too long  
8   Exec format error  
9   Bad file number  
10    
11    
12  Not enough core  
13  Permission denied  
14    
15    
16    
17  File exists  
18  Cross-device link  
19    
20    
21    
22  Invalid argument  
23    
24  Too many open files  
25

27    
28  No space left on device  
29    
30

33  Math argument  
34  Result too large  
35    
36  Resource deadlock would occur  
```
