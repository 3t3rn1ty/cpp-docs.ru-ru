# Верхний и нижний регистры
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.1.2** Учитываются ли различия регистров символов  
  
 Microsoft C обрабатывает идентификаторы в единице компиляции с учетом регистра.  
  
 Компоновщик Microsoft учитывает регистр.  Необходимо указывать все идентификаторы единообразно с учетом регистра.
