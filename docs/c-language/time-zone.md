[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.12.1** Местный часовой пояс и летнее время  
  
 В качестве местного часового пояса установлено стандартное тихоокеанское время.  Microsoft C поддерживает летнее время.
