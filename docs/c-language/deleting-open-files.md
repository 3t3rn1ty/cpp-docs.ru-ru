# <a name="deleting-open-files"></a>Удаление открытых файлов
**ANSI 4.9.4.1** Результат функции удаления для открытого файла  
  
 Функция удаления удаляет файл. Если файл открыт, эта функция не выполняется и возвращает значение -1.
