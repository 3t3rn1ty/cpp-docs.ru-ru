# Изменение существующего имени
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.9.4.2** Действия в случае, если файл с новым именем уже существует до вызова функции **rename**  
  
 При попытке переименовать файл с использованием уже существующего имени функция **rename** завершается неудачей и возвращает код ошибки.
