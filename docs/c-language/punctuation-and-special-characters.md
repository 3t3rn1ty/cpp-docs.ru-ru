# Знаки препинания и специальные символы
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Знаки пунктуации и специальные символы в наборе символов языка C имеют различные применения, от организации текста программы до определения задач, выполняемых компилятором или скомпилированной программой.  Эти знаки не определяют операции, подлежащие выполнению.  Некоторые знаки пунктуации также являются операторами \(см. раздел [Операторы](../c-language/c-operators.md)\).  Компилятор определяет их использование в зависимости от контекста.  
  
## Синтаксис  
 `punctuator`: один из следующих символов:  
 **\[ \]   \( \)   { }   \*   ,   :   \=   ;   ... \#**  
  
 Эти символы имеют специальное значение в языке C.  Их применение описывается в настоящей книге.  Знак решетки \(**\#**\) может использоваться только в [директивах препроцессора](../preprocessor/preprocessor-directives.md).
