# Константы в первичных выражениях
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Константный операнд имеет значение и тип того константного значения, которое он представляет.  Символьная константа имеет тип `int`.  Целочисленная константа имеет тип `int`, **long**, `unsigned int` или `unsigned long` \(в зависимости от размера целого числа и от способа указания значения\).  Дополнительные сведения см. в разделе [Константы](../c-language/c-constants.md).
