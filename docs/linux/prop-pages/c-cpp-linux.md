# <a name="cc-properties-linux-c"></a>Свойства C/C++ (Linux)

## <a name="general"></a>Общие
Свойство. | Описание: | Варианты
--- | ---| ---
Дополнительные каталоги включаемых файлов | Задает один или несколько каталогов для добавления в качестве пути включения; несколько значений разделяются точками с запятой. (-I[path]).
Формат отладочной информации | Определяет тип отладочной информации, создаваемой компилятором. | **Нет** — отладочная информация не создается для ускорения компиляции.<br>**Минимальные сведения об отладке** — создание минимальных сведений об отладке.<br>**Полная отладочная информация (DWARF2)** — создание отладочной информации DWARF2.<br>
Имя объектного файла | Задает имя, переопределяющее имя объектного файла по умолчанию; может быть именем файла или каталога. (-o [name]).
Уровень предупреждений | Выберите, как строго компилятор должен воспринимать ошибки в коде.  Другие флаги должны быть добавлены непосредственно в раздел дополнительных параметров. (/w, /Weverything). | **Отключить все предупреждения** — отключает все предупреждения компилятора.<br>**Включить все предупреждения** — включает все предупреждения, включая предупреждения, отключенные по умолчанию.<br>
Обрабатывать предупреждения как ошибки | Интерпретирует все предупреждения компилятора как ошибки. Для нового проекта рекомендуется использовать параметр /Werror во всех компиляциях; разрешение всех предупреждений обеспечит минимальное число трудновыявляемых дефектов кода.
Дополнительные предупреждения C | Определяет набор дополнительных предупреждений.
Дополнительные предупреждения C++ | Определяет набор дополнительных предупреждений.
Включить подробный режим | При включении подробного режима этот инструмент выводит более подробную информацию, чем для диагностики сборки.
Компилятор C | Задает программу, вызываемую при компиляции файлов исходного кода C, или путь к компилятору C в удаленной системе.
Компилятор C++ | Задает программу, вызываемую при компиляции файлов исходного кода C++, или путь к компилятору C++ в удаленной системе.
Время ожидания компиляции | Время ожидания удаленной компиляции (в миллисекундах).
Копировать объектные файлы | Указывает, следует ли копировать скомпилированные объектные файлы с удаленной системы на локальный компьютер.

## <a name="optimization"></a>Оптимизация
Свойство. | Описание: | Варианты
--- | ---| ---
Оптимизация | Указывает уровень оптимизации для приложения. | **Настраиваемый** — настраиваемая оптимизация.<br>**Отключено** — отключить оптимизацию.<br>**Минимизировать размер** — оптимизировать по размеру.<br>**Максимизировать скорость** — оптимизировать по скорости.<br>**Полная оптимизация** — дорогостоящие оптимизации.<br>
Строгое присвоение псевдонима | Использовать самые строгие правила присвоения псевдонима.  Объект одного типа никогда не будет считаться находящимся по тому же адресу, что и объект другого типа.
Развернуть циклы | Развернуть циклы, чтобы приложение работало быстрее благодаря уменьшению числа выполняемых ветвей за счет увеличения размера кода.
Оптимизация времени компоновки | Включить межпроцедурные оптимизации с помощью разрешения оптимизатору просматривать объектные файлы в приложении.
Опустить указатель на фреймы | Отменяет создание указателей на фреймы в стеке вызовов.
Нет общих блоков | Выделить даже не инициализированные глобальные переменные в разделе данных объектного файла, вместо того чтобы создавать их в виде общих блоков

## <a name="preprocessor"></a>Препроцессор
Свойство. | Описание: | Варианты
--- | ---| ---
Определения препроцессора | Определяет символы предварительной обработки в исходном файле. (-D)
Отменить определения препроцессора | Задает одну или несколько отмен определений препроцессора.  (-U [macro])
Отменить все определения препроцессора | Отменить все объявленные ранее определения препроцессора.  (-undef)
Показывать включаемые файлы | Создает список включаемых файлов с выводом компилятора.  (-H)

## <a name="code-generation"></a>Создание кода
Свойство. | Описание: | Варианты
--- | ---| ---
Позиционно-независимый код | Создать позиционно-независимый код (PIC) для использования в общей библиотеке.
Статические элементы являются потокобезопасными | Выделить дополнительный код для использования процедур, указанных в C++ ABI, для потокобезопасной инициализации локальных статических элементов. | **Нет** — отключение потокобезопасных статических элементов.<br>**Да** — включение потокобезопасных статических элементов.<br>
Оптимизация операций с плавающей запятой | Разрешает оптимизацию операций с плавающей запятой с помощью послабления требований к соответствию спецификации IEEE-754.
Встроенные методы скрыты | Когда включено, смещенные копии встроенных методов объявляются "частными внешними".
Символы скрыты по умолчанию | Все символы объявляются "частными внешними", кроме явным образом помеченных для экспорта с помощью макроса "__attribute".
Включить исключения C++ | Задает модель обработки исключений, используемую компилятором. | **Нет** — отключение обработки исключений.<br>**Да** — включение обработки исключений.<br>

--- | ---| ---
Включить информацию о типах времени выполнения | Добавляет код для проверки типов объектов C++ во время выполнения (информация о типах времени выполнения).     (frtti, fno-rtti)
Стандарт языка C | Определяет стандарт языка C. | **Default**<br>**C89** — стандарт языка C89.<br>**C99** — стандарт языка C99.<br>**C11** — стандарт языка C11.<br>**C99 (диалект GNU)** — стандарт языка C99 (диалект GNU).<br>**C11 (диалект GNU)** — стандарт языка C11 (диалект GNU).<br>
Стандарт языка C++ | Определяет стандарт языка C++. | **Default**<br>**C++03** — стандарт языка C++03.<br>**C++11** — стандарт языка C++11.<br>**C++14** — стандарт языка C++14.<br>**C++03 (диалект GNU)** — стандарт языка C++03 (диалект GNU).<br>**C++11 (диалект GNU)** — стандарт языка C++11 (диалект GNU).<br>**C++14 (диалект GNU)** — стандарт языка C++14 (диалект GNU).<br>

## <a name="advanced"></a>Дополнительно
Свойство. | Описание: | Варианты
--- | ---| ---
Компилировать как | Выбор языка компиляции C-файлов и CPP-файлов.  При использовании значения "по умолчанию" определение будет производиться по расширениям C и CPP. (-x c, -x c++) | **По умолчанию** — по умолчанию.<br>**Компилировать как код C** — компилировать как код C.<br>**Компилировать как код C++** — компилировать как код C++.<br>
Принудительно включенные файлы | Один или несколько принудительно включенных файлов (-include [имя])
