# <a name="vc-directories-linux-c"></a>Каталоги VC++ (Linux C++)

Свойство. | Описание:
--- | ---
Каталоги включения | Путь, используемый при поиске включаемых файлов во время сборки проекта VC++.  Соответствует переменной среды INCLUDE.
Каталоги библиотек | Путь, используемый при поиске файлов библиотек во время сборки проекта VC++.  Соответствует переменной среды LIB.
Каталоги исходного кода | Путь, который следует использовать при поиске файлов исходного кода, нужных для работы IntelliSense.
Исключаемые каталоги | Путь, который нужно пропустить при поиске зависимостей.
