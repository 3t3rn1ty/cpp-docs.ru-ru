# Поддержка многопоточности библиотеками
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Все версии библиотек CRT теперь поддерживают многопоточность за исключением неблокирующих версий некоторых функции.  Дополнительные сведения см. в разделе [Производительность многопотоковых библиотек](../../c-runtime-library/multithreaded-libraries-performance.md).  
  
 Дополнительные сведения о версиях библиотек CRT см. в разделе [Особенности библиотеки CRT](../../c-runtime-library/crt-library-features.md).
