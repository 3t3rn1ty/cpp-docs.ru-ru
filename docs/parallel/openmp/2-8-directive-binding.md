# 2.8 Directive Binding
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Динамическая привязка директив должна соответствовать следующим правилам:  
  
-   **для**"  **Разделы**"  **Одинарный**"  **Образец**и  **барьер** привязка директив динамически заключать  **Параллельно**, если он существует, независимо от значения любых  **If** предложение, которое может присутствовать на котором директивном.  Если параллельная область в данный момент не выполняется, то рекомендации выполняются командой структурной только главного потока.  
  
-   **Упорядочено** директива привязывает динамически заключать  **для**.  
  
-   **атомарный** директива обеспечивает монопольный доступ по отношению к  **атомарный** правила во всех потоках, а не только текущей рабочей группе.  
  
-   **Критические** директива обеспечивает монопольный доступ по отношению к  **Критические** правила во всех потоках, а не только текущей рабочей группе.  
  
-   Директива не может выполнить все директивным вне ближайший динамически заключать **Параллельно**.
