# <a name="266-ordered-construct"></a>2.6.6 Конструкция ordered
Следующие структурированный блок **упорядоченные** директива выполняются в порядке, в котором будет выполняться итераций в последовательном цикле. Синтаксис **упорядоченные** директивы таков:  
  
```  
#pragma omp ordered new-linestructured-block  
```  
  
 **Упорядоченные** директива должна быть внутри динамического экстент **для** или **параллельных для** построения. **Для** или **параллельных для** директивы, к которому **упорядоченные** конструкция привязки должен иметь **упорядоченные** предложение, указанное как описано в [Раздел 2.4.1](../../parallel/openmp/2-4-1-for-construct.md) на стр. В процессе выполнения **для** или **параллельных для** построения с **упорядоченные** предложение, **упорядоченные** конструкции, выполняются в строго порядок, в котором они будут выполняться в последовательного выполнения цикла.  
  
 Ограничения для **упорядоченные** директивы, следующим образом:  
  
-   Итерации цикла с **для** конструкция не может выполняться одной директиве упорядоченный более одного раза, и он не может выполняться несколько **упорядоченные** директивы.
