# OpenMP Data Types
[!INCLUDE[vs2017banner](../../../assembler/inline/includes/vs2017banner.md)]

Ссылки на типы данных, используемые в API модели OpenMP.  
  
 Реализация Visual C\+\+ стандарта OpenMP включает следующие типы данных.  
  
|Тип данных|Описание|  
|----------------|--------------|  
|[omp\_lock\_t](../../../parallel/openmp/reference/omp-lock-t.md)|Тип, в котором хранится состояние блокировки, доступна ли блокировка или если поток имеет блокировку.|  
|[omp\_nest\_lock\_t](../Topic/omp_nest_lock_t.md)|Тип, содержащий одну из следующих информации о блокировки: доступна ли блокировка и идентификаторы потока, который имеет блокировку и количество вложений.|
