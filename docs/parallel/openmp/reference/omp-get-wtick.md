# <a name="ompgetwtick"></a>omp_get_wtick

Возвращает количество секунд между тактов процессора.

## <a name="syntax"></a>Синтаксис

```
double omp_get_wtick( );
```

## <a name="example"></a>Пример

См. в разделе [omp_get_wtime](../../../parallel/openmp/reference/omp-get-wtime.md) пример использования `omp_get_wtick`.
