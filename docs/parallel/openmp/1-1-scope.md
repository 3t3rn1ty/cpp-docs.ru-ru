# 1.1 Scope
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Эта спецификация рассматриваются только пользователь\-направленный параллелизации, при котором пользователь явно указывает действие, выполняемое компилятором и системой среды выполнения для выполнения программы в параллельном режиме.  Необходимо, чтобы проверить реализации OpenMP c и C\+\+ для зависимостей конфликтов взаимоблокировки, состояний гонки или других проблем, которые приводятся в неправильном выполнении программы.  Пользователь отвечает за обеспечение, что приложение с помощью конструкций API OpenMP c и C\+\+ выполняется правильно.  Автоматические параллелизации, создаваемые компилятором и директивы компилятору для содействия такого параллелизации, не описанные в этом документе.
