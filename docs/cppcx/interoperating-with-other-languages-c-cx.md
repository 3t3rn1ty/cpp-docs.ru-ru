# Взаимодействие с другими языками (C++/CX)
В этой части документации описано, как использовать [!INCLUDE[cppwrt](../cppcx/includes/cppwrt-md.md)] \([!INCLUDE[cppwrt_short](../cppcx/includes/cppwrt-short-md.md)]\) для создания компонентов [!INCLUDE[wrt](../cppcx/includes/wrt-md.md)], которые могут использоваться другими языками программирования и библиотеками. Можно также создавать компоненты, которые могут использоваться не всеми языками. В этом разделе описываются различные аспекты, которые необходимо учитывать при взаимодействии приложения [!INCLUDE[cppwrt_short](../cppcx/includes/cppwrt-short-md.md)] с компонентами, написанными с помощью JavaScript, — управляемого языка платформы .NET Framework — или библиотеки шаблонов C\+\+ [!INCLUDE[wrt](../cppcx/includes/wrt-md.md)].
