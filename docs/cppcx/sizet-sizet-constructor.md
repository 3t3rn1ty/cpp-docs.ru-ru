# Конструктор SizeT::SizeT
Инициализирует новый экземпляр класса SizeT указанным значением.  
  
## Синтаксис  
  
```cpp  
SizeT( uint32 value1 );   SizeT( void* value2 );  
```  
  
## Параметры  
 value1  
 32\-битовое значение без знака.  
  
 value2  
 Указатель на 32\-разрядное значение без знака.
