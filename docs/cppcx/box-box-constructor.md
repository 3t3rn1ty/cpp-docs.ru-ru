# Конструктор Box::Box
Создает объект `Box`, который может инкапсулировать значение указанного типа.  
  
## Синтаксис  
  
```cpp  
Box(T valueArg);  
```  
  
#### Параметры  
 `valueArg`  
 Тип упаковываемого значения — например, `int`, `bool`, `float64`, `DateTime`.  
  
## Требования  
 **Заголовок:** vccorlib.h  
  
 **Пространство имен:** Platform
