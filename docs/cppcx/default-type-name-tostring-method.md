# Метод default::(имя_типа)::ToString
Возвращает строку, которая представляет текущий тип.  
  
## Синтаксис  
  
```cpp  
  
String^ ToString()  
```  
  
## Возвращаемое значение  
 Строка, представляющая текущий объект.
