# Метод StringReference::GetHSTRING
Возвращает содержимое строки как `__abi_HSTRING`.  
  
## Синтаксис  
  
```scr  
__abi_HSTRING GetHSTRING() const  
  
```  
  
## Возвращаемое значение  
 Объект `__abi_HSTRING`, который содержит строковые данные.
