# Метод Map::Remove
Удаляет указанную пару "ключ\-значение" из текущего объекта Map.  
  
## Синтаксис  
  
```  
  
virtual void Remove(  
   K key);  
```  
  
#### Параметры  
 `key`  
 Ключ из пары "ключ\-значение". Тип `key` является именем типа *K*.  
  
## Требования  
 **Заголовок:** collection.h  
  
 **Пространство имен:** Platform::Collections
