# Метод Map::Clear
Удаляет все пары "ключ\-значение" из текущего объекта Map.  
  
## Синтаксис  
  
```  
  
virtual void Clear();   
```  
  
## Требования  
 **Заголовок:** collection.h  
  
 **Пространство имен:** Platform::Collections
