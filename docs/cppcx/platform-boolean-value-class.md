# Класс значений Platform::Boolean
Представляет логическое значение. Эквивалент `bool`.  
  
## Синтаксис  
  
```cpp  
public value struct Boolean  
```  
  
## Участники  
 Класс Boolean имеет методы Equals\(\), GetHashCode\(\) и ToString\(\), наследуемые от [Класс Platform::Object](../cppcx/platform-object-class.md), и метод GetTypeCode\(\), наследуемый от [Класс Platform::Type](../cppcx/platform-type-class.md).
