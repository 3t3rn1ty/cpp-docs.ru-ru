# Интерфейс Platform::IValueType
`Platform::IValueType` — это интерфейс инфраструктуры, реализуемый классами и структурами значений. Не должен использоваться в коде явным образом.  
  
## Синтаксис  
  
```cpp  
interface class IValueType  
```
