# Метод StringReference::Data
Возвращает содержимое этой строки `StringReference` как массив значений char16.  
  
## Синтаксис  
  
```cpp  
const ::default::char16 * Data() const  
```  
  
## Возвращаемое значение  
 Массив текстовых символов ЮНИКОДА char16.
