# Класс Platform::STAThreadAttribute
Указывает, что потоковая модель для приложения является однопотоковым подразделением \(STA\).  
  
## Синтаксис  
  
```  
public ref class STAThreadAttribute sealed : Attribute  
```  
  
## Члены  
  
### Открытые конструкторы  
  
|Имя|Описание|  
|---------|--------------|  
|[Конструктор STAThreadAttribute](../cppcx/stathreadattribute-constructor-1.md)|Инициализирует новый экземпляр класса.|  
  
### Открытые методы  
 Атрибут STAThreadAttribute наследует от [Класс Platform::Object](../cppcx/platform-object-class.md). Атрибут STAThreadAttribute также перегружает или имеет следующие члены:  
  
|Имя|Описание|  
|---------|--------------|  
|[STAThreadAttribute::Equals](../cppcx/stathreadattribute-equals.md)|Определяет, равен ли заданный объект текущему объекту.|  
|[STAThreadAttribute::GetHashCode](../cppcx/stathreadattribute-gethashcode.md)|Возвращает хэш\-код данного экземпляра.|  
|[STAThreadAttribute::ToString](../cppcx/stathreadattribute-tostring.md)|Возвращает строку, представляющую текущий объект.|  
  
## Иерархия наследования
