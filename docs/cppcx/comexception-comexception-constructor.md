# COMException::COMException - конструктор
Инициализирует новый экземпляр класса COMException.  
  
## Синтаксис  
  
```cpp  
COMException( int hresult )  
```  
  
#### Параметры  
 hresult  
 HRESULT ошибки, представляемый этим исключением.
