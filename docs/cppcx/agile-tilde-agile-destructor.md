# Деструктор Agile::~Agile
Ликвидирует текущий экземпляр класса Agile.  
  
## Синтаксис  
  
```cpp  
  
~Agile();  
```  
  
## Заметки  
 Этот деструктор также освобождает объект, представленный текущим объектом Agile.
