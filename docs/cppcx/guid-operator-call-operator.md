# Оператор Guid::operator()
Выполняет неявное преобразование [структуры GUID](http://msdn.microsoft.com/library/windows/desktop/aa373931\(v=vs.85\).aspx) в Platform::Guid.  
  
## Синтаксис  
  
```cpp  
Platform::Guid operator()  
```  
  
## Возвращаемое значение  
 Структура Guid.
