# Метод IntPtr::ToInt32
Преобразует текущее значение IntPtr в 32\-битовое целое число.  
  
## Синтаксис  
  
```cpp  
int32 IntPtr::ToInt32();  
```  
  
## Возвращаемое значение  
 32\-битовое целое число.
