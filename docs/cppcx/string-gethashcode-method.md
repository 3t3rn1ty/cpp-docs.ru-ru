# Метод String::GetHashCode
Возвращает хэш\-код данного экземпляра.  
  
## Синтаксис  
  
```cpp  
  
virtual override int GetHashCode()  
```  
  
## Возвращаемое значение  
 Хэш\-код данного экземпляра.
