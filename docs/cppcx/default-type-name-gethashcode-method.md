# Метод default::(имя_типа)::GetHashCode
Возвращает хэш\-код данного экземпляра.  
  
## Синтаксис  
  
```cpp  
public:int GetHashCode()  
```  
  
## Возвращаемое значение  
 Хэш\-код данного экземпляра.
