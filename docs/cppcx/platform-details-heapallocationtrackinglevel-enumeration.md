# Перечисление Platform::Details::HeapAllocationTrackingLevel
Это перечисление предназначено только для внутреннего использования \(не для использования при разработке\).  
  
## Синтаксис  
  
```cpp  
  
enumm class HeapAllocationTrackingLevel;  
```  
  
## Примечания  
 Это перечисление предоставляется исключительно для полноты информации, поскольку его данные можно изучить с помощью браузеров.  
  
## Иерархия наследования  
 `Platform`
