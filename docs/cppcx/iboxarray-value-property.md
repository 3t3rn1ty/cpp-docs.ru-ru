# Свойство IBoxArray::Value
Возвращает значение, которое было изначально сохранено в этом объекте.  
  
## Синтаксис  
  
```scr  
property T Value {T get();}  
```  
  
```csharp  
  
```  
  
#### Параметры  
 `T`  
 Введите запакованное значение.  
  
## Значение свойства, возвращаемое значение  
 Возвращает значение, которое было изначально сохранено в этом объекте.
