# Метод UnorderedMapView::Size
Возвращает количество элементов [Windows::Foundation::Collections::IKeyValuePair\<K,V\>](http://msdn.microsoft.com/library/windows/apps/br226031.aspx) в UnorderedMapView.  
  
## Синтаксис  
  
```cpp  
  
virtual property unsigned int Size;  
```  
  
## Возвращаемое значение  
 Количество элементов в Unordered MapView.  
  
## Требования  
 **Заголовок:** collection.h  
  
 **Пространство имен:** Platform::Collections
