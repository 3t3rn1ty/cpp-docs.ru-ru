# Свойство Array::Value
Получает дескриптор текущего массива.  
  
## Синтаксис  
  
```  
  
property Array^ Value;  
```  
  
## Возвращаемое значение  
 Дескриптор текущего массива.  
  
## Требования  
 **Заголовок:** vccorlib.h  
  
 **Пространство имен:** Platform
