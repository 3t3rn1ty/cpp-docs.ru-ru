# Свойство WriteOnlyArray::Data
Указатель на буфер данных.  
  
## Синтаксис  
  
```cpp  
property T* Data{  
   T* get() const;  
}  
```  
  
## Возвращаемое значение  
 Указатель на необработанные байты массива.

 **Пространство имен:** Platform
