# <a name="intel39s-mmx-instruction-set"></a>Intel &#39; s набор инструкций MMX
## <a name="microsoft-specific"></a>Блок, относящийся только к системам Microsoft  
 Компилятор Visual C++ позволяет использовать набор инструкций MMX Intel (мультимедийное расширение), заданных во встроенном ассемблере. Инструкции MMX также поддерживаются дизассемблированным кодом отладчика. Компилятор выдает предупреждение, если функция содержит инструкции MMX, но не содержит инструкцию EMMS для очистки мультимедийного состояния. Дополнительные сведения см. на веб-сайте Intel.
