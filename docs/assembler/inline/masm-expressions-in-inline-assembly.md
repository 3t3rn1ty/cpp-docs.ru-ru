# <a name="masm-expressions-in-inline-assembly"></a>Выражения MASM во встроенном коде на языке ассемблера
## <a name="microsoft-specific"></a>Блок, относящийся только к системам Microsoft  
 Во встроенном коде на ассемблере может использоваться любое выражение MASM с произвольным сочетанием операндов и операторов, при вычислении которого получается одно значение или адрес.
