# <a name="dt"></a>DT
Выделяет и при необходимости инициализирует 10 байт хранилища для каждого `initializer`. Можно также использовать как спецификатор типа в любом является допустимым типом. `DT`— синоним [TBYTE](../../assembler/masm/tbyte.md).  
  
## <a name="syntax"></a>Синтаксис  
  
```  
[[name]] DT initializer [[, initializer]]...  
```
