# DT
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Выделяет и инициализирует 10 байт хранения для каждого `initializer`.  Может также использоваться как описатель типа в любом месте, где допустим тип.  `DT` синоним   [TBYTE](../../assembler/masm/tbyte.md).  
  
## Синтаксис  
  
```  
[[name]] DT initializer [[, initializer]]...  
```
