# <a name="public-masm"></a>PUBLIC (MASM)
Делает каждой переменной, метки или указан как абсолютный символ *имя* доступны для всех модулей в программе.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
PUBLIC [[langtype]] name [[, [[langtype]] name]]...  
```
