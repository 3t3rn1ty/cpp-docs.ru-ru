# <a name="operator-high"></a>Оператор HIGH
Возвращает старшие 8 разрядов младшие 16 разрядов *выражение*. Выражения MASM являются 64-разрядными значениями.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
HIGH expression  
```
