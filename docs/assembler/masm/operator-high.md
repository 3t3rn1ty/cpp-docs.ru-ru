# <a name="operator-high"></a>Оператор HIGH

Возвращает старшие 8 разряда младшие 16 разрядов *выражение*. Выражения MASM являются 64-разрядными значениями.

## <a name="syntax"></a>Синтаксис

> Высокий уровень выражения
