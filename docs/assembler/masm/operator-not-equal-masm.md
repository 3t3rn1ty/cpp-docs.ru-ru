# <a name="operator--masm"></a>Оператор != (MASM)
Не равно. Используется только в пределах [. Если](../../assembler/masm/dot-if.md), [. ХОТЯ](../../assembler/masm/dot-while.md), или [. ПОВТОРИТЕ](../../assembler/masm/dot-repeat.md) блокирует и вычисляется во время выполнения, а не во время сборки.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
expression1 != expression2  
```
