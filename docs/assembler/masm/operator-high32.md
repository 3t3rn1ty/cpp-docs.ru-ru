# <a name="operator-high32"></a>Оператор HIGH32
Возвращает старшие 32 разряда *выражение*. Выражения MASM являются 64-разрядными значениями.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
HIGH32 expression  
```
