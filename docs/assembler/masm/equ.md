# <a name="equ"></a>EQU
Числовое значение присваивается первой директивы *выражение* для *имя*.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
      name EQU expression  
name EQU <text>  
```  
  
## <a name="remarks"></a>Примечания  
 *Имя* не может быть переопределен.  
  
 Вторая директива назначает указан *текст* для *имя*. *Имя* можно назначить другой *текст* позже. В разделе [TEXTEQU](../../assembler/masm/textequ.md).
