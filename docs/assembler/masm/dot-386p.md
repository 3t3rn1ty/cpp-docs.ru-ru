# <a name="386p"></a>.386P
Включает сборку всех инструкций (включая привилегий) 80386 процессора; Отключает сборку инструкций, процессоры, более поздней версии.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
.386P  
  
```  
  
## <a name="remarks"></a>Примечания  
 Также позволяет 80387 инструкции.
