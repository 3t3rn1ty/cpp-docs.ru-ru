# .386P
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Включает сборку всех инструкций \(включая привилигированное\) для процессора равно 80386 %. запрещает сборка инструкций с более процессорами, вставленные последними.  
  
## Синтаксис  
  
```  
  
.386P  
  
```  
  
## Заметки  
 Также включает 80387 инструкций.
