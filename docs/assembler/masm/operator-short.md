# operator SHORT
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Задает тип *метка* в шорты.  Все гиперссылки в *метка* short \(должна быть в диапазоне от 128 до \+127 байтов из инструкций гиперссылок в *метка*\).  
  
## Синтаксис  
  
```  
  
SHORT label  
```
