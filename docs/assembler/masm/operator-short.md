# <a name="operator-short"></a>оператор SHORT
Задает тип *метка* в короткие. Все ссылки на *метка* должен быть коротким (в диапазоне от -128 до + 127 байт от инструкции прыжка до *метка*).  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
SHORT label  
```
