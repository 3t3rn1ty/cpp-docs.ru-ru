# IFDIF]
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Предоставляет сборку, если элементы текста различаются.  
  
## Синтаксис  
  
```  
  
IFDIF[[I]] textitem1, textitem2  
```  
  
## Заметки  
 If **е** обеспечивает сравнение нечувствительно к регистру.  См. IF для полного синтаксиса.
