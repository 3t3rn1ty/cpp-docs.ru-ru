# <a name="endprolog"></a>.ENDPROLOG
Сигнализирует о завершении объявлений в прологе.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
.ENDPROLOG  
```  
  
## <a name="remarks"></a>Примечания  
 Это ошибка для использования любого из объявлений в прологе за пределами области между [PROC](../../assembler/masm/proc.md) КАДРОВ и. ENDPROLOG.  
  
 Дополнительные сведения см. в разделе [MASM для x64 (ml64.exe)](../../assembler/masm/masm-for-x64-ml64-exe.md).
