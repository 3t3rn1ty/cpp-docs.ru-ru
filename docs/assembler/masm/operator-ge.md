# <a name="operator-ge"></a>Оператор GE
Возвращает значение true (-1), если *expression1* больше или равно *expression2*, или значение false (0), если это не так.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
expression1 GE expression2  
```
