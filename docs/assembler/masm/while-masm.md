# <a name="while-masm"></a>WHILE (MASM)
Повторяет сборки блока `statements` при условии, что *выражение* остается истинным.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
   WHILE expression  
statements  
ENDM  
```
