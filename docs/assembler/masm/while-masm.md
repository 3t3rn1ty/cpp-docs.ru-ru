# <a name="while-masm"></a>WHILE (MASM)

Повторяет сборку блока *инструкций* , пока *выражение* остается истинным.

## <a name="syntax"></a>Синтаксис

> ХОТЯ *выражение*<br/>
> *Инструкции*<br/>
> ENDM
