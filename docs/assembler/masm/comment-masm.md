# <a name="comment-masm"></a>COMMENT (MASM)
Обрабатывает все *текст* между или на той же строке, в качестве разделителей как комментарий.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
      COMMENT delimiter [[text]]   
[[text]]   
[[text]] delimiter [[text]]  
```
