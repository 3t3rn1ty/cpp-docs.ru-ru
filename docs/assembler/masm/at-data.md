# <a name="data"></a>@data
Имя по умолчанию группу данных. Равняется DGROUP для всех моделей, за исключением **плоский**. Равно **плоский** под **плоский** модели памяти (макрос текста).  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
@data  
  
```
