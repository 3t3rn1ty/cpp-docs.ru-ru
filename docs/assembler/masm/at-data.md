# @data
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Имя группы данных по умолчанию.  Результатом которого является DGROUP для всех моделей, отличный от **Плоский**.  Равно **Плоский** под  **Плоский** модель памяти \(макрос текста\).  
  
## Синтаксис  
  
```  
  
@data  
  
```
