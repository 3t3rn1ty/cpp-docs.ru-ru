# <a name="386"></a>.386

Включает сборку непривилегированному инструкций для 80386 процессора; Отключает сборку из инструкций, процессоры, более поздней версии.

## <a name="syntax"></a>Синтаксис

> .386

## <a name="remarks"></a>Примечания

Также позволяет 80387 инструкции.
