# <a name="nolistif"></a>.NOLISTIF
Подавляет перечень условных блоков которого условие имеет значение false (0).  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
.NOLISTIF  
  
```  
  
## <a name="remarks"></a>Примечания  
 Это значение по умолчанию. То же, что [. SFCOND](../../assembler/masm/dot-sfcond.md).
