# ML Nonfatal Error A2063
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**можно ВЫРАВНИВАТЬ только в степень 2. Выражение**  
  
 Выражение, указанное в [Выровнять](../../assembler/masm/align-masm.md) недопустимая директива.  
  
 **Выровнять** выражение должно быть мощью 2 между 2 и 256, а должно быть меньше или равно выравнивание текущих сегмента, структуры или объединения.
