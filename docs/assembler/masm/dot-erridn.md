# .ERRIDN]
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Формирует ошибку, если элементы текста совпадают.  
  
## Синтаксис  
  
```  
  
.ERRIDN[[I]] <textitem1>, <textitem2> [[, message]]  
```  
  
## Заметки  
 If **е** обеспечивает сравнение нечувствительно к регистру.
