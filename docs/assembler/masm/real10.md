# <a name="real10"></a>REAL10
Выделяет и при необходимости инициализирует 10-байтовое число с плавающей запятой для каждого *инициализатора*.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
name REAL10 initializer [[, initializer]]...  
```
