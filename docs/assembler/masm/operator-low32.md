# <a name="operator-low32"></a>Оператор LOW32
Возвращает младшие 32 разряда *выражение*. Выражения MASM являются 64-разрядными значениями.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
LOW32 expression  
```
