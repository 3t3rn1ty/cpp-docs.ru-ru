# <a name="nocref"></a>.NOCREF
Подавляет список символов в файле символов таблицы и браузера.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
.NOCREF [[name[[, name]]...]]  
```  
  
## <a name="remarks"></a>Примечания  
 Если имена заданы, заданными именами, подавляются. То же, что [. XCREF](../../assembler/masm/dot-xcref.md).
