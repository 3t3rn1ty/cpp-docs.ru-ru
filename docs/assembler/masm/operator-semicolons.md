# <a name="operator-"></a>Оператор ;;
Рассматривает *текст* как комментарии в макрос, который появляется только в определении макроса. Список не содержит *текст* где макрос расширяется.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
;;  
text  
  
```
