# <a name="ml-nonfatal-error-a2034"></a>Некритичная ошибка ML A2034
**должен находиться в блоке сегмента**  
  
 Одно из следующих найден вне блока сегмента:  
  
-   Инструкции  
  
-   Определение метки  
  
-   Объект [THIS](../../assembler/masm/operator-this.md) оператор  
  
-   Объект [ $ ](../../assembler/masm/dollar.md) оператор  
  
-   Определение процедуры  
  
-   [ВЫРОВНЯТЬ](../../assembler/masm/align-masm.md) директивы  
  
-   [ORG](../../assembler/masm/org.md) директивы
