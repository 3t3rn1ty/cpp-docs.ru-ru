# <a name="fardata"></a>.FARDATA?
При использовании с [. МОДЕЛЬ](../../assembler/masm/dot-model.md), начинается сегмент дальней данных для неинициализированных данных (имя сегмента FAR_BSS или *имя*).  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
.FARDATA? [[name]]  
```
