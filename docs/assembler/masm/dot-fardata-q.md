# .FARDATA?
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

При использовании с .MODELсегмент начинается далекий данных для неинициализированных данных \(имя сегмента FAR\_BSS или Имя\).  
  
## Синтаксис  
  
```  
  
.FARDATA? [[name]]  
```
