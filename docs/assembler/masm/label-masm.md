# <a name="label-masm"></a>LABEL (MASM)

Создает новую метку, назначив текущее значение счетчика расположение и заданного *тип* для *имя*.

## <a name="syntax"></a>Синтаксис

> *имя* МЕТКА *тип*

> *имя* МЕТКА [[NEAR | ДАЛЕКО | PTR PROC]] [[*тип*]]
