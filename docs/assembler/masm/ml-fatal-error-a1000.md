# ML Fatal Error A1000
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**не удается открыть файл. fileName**  
  
 Ассемблер не смог открыть источник, включает или выходной файл.  
  
 Причиной может быть одно из следующих действий:  
  
-   Файл не существует.  
  
-   Файл используется другим процессом.  
  
-   Имя файла недопустимо.  
  
-   Файл, доступный только для чтения с именем файла вывода уже существует.  
  
-   Текущий диск заполнен.  
  
-   Текущая папка является корневым и переполнена.  
  
-   Устройство не может быть записан.  
  
-   Диск не готов.
