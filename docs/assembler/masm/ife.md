# IFE
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Предоставляет сборку, если *Выражение* значение false \(0\).  
  
## Синтаксис  
  
```  
  
IFE expression  
```  
  
## Заметки  
 См. IF для полного синтаксиса.
