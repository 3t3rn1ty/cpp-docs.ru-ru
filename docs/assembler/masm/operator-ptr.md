# <a name="operator-ptr"></a>Оператор PTR

Первый оператор силы *выражение* будет считаться с указанным *тип*. Второй оператор устанавливает указатель на *тип*.

## <a name="syntax"></a>Синтаксис

> *Тип* PTR *выражение*

> [[*расстояние*]] PTR *типа*
