# <a name="operator-ptr"></a>Оператор PTR
Первый оператор принудительно *выражение* следует рассматривать как с указанным `type`. Второй оператор устанавливает указатель на `type`.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
      type PTR expression  
[[distance]] PTR type  
```
