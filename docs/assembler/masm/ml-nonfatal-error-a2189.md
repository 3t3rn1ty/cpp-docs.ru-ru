# <a name="ml-nonfatal-error-a2189"></a>Некритичная ошибка ML A2189
**Недопустимое сочетание с выравниванием сегмента**  
  
 Выравнивание, определяемое **ВЫРОВНЯТЬ** или `EVEN` директива было больше, чем текущее выравнивание сегмента в соответствии с **СЕГМЕНТ** директивы.
