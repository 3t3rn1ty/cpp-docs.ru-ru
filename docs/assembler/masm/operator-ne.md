# <a name="operator-ne"></a>Оператор NE
Возвращает значение true (-1), если *expression1* не равно *expression2*, или возвращает значение false (0), если это так.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
expression1 NE expression2  
```
