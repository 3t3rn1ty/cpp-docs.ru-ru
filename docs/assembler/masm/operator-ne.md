# operator NE
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Возвращает значение true \(\- 1\), если *expression1* не равно *expression2*или возвращает значение false \(0\), если он используется.  
  
## Синтаксис  
  
```  
  
expression1 NE expression2  
```
