# <a name="dw"></a>DW
Выделяет и при необходимости инициализирует word (2 байта) хранилища для каждого `initializer`. Можно также использовать как спецификатор типа в любом является допустимым типом. `DW`— синоним [WORD](../../assembler/masm/word.md).  
  
## <a name="syntax"></a>Синтаксис  
  
```  
[[name]] DW initializer [[, initializer]]...  
```
