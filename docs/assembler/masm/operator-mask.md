# <a name="operator-mask"></a>Оператор MASK
Возвращает Битовая маска в котором биты в *recordfieldname* или *записи* устанавливаются и удаляются все биты.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
MASK {recordfieldname | record}  
```
