# <a name="operator-mask"></a>Оператор MASK

Возвращает немного маски, в котором биты в *recordfieldname* или *записи* заданы и все остальные биты, очищаются.

## <a name="syntax"></a>Синтаксис

> Маска {recordfieldname | запись}
