# <a name="operator-lt"></a>Оператор LT
Возвращает значение true (-1), если *expression1* — меньше, чем *expression2*, или значение false (0), если это не так.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
expression1 LT expression2  
```
