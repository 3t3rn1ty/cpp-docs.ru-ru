# operator LT
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Возвращает значение true \(\- 1\), если *expression1* меньше *expression2*или возвращает значение false \(0\), если объект отсутствует.  
  
## Синтаксис  
  
```  
  
expression1 LT expression2  
```
