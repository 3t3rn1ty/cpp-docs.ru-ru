# operator SECTIONREL
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Возвращает смещение relative раздела выражения по отношению к шагу, содержащий целевой объект в конечный исполняемом файле.  
  
## Синтаксис  
  
```  
SECTIONREL expression  
```  
  
## Заметки  
 SECTIONREL доступно только при использовании излучением объектов COFF.
