# <a name="operator-sectionrel"></a>Оператор SECTIONREL

Возвращает смещение относительного разделе выражения относительно раздел, содержащий целевой объект в конечный исполняемый файл.

## <a name="syntax"></a>Синтаксис

> Выражение SECTIONREL

## <a name="remarks"></a>Примечания

SECTIONREL доступен только с помощью эмиссии объект COFF.
