# <a name="operator-sectionrel"></a>Оператор SECTIONREL
Возвращает смещение относительного раздел выражения относительно раздела, содержащего целевой объект в конечный исполняемый файл.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
SECTIONREL expression  
```  
  
## <a name="remarks"></a>Примечания  
 SECTIONREL доступен только в вывод объекта COFF.
