# <a name="ml-fatal-error-a1009"></a>Неустранимая ошибка ML A1009

**слишком длинная строка**

Строки в исходном файле превышает ограничение в 512 символов.

Если несколько физических строк сцепляются символ продолжения строки (\), результирующее логической строке по-прежнему ограничена 512 символами.
