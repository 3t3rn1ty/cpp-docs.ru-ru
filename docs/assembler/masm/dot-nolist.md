# <a name="nolist"></a>.NOLIST
Подавляет список программ.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
.NOLIST  
  
```  
  
## <a name="remarks"></a>Примечания  
 То же, что [. XLIST](../../assembler/masm/dot-xlist.md).
