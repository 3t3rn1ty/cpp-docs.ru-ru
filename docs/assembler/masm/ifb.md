# IFB
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Предоставляет сборку, если *textitem* пустыми.  
  
## Синтаксис  
  
```  
  
IFB   
textitem  
  
```  
  
## Заметки  
 См. IF для полного синтаксиса.
