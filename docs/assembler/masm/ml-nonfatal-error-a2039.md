# <a name="ml-nonfatal-error-a2039"></a>Некритичная ошибка ML A2039
**слишком длинная строка**  
  
 Строка исходного файла превышает ограничение в 512 символов.  
  
 Если несколько физических строк сцепляются знак продолжения строки (\), результирующее логической строки по-прежнему ограничена 512 символами.
