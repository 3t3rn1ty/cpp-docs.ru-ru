# <a name="operator-gt"></a>Оператор GT
Возвращает значение true (-1), если *expression1* больше, чем *expression2*, или значение false (0), если это не так.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
expression1 GT expression2  
```
