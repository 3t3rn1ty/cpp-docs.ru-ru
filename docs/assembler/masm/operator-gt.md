# operator GT
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Возвращает значение true \(\- 1\), если *expression1* больше *expression2*или возвращает значение false \(0\), если объект отсутствует.  
  
## Синтаксис  
  
```  
  
expression1 GT expression2  
```
