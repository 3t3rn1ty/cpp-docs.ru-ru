# IFDEF
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Предоставляет сборку, если *Имя* ранее определенные метка, переменная или символа.  
  
## Синтаксис  
  
```  
  
IFDEF name  
```  
  
## Заметки  
 См. IF для полного синтаксиса.
