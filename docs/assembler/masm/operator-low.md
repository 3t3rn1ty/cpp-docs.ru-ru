# <a name="operator-low"></a>Оператор LOW
Возвращает младшие 8 разрядов *выражение*. Выражения MASM являются 64-разрядными значениями.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
LOW expression  
```
