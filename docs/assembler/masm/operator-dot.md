# <a name="operator-"></a>Оператор .
Первый оператор возвращает *выражение* плюс смещение *поле* в пределах его структуры или объединения. Второй оператор возвращает значение в расположении, указанном *зарегистрировать* плюс смещение *поле* в пределах его структуры или объединения.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
      expression  
      . field [[. field]]...  
[register]. field [[. field]]...  
```
