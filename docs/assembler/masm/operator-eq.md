# operator EQ
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Возвращает значение true \(\- 1\), если *expression1* equals *expression2*или возвращает значение false \(0\), если это не так.  
  
## Синтаксис  
  
```  
  
expression1 EQ expression2  
```
