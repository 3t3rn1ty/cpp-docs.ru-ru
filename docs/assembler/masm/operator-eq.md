# <a name="operator-eq"></a>Оператор EQ
Возвращает значение true (-1), если *expression1* равняется *expression2*, или значение false (0), если это не так.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
expression1 EQ expression2  
```
