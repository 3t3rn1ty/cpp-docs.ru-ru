# IF2
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**IF** блок вычисляется при каждом проходе, если сборки  **ПАРАМЕТР: SETIF2** существует  **True**.  
  
## Синтаксис  
  
```  
  
IF2   
expression  
  
```  
  
## Заметки  
 См. IF для полного синтаксиса.
