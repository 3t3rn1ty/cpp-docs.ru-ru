# Microsoft Macro Assembler Reference
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Microsoft Macro Assembler \(MASM\) предоставляет несколько преимуществ по сравнению с встроенной сборки.  MASM содержит макрос языка, который имеет такие функции, как обработка строк циклов, арифметические операции и текст.  MASM также дает большие возможности управления оборудованием, так как он поддерживает наборы инструкций процессоры Pentium, 386 и 486.  С помощью MASM, можно уменьшить затраты времени и памяти.
