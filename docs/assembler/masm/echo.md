# <a name="echo"></a>ECHO

Отображает *сообщение* стандартное устройство вывода (по умолчанию, экрана).

## <a name="syntax"></a>Синтаксис

> ECHO *сообщения*

## <a name="remarks"></a>Примечания

Совпадение с кодом [% OUT](../../assembler/masm/percent-out.md).
