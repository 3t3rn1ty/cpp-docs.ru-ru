# <a name="echo"></a>ECHO
Отображает *сообщение* стандартное устройство вывода (по умолчанию экрана).  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
ECHO   
message  
  
```  
  
## <a name="remarks"></a>Примечания  
 То же, что [% OUT](../../assembler/masm/percent-out.md).
