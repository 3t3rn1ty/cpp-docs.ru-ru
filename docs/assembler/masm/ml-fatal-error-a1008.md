# ML Fatal Error A1008
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**несогласованная директива вложение макросов**  
  
 Любые макрос не завершен до конца файла или закрывающей рекомендации ENDM найти вне блока макроса.  
  
 Одной из причин этой ошибки подавление с многоточием before [.REPEAT](../../assembler/masm/dot-repeat.md) OR  [.WHILE](../../assembler/masm/dot-while.md).
