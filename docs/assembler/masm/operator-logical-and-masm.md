# <a name="operator-ampamp-masm"></a>оператор &amp; &amp; (MASM)
Заменяет `parameter` с его соответствующее значение аргумента.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
&  
parameter  
&  
  
```
