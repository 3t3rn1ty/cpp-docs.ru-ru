# <a name="ml-nonfatal-error-a2038"></a>Некритичная ошибка ML A2038
**операнд оператора макрос**  
  
 Код на языке ассемблера обнаружен конец списка параметров макроса сразу после [!](../../assembler/masm/operator-logical-not-masm-run-time.md) или [ % ](../../assembler/masm/operator-percent.md) оператор.
