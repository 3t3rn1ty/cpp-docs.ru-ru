# ML Nonfatal Error A2047
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**пустая строка \(иметь нулевую\)**  
  
 Строка разделителя состояла из пары кавычек и никаких символов.  
  
 Для строк, которые должны быть допустима, она должна содержать 1\-255 символов.
