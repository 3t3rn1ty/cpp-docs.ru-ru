# <a name="ml-nonfatal-error-a2004"></a>Некритичная ошибка ML A2004
**конфликт типов символов: идентификатор**  
  
 [EXTERNDEF](../../assembler/masm/externdef.md) или [МЕТКА](../../assembler/masm/label-masm.md) директива была использована для переменной, символ, структуру данных или метки, которое было определено в том же модуле, но с другим типом.
