# operator ! (MASM Run Time)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Логическое отрицание.  Используется только в пределах [.IF](../Topic/.IF.md)"  [.WHILE](../../assembler/masm/dot-while.md)или  [.REPEAT](../../assembler/masm/dot-repeat.md) блоки и вычисляется во время выполнения, а не во время сборки.  
  
## Синтаксис  
  
```  
  
!  
expression  
  
```
