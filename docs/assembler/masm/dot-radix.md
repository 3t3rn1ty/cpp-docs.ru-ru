# <a name="radix"></a>.RADIX

Задает основание системы счисления по умолчанию, в диапазоне 2 до 16, значение *выражение*.

## <a name="syntax"></a>Синтаксис

> . Выражение основания системы СЧИСЛЕНИЯ
