# ML Nonfatal Error A2019
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**операнд должен быть типом записей или полем**  
  
 Далее приводится операнд [Ширина](../../assembler/masm/operator-width.md) OR  [Маска](../../assembler/masm/operator-mask.md) недопустимый оператор.  
  
 [Ширина](../../assembler/masm/operator-width.md) оператор операнд, который принимает имя поля или записи.  [Маска](../../assembler/masm/operator-mask.md) оператор операнд, который принимает имя поля или типа записи.
