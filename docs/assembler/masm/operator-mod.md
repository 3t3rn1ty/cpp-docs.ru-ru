# <a name="operator-mod"></a>Оператор MOD
Возвращает целочисленное значение остатка (остаток от деления) при делении *expression1* по *expression2*.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
expression1 MOD expression2  
```
