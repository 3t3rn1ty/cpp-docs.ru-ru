# <a name="df"></a>DF
Выделяет и при необходимости инициализирует 6 байтов памяти для каждого `initializer`. `DF`— синоним [FWORD](../../assembler/masm/fword.md).  
  
## <a name="syntax"></a>Синтаксис  
  
```  
[[name]] DF initializer [[, initializer]]...  
```  
  
## <a name="remarks"></a>Примечания  
 Также можно использовать в качестве спецификатора типа в любом является допустимым типом.
