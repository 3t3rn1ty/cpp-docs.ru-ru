# EXITM
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Завершает расширение текущего блока повторения или макроса и начинает сборку следующей выписки вне блока.  
  
## Синтаксис  
  
```  
  
EXITM [[textitem]]   
```  
  
## Заметки  
 В функции макроса *textitem* возвращаемое значение.
