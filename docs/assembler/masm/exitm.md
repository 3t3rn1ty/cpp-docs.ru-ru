# <a name="exitm"></a>EXITM
Прекращает расширения текущего блока повтора или макрос и запускает сборку операторе, следующем за пределами блока.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
EXITM [[textitem]]   
```  
  
## <a name="remarks"></a>Примечания  
 В функцию макрос *textitem* возвращается значение.
