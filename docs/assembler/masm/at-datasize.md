# <a name="datasize"></a>@DataSize

0 для **КРОШЕЧНЫЕ**, **НЕБОЛЬШОЙ**, **Средний**, и **НЕСТРУКТУРИРОВАННЫЙ** моделирует, 1 для **COMPACT** и  **БОЛЬШОЙ** моделей и 2 для **БОЛЬШОЙ** модели (уравнять числовые).

## <a name="syntax"></a>Синтаксис

> @DataSize
