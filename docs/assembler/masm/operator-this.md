# <a name="operator-this"></a>Оператор THIS
Возвращает указанный операнд `type` , смещение и сегмент значения равны значения счетчика текущего расположения.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
THIS type  
```
