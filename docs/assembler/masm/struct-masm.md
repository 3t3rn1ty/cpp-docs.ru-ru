# <a name="struct-masm"></a>STRUCT (MASM)
Объявляет тип структуры, с указанным *fielddeclarations*. Каждое поле должно быть определение допустимые данные. То же, что [STRUC](../../assembler/masm/struc.md).  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
   name STRUCT [[alignment]] [[, NONUNIQUE]]  
fielddeclarations  
name ENDS  
```
