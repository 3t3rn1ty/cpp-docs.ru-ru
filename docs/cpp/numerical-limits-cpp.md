# <a name="numerical-limits-c"></a>Числовые пределы (C++)
Два стандартных включаемых файла, LIMITS.H и FLOAT.H, определяют числовые ограничения, то есть минимальные и максимальные значения, которые могут удерживаться переменной заданного типа. Эти минимальные и максимальные значения гарантированно перенести в любой компилятор C++, который использует то же представление данных в ANSI C. ОГРАНИЧЕНИЯ. H включаемый файл определяет [числовые ограничения для целочисленных типов](../cpp/integer-limits.md)и число с плавающей запятой. Определяет H [числовые ограничения для типов с плавающей запятой](../cpp/floating-limits.md).
