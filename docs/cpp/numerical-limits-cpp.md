# <a name="numerical-limits-c"></a>Числовые пределы (C++)
Два стандартных включаемых файлов, \<limits.h > и \<float.h >, определяют числовые ограничения или минимального и максимального значений, содержащихся в переменной заданного типа. Эти минимальные и максимальные значения гарантированно перенести в любой компилятор C++, который использует то же представление данных в ANSI C. \<Limits.h > включить файл определяет [числовые ограничения для целочисленных типов](../cpp/integer-limits.md), и \<float.h > определяет [числовые ограничения для типов с плавающей запятой](../cpp/floating-limits.md).
