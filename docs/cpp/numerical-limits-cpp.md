# Числовые пределы (C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Два стандартных включаемых файла, LIMITS.H и FLOAT.H, определяют числовые ограничения, то есть минимальные и максимальные значения, которые могут удерживаться переменной заданного типа.  Гарантируется, что эти минимальные и максимальные значения можно перенести в любой компилятор С\+\+, который использует то же представление данных, что и ANSI C.  Включаемый файл LIMITS.H определяет [числовые ограничения для целочисленных типов](../Topic/Integer%20Limits.md), а файл FLOAT.H — [числовые ограничения для типов с плавающей запятой](../cpp/floating-limits.md).
