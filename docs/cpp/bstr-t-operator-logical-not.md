# <a name="bstrtoperator-"></a>_bstr_t::operator !
**Блок, относящийся только к системам Microsoft**  
  
 Проверяет, если инкапсулированный `BSTR` — **NULL** строки.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
bool operator!( ) const throw( );  
  
```
