# Программа и компоновка (C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Программа состоит из одной или нескольких записей преобразования, связанных друг с другом.  Выполнение \(по существу\) начинается в записи преобразования, содержащей функцию **main**.  \(Дополнительные сведения о записях преобразования см. в разделе [Этапы преобразования](../preprocessor/phases-of-translation.md) *Справочника по препроцессору*.\) Дополнительные сведения о функции **main** см. в разделе [Запуск программы. Функция main](../Topic/main:%20Program%20Startup.md).
