**Блок, относящийся только к системам Microsoft**  
  
 Доступны следующие процедуры.  
  
|Функция|Описание|  
|-------------|--------------|  
|[\_com\_raise\_error](../cpp/com-raise-error.md)|Создает ошибку [\_com\_error](../cpp/com-error-class.md) в ответ на сбой.|  
|[\_set\_com\_error\_handler](../cpp/set-com-error-handler.md)|Заменяет функцию по умолчанию, используемую для обработки ошибок COM.|  
|[ConvertBSTRToString](../cpp/convertbstrtostring.md)|Преобразует значение `BSTR` в **char \***.|  
|[ConvertStringToBSTR](../cpp/convertstringtobstr.md)|Преобразует значение **char \*** в строку `BSTR`.|
