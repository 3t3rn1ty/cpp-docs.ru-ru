# Операторы перехода (C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Оператор перехода C\+\+ выполняет немедленную локальную передачу контроля.  
  
## Синтаксис  
  
```  
break;  
continue;  
return [expression];  
goto identifier;  
```
