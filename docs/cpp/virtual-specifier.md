# Спецификатор virtual
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Ключевое слово [virtual](../cpp/virtual-cpp.md) может использоваться только для нестатических функций\-членов класса.  Это означает, что привязка вызовов функции откладывается до времени выполнения.  Дополнительные сведения см. в разделе [Виртуальные функции](../cpp/virtual-functions.md).
