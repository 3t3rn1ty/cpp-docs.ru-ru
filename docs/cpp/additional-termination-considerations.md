# Дополнительные сведения о завершении
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Программу на языке C\+\+ можно завершить с помощью **exit**, `return` или **abort**.  Можно добавить обработку выхода с помощью функции `atexit`.  Эти темы рассматриваются в следующих разделах.
