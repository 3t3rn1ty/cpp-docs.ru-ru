# <a name="program-termination"></a>Завершение программы

В C++ существует несколько способов завершения работы программы:

- Вызовите [выйти из](../cpp/exit-function.md) функции.

- Вызовите [прервать](../cpp/abort-function.md) функции.

- Выполнение [возвращают](../cpp/return-statement-in-program-termination-cpp.md) инструкции от `main`.
