# В каких случаях следует использовать классы баз данных?
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Во многих приложениях используются функции хранения данных в базах данных.  База данных представляет собой гибкое хранилище данных, к которому в большинстве случаев могут обращаться несколько пользователей и приложений.  В базах данных может храниться большой объем данных, а также поддерживаются эффективные функции запроса и обновления данных.
