# Допустимость непосредственного вызова объектов DAO или ODBC
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Для большей точности управления в библиотеке MFC поддерживается непосредственный вызов объектов DAO или ODBC, а также обращение к ним с помощью классов.  Библиотека MFC применяется в целях упрощения программирования для Windows, однако не используется при обращении к базовым интерфейсам API.
