# Где в базе знаний Майкрософт можно найти статьи, посвященные базам данных?
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Чтобы найти в базе знаний статьи, посвященные ODBC MFC, следует выполнить поиск в базе знаний по ключевому слову MFCDATABASE.  
  
 Чтобы найти в базе знаний статьи, посвященные DAO MFC, следует выполнить поиск в базе знаний по ключевому слову MFCDAO.
