# <a name="catalog-information--mfc-data-access"></a>Сведения о каталоге (доступ к данным MFC)
Сведения о таблицах источника данных могут включать имена таблиц и столбцов в них, привилегии таблиц, имена первичных и внешних ключей, сведения о предопределенных запросах и хранимых процедурах, сведения об индексах таблиц и статистику таблиц.  
  
 Дополнительные сведения см. в разделе [источника данных: определение схемы источника данных (ODBC)](../data/odbc/data-source-determining-the-schema-of-the-data-source-odbc.md).
