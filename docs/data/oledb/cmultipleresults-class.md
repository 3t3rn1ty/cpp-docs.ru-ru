# Класс CMultipleResults
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Если требуется команду обрабатывать несколько результирующих наборов, используйте `CMultipleResults` для аргумента *TMultiple* шаблона `CCommand`.  
  
## Синтаксис  
  
```  
class CMultipleResults  
```  
  
## Заметки  
 Для обработки нескольких результирующих наборов [CCommand](../../data/oledb/ccommand-class.md) должен наследовать от этого класса.
