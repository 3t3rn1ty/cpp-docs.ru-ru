# <a name="endprovidercolumnmap"></a>END_PROVIDER_COLUMN_MAP
Отмечает конец карты записей столбцов поставщика.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp
END_PROVIDER_COLUMN_MAP()  
  
```  
  
## <a name="example"></a>Пример
