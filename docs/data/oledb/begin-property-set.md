# BEGIN_PROPERTY_SET
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Обозначает начало набора свойств в сопоставлении набора свойств.  
  
## Синтаксис  
  
```  
  
BEGIN_PROPERTY_SET(  
guid   
)  
  
```  
  
#### Параметры  
 `guid`  
 \[in\] свойства GUID.  
  
## Пример  
 В разделе [BEGIN\_PROPSET\_MAP](../Topic/BEGIN_PROPSET_MAP.md).  
  
## Требования  
 **Header:** atldb.h
