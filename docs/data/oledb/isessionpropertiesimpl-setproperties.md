# ISessionPropertiesImpl::SetProperties

## Синтаксис  
  
```  
  
      STDMETHOD(SetProperties)(   
   ULONG cPropertySets,   
   DBPROPSET rgPropertySets[]    
);  
```  
  
#### Параметры  
 В разделе [ISessionProperties::SetProperties](https://msdn.microsoft.com/en-us/library/ms714405.aspx) справочника *программиста OLE DB*.
