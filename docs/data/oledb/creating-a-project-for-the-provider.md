# Создание проекта для поставщика
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

#### Создание проекта для размещения поставщика OLE DB  
  
1.  В меню **Файл** выберите команду `New` и затем пункт **Проект**.  
  
     Откроется диалоговое окно **Создание проекта**.  
  
2.  В области "Типы проектов" выберите папку "Проекты Visual C\+\+".  В области "Шаблоны" выберите **Проект ATL**.  
  
3.  В поле **Имя** введите имя проекта и нажмите кнопку **ОК**.  
  
     Открывается окно мастера проектов ATL.  
  
4.  В мастере проектов ATL для параметра **Тип сервера** установите значение **Библиотека динамической компоновки \(DLL\)**.  
  
5.  Нажмите кнопку **Готово**.
