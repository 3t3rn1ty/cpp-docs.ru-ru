# Реализация простого поставщика, предназначенного только для чтения
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Подробный пример, рассматриваемый в этом разделе, демонстрирует, как изменить файлы, созданные мастером, с целью создания простого поставщика, предназначенного только для чтения, который считывает из текстового файла набор из двух строк.  Чтобы создать такой поставщик, измените созданные мастером файлы так, как указано в следующих разделах:

-   [Хранение строк в поставщике](../../data/oledb/storing-strings-in-the-ole-db-provider.md)
