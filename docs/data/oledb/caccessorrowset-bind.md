# CAccessorRowset::Bind
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Создает привязки, если указано **bBind** как false в [CCommand::Open](../../data/oledb/ccommand-open.md).  
  
## Синтаксис  
  
```  
  
HRESULT Bind( );  
  
```  
  
## Возвращаемое значение  
 Стандартное `HRESULT`.  
  
## Требования  
 **Заголовок:** atldbcli.h
