# <a name="beginparammap"></a>BEGIN_PARAM_MAP
Отмечает начало карты записей параметра.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp
BEGIN_PARAM_MAP(x)  
  
```  
  
#### <a name="parameters"></a>Параметры  
 *x*  
 [входные данные] Имя класса записей пользователя.  
  
## <a name="remarks"></a>Примечания  
 Используются параметры [команды](https://msdn.microsoft.com/en-us/library/ms724608.aspx).
