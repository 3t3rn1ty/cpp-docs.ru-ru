# <a name="cenumeratoraccessormntype"></a>CEnumeratorAccessor::m_nType
Переменная, позволяющее определить, описывает ли строка источника данных или перечислителя.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp
USHORT m_nType;  
  
```  
  
## <a name="remarks"></a>Примечания  
 В разделе [ISourcesRowset::GetSourcesRowset](https://msdn.microsoft.com/en-us/library/ms711200.aspx) в *Справочник программиста OLE DB* для получения дополнительной информации.
