# <a name="schema-rowset-classes-and-typedef-classes"></a>Классы схемы Rowset и Typedef
Схема является коллекцией объектов базы данных, которые принадлежат либо были созданы, определенным пользователем. Каталог может содержать один или несколько схем, но всегда должен содержать схема с именем INFORMATION_SCHEMA, содержащей домены информационной схемы и представления. Сведения о схеме в OLE DB получаются с помощью предопределенные наборы строк схемы и включает типы, таблиц, столбцов, индексов, представлений, утверждения и ограничения, статистики, наборов символов, параметры сортировки и домены.  
  
 Наборы строк схемы — это предопределенные наборы строк, представляющих метаданные. Наборы строк схемы, обычно используются в динамическом программировании, когда структура базы данных не известен во время компиляции. Эти наборы строк схемы можно использовать для получения сведений о базе данных во время выполнения.  
  
 Используйте классы typedef для создания наборов строк схемы. Ниже перечислены соответствующего typedef и классы набора строк схемы. Необходимо вызвать [CRestrictions::Open](../../data/oledb/crestrictions-open.md) после создания экземпляра набора строк схемы. Этот метод возвращает результирующий набор, основанный на заданными ограничениями. В разделе [IDBSchemaRowset](https://msdn.microsoft.com/en-us/library/ms713686.aspx) сведения о столбцах ограничений, связанных с каждого набора строк схемы.
