# <a name="cenumeratoraccessormbisparent"></a>CEnumeratorAccessor::m_bIsParent
Переменная, показывающего, является Перечислитель перечислитель родительский, если строка находится перечислитель.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
VARIANT_BOOL m_bIsParent;  
  
```  
  
## <a name="remarks"></a>Примечания  
 В разделе [ISourcesRowset::GetSourcesRowset](https://msdn.microsoft.com/en-us/library/ms711200.aspx) в *Справочник программиста OLE DB* для получения дополнительной информации.
