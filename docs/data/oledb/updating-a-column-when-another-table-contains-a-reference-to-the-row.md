# Обновление столбца при наличии ссылки в другой таблице на данную строку
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Некоторые поставщики могут обнаружить измененный столбец в строке, но большинство поставщиков не имеют данной возможности.  В результате обновление столбца при наличии ссылки на строку, над которой выполняется попытка обновления, может вызвать ошибку.  Чтобы решить данную проблему, необходимо создать отдельный метод доступа, содержащий только те столбцы, которые необходимо изменить.  Далее необходимо передать число методов доступа методу `SetData`.
