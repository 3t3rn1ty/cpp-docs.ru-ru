# <a name="ccollations-ccollationinfo"></a>CCollations, CCollationInfo
Вызовите typedef-класс **CCollations** реализации класса своего параметра **CCollationInfo**.  
  
## <a name="remarks"></a>Примечания  
 В разделе [классы набора строк схемы и классы Typedef](../../data/oledb/schema-rowset-classes-and-typedef-classes.md) Дополнительные сведения об использовании классов typedef.  
  
 Этот класс определяет символьные параметры сортировки, определенные в каталоге, которые доступны для данного пользователя.

|Члены данных|Столбцы OLE DB|  
|------------------|--------------------|  
|m_szCatalog|COLLATION_CATALOG|  
|m_szSchema|COLLATION_SCHEMA|  
|m_szName|COLLATION_NAME|
