# <a name="cenumeratoraccessormszparsename"></a>CEnumeratorAccessor::m_szParseName
Строка для [IParseDisplayName](http://msdn.microsoft.com/library/windows/desktop/ms680604) для получения моникера для источника данных или перечислителя.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp
WCHAR m_szParseName[129];  
  
```  
  
## <a name="remarks"></a>Примечания  
 В разделе [ISourcesRowset::GetSourcesRowset](https://msdn.microsoft.com/en-us/library/ms711200.aspx) в *Справочник программиста OLE DB* для получения дополнительной информации.
