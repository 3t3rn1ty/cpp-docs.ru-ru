# BOOKMARK_ENTRY
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Привязывает столбца закладки.  
  
## Синтаксис  
  
```  
  
BOOKMARK_ENTRY(  
variable  
 )  
  
```  
  
#### Параметры  
 *variable*  
 \[in\] переменная, привязываемая к столбцу закладки.
