# Создание поставщика
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

#### Создание поставщика OLE DB с помощью мастера поставщиков OLE DB библиотеки ATL  
  
1.  Щелкните проект правой клавишей мыши.  
  
2.  В контекстном меню выберите команду **Добавить**, а затем щелкните **Добавить класс**.  
  
3.  В диалоговом окне **Добавление класса** выберите значок **Поставщик ATL OLE DB** и нажмите кнопку **Открыть**.  
  
4.  В мастере поставщиков OLE DB библиотеки ATL введите короткое имя поставщика в поле **Краткое имя**.  В последующих разделах используется короткое имя "MyProvider", но вы можете указать и другое имя.  Остальные поля заполняются автоматически в соответствии с введенным именем.  
  
5.  При необходимости отредактируйте содержимое этих полей.  Кроме имен объектов и файлов, для редактирования доступны следующие поля:  
  
    -   **Coclass** — имя, которое COM использует для создания поставщика.  
  
    -   **ProgID** — программный идентификатор, представляющий собой текстовую строку, которую можно использовать вместо идентификатора GUID.  
  
    -   **Версия** — используется совместно с ProgID и coclass для создания программного идентификатора, зависящего от версии.  
  
6.  Нажмите кнопку **Готово**.
