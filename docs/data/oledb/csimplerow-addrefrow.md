# <a name="csimplerowaddrefrow"></a>CSimpleRow::AddRefRow
Добавляет счетчик ссылок в дескриптор существующей строки в потокобезопасным способом.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp
DWORD AddRefRow();  
  
```  
  
## <a name="requirements"></a>Требования  
 **Заголовок:** atldb.h
