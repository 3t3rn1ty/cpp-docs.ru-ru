# Установка поддержки баз данных ATL
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Программа установки устанавливает все необходимые компоненты, позволяющие вести разработку, при базовой установке OLE DB.  Интерфейсы OLE DB устанавливаются вместе с пакетом компонентов MDAC SDK.  Шаблоны OLE DB, которые поддерживают разработку OLE DB в Visual C\+\+ .NET, устанавливаются как часть библиотеки ATL.  Программа установки Visual C\+\+ .NET устанавливает как пакет компонентов MDAC SDK \(версии 2.8\), так и библиотеку ATL.
