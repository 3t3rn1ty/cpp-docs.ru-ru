# Источник данных. Определение схемы источника данных (ODBC)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Данный раздел относится к классам ODBC библиотеки MFC.  
  
 Чтобы задать элементы данных в объектах `CRecordset`, необходимо знать схему источника данных, к которому выполняется подключение.  Определение схемы источника данных предполагает получение списка таблиц в источнике данных и столбцов в каждой таблице, определение типа данных в каждом столбце, а также определение наличия индексов.
