# <a name="net-framework-equivalents-to-c-native-types-ccli"></a>Эквиваленты собственным типам C++ в .NET Framework (C++/CLI)
В следующей таблице показаны ключевые слова для встроенных типов Visual C++, которые являются псевдонимами предопределенных типов в **системы** пространства имен.  
  
|Тип Visual C++|Тип платформы .NET Framework|  
|-----------------------|-------------------------|  
|**bool**|**System.Boolean**|  
|**подписанные char** (см. [/j.](../build/reference/j-default-char-type-is-unsigned.md) Дополнительные сведения)|**System.SByte**|  
|**unsigned char**|**System.Byte**|  
|**wchar_t**|**System.Char**|  
|**двойные** и **long double**|**System.Double**|  
|**float**|**System.Single**|  
|**int**, **типа signed int**, **длинные**, и **длинное со знаком**|**System.Int32**|  
|**Тип unsigned int** и **длинного без знака**|**System.UInt32**|  
|**__int64** и **подписан __int64**|**System.Int64**|  
|**unsigned __int64**|**System.UInt64**|  
|**Краткое** и **подписан short**|**System.Int16**|  
|**unsigned short**|**System.UInt16**|  
|**void**|**System.Void**|
