# swap (STL/CLR)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Меняет местами значения элементов между 2 типы объектов, присвоению содержимое первого объекта к второго объекта и содержимое второго с первым.  
  
## Синтаксис  
  
```  
<class _Ty> inline  
    void swap(_Ty% _Left, _Ty% _Right);  
```  
  
## Заметки  
 Эта функция работает аналогично функции `swap` STL.  Для получения дополнительной информации см. [swap](../Topic/swap.md).

 **Пространство имен:** cliext
