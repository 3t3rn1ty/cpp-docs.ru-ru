# <a name="swap-stlclr"></a>swap (STL/CLR)
Меняет местами значения элементов между двумя типами объектов, присваивая содержимое первого объекта второму объекту, а содержимое второго — первому.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
<class _Ty> inline  
    void swap(_Ty% _Left, _Ty% _Right);  
```  
  
## <a name="remarks"></a>Примечания  
 Эта функция работает так же, как функция стандартной библиотеки C++ `swap`. Дополнительные сведения см. в разделе [swap](http://msdn.microsoft.com/Library/b471a2de-035e-4aff-b1c7-345d85d93972).
