# <a name="utility-stlclr"></a>utility (STL/CLR)
Включать заголовок STL/CLR `<cliext/utility>` для определения шаблона класса `pair` и некоторые вспомогательные функции шаблона.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
#include <utility>  
```
