# utility (STL/CLR)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Включать заголовок `<cliext/utility>` STL\/CLR для определения класса шаблона `pair` и несколько шаблонных вспомогательных функций.  
  
## Синтаксис  
  
```  
#include <utility>  
```  
  
## Объявления  
  
|Класс|Описание|  
|-----------|--------------|  
|[pair](../dotnet/pair-stl-clr.md)|Создается пара элементов.|  
  
|Оператор|Описание|  
|--------------|--------------|  
|[operator\=\= \(pair\)](../dotnet/operator-equality-pair-stl-clr.md)|Сравнение пар равное.|  
|[operator\!\= \(pair\)](../dotnet/operator-inequality-pair-stl-clr.md)|Сравнение пар неравное.|  
|[operator\< \(pair\)](../dotnet/operator-less-than-pair-stl-clr.md)|Пара меньше сравнения.|  
|[operator\<\= \(pair\)](../dotnet/operator-less-or-equal-pair-stl-clr.md)|Пара меньше или равное сравнение.|  
|[operator\> \(pair\)](../dotnet/operator-greater-than-pair-stl-clr.md)|Пары больше сравнение.|  
|[operator\>\= \(pair\)](../dotnet/operator-greater-or-equal-pair-stl-clr.md)|Пары больше или равное сравнение.|  
  
|Функция|Описание|  
|-------------|--------------|  
|[make\_pair](../dotnet/make-pair-stl-clr.md)|Создайте пару из пары значений.|  
  
## Требования  
 **Заголовок:**\<cliext\/utility\>  
  
 **Пространство имен:** cliext
