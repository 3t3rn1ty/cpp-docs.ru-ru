# <a name="algorithm-stlclr"></a>algorithm (STL/CLR)
Определяет шаблонные функции контейнера STL/CLR, выполняющие алгоритмы.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
#include <cliext/algorithm>  
```  
  
## <a name="functions"></a>Функции

|--------------|-----------------|  
|[adjacent_find (STL/CLR)](../dotnet/adjacent-find-stl-clr.md)|Поиск двух соседних элементов, которые равны.|  
|[binary_search (STL/CLR)](../dotnet/binary-search-stl-clr.md)|Проверяет, является ли заданное значение содержит упорядоченную последовательность.|  
|[copy (STL/CLR)](../dotnet/copy-stl-clr.md)|Копирует значения из исходного диапазона в диапазон назначения, прохода в прямом направлении.|

|[next_permutation (STL/CLR)](../dotnet/next-permutation-stl-clr.md)|Изменяет порядок элементов в диапазоне, чтобы исходный порядок был заменен перестановкой лексикографически Следующий больший, если он существует.|  
|[nth_element (STL/CLR)](../dotnet/nth-element-stl-clr.md)|Разделяет последовательность элементов, правильно находя `n`-ый элемент последовательности, чтобы все элементы перед ним были меньше или равно к нему и все элементы, следующие за его, больше или равны ему.|  
|[partial_sort (STL/CLR)](../dotnet/partial-sort-stl-clr.md)|Упорядочивает указанный число меньших элементов в диапазоне в не нисходящем порядке.|  
|[partial_sort_copy (STL/CLR)](../dotnet/partial-sort-copy-stl-clr.md)|Копирует элементы из исходного диапазона в диапазон назначения, таким образом, что все элементы из исходного диапазона упорядочены.|  
|[partition (STL/CLR)](../dotnet/partition-stl-clr.md)|Упорядочивает элементы в диапазоне, таким образом, что эти элементы, удовлетворяющие унарному предикату, расположены перед те, которые ему не удовлетворяют.|

|[remove_copy (STL/CLR)](../dotnet/remove-copy-stl-clr.md)|Копирует элементы из исходного диапазона в диапазон назначения за исключением того, что элементы с заданным значением не копируются, не нарушая порядок остальных элементов.|  
|[remove_copy_if (STL/CLR)](../dotnet/remove-copy-if-stl-clr.md)|Копирует элементы из исходного диапазона в диапазон назначения за исключением тех, которые удовлетворяют предикату, не нарушая порядок остальных элементов.|  
|[remove_if (STL/CLR)](../dotnet/remove-if-stl-clr.md)|Удаляет элементы, соответствующие предикату, из заданного диапазона без нарушения порядка остальных элементов. .|

|[replace_copy_if (STL/CLR)](../dotnet/replace-copy-if-stl-clr.md)|Проверяет каждый элемент в исходном диапазоне и заменяет его, если он соответствует заданному предикату, одновременно копируя результат в новый диапазон назначения.|  
|[replace_if (STL/CLR)](../dotnet/replace-if-stl-clr.md)|Проверяет каждый элемент в диапазоне и заменяет его, если он соответствует заданному предикату.|
