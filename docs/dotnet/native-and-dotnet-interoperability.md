# <a name="native-and-net-interoperability"></a>Взаимодействие исходного кода и платформы.NET
Visual C++ поддерживает возможности взаимодействия, позволяющие управляемым и неуправляемым конструкциям сосуществовать и взаимодействовать в этой сборке и даже в том же файле. Небольшое подмножество этих функциональных возможностей, например P/Invoke поддерживается с другими языками .NET, но большая часть поддержки взаимодействия, предоставляемые Visual C++ недоступна на других языках.
