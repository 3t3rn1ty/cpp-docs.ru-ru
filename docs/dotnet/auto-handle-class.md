# Класс auto_handle
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Автоматическое управление ресурсами, который может быть использован для встраивания виртуальный дескриптор в управляемый тип.  
  
## Синтаксис  
  
```  
template<typename _element_type>  
ref class auto_handle;  
```  
  
#### Параметры  
 `_element_type`  
 Управляемый тип для внедрения.  
  
## Требования  
 **Файл заголовка**\<msclr\\auto\_handle.h\>  
  
 **Пространство имен** msclr
